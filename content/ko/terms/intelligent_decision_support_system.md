## Definition

지능형 의사결정 지원 시스템(IDSS)은 머신러닝과 자연어 처리와 같은 인공지능 기술을 전통적인 의사결정 지원 프레임워크와 통합합니다. 이는 방대한 데이터를 실시간으로 처리하고 시나리오를 시뮬레이션하여 의사결정자에게 더 정확하고 신속한 선택을 가능하게 합니다.

### Summary

데이터를 분석하고 실행 가능한 인사이트를 제공하여 인간의 복잡한 의사결정을 돕는 AI 강화 시스템입니다.

## Key Concepts

- AI 통합
- 데이터 분석
- 인간 중심 루프(Human-in-the-loop)
- 예측 모델링

## Use Cases

- 의료 진단 보조
- 금융 리스크 평가
- 공급망 최적화
