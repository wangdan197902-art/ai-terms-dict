## Definition

계층적 제어 시스템은 제어 논리를 여러 계층으로 조직합니다. 일반적으로 고수준의 전략적 계획부터 저수준의 실시간 실행까지 다양하며, 상위 계층은 목표와 명령을 정의하고 하위 계층은 이를 구체적이고 실시간으로 실행합니다. 이는 복잡한 시스템의 관리를 단순화하고 효율성을 높입니다.

### Summary

의사 결정이 계층으로 조직되며, 상위 레벨이 하위 레벨 컨트롤러의 목표를 설정하는 제어 아키텍처입니다.

## Key Concepts

- 계층적 아키텍처
- 추상화
- 피드백 루프
- 분해(Decomposition)

## Use Cases

- 로봇공학
- 산업 자동화
- 자율 주행
