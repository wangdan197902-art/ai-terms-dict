## Definition

다국어 모델은 각 언어별로 별도의 모델을 필요로 하지 않고 다양한 언어 입력을 처리하도록 설계되었습니다. 이러한 시스템은 일반적으로 공유 임베딩이나 교차 언어 정렬을 활용합니다.

### Summary

AI에서 다국어는 여러 자연어로 콘텐츠를 처리, 이해하거나 생성할 수 있는 모델을 의미합니다.

## Key Concepts

- 교차 언어 전이
- 공유 어휘
- 언어 식별
- 제로샷 번역

## Use Cases

- 기계 번역 서비스
- 글로벌 고객 지원 챗봇
- 교차 언어 검색 엔진
