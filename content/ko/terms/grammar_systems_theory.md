## Definition

이론 컴퓨터 과학과 언어학에서 유래한 이 분야는 고전적인 촘스키 계층 구조 개념을 다중 구성 요소 시스템으로 확장합니다. 이는 여러 문법이나 구성 요소가 어떻게 상호작용하며 정보를 처리하는지 조사합니다.

### Summary

문법 시스템 이론은 병렬 또는 분산 환경에서 상호 작용하는 문법을 기반으로 하는 계산 모델을 연구하는 형식 언어 이론의 한 분야입니다.

## Key Concepts

- 형식 언어
- 병렬 계산
- 촘스키 계층 구조
- 막 계산(Membrane Computing)

## Use Cases

- 이론 컴퓨터 과학 연구
- 생물학적 시스템 모델링
- 분산 알고리즘 설계
