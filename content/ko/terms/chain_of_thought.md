## Definition

생각의 사슬(Chain-of-Thought, CoT) 프롬프팅은 대규모 언어 모델이 최종 답안에 도달하기 전에 단계별 추론 설명을 생성하도록 안내하는 전략입니다. 복잡한 문제를 작은 부분으로 나누어 모델이 논리적 과정을 거치도록 함으로써 정확도를 향상시킵니다.

### Summary

대형 언어 모델(LLM)이 답변하기 전에 중간 추론 단계를 생성하도록 유도하는 프롬프팅 기법입니다.

## Key Concepts

- 프롬프트 엔지니어링
- 추론 단계
- 해석 가능성
- 제로샷 CoT

## Use Cases

- 복잡한 수학 응용문제 해결
- 논리적 추론 작업
- 코드 스니펫 디버깅
