## Definition

MS MARCO(Microsoft Machine Reading Comprehension)는 자연어 처리, 특히 정보 검색 및 질문 응답 분야에서 널리 사용되는 데이터셋입니다. 익명화된 실제 웹 검색 쿼리와 해당 쿼리에 대한 관련 문서 구절로 구성되어 있으며, 검색 엔진의 정확도를 높이는 데 핵심적인 역할을 합니다.

### Summary

마이크로소프트 머신 리딩 컴프리헨션 데이터셋으로, 정보 검색 시스템 훈련을 위해 실제 검색 쿼리와 관련 문서 구절의 대규모 컬렉션입니다.

## Key Concepts

- 정보 검색(Information Retrieval)
- 구절 순위 매기기(Passage Ranking)
- 검색 쿼리(Search Queries)
- 머신 리딩 컴프리헨션(Machine Reading Comprehension)

## Use Cases

- 검색 엔진 훈련
- 질문 응답 시스템 개발
- 검색 모델 벤치마킹
