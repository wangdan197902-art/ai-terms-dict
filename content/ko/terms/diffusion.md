## Definition

디퓨전 모델은 데이터에 노이즈를 추가하는 확률적 과정을 역학습하는 생성형 AI 클래스입니다. 신경망이 단계별로 노이즈를 예측하고 제거하도록 훈련시킴으로써, 최종적으로 원본과 유사하거나 새로운 고품질 데이터를 생성합니다.

### Summary

점진적인 노이즈 추가 과정을 역전시켜 깨끗한 샘플을 재구성함으로써 데이터를 생성하는 생성 모델링 기법.

## Key Concepts

- 확률적 과정(Stochastic Process)
- 노이즈 스케줄(Noise Schedule)
- 역과정(Reverse Process)
- 잠재 공간(Latent Space)

## Use Cases

- 고충실도 이미지 생성
- 오디오 합성
- 신약 발견을 위한 분자 설계
