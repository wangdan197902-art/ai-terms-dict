## Definition

이 개념은 연산의 기본 단위가 단일 문자인 텍스트 조작에 중점을 둡니다. 철자 검사, 암호 해독, 손글씨 인식 등 세밀한 텍스트 분석이 필요한 작업에서 주로 사용됩니다.

### Summary

문자 컴퓨팅은 단어나 문단 단위가 아닌 개별 문자 수준에서 텍스트를 처리, 생성 또는 분석하는 것을 포함합니다.

## Key Concepts

- 토큰화(Tokenization)
- 텍스트 생성(Text Generation)
- 세밀한 분석(Fine-grained Analysis)
- 서브워드 단위(Subword Units)

## Use Cases

- 손글씨 인식(OCR)
- 저자원 언어 모델링
- 암호문 분석
