## Definition

지능형 에이전트는 센서나 데이터 입력을 통해 주변 환경을 지각하고, 추론 알고리즘을 사용하여 정보를 처리하며, 액추에이터를 통해 환경에 작용하는 시스템입니다. 이는 목표 지향적이고 반응적인 행동을 통해 자율적으로 작동합니다.

### Summary

환경을 지각하고, 행동에 대해 추론하며, 특정 목표를 달성하기 위해 작업을 실행하는 자율 소프트웨어 개체입니다.

## Key Concepts

- 자율성
- 지각-행동 루프
- 목표 지향성
- 반응성

## Use Cases

- 시장 전략 실행 자율 거래 봇
- 다단계 워크플로우 처리 로봇 프로세스 자동화(RPA)
- 캘린더 및 이메일 작업 관리 개인 비서
