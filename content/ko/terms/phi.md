## Definition

Phi는 'Teaching-Learning Paradigm(교수-학습 패러다임)'을 기반으로 한 파운데이션 모델을 줄인 이름으로, 마이크로소프트에서 개발한 컴팩트한 대규모 언어 모델(Large Language Model) 계열입니다. 거대한 웹 코퍼스(web corpus)로 훈련된 기존 LLM과는 달리, Phi는 고품질 교육용 데이터와 합성 데이터를 사용하여 높은 추론 능력과 지식 밀도를 달성합니다.

### Summary

지식 밀도와 추론 능력을 중점으로 하는 마이크로소프트 리서치(Microsoft Research)에서 개발한 소형이지만 매우 효율적인 대규모 언어 모델 시리즈.

## Key Concepts

- 소형 언어 모델(Small Language Models)
- 합성 데이터(Synthetic Data)
- 추론(Reasoning)
- 지식 밀도(Knowledge Density)

## Use Cases

- 엣지 디바이스(AI) 배포
- 복잡한 논리적 추론 작업
- 비용 효율적인 API 서비스
