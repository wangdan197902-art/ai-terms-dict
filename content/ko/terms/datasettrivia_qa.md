## Definition

TriviaQA는 오픈 도메인 질문 답변을 위해 설계된 데이터셋으로, 100만 개 이상의 질문과 해당 답변을 특징으로 합니다. 기존 모델에 도전하기 위해 제작되었으며, 모델이 광범위한 지식을 통합하고 추론할 수 있도록 요구합니다.

### Summary

다양한 퀴즈 도메인에 걸쳐 수백만 개의 질문과 답변을 포함하는 오픈 도메인 질문 답변을 위한 대규모 데이터셋입니다.

## Key Concepts

- 오픈 도메인 질문 답변
- 다중 단계 추론
- 지식 통합

## Use Cases

- 검색 증강 생성 모델 훈련
- LLM의 사실적 일관성 평가
- 지식 기반 질문 답변 시스템 벤치마킹
