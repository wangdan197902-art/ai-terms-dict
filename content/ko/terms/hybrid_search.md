## Definition

하이브리드 검색은 밀도 벡터 검색(의미와 맥락을 포착)과 희소 벡터(키워드) 검색(정확한 용어 일치)이라는 두 가지 서로 다른 검색 방법을 통합합니다. 이를 통해 단일 방법만 사용할 때보다 더 정확하고 포괄적인 결과를 제공합니다.

### Summary

정확도와 관련성을 높이기 위해 의미론적 벡터 검색과 전통적인 키워드 기반 인덱싱을 결합한 검색 전략.

## Key Concepts

- 벡터 검색
- 키워드 매칭
- 재순위화(Reranking)
- 역순위 융합(Reciprocal Rank Fusion)

## Use Cases

- 기업 문서 검색
- 전자상거래 제품 검색
- 고급 RAG(검색 증강 생성) 파이프라인
