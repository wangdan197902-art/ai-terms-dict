## Definition

크로스 모달 AI는 시각, 청각, 텍스트 입력 등 서로 다른 모드에서 데이터를 처리하고 상관관계를 맺습니다. 이러한 시스템은 관련성을 이해하기 위해 공유 표현을 학습합니다.

### Summary

텍스트와 이미지 등 서로 다른 감각 데이터 유형 간에 정보를 통합하고 처리하는 기술.

## Key Concepts

- 멀티모달 학습
- 데이터 퓨전
- 공유 임베딩
- 인터모달 매핑

## Use Cases

- 이미지 캡셔닝
- 시각 질문 답변
- 멀티모달 검색
