## Definition

멀티 에이전트 시스템은 여러 독립적인 에이전트로 구성되며, 각 에이전트는 서로 다른 작업이나 도메인에 특화될 수 있습니다. 이러한 에이전트들은 공통의 목표를 달성하기 위해 상호 작용하고 행동을 조정합니다.(문단이 잘려 있어 완전한 번역은 생략됨)

### Summary

여러 자율적 AI 에이전트가 협력, 경쟁 또는 조정하여 개별 능력으로는 해결하기 어려운 복잡한 문제를 해결하는 아키텍처 접근 방식입니다.

## Key Concepts

- 에이전트 협업(Agent Collaboration)
- 특화(Specialization)
- 조정(Coordination)
- 분산화(Decentralization)

## Use Cases

- 자동화된 소프트웨어 개발
- 복잡한 시뮬레이션 환경
- 분산 의사 결정
