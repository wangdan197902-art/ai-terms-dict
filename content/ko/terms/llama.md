## Definition

라마(Large Language Model Meta AI)는 메타에서 출시한 일련의 기반 대규모 언어 모델입니다. 많은 독점 모델과 달리 라마 모델은 종종 오픈 가중치로 출시되어 연구자와 개발자가 모델을 자유롭게 수정하고 배포할 수 있게 합니다.

### Summary

메타 AI에서 개발한 대규모 언어 모델 패밀리로, 오픈 가중치와 뛰어난 성능으로 알려져 있습니다.

## Key Concepts

- 대규모 언어 모델(Large language model)
- 오픈 가중치(Open weights)
- 메타 AI(Meta AI)
- 파운데이션 모델(Foundation model)

## Use Cases

- 챗봇 개발(Chatbot development)
- 코드 생성 지원(Code generation assistance)
- 연구 프로토타이핑(Research prototyping)
