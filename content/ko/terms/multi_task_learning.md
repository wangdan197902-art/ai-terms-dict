## Definition

이 기법은 관련 작업 간에 공유되는 귀납적 편향을 활용하여 학습 효율성과 성능을 향상시킵니다. 단일 모델을 사용하여 여러 작업을 동시에 수행하도록 학습함으로써, 모델은 공통된 표현을 배우게 됩니다.

### Summary

모델이 일반화를 향상시키기 위해 여러 관련 작업을 동시에 학습하는 머신러닝 패러다임입니다.

## Key Concepts

- 공유 표현
- 귀납적 편향
- 작업 간섭
- 일반화

## Use Cases

- 자연어 처리 (예: 개체명 인식 및 품사 태깅)
- 컴퓨터 비전 (예: 객체 탐지 및 분할)
- 음성 인식 및 합성
