## Definition

생성형 사전 훈련 변환기 2(GPT-2)는 트랜스포머 아키텍처를 사용하여 인간과 유사한 텍스트를 생성하는 자기회귀 언어 모델입니다. 이는 방대한 양의 인터넷 텍스트 데이터셋으로 학습되었습니다.

### Summary

GPT-2는 텍스트 생성 및 이해를 위해 OpenAI에서 개발한 대규모 트랜스포머 기반 언어 모델입니다.

## Key Concepts

- 트랜스포머 아키텍처
- 자기회귀 모델링
- 자연어 처리
- 퓨샷 학습

## Use Cases

- 텍스트 완성 및 생성
- 기계 번역
- 대화형 에이전트
