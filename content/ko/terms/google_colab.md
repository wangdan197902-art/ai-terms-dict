## Definition

구글 코랩(Google Colaboratory)은 별도의 설정 없이 사용할 수 있는 호스팅 Jupyter 노트북 서비스로, 그래픽 처리 장치(GPU) 및 기타 컴퓨팅 리소스에 대한 무료 접근 권한을 제공합니다.

### Summary

브라우저를 통해 Python 코드를 작성하고 실행할 수 있는 클라우드 기반 Jupyter 노트북 환경으로, 무료 GPU 접근 권한을 제공합니다.

## Key Concepts

- 클라우드 컴퓨팅
- Jupyter Notebook
- GPU 가속화
- 파이썬 프로그래밍

## Use Cases

- 머신러닝 프로토타이핑
- 데이터 사이언스 교육
- 협업 코딩
