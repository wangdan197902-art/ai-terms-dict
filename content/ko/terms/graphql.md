## Definition

Facebook에서 개발한 GraphQL은 API의 데이터를 완전하고 이해하기 쉽게 설명하여 클라이언트가 필요 이상의 데이터 없이 필요한 것만 요청할 수 있는 힘을 줍니다. 이는 여러 엔드포인트를 통해 데이터를 가져오는 전통적인 REST API의 단점을 대체합니다.

### Summary

GraphQL은 API를 위한 쿼리 언어이자 기존 데이터를 사용하여 이러한 쿼리를 수행하기 위한 런타임으로, 클라이언트가 필요한 데이터만 정확히 요청할 수 있게 합니다.

## Key Concepts

- 스키마 정의(Schema Definition)
- 쿼리 언어
- 단일 엔드포인트
- 타입 안전성(Type Safety)

## Use Cases

- 모던 웹 애플리케이션 백엔드
- 모바일 앱 데이터 페칭
- 마이크로서비스 오케스트레이션
