## Definition

이 지표는 카테고리 집합이 해당 카테고리 내 속성 값을 얼마나 잘 예측할 수 있는지를 정량화합니다. 이는 카테고리의 크기와 그 내부 구성 요소의 동질성(균일성) 사이의 균형을 맞추어 분류의 유용성을 측정합니다.

### Summary

카테고리 유틸리티는 속성 값에 대해 제공하는 정보 이득을 기반으로 분류 체계의 효과성을 평가하는 수학적 측정치입니다.

## Key Concepts

- 정보 이득(Information Gain)
- 클러스터링 평가(Clustering Evaluation)
- 예측 정확도(Predictive Accuracy)
- 개념 학습(Concept Learning)

## Use Cases

- 클러스터링 품질 평가
- 개념 획득 연구
- 특징 선택 최적화
