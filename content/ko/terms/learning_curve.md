학습 곡선은 모델의 성능 지표를 훈련 데이터 양이나 훈련 에포크 수에 대해 그래프로 그려 학습 진행 상황을 시각화합니다.

- 훈련 점수
- 검증 점수
- 과소적합 vs 과대적합

- 모델 성능 문제 진단
- 샘플 크기 요구사항 결정
