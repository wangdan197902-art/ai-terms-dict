## Definition

테스트 세트는 최종 모델의 일반화 능력을 평가하기 위해 훈련 과정에서 제외된 데이터의 일부입니다. 하이퍼파라미터 튜닝에 사용되는 검증 세트와 달리 테스트 세트는 모델의 실제 성능을 unbiased하게 추정하는 데 사용됩니다.

### Summary

테스트는 훈련된 AI 모델을 보이지 않는 데이터로 평가하여 성능을 측정하는 평가 단계를 의미합니다.

## Key Concepts

- 일반화(Generalization)
- 미사용 데이터(Unseen Data)
- 모델 평가(Model Evaluation)
- 과적합 방지(Overfitting Prevention)

## Use Cases

- 분류 모델의 정확도 측정
- 다양한 알고리즘 버전의 벤치마킹
- 배포 전 최종 검증
