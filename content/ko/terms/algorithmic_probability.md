## Definition

콜모고로프 복잡성과 솔로모노프 귀납에 뿌리를 둔 알고리즘 확률은 더 짧은 프로그램에서 생성된 출력에 더 높은 확률을 할당합니다. 이는 더 간단한 설명이 더 그럴듯하다는 주장을 펼칩니다.

### Summary

무작위 프로그램이 특정 출력 문자열을 생성할 가능성의 이론적 측정값입니다.

## Key Concepts

- 콜모고로프 복잡성
- 솔로모노프 귀납
- 오컴의 면도날
- 보편적 사전 확률

## Use Cases

- AI의 이론적 기초
- 데이터 압축 알고리즘
- 귀납적 추론 모델
