## Definition

통계 모델링에서 GLM은 일반화 선형 모델(Generalized Linear Models)의 약자로, 정규분포 이외의 오차 분포 모델을 허용하도록 선형 회귀를 확장한 것입니다. 반면, 인공지능 분야에서는 지푸 AI(Zhipu AI)가 개발한 대규모 언어 모델인 GLM 시리즈를 지칭하기도 합니다.

### Summary

GLM은 통계학의 일반화 선형 모델을 지칭하거나, 지푸 AI(Zhipu AI)에서 개발한 일반 언어 모델 시리즈를 의미합니다.

## Key Concepts

- 일반화 선형 모델
- 대규모 언어 모델
- 프리픽스 LM
- 양방향 인코딩

## Use Cases

- 통계적 예측
- 자연어 처리
- 텍스트 생성
