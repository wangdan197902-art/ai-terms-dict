## Definition

조건부 랜덤 필드(CRF)는 자연어 처리 및 생물정보학 분야에서 일반적으로 사용되는 판별 모델 클래스입니다. 생성 모델과 달리 CRF는 직접적으로 조건부 확률을 모델링하므로, 주변 특징 간의 복잡한 상호 작용을 포착하는 데 특히 효과적입니다.

### Summary

조건부 랜덤 필드(CRF)는 시퀀스 레이블링과 같은 구조적 예측 작업을 위해 사용되는 판별적 확률 모델입니다.

## Key Concepts

- 판별 모델 (Discriminative Model)
- 구조적 예측 (Structured Prediction)
- 시퀀스 레이블링 (Sequence Labeling)
- 전역 정규화 (Global Normalization)

## Use Cases

- 개체명 인식 (Named Entity Recognition)
- 품사 태깅 (Part-of-Speech Tagging)
- 생물정보학 시퀀스 분석
