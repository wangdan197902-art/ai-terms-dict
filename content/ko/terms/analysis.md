## Definition

AI의 맥락에서 분석은 근본적인 패턴을 이해하거나 문제를 진단하며, 실행 가능한 통찰을 도출하기 위해 데이터, 모델 예측 또는 시스템 동작을 체계적으로 검토하는 과정을 의미합니다.

### Summary

데이터나 모델 출력을 검토하여 의미 있는 통찰과 패턴을 추출하는 과정.

## Key Concepts

- 통찰 추출
- 모델 해석 가능성
- 데이터 검토
- 진단 평가

## Use Cases

- 모델 디버깅
- 비즈니스 인텔리전스
- 설명 가능한 AI (XAI)
