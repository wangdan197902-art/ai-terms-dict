## Definition

인공지능에서 벡터는 정보를 수치적으로 표현하는 데 사용되는 기본 데이터 구조입니다. 이는 엔티티의 특징을 매핑하는 순서 있는 숫자 목록(요소)으로 구성됩니다.

### Summary

크기와 방향을 모두 나타내는 수학적 객체로, AI에서는 종종 고차원 공간에서 데이터 포인트를 인코딩하는 데 사용됩니다.

## Key Concepts

- 차원
- 임베딩
- 크기
- 방향

## Use Cases

- 데이터베이스의 의미론적 검색
- 자연어 처리 임베딩
- 이미지 인식 특징 추출

## Code Example

```python
import numpy as np
vec = np.array([1.0, 2.0, 3.0])
print(vec.shape)
```
