## Definition

MXFP4(Mixed eXtended Floating Point 4-bit)는 AI 워크로드에서 성능을 최적화하고 메모리 대역폭 사용을 줄이기 위해 도입된 특수 데이터 타입 형식입니다. 혼합 정밀도 연산을 허용함으로써 계산 효율성을 높이고, 특히 에지 디바이스나 대규모 트랜스포머 모델 배포 시 메모리 사용량을 획기적으로 줄일 수 있습니다.

### Summary

AI 하드웨어 가속기에서 효율적인 행렬 곱셈을 위해 최적화된 혼합 정밀도 부동 소수점 형식입니다.

## Key Concepts

- 양자화(Quantization)
- 혼합 정밀도(Mixed Precision)
- 하드웨어 가속(Hardware Acceleration)
- 행렬 곱셈(Matrix Multiplication)

## Use Cases

- 에지 디바이스에서의 LLM 추론 최적화
- 대규모 훈련 시 메모리 발자국 축소
- 트랜스포머 기반 모델 배포 가속화
