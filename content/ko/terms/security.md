## Definition

AI 보안은 적대적 공격, 데이터 독성 및 모델 추출과 같은 위협으로부터 머신러닝 모델, 데이터 파이프라인 및 배포 인프라를 보호하기 위한 조치를 포괄합니다.

### Summary

AI 시스템을 무단 접근, 오용 및 악의적인 공격으로부터 보호하는 관행입니다.

## Key Concepts

- 적대적 견고성
- 데이터 프라이버시
- 모델 무결성
- 액세스 제어

## Use Cases

- 금융 사기 탐지 모델 보호
- 의료 진단 알고리즘 보안 강화
- 자율주행 차량 지각 시스템 방어
