## Definition

모국어 식별(NLI)은 화자가 처음 습득한 언어를 인식하는 데 중점을 두는 자연어 처리의 하위 분야입니다. 일반적인 언어 감지와 달리 NLI는 화자의 언어적 특징을 심층 분석하여...

### Summary

화자의 발화나 텍스트 샘플을 바탕으로 화자의 모국어를 자동으로 판별하는 과정입니다.

## Key Concepts

- 화자 프로파일링
- 언어적 특징
- 액센트 인식
- 자연어 처리(NLP)

## Use Cases

- 생체 인증 보안
- 고객 서비스 상호작용 개인화
- 사회언어학적 인구 통계 분석
