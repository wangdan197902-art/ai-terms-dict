## Definition

다단계 방법은 복잡한 쿼리나 작업을 더 작고 실행 가능한 단계로 분해하는 과정을 포함합니다. 이 접근 방식은 수학 문제 풀이나 코드 생성과 같은 추론 작업에서 매우 중요합니다.

### Summary

최종 해결책에 도달하기 위해 일련의 논리적 연산이나 계산을 수행해야 하는 문제 해결 전략입니다.

## Key Concepts

- 추론(reasoning)
- 중간 단계(intermediate steps)
- 생각의 사슬(chain of thought)
- 분해(decomposition)

## Use Cases

- 복잡한 수학 응용 문제
- 종속성이 있는 코드 생성
- 논리적 추론 작업
