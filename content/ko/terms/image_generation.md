## Definition

이 패러다임은 Stable Diffusion이나 DALL-E와 같은 모델을 활용하여 텍스트 프롬프트나 기타 입력 기반 고품질 이미지를 생성합니다. 이는 복잡한 데이터 분포를 학습하여 현실적인 이미지를 합성하는 기술을 포함합니다.

### Summary

이미지 생성은 생성형 AI 모델을 사용하여 처음부터 새로운 시각적 콘텐츠를 만들거나 기존 이미지를 수정하는 과정입니다.

## Key Concepts

- 텍스트-이미지 변환(Text-to-Image)
- 디퓨전 모델(Diffusion Models)
- 잠재 디퓨전(Latent Diffusion)
- 합성 데이터(Synthetic Data)

## Use Cases

- 디지털 아트 창작
- 마케팅 자료 디자인
- 모델 학습을 위한 데이터 증강
