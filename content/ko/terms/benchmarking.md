## Definition

벤치마킹은 미리 정의된 벤치마크를 사용하여 특정 작업에서 AI 모델이 얼마나 잘 수행되는지 측정하기 위해 실험을 수행하는 적극적인 관행입니다. 이 과정은 표준화된 프로토콜을 통해 모델을 실행하는 것을 포함합니다.

### Summary

AI 모델의 성능을 정량화하고 개선 영역을 식별하기 위해 벤치마크를 대상으로 모델을 체계적으로 테스트하는 과정입니다.

## Key Concepts

- 성능 테스트
- 정량적 분석
- 최적화
- 검증

## Use Cases

- 서로 다른 하드웨어 가속기의 추론 속도 비교.
- 파인튜닝된 모델의 정확도를 사전 훈련된 기준선과 비교 측정.
- 다양한 인구통계학적 그룹 전반에 걸쳐 모델의 공정성과 편향성 감사.
