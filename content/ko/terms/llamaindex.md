## Definition

원래 GPT Index로 알려졌던 LlamaIndex는 구조화되고 비구조화된 데이터를 LLM이 흡수하고 상호작용할 수 있게 하는 강력한 데이터 프레임워크입니다. 인덱싱, 쿼리, 관리를 위한 도구를 제공하여 RAG 애플리케이션 개발을 용이하게 합니다.

### Summary

LlamaIndex는 커스텀 데이터 소스를 대형 언어 모델(LLM)에 연결하여 검색 증강 생성(RAG) 애플리케이션 구축을 지원하는 프레임워크입니다.

## Key Concepts

- RAG (검색 증강 생성)
- 데이터 인덱싱
- 벡터 데이터베이스
- 컨텍스트 주입

## Use Cases

- 사적 데이터 기반 챗봇 구축
- 문서 Q&A 시스템
- 기업용 지식 베이스
