## Definition

'드라이븐'이라는 용어는 일반적으로 AI 접근 방식 뒤의 주요 힘이나 메커니즘을 나타내는 접미사로 사용됩니다. 예를 들어, '데이터 기반(data-driven)'은 결정이 통계적 패턴을 기반으로 이루어짐을 의미합니다.

### Summary

드라이븐은 의사결정이나 모델 아키텍처가 데이터나 목표와 같은 특정 입력 유형에 주로 의해 주도되는 AI 시스템이나 방법론을 설명합니다.

## Key Concepts

- 데이터 중심 접근법
- 목적 최적화
- 패러다임 식별
- 강화 학습

## Use Cases

- 데이터 중심 AI 워크플로우 설명
- 목표 지향적인 강화 학습 에이전트 정의
- 지식 기반 또는 데이터 기반 하이브리드 시스템 분류
