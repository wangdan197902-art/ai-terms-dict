## Definition

머신러닝, 특히 이전 학습(transfer learning)에서 도메인은 두 가지 구성 요소로 정의됩니다: 특징 공간(모든 가능한 입력의 집합)과 해당 입력의 주변 확률 분포(marginal probability distribution).

### Summary

도메인은 특징 공간과 확률 분포로 특징지어지는 특정 컨텍스트나 데이터 분포를 나타냅니다.

## Key Concepts

- 특징 공간
- 주변 분포
- 도메인 시프트
- 소스와 타겟

## Use Cases

- 합성 데이터로 학습된 모델을 실제 세계 시나리오에 적응시키는 것
- 문화적 또는 언어적 차이를 넘어선 자연어 처리(NLP) 작업 처리
- 서로 다른 병원 장비 간 의료 영상 분석
