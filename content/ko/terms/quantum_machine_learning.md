## Definition

양자 머신러닝(QML)은 양자 컴퓨팅 능력과 머신러닝 기술을 통합하는 새로운 융합 학문 분야입니다. 이는 얽힘(entanglement)과 같은 양자 현상을 활용하여 기존 컴퓨터로는 처리하기 어려운 복잡한 문제를 해결하는 것을 목표로 합니다.

### Summary

양자 머신러닝은 양자 컴퓨팅과 머신러닝 알고리즘을 결합하여 문제를 더 빠르거나 효율적으로 해결하는 분야입니다.

## Key Concepts

- 양자 알고리즘(Quantum Algorithms)
- 하이브리드 시스템(Hybrid Systems)
- 지수적 속도 향상(Exponential Speedup)
- 양 커널(Quantum Kernels)

## Use Cases

- 분자 시뮬레이션
- 금융 포트폴리오 최적화
- 고차원 데이터 분류
