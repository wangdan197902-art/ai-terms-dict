## Definition

학습 기반 접근 방식은 명시적 규칙 시스템과 대조적으로, 데이터 노출을 기반으로 패턴을 식별하고 결정을 내리기 위해 통계적 알고리즘에 의존합니다. 이 범주에는 지도 학습, 비지도 학습 등이 포함됩니다.

### Summary

명시적인 프로그래밍 규칙이 아닌 경험을 통해 알고리즘의 성능이 향상되는 방법을 지칭합니다.

## Key Concepts

- 패턴 인식
- 최적화
- 적응
- 데이터 중심

## Use Cases

- 예측 유지보수 시스템
- 추천 엔진
- 자율 주행 인지
