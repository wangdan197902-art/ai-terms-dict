## Definition

질문 응답(Question Answering, QA)은 주어진 문맥이나 지식베이스에서 사용자 쿼리에 대한 정확한 응답을 검색하거나 생성하는 과정입니다. 이는 특정 문서에 의존하는 폐쇄형 도메인 QA부터...

### Summary

시스템이 자연어로 제시된 질문에 대해 자동으로 정확한 답변을 제공하는 자연어 처리(NLP) 작업입니다.

## Key Concepts

- 정보 검색(Information Retrieval)
- 의미론적 이해(Semantic Understanding)
- 맥락 기반 추출(Contextual Extraction)
- 자연어 처리(Natural Language Processing)

## Use Cases

- 가상 비서
- 검색 엔진
- 고객 지원 자동화
