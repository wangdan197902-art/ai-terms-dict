## Definition

DeepSeek는 DeepSeek 회사에서 만든 인공지능 모델 패밀리를 지칭합니다. 이러한 모델들은 코드 생성, 복잡한 자연어 처리 작업 등 다양한 고급 작업을 처리하도록 설계되었으며, 특히 비용 효율성과 성능 면에서 주목받고 있습니다.

### Summary

DeepSeek는 높은 효율성, 강력한 코딩 능력, 오픈 가중치 공개로 알려진 DeepSeek社에서 개발한 대형 언어 모델 시리즈입니다.

## Key Concepts

- 대형 언어 모델(LLM)
- 오픈 가중치(Open Weights)
- 코드 생성
- 전문가 혼합(MoE)
- 자연어 처리(NLP)

## Use Cases

- 소프트웨어 개발 지원
- 데이터 분석 자동화
- 다국어 챗봇
