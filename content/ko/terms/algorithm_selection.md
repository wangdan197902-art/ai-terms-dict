## Definition

알고리즘 선택은 주어진 작업을 효율적으로 해결하는 데 가장 적합한 접근 방식을 결정하기 위해 다양한 계산 방법을 평가하는 것을 포함합니다. 이 과정에는 시간 복잡도, 공간 복잡도 등의 요인을 고려합니다.

### Summary

알고리즘 선택은 성능 지표와 제약 사항에 기반하여 특정 문제에 가장 적합한 계산 방법을 선택하는 과정입니다.

## Key Concepts

- 성능 지표
- 복잡도 분석
- 벤치마킹

## Use Cases

- 대규모 데이터셋용 정렬 알고리즘 선택
- 머신러닝 파이프라인용 분류기 선택
- 데이터베이스 쿼리 실행 계획 최적화
