## Definition

이 용수는 수십억 개의 매개변수를 가진 모델을 활용하여 다양한 언어 작업에 제로샷 또는 퓨샷 학습을 적용하는 더 넓은 응용 패러다임을 지칭합니다. 이는 특화된 모델들과 달리 범용성을 가집니다.

### Summary

광범위한 자연어 이해 및 생성 작업을 위해 확장된 신경망을 사용하는 패러다임입니다.

## Key Concepts

- 제로샷 학습
- 프롬프트 엔지니어링
- 일반화
- 매개변수 규모

## Use Cases

- 다중 턴 대화 시스템
- 시맨틱 검색
- 데이터 추출
