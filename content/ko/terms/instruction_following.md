## Definition

명령 따름은 대규모 언어 모델 및 기타 AI 시스템이 프롬프트 내의 미묘한 인간 지시를 이해하고 명시적 제약을 준수하는 능력을 의미합니다. 이는 모델이 사용자의 의도를 더 잘 따르도록 하는 패러다임 전환을 나타냅니다.

### Summary

AI 모델이 인간의 특정 명령이나 제약 조건을 정확하게 해석하고 실행하는 능력입니다.

## Key Concepts

- 프롬프트 엔지니어링
- 제약 조건 만족
- 정렬(Alignment)
- 작업 특이성

## Use Cases

- 특정 라이브러리 제약 조건이 있는 자동 코드 생성
- 엄격한 스키마 형식을 따르는 데이터 추출
- 규정 준수를 따르는 고객 서비스 챗봇
