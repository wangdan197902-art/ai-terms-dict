## Definition

AI 모드는 디지털 플랫폼이나 애플리케이션 내에서 사용자 상호작용을 향상시키기 위해 AI 기능이 활성화된 특정 운영 상태를 의미합니다. 이 모드는 일반적으로 자연어

### Summary

소프트웨어 인터페이스에서 인공지능이 작업을 적극적으로 보조하거나 자동화하는 기능적 상태입니다.

## Key Concepts

- 적극적 보조
- 사용자 상호작용
- 기능 활성화
- 맥락 인식

## Use Cases

- 고객 서비스 앱에서 챗봇 응답 활성화
- 브라우저에서 스마트 검색 제안 활성화
- 생산성 소프트웨어에서 생성형 작성 도구 켜기
