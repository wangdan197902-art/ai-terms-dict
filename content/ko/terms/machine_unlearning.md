## Definition

이 기법은 모델이 일반적인 지식은 유지하면서 특정 사용자 데이터를 잊도록 허용함으로써 GDPR의 '잊힐 권리'와 같은 개인정보 보호 규정을 충족시킵니다. 이는 성능을 근사적으로 유지하며...

### Summary

머신 언러닝은 모델을 처음부터 다시 학습시키지 않고도 특정 데이터 포인트나 그 영향력을 훈련된 모델에서 제거하는 과정입니다.

## Key Concepts

- 잊힐 권리
- 모델 재학습 근사
- 데이터 프라이버시
- 그라디언트 업데이트

## Use Cases

- 데이터 삭제 요청 준수
- 모델에서 편향되거나 오류가 있는 데이터 제거
- 데이터 독성 공격 완화
