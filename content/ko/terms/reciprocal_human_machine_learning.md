## Definition

このアプローチは、単純なヒューマンインザループ（人間によるラベリング）を超えたものです。双方向の知識転送が含まれます：人間はモデルのエラーを修正し、モデルはパターン識別の支援や自動化を通じて人間...

### Summary

人間と機械が互いに教え合い、学び合うことでパフォーマンスを継続的に向上させる協調的学習パラダイム。

## Key Concepts

- 双方向学習
- 人間-AI協働
- 適応型システム
- 継続的改善

## Use Cases

- インタラクティブ注釈ツール
- パーソナライズドレコメンデーションの調整
- 専門家関与型の医療診断
