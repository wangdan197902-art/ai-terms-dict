## Definition

큐웬 에디트는 큐웬 시리즈 내에서 텍스트 콘텐츠를 편집, 다듬기, 재구성하기 위해 최적화된 특정 기능이나 모델 버전을 의미합니다. 이러한 기능을 사용하면 사용자는 문장을 다시 작성하거나

### Summary

큐웬 에디트는 텍스트 및 콘텐츠 편집에 중점을 둔 큐웬 생태계 내의 기능이나 모델을 지칭합니다.

## Key Concepts

- 텍스트 편집
- 콘텐츠 정제
- 문법 교정
- 스타일 전환

## Use Cases

- 학술 논문 교정
- 마케팅 카피 재작성
- 회의록 요약
