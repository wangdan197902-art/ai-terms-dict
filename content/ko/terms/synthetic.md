## Definition

AI에서 합성 데이터는 실제 세계의 데이터를 모방하지만 실제 개인 정보나 민감한 기록을 포함하지 않는 인공적으로 생성된 정보입니다. 이는 실제

### Summary

합성은 자연 출처에서 수집된 것이 아니라 알고리즘에 의해 인공적으로 생성된 데이터나 콘텐츠를 지칭합니다.

## Key Concepts

- 데이터 생성(Data Generation)
- 개인정보 보호(Preservation)
- GANs (생성적 적대 신경망)
- 시뮬레이션(Simulation)

## Use Cases

- 시뮬레이션 환경에서 자율주행차 훈련(Training autonomous vehicles in simulated environments)
- 불균형 의료 데이터셋 보완(Augmenting imbalanced medical datasets)
- 실제 사용자 데이터를 노출하지 않고 소프트웨어 테스트(Testing software without exposing real user data)
