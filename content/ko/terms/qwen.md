## Definition

큐웜(Qwen)은 알리바바 그룹의 통이 연구소에서 만든 고급 대규모 언어 모델 패밀리를 나타냅니다. 자연어 이해, 코드 생성, 논리적 추론 등 다양한 작업에 최적화된 여러 버전을 포함하고 있습니다.

### Summary

큐웜(Qwen)은 알리바바 그룹의 통이 연구소(Tongyi Lab)에서 개발한 대규모 언어 모델 시리즈입니다.

## Key Concepts

- 대규모 언어 모델(Large Language Model)
- 알리바바 통이 연구소(Alibaba Tongyi Lab)
- 자연어 처리(Natural Language Processing)
- 파운데이션 모델(Foundation Model)

## Use Cases

- 일반 대화형 AI 어시스턴트
- 콘텐츠 생성 및 요약
- 복잡한 논리적 추론 작업
