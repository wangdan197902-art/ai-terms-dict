## Definition

Mistral Common은 Mistral AI에서 유지 관리하는 Python 패키지로, 해당 모델과 상호 작용하기 위한 표준화된 도구를 제공합니다. 주로 텍스트를 토큰으로 변환하는 데 필요한 토크나이저 구현을 제공합니다.

### Summary

다양한 Mistral AI 모델 변형과 호환되는 토큰화 및 유틸리티 함수를 제공하는 공유 라이브러리입니다.

## Key Concepts

- 토크나이저
- 전처리
- 라이브러리
- 통합

## Use Cases

- 텍스트 토큰화
- 모델 입력 준비
- 출력 디코딩
