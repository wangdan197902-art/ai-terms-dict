## Definition

Natural Questions(NQ)는 구글이 오픈 도메인 질문 응답 연구의 발전을 위해 도입한 벤치마크 데이터셋입니다. 익명화된 실제 구글 검색 쿼리를 길고 상세한 답변(주로 위키백과 페이지의 특정 섹션)과 매핑하여 구성됩니다.

### Summary

오픈 도메인 질문 응답을 위한 대규모 데이터셋으로, 실제 구글 검색 쿼리와 이에 대응하는 주석 처리된 위키백과 문서 조각으로 구성됩니다.

## Key Concepts

- 오픈 도메인 QA
- 위키백과
- 구글 검색
- 긴 형식 답변

## Use Cases

- 지식 기반 질문 응답 시스템 구축
- 검색 증강 생성(RAG) 학습
- 사실 정확도 평가
