## Definition

비지도 학습은 원시 데이터 내에서 숨겨진 구조, 클러스터 또는 분포를 자율적으로 식별합니다. 일반적인 방법에는 클러스터링, 차원 축소 및 생성 모델링이 포함됩니다.

### Summary

명확한 지도나 정답 없이 레이블이 지정되지 않은 데이터에서 패턴을 학습하는 머신러닝 기법입니다.

## Key Concepts

- 클러스터링
- 차원 축소
- 패턴 인식
- 레이블 데이터 부재

## Use Cases

- 고객 세분화
- 이상 탐지
- NLP의 주제 모델링
