## Definition

퍼지 에이전트는 데이터가 종종 모호하거나 불완전한 환경에서 작동하며, 이진 참/거짓 상태 대신 퍼지 논리 시스템을 사용합니다. 소속 함수(membership functions)와 언어적 변수(linguistic variables)를 사용하여(사용함) 불확실성을 관리합니다.

### Summary

의사결정 과정에서 불확실성과 부정확한 정보를 처리하기 위해 퍼지 논리를 사용하는 자율적 실체.

## Key Concepts

- 퍼지 논리
- 불확실성 처리
- 자율 의사결정
- 소속 함수

## Use Cases

- 비정형 환경에서의 로봇 항법
- 가변 입력이 있는 산업 공정 제어
- 편안함 수준 기반 스마트 온도 조절 장치 규제
