### Summary

Microsoft Azure 클라우드 인프라 서비스 위에 머신러닝 모델을 호스팅하고 실행하는 과정입니다.

## Key Concepts

- 클라우드 인프라(Cloud Infrastructure)
- 모델 서빙(Model Serving)
- 확장성(Scalability)
- CI/CD 파이프라인(CI/CD Pipelines)

## Use Cases

- 추론을 위한 실시간 API 엔드포인트
- 대규모 데이터셋의 배치 스코어링
- 엔터프라이즈급 모델 관리
