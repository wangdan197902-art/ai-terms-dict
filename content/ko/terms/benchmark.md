## Definition

인공지능에서 벤치마크는 머신러닝 모델의 능력을 측정하도록 설계된 표준화된 테스트 스위트 또는 데이터셋입니다. 이는 서로 다른 모델을 비교하기 위한 일관된 프레임워크를 제공합니다.

### Summary

AI 모델의 성능을 확립된 기준선과 비교하여 평가하는 데 사용되는 표준 참조점 또는 지표입니다.

## Key Concepts

- 표준화된 평가
- 성능 지표
- 비교 분석
- 기준선 비교

## Use Cases

- 기존 최첨단 방법론 대비 새로운 모델 아키텍처 평가.
- 다양한 조건이나 데이터셋 하에서 모델의 강건성 평가.
- 공정한 비교를 보장하기 위해 학술 연구 결과를 보고함.
