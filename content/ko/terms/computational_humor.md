## Definition

계산적 유머는 기계가 농담, 말장난, 재치 있는 발언 등을 생성하거나 해석하는 방법을 연구합니다. 이는 일반적으로 자연어 처리(NLP)를 활용하여 불일치, 의미의 변화, 또는 예기치 않은

### Summary

컴퓨터적 방법을 통해 유머러스한 콘텐츠를 생성, 이해, 감상하는 데 중점을 두는 AI의 하위 분야입니다.

## Key Concepts

- 불일치 이론 (Incongruity Theory)
- 자연어 처리 (Natural Language Processing)
- 의미 변화 (Semantic Shift)
- 생성 모델 (Generative Models)

## Use Cases

- 챗봇의 개성 강화
- 자동 농담 생성
- 창작 글쓰기 보조 도구
