## Definition

모델 레지스트리는 MLOps의 핵심 구성 요소로, ML 모델을 저장, 버전 관리 및 관리하기 위한 통합 저장소 역할을 합니다. 이를 통해 팀은 모델 메타데이터, 성능 지표 등을 추적할 수 있습니다.

### Summary

모델 레지스트리는 머신러닝 모델의 전체 수명 주기를 추적하기 위한 중앙 집중식 저장 및 관리 시스템입니다.

## Key Concepts

- 버전 제어
- 메타데이터 관리
- 수명 주기 추적
- MLOps

## Use Cases

- 개발부터 프로덕션까지 모델 버전 추적
- 모델 성능 및 계보 감사
- 모델 아티팩트 관련 팀 협업 촉진
