## Definition

유사도 학습은 유사한 항목이 서로 가깝고 유사하지 않은 항목이 멀리 떨어지도록 입력을 벡터 공간으로 매핑하는 모델을 훈련하는 데 중점을 둡니다. 사이먼 네트워크(Siamese networks)나 트리플릿 손실(Triplet loss)과 같은 기법이 사용됩니다.

### Summary

두 객체가 얼마나 유사하거나 다른지 결정하기 위한 거리 측정법을 학습하는 머신러닝 접근 방식입니다.

## Key Concepts

- 임베딩 공간(Embedding space)
- 거리 측정법(Distance metrics)
- 트리플릿 손실(Triplet loss)
- 사이먼 네트워크(Siamese networks)

## Use Cases

- 얼굴 인식
- 중복 감지
- 추천 시스템
