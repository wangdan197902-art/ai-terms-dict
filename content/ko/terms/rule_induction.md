## Definition

규칙 유도는 데이터에서 직접 if-then 규칙을 도출하는 상징적 기계 학습 방법입니다. 불투명한 가중치를 생성하는 신경망과 달리, 규칙 유도는 해석 가능한 모델을 제공합니다.

### Summary

라벨이 지정된 학습 데이터로부터 논리적 규칙 세트를 자동으로 생성하여 새로운 인스턴스를 분류하는 기계 학습 기법.

## Key Concepts

- if-then 규칙
- 의사결정 목록
- 해석 가능성
- 상징적 학습

## Use Cases

- 의료 진단 가이드라인
- 신용 승인 기준
- 제조 공정 결함 감지
