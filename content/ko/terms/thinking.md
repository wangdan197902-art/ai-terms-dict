## Definition

인간이 생물학적으로 사고하는 반면, AI의 '사고'는 인지 기능을 모방하는 계산적 연산을 포함합니다. 여기에는 논리적 귀납, 패턴 인식, 추론 등이 포함됩니다. 최신 대규모 언어 모델(LLM)은...

### Summary

AI에서 사고란 알고리즘에 의해 시뮬레이션되는 추론, 문제 해결 및 의사 결정의 인지 과정을 의미합니다.

## Key Concepts

- 추론
- 인지 시뮬레이션
- 추론(Inference)
- 문제 해결

## Use Cases

- 자연어 이해
- 자동화된 의사 결정
- 복잡한 문제 해결
