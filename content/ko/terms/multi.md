## Definition

접두사 'multi-'는 AI에서 여러 개의 병렬 구성 요소가 관여하는 아키텍처나 프로세스를 지칭하는 데 자주 사용됩니다. 예시로는 모델이 서로 다른 부분에 집중할 수 있게 하는 멀티 헤드 어텐션(Multi-Head Attention) 등이 있습니다.

### Summary

시스템이나 모델 내에 여러 개의 인스턴스, 헤드(Head), 또는 모달리티가 존재함을 나타내는 접두사.

## Key Concepts

- 병렬 처리
- 아키텍처 설계
- 통합

## Use Cases

- 트랜스포머 기반 모델 설계
- 센서 데이터 스트림 통합
- 앙상블 학습 시스템 구축
