## Definition

제로샷 프롬프팅은 사전 훈련된 언어 모델에게 소수 예시(few-shot examples)를 제공하거나 추가적인 파인튜닝을 수행하지 않고도, 텍스트 기반 프롬프트를 통해 작업을 직접 완료하도록 요청하는 방식입니다. 모델은 학습된 지식과 문맥 이해 능력을 바탕으로 지시사항만으로도 적절한 출력을 생성합니다.

### Summary

사전 예시나 파인튜닝 없이 자연어 지시사항만으로 대형 언어 모델이 작업을 수행하는 기법입니다.

## Key Concepts

- 프롬프트 엔지니어링
- 창발적 능력(Emergent Abilities)
- 문맥 내 학습(In-Context Learning)
- 지시사항 튜닝(Instruction Tuning)

## Use Cases

- AI 애플리케이션의 신속한 프로토타이핑
- 동적 작업 전환
- 데이터 주석 비용 절감
