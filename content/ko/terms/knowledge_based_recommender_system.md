## Definition

과거 사용자 행동에 의존하는 협업 필터링과 달리, KBRS는 항목과 사용자 선호도에 대한 명시적인 지식을 사용하여 추천을 도출합니다. 이는 데이터가 희소하거나 과거 기록이 없는 시장에서 특히 효과적입니다.

### Summary

과거 데이터 대신 명시적인 도메인 지식과 사용자 제약을 추론하여 추천을 생성하는 엔진.

## Key Concepts

- 명시적 제약 조건
- 추론
- 설명 가능성
- 희소 데이터 처리

## Use Cases

- 부동산 검색
- 복잡한 B2B 장비 조달
- 개인화된 여행 일정 계획
