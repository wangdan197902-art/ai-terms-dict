## Definition

머신러닝과 최적화에서 원스텝 방법은 수렴하기 위해 여러 반복이나 업데이트를 필요로 하지 않고 문제를 직접 해결합니다. 많은 단계를 거쳐 최소화하는 경사 하강법과는 대조적입니다.

### Summary

반복적 정교화 없이 단일 반복으로 작업 또는 의사결정 주기를 완료하는 알고리즘이나 프로세스를 지칭합니다.

## Key Concepts

- 폐쇄형 해(Closed-form solution)
- 계산 효율성(Computational efficiency)
- 비반복적(Non-iterative)
- 직접 매핑(Direct mapping)

## Use Cases

- 정규 방정식을 통한 선형 회귀
- 실시간 신호 처리
- 단순 분류 규칙
