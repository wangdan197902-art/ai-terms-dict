## Definition

계산 학습 이론에서 샘플 복잡성은 모델을 효과적으로 훈련하는 데 필요한 데이터 양을 정량화합니다. 이는 모델 용량과 데이터 가용성 사이의 균형을 맞추어 일반화 오차를 최소화하는 데 기여합니다.

### Summary

샘플 복잡성은 머신러닝 알고리즘이 높은 확률로 특정 수준의 성능을 달성하는 데 필요한 훈련 예제의 수를 의미합니다.

## Key Concepts

- 일반화 오차
- 훈련 데이터 크기
- 수렴
- 과적합 방지

## Use Cases

- 새로운 모델의 데이터셋 요구사항 결정
- 다양한 학습 알고리즘의 효율성 비교
- 데이터 수집 노력의 예산 책정
