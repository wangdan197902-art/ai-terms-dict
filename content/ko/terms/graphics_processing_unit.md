## Definition

GPU는 그래픽 렌더링 작업을 처리하기 위해 처음 개발된 고성능 프로세서입니다. 순차적 직렬 처리에 최적화된 소수의 코어를 가진 CPU와 달리, GPU는 수천 개의 코어를 갖추고 있어 대규모 병렬 연산을 효율적으로 수행할 수 있습니다.

### Summary

원래 빠른 이미지 조작 및 렌더링을 위해 설계된 특수 전자 회로로, 이제 AI의 병렬 컴퓨팅에 널리 사용됩니다.

## Key Concepts

- 병렬 처리
- CUDA 코어
- 행렬 곱셈
- 하드웨어 가속

## Use Cases

- 대규모 신경망 학습
- 실시간 컴퓨터 비전 추론
- 시뮬레이션에서의 3D 그래픽 렌더링
