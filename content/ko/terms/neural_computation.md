## Definition

신경 계산은 인공 뉴런이 입력 신호를 출력 응답으로 변환하기 위해 수행하는 수학적 연산을 참조합니다. 여기에는 가중 합, 활성화 함수, 역전파(Backpropagation) 등이 포함됩니다.

### Summary

생물학적 뉴런에서 영감을 받은 인공 신경망 내에서의 정보 처리 과정을 의미합니다.

## Key Concepts

- 활성화 함수
- 가중 합
- 역전파
- 분산 표현

## Use Cases

- 이미지 인식 시스템
- 자연어 처리 모델
- 예측 분석
