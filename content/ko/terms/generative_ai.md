## Definition

대규모 언어 모델(LLM)이나 확산 모델(Diffusion Model)과 같은 이러한 시스템은 기존 정보를 단순히 검색하는 것을 넘어 새로운 출력을 종합합니다. 훈련 데이터를 통해 데이터의 내재적 구조와 스타일을 학습하여, 입력 프롬프트나 조건에 따라 인간과 유사하거나 독창적인 콘텐츠 샘플을 생성해냅니다.

### Summary

생성형 AI는 학습된 패턴을 기반으로 텍스트, 이미지, 오디오, 코드 등 새로운 콘텐츠를 생성할 수 있는 인공지능 유형입니다.

## Key Concepts

- 콘텐츠 합성
- 패턴 학습
- LLM (대규모 언어 모델)
- 확산 모델

## Use Cases

- 텍스트 생성
- 이미지 제작
- 코드 완성
