## Definition

이 관행은 대규모 언어 모델(LLM)과 같은 AI 모델을 VS Code나 IntelliJ와 같은 소프트웨어 개발 환경에 연결하는 것을 포함합니다. 이는 지능형 코드 완성, 자동 리팩토링 등의 기능을 가능하게 합니다.

### Summary

프로그래머의 코딩 작업을 지원하기 위해 AI 기반 도구를 통합 개발 환경(IDE)에 직접 내장하는 것을 의미합니다.

## Key Concepts

- 자동 완성
- 코드 완성
- LLM (대규모 언어 모델)
- 개발자 경험

## Use Cases

- 지능형 코드 제안
- 자동 리팩토링
- 자연어 쿼리 실행
