## Definition

차등 프라이버시는 쿼리 결과나 모델 매개변수에 보정된 통계적 노이즈를 추가하여 강력한 프라이버시 보장을 제공합니다. 이는 특정 개인에 대한 정보 유출의 최대량을 정량화하며...

### Summary

특정 개인의 데이터 포함 여부가 분석 결과에 유의미한 영향을 미치지 않도록 보장하는 엄격한 수학적 프레임워크입니다.

## Key Concepts

- 프라이버시 예산(Epsilon)
- 노이즈 보정
- 조합 정리(Composition Theorem)
- 영지식 증명

## Use Cases

- 민감한 의료 기록을 기반으로 모델 학습
- 개별 신원 노출 없이 집계 통계 게시
- 신뢰할 수 없는 당사 간 보안 데이터 공유
