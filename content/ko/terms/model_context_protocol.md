## Definition

모델 컨텍스트 프로토콜(MCP)은 AI 애플리케이션이 데이터베이스, API, 파일 시스템 등 다양한 데이터 소스에 균일한 방식으로 연결할 수 있게 하는 오픈 표준입니다. 이는 복잡한 통합 코드를 추상화하여 개발자가 모델에 외부 도구와 데이터를 쉽게 연결할 수 있도록 합니다.

### Summary

AI 모델과 외부 데이터 소스 또는 도구 간에 안전하고 효율적인 통신을 촉진하도록 설계된 표준화된 프레임워크입니다.

## Key Concepts

- 상호 운용성
- 표준화
- 데이터 통합
- 도구 호출

## Use Cases

- 엔터프라이즈 데이터에 LLM 연결
- 이식 가능한 AI 어시스턴트 구축
- 보안 API 통합
