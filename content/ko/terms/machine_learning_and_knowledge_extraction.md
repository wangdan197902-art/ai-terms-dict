## Definition

이 분야는 머신러닝 기법을 자연어 처리(NLP) 및 데이터 마이닝과 결합하여 원시 데이터를 실행 가능한 지식으로 변환합니다. 여기에는 엔티티 인식, 관계 추출, 개념 분류 등을 수행하도록 모델을 훈련시키는 작업이 포함되며, 방대한 양의 텍스트나 데이터에서 의미 있는 인사이트를 자동으로 발견하는 데 중점을 둡니다.

### Summary

대규모 비정형 데이터셋에서 패턴을 자동으로 식별하고 구조화된 정보를 도출하기 위해 머신러닝 알고리즘을 사용하는 과정입니다.

## Key Concepts

- 패턴 인식
- 자연어 처리
- 데이터 마이닝
- 피처 엔지니어링

## Use Cases

- 소셜 미디어 게시글에서 고객 감정 추출
- 임상 기록에서 의학적 상태 식별
- 법률 사무소의 문서 분류 자동화
