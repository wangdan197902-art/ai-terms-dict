## Definition

웹 인텔리전스는 방대한 양의 인터넷 상 비정형 데이터를 처리하기 위해 데이터 마이닝, 머신러닝, 시맨틱 기술을 사용합니다. 이는 원시 웹 데이터를 변

### Summary

월드 와이드 웹에서 정보를 추출, 관리 및 분석하기 위해 지능적인 기법을 적용하는 것입니다.

## Key Concepts

- 데이터 마이닝
- 시맨틱 웹
- 링크 분석
- 콘텐츠 분류

## Use Cases

- 경쟁 시장 분석
- 소셜 미디어 감정 모니터링
- 검색 엔진 최적화
