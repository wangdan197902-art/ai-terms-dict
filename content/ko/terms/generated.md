## Definition

'생성된'이라는 용어는 텍스트, 이미지, 오디오 또는 코드와 같은 생성형 AI 모델이 생산한 출력을 설명합니다. 기존 데이터를 가져오는 검색 기반 시스템과 달리, 생성형 모델은 새로운 데이터를 합성하여 만들어냅니다.

### Summary

정적 데이터셋에서 직접 검색되는 것이 아니라 AI 모델에 의해 창작된 콘텐츠를 지칭합니다.

## Key Concepts

- 생성형 AI
- 합성
- 창작
- 출력

## Use Cases

- 텍스트 요약
- 이미지 합성
- 코드 생성
