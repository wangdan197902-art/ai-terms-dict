## Definition

광학 문자 인식(OCR)은 디지털 이미지 내의 텍스트를 식별하기 위해 이미지 처리 및 패턴 인식 알고리즘을 사용합니다. 인쇄되거나 손으로 쓴 문자를 기계가 읽을 수 있는 인코딩된 텍스트로 변환하여 데이터 아카이빙 및 자동화된 문서 처리를 지원합니다.

### Summary

스캔된 종이 문서나 이미지와 같은 다양한 유형의 문서를 편집 가능하고 검색 가능한 데이터로 변환하는 기술입니다.

## Key Concepts

- 텍스트 탐지
- 문자 인식
- 이미지 처리
- 디지털화
- 패턴 매칭

## Use Cases

- 역사적 아카이브 디지털화
- 송장 처리 자동화
- 스크린샷에서 텍스트 추출
