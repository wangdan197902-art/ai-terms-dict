## Definition

오픈 소스는 소프트웨어 프로젝트의 기반 코드가 공개적으로 접근 가능한 개발 모델을 의미합니다. AI 분야에서 이는 연구자와 개발자가 알고리즘을 검사하고, 수정하며, 재배포할 수 있게 하여 혁신과 협업을 촉진합니다.

### Summary

누구나 자유롭게 사용, 수정 및 배포할 수 있도록 소스 코드가 공개된 소프트웨어.

## Key Concepts

- 투명성
- 협업
- 접근성
- 커뮤니티 주도형

## Use Cases

- PyTorch 프레임워크 개발
- Hugging Face 모델 라이브러리
- 학술 연구 재현성
