## Definition

AI에서의 번역은 신경 기계 번역(Neural Machine Translation)을 의미하며, 딥러닝 모델이 언어 간 의미 표현을 매핑합니다. 규칙 기반 시스템과 달리 현대적 접근법은 문맥상의 뉘앙스를 학습합니다.

### Summary

의미를 보존하면서 소스 자연어를 대상 자연어로 텍스트를 변환하는 과정입니다.

## Key Concepts

- 신경 기계 번역
- 소스 언어
- 대상 언어
- 의미 보존

## Use Cases

- 글로벌 콘텐츠 현지화
- 실시간 채팅 통역
- 문서의 다국어화
