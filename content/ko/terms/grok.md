## Definition

Grok은 일론 머스크의 회사인 xAI가 만든 대형 언어 모델 기반 챗봇입니다. 주로 X 플랫폼(구 트위터) 구독자에게 제공됩니다. Grok은 실시간으로 최신 소셜 미디어 데이터에 접근할 수 있어 다른 LLM들과 차별화됩니다.

### Summary

xAI에서 개발하여 X 플랫폼에 통합된 AI 챗봇으로, 실시간 소셜 미디어 데이터 접근성과 유머러스한 성격으로 유명합니다.

## Key Concepts

- 대형 언어 모델(LLM)
- 실시간 데이터 접근
- xAI
- 대화형 AI

## Use Cases

- 최신 사건을 바탕으로 한 질문 답변
- 창의적이고 유머러스한 콘텐츠 생성
- 소셜 미디어 트렌드 요약
