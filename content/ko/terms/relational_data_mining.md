## Definition

관계형 데이터 마이닝은 단일 평평한 테이블이 아니라 여러 관련 테이블로 조직화된 데이터베이스에서 유용한 정보를 추출하는 데 중점을 둡니다. 이는 테이블 간의 관계와 연결성을 활용하여 더 풍부한 인사이트를 도출합니다.

### Summary

여러 개의 서로 연관된 테이블이나 엔터티로 구성된 데이터에서 패턴과 지식을 발견하는 과정.

## Key Concepts

- 엔터티-관계
- 다중 관계 학습
- 그래프 마이닝
- 데이터베이스 통합

## Use Cases

- 소셜 네트워크 분석
- 금융 거래 사기 탐지
- 신약 개발 및 생정보학
