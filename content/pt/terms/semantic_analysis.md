## Definition

Vai além da estrutura sintática para interpretar a intenção e o significado real dos inputs de linguagem. Isso envolve desambiguar significados de palavras com base no contexto, identificar entidades e compreender a intenção subjacente das mensagens.

### Summary

A análise semântica é o processo de extrair significado do texto, compreendendo as relações entre palavras e o contexto dentro do processamento de linguagem natural.

## Key Concepts

- Compreensão contextual
- Desambiguação de sentido de palavras
- Reconhecimento de intenção
- PLN (Processamento de Linguagem Natural)

## Use Cases

- Análise de sentimento
- Detecção de intenção em chatbots
- Sistemas de recuperação de informação
