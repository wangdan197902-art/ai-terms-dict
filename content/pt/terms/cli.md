## Definition

Uma Interface de Linha de Comando (CLI) permite que os usuários controlem o software inserindo comandos textuais em vez de usar elementos gráficos. No desenvolvimento de IA, as CLIs são essenciais para executar scripts, gerenciar...

### Summary

A Interface de Linha de Comando é um método baseado em texto para interagir com programas de software digitando comandos em um terminal.

- Entrada de Texto
- Emulação de Terminal
- Scripting
- Automação

## Use Cases

- Execução de scripts de inferência de modelos
- Gerenciamento de recursos de IA na nuvem
- Automação de pipelines de pré-processamento de dados
