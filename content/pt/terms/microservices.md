## Definition

No contexto da engenharia de IA, os microsserviços permitem que diferentes componentes de um pipeline de IA, como pré-processamento de dados, inferência de modelos e armazenamento de resultados, sejam desenvolvidos, escalados e mantidos

### Summary

Microsserviços é um estilo arquitetônico que estrutura uma aplicação como uma coleção de serviços fracamente acoplados e implantáveis independentemente, organizados em torno de capacidades de negócios.

## Key Concepts

- Acoplamento Fraco
- Implantação Independente
- Decomposição de Serviços
- Gateways de API

## Use Cases

- Endpoints de inferência de IA escaláveis
- Pipelines modulares de aprendizado de máquina
- Sistemas de recomendação resilientes
