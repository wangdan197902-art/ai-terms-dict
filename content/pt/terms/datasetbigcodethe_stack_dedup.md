## Definition

The Stack Dedup é um subconjunto especializado do The Stack, um repositório massivo de código aberto. Ele aplica técnicas rigorosas de deduplicação para eliminar trechos de código redundantes que poderiam enviesar grandes modelos de linguagem.

### Summary

Uma versão deduplicada do conjunto de dados The Stack, curada pela BigCode para remover trechos de código quase duplicados e obter dados de treinamento mais limpos.
