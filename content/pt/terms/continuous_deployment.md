## Definition

A Implantação Contínua é uma extensão da entrega contínua que automatiza todo o processo de lançamento. Uma vez que as alterações no código passam por todas as barreiras de qualidade, incluindo testes unitários, testes de integração e segurança...

### Summary

Uma prática de engenharia de software na qual cada alteração de código que passa nos testes automatizados é lançada automaticamente em produção.

## Key Concepts

- Testes Automatizados
- Pipelines CI/CD
- Implantação com Tempo de Inatividade Zero
- Bandeiras de Recursos (Feature Flags)

## Use Cases

- Plataformas de negociação de alta frequência
- Aplicações web em grande escala
- Gerenciamento de arquitetura de microsserviços
