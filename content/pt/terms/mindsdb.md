## Definition

O MindsDB atua como uma ponte entre bancos de dados relacionais tradicionais e fluxos de trabalho modernos de aprendizado de máquina. Permite aos usuários criar modelos preditivos usando consultas SQL padrão, eliminando a necessidade de mover dados para plataformas externas ou escrever código complexo em Python para tarefas básicas de previsão.

### Summary

O MindsDB é uma plataforma de código aberto que permite aos desenvolvedores criar, treinar e implantar modelos de aprendizado de máquina diretamente em seus bancos de dados existentes usando SQL.

## Key Concepts

- ML baseado em SQL
- Integração de banco de dados
- AutoML
- Modelagem preditiva

## Use Cases

- Detecção de fraude em tempo real em bancos de dados
- Previsão de rotatividade de clientes (churn)
- Relatórios automatizados com previsões
