## Definition

Um linter é um utilitário que realiza análise estática no código-fonte sem executá-lo. Ele verifica erros de sintaxe, possíveis bugs, 'code smells' (cheiros de código) e desvios de guias de estilo ou melhores práticas.

### Summary

Uma ferramenta de análise estática de código usada para sinalizar erros de programação, bugs, erros estilísticos e construções suspeitas.

## Key Concepts

- Análise estática
- Qualidade do código
- Aplicação de estilo
- Detecção de bugs

## Use Cases

- Revisão automatizada de código
- Hooks pré-commit
- Validação em pipelines CI/CD
