## Definition

A Modelagem Dinâmica Empírica (EDM) é um framework para analisar sistemas dinâmicos não lineares usando dados observacionais, sem assumir uma forma paramétrica específica. Ela depende do método de embutimento de Takens para reconstruir o espaço de estados a partir de uma única série temporal.

### Summary

Uma abordagem estatística não paramétrica usada para reconstruir a dinâmica de sistemas complexos a partir de dados de séries temporais observacionais.

## Key Concepts

- Reconstrução do espaço de estados
- Teorema de Takens
- Não linearidade
- Inferência causal

## Use Cases

- Previsão populacional ecológica
- Análise de sinais em neurociência
- Previsão de volatilidade em mercados financeiros
