## Definition

O pós-treinamento é uma etapa crítica no ciclo de vida do aprendizado de máquina que ocorre após o treinamento inicial (pré-treinamento) de um modelo em dados gerais de grande escala. Durante essa fase, o modelo passa por ajustes finos, alinhamento ou quantização.

### Summary

O pós-treinamento refere-se à fase de refinamento de um modelo pré-treinado em conjuntos de dados específicos para adaptá-lo a tarefas particulares ou otimizar seu desempenho.

## Key Concepts

- Ajuste fino
- RLHF (Aprendizado por Reforço com Feedback Humano)
- Quantização
- Adaptação

## Use Cases

- Alinhamento de LLMs com preferências humanas
- Otimização do tamanho do modelo para dispositivos de borda
- Especialização de modelos para diagnóstico médico
