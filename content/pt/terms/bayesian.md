## Definition

Abordagens bayesianas na IA usam teoria da probabilidade para atualizar a probabilidade de hipóteses à medida que mais evidências se tornam disponíveis. Esse método permite que os modelos quantifiquem a incerteza e refinem as previsões dinamicamente

### Summary

Refere-se a métodos estatísticos baseados no Teorema de Bayes para atualizar probabilidades com novas evidências.

## Key Concepts

- Teorema de Bayes
- Probabilidade a priori
- Probabilidade a posteriori
- Quantificação da incerteza

## Use Cases

- Filtragem de spam em e-mails
- Sistemas de diagnóstico médico
- Análise de testes A/B

## Code Example

```python
from sklearn.naive_bayes import GaussianNB
model = GaussianNB()
model.fit(X_train, y_train)
```
