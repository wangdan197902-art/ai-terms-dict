## Definition

Um modelo orientado por dados é um tipo de sistema de inteligência artificial onde o comportamento e as previsões emergem de padrões identificados dentro de dados históricos, em vez de serem definidos por regras codificadas rigidamente.

### Summary

Um modelo computacional cujos parâmetros e estrutura são aprendidos diretamente a partir de dados empíricos, em vez de programação explícita.

## Key Concepts

- Aprendizado Empírico
- Reconhecimento de Padrões
- Inferência Estatística
- Modelos de Caixa Preta

## Use Cases

- Prever tendências do mercado de ações usando dados históricos de preços
- Diagnosticar doenças a partir de exames de imagem médica
- Prever padrões climáticos usando observações de satélite
