## Definition

Os Server-Sent Events (SSE) permitem comunicação unidirecional do servidor para o cliente, onde o servidor pode transmitir dados continuamente sem que o cliente precise fazer polling repetidamente. Ele usa HTTP simples, tornando-o...

### Summary

Um protocolo padrão que permite que servidores web enviem atualizações em tempo real para clientes através de uma única conexão HTTP.

## Key Concepts

- Stream de eventos
- Comunicação unidirecional
- Reconexão automática
- Baseado em HTTP

## Use Cases

- Feed ao vivo de preços de ações
- Notificações em tempo real
- Atualizações de acompanhamento de progresso
