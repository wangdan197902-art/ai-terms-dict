## Definition

Desenvolvido pelo Facebook, o GraphQL fornece uma descrição completa e compreensível dos dados na sua API, dando aos clientes o poder de pedir exatamente o que precisam e nada mais. Ele substitui múltiplas chamadas de API por uma única estrutura flexível.

### Summary

GraphQL é uma linguagem de consulta para APIs e um tempo de execução para atender a essas consultas com dados existentes, permitindo que os clientes solicitem exatamente o que precisam.

## Key Concepts

- Definição de Schema
- Linguagem de Consulta
- Único Ponto de Entrada
- Segurança de Tipos

## Use Cases

- Back-ends de aplicativos web modernos
- Busca de dados em aplicativos móveis
- Orquestração de microsserviços
