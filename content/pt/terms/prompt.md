## Definition

Um prompt serve como a interface principal para interagir com grandes modelos de linguagem e outros sistemas de IA generativa. Ele define o contexto, o tom e as restrições para a saída do modelo. Prompts eficazes são essenciais para guiar o modelo a gerar resultados precisos e relevantes.

### Summary

Um texto de entrada ou instrução fornecido a um modelo de IA generativa para elicitar uma resposta ou comportamento específico.

## Key Concepts

- Ajuste por Instrução
- Janela de Contexto
- Aprendizado Few-Shot
- Tokenização

## Use Cases

- Geração de conteúdo criativo
- Autocompletar código e depuração
- Interações com chatbots de atendimento ao cliente
