## Definition

Os modelos de difusão são uma classe de IA generativa que aprende a reverter um processo estocástico de adição de ruído aos dados. Ao treinar uma rede neural para prever e remover esse ruído passo a passo, eles podem gerar novas amostras de dados de alta qualidade.

### Summary

Uma técnica de modelagem generativa que cria dados revertendo um processo gradual de adição de ruído para reconstruir amostras limpas.

## Key Concepts

- Processo Estocástico
- Agenda de Ruído (Noise Schedule)
- Processo Reverso
- Espaço Latente

## Use Cases

- Geração de imagens de alta fidelidade
- Síntese de áudio
- Projeto molecular para descoberta de fármacos
