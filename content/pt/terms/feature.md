## Definition

No aprendizado de máquina, um feature é um atributo ou variável distinto que descreve uma instância dentro de um conjunto de dados. Os features podem ser numéricos, categóricos ou textuais e servem como a base fundamental para os modelos preditivos.

### Summary

Uma propriedade ou característica mensurável individual de um fenômeno observado, servindo como dados de entrada para modelos de aprendizado de máquina.

## Key Concepts

- Variável de Entrada
- Atributo
- Coluna do Conjunto de Dados
- Entrada de ML

## Use Cases

- Entradas para modelagem preditiva
- Análise e visualização de dados
- Processos de seleção de features
