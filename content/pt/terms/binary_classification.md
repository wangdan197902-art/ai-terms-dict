## Definition

A classificação binária é um problema fundamental de aprendizado de máquina onde a variável de saída é categórica e possui exatamente dois resultados possíveis, como verdadeiro/falso ou spam/não spam. Algoritmos como a regressão logística são comumente utilizados.

### Summary

Uma tarefa de aprendizado supervisionado onde o objetivo é prever uma de duas classes possíveis para cada instância de entrada.

## Key Concepts

- Limiar de Decisão
- Matriz de Confusão
- Regressão Logística
- Métricas de Precisão

## Use Cases

- Detecção de Spam
- Diagnóstico Médico
- Detecção de Fraudes
