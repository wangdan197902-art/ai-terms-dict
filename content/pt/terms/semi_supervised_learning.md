## Definition

O aprendizado semissupervisionado é um paradigma híbrido de treinamento que utiliza uma pequena quantidade de dados rotulados junto com um grande volume de dados não rotulados. A premissa central é que a estrutura dos dados não rotulados pode ajudar a inferir padrões mais robustos.

### Summary

Uma abordagem de aprendizado de máquina que utiliza tanto dados rotulados quanto não rotulados para melhorar a precisão e a generalização do modelo.

## Key Concepts

- Dados rotulados
- Dados não rotulados
- Auto-treinamento
- Hipótese da variedade (Manifold assumption)

## Use Cases

- Classificação de imagens com anotações limitadas
- Análise de sentimento textual com rótulos esparsos
- Previsão de diagnóstico médico com dados escassos de especialistas
