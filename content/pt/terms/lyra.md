## Definition

No contexto da terminologia moderna de IA, Lyra frequentemente denota sistemas de IA especializados focados em melhorar a interação do usuário por meio de processamento de linguagem natural. Pode referir-se a um LLM de código aberto desenvolvido

### Summary

Lyra refere-se a várias iniciativas ou modelos de IA, mais notavelmente um modelo de linguagem grande de código aberto ou uma ferramenta específica de busca e descoberta baseada em IA projetada para melhorar a recuperação de informações.

## Key Concepts

- Modelo de Linguagem Grande
- Recuperação de Informações
- IA de Código Aberto
- Busca Semântica

## Use Cases

- Alimentar agentes conversacionais
- Melhorar a relevância dos resultados de mecanismos de busca
- Fornecer capacidades de PLN de código aberto para desenvolvedores
