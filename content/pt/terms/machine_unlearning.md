## Definition

Esta técnica atende a regulamentações de privacidade, como o 'direito ao esquecimento' do GDPR, permitindo que os modelos esqueçam dados específicos dos usuários enquanto retêm o conhecimento geral. O objetivo é aproximar o desempenho do modelo como se esses dados nunca tivessem sido usados no treinamento.

### Summary

O desaprendizado de máquina é o processo de remover pontos de dados específicos ou sua influência de um modelo treinado, sem a necessidade de retreiná-lo do zero.

## Key Concepts

- Direito ao Esquecimento
- Aproximação de Retreinamento de Modelo
- Privacidade de Dados
- Atualizações de Gradiente

## Use Cases

- Cumprimento de solicitações de exclusão de dados
- Remoção de dados tendenciosos ou errôneos dos modelos
- Mitigação de ataques de envenenamento de dados
