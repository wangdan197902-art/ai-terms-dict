## Definition

O viés nos algoritmos geralmente origina-se de dados de treinamento não representativos, escolhas de design subjetivas ou ciclos de feedback que amplificam preconceitos sociais existentes. Manifesta-se como previsões distorcidas

### Summary

O viés algorítmico refere-se a erros sistemáticos e repetitivos em um sistema de computador que criam resultados injustos, como privilegiar um grupo arbitrário em detrimento de outros.

## Key Concepts

- Representação de Dados
- Erro Sistemático
- Estratégias de Mitigação

## Use Cases

- Correção das classificações nos resultados de busca
- Equilíbrio de conjuntos de dados para diagnóstico médico
- Diversificação dos conjuntos de treinamento para reconhecimento de imagens
