## Definition

As RNNs são projetadas para reconhecer padrões em sequências de dados, como texto, genomas, escrita à mão ou palavras faladas. Diferentemente das redes feedforward, elas possuem memória interna que captura informações sobre entradas anteriores, permitindo que o contexto atual influencie a saída.

### Summary

Uma RNN é uma classe de redes neurais artificiais onde as conexões entre nós formam um grafo direcionado ao longo de uma sequência temporal.

## Key Concepts

- Dados Sequenciais
- Estado Oculto
- Dinâmica Temporal
- Backpropagation Through Time

## Use Cases

- Modelagem de Linguagem
- Previsão de Séries Temporais
- Reconhecimento de Fala
