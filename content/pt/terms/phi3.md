## Definition

Phi-3 é uma série de modelos de linguagem pequenos (SLMs) lançados pela Microsoft, projetados para oferecer alto desempenho comparável a modelos maiores, enquanto requer significativamente menos recursos computacionais. Eles são otimizados para execução eficiente em hardware limitado.

### Summary

Uma família de modelos de linguagem grandes (LLMs) de pesos abertos desenvolvida pela Microsoft Research.

## Key Concepts

- Modelos de Linguagem Pequenos
- Microsoft Research
- IA de Borda
- Treinamento com Dados Sintéticos

## Use Cases

- Execução de LLMs em dispositivos móveis ou laptops
- Inferência com custo-benefício para aplicações empresariais
- Processamento em tempo real com requisitos de baixa latência
