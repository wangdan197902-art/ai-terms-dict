## Definition

Modelos de pesos abertos diferem da IA totalmente de código aberto porque apenas os parâmetros aprendidos finais são liberados, não necessariamente a infraestrutura ou os dados usados para criá-los. Isso permite que os usuários executem inferências com o modelo.

### Summary

Modelos de IA cujos parâmetros treinados (pesos) são publicados, mas o código de treinamento e o conjunto de dados podem permanecer privados.

## Key Concepts

- Pesos do modelo
- Inferência
- Ajuste fino
- Transparência parcial

## Use Cases

- Implantação de LLMs em hardware local
- Personalização de modelos pré-treinados
- Redução de custos de API
