## Definition

Pyannote é uma biblioteca Python de código aberto desenvolvida pelo pyannote.audio, especializada em diarização de falantes, que é o processo de determinar 'quem falou quando' em um fluxo de áudio. Ela aproveita redes profundas

### Summary

Pyannote é uma biblioteca Python que oferece soluções de ponta para tarefas de diarização de falantes e segmentação de áudio.

## Key Concepts

- Diarização de Falantes
- Segmentação de Áudio
- Detecção de Atividade de Voz
- Agrupamento de Embeddings

## Use Cases

- Transcrição de reuniões com múltiplos falantes
- Análise de conteúdo de rádio transmitido
- Pesquisa sobre dinâmicas conversacionais
