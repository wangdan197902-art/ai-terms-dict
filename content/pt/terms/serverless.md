## Definition

A arquitetura Serverless permite que os desenvolvedores criem e executem aplicativos sem gerenciar a infraestrutura de servidores. O provedor de nuvem dimensiona automaticamente os recursos para cima ou para baixo com base na demanda, cobrando apenas pelo uso efetivo.

### Summary

Um modelo de execução em computação em nuvem no qual o provedor gerencia dinamicamente a alocação de recursos de máquina, em vez do usuário.

## Key Concepts

- Função como Serviço (FaaS)
- Dimensionamento automático (Auto-scaling)
- Pagamento por uso
- Orientado a eventos

## Use Cases

- Pipelines de processamento de imagens
- Backend para aplicativos móveis
- Tarefas de dados agendadas
