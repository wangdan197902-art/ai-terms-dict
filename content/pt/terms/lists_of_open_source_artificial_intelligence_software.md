## Definition

Estes se referem a repositórios organizados, como tópicos do GitHub, listas 'Awesome' ou wikis mantidas pela comunidade, que agregam software de código aberto relacionado à inteligência artificial. Eles servem como recursos essenciais para descoberta.

### Summary

Coleções curadas ou diretórios que catalogam ferramentas, bibliotecas e estruturas de IA publicamente disponíveis.

## Key Concepts

- Código aberto
- Curadoria de recursos
- Manutenção comunitária
- Descoberta de software

## Use Cases

- Descoberta de ferramentas de pesquisa
- Seleção de stack tecnológico para startups
- Coleta de recursos educacionais
