## Definition

Ruído de rótulos refere-se às discrepâncias entre os rótulos de classe verdadeiros das instâncias de dados e os rótulos fornecidos no conjunto de treinamento. Isso pode surgir de erros de anotação humana, pontos de dados ambíguos ou processos de coleta de dados defeituosos, afetando a performance do modelo.

### Summary

Erros ou inconsistências nos rótulos alvo de um conjunto de dados utilizado para treinamento de aprendizado de máquina supervisionado.

## Key Concepts

- Qualidade dos Dados
- Aprendizado Robusto
- Erros de Anotação
- Ruído Simétrico/Assimétrico

## Use Cases

- Treinamento de modelos com dados obtidos por multidão (crowdsourcing)
- Manipulação de conjuntos de dados imperfeitos do mundo real
- Melhoria da robustez do modelo
