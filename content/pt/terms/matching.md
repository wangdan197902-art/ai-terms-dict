## Definition

A correspondência é uma técnica crítica em aprendizado de máquina usada para estabelecer relações entre entidades de dados distintas. Na visão computacional, a correspondência de características identifica pontos correspondentes entre imagens.

### Summary

A correspondência envolve alinhar dois conjuntos de pontos de dados ou características para identificar correspondências, similaridades ou pareamentos ótimos entre eles.

## Key Concepts

- Correspondência de Características
- Métricas de Similaridade
- Espaços de Embedding
- Busca por Vizinho Mais Próximo

## Use Cases

- Rastreamento de objetos em vídeo
- Sistemas de recomendação
- Detecção de duplicatas em bancos de dados
