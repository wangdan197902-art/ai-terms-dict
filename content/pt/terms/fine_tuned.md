## Definition

O ajuste fino (fine-tuning) envolve pegar um modelo que já foi treinado em um grande conjunto de dados geral e continuar seu treinamento em um conjunto de dados menor e específico para a tarefa. Essa técnica aproveita os recursos gerais aprendidos anteriormente para melhorar o desempenho em tarefas especializadas.

### Summary

O processo de treinar adicionalmente um modelo pré-treinado em um conjunto de dados específico para adaptá-lo a uma tarefa downstream particular.

## Key Concepts

- transferência de aprendizado
- atualização de pesos
- específico para a tarefa
- modelo pré-treinado

## Use Cases

- Adaptação de LLMs para revisão de documentos jurídicos
- Personalização de modelos de visão para detecção de defeitos industriais
- Especialização de reconhecimento de fala para sotaques específicos
