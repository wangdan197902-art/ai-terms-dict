## Definition

Código aberto refere-se a um modelo de desenvolvimento no qual o código subjacente de um projeto de software é publicamente acessível. Na IA, isso permite que pesquisadores e desenvolvedores inspecionem, modifiquem e redistribuam os algoritmos.

### Summary

Software cujo código-fonte é disponibilizado gratuitamente para uso, modificação e distribuição por qualquer pessoa.

## Key Concepts

- Transparência
- Colaboração
- Acessibilidade
- Impulsionado pela comunidade

## Use Cases

- Desenvolvimento do framework PyTorch
- Bibliotecas de modelos da Hugging Face
- Reprodutibilidade em pesquisa acadêmica
