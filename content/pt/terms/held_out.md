## Definition

Um conjunto de dados 'held-out' consiste em exemplos intencionalmente excluídos da fase de treinamento de um modelo de aprendizado de máquina. Esse subconjunto é usado para avaliar quão bem o modelo generaliza para dados não vistos, fornecendo uma estimativa imparcial do desempenho real.

### Summary

Amostras de dados reservadas do conjunto de treinamento para avaliar o desempenho do modelo e evitar o sobreajuste durante o desenvolvimento.

## Key Concepts

- Generalização
- Sobreajuste
- Conjunto de validação
- Avaliação imparcial

## Use Cases

- Ajuste de hiperparâmetros
- Comparação de diferentes arquiteturas de modelos
- Estimativa final de desempenho antes da produção
