## Definition

O MS MARCO (Microsoft Machine Reading Comprehension) é um conjunto de dados amplamente utilizado em processamento de linguagem natural, particularmente para recuperação de informação e resposta a perguntas. Consiste em buscas anonimizadas

### Summary

Conjunto de Dados de Compreensão de Leitura por Máquina da Microsoft, uma coleção em larga escala de consultas de pesquisa reais e trechos de documentos relevantes, usada para treinar sistemas de recuperação de informação.

## Key Concepts

- Recuperação de Informação
- Classificação de Passagens
- Consultas de Pesquisa
- Compreensão de Leitura por Máquina

## Use Cases

- Treinamento de mecanismos de busca
- Desenvolvimento de sistemas de resposta a perguntas
- Avaliação de benchmarks de modelos de recuperação
