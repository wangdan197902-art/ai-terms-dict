## Definition

A Automação de Processos Robóticos (RPA) emprega robôs de software, muitas vezes aprimorados com IA, para imitar interações humanas com sistemas digitais. É usada para otimizar fluxos de trabalho, como entrada de dados e processamento de faturas.

### Summary

O uso de bots de software para automatizar tarefas digitais repetitivas e de alto volume, tradicionalmente realizadas por humanos em processos de negócios.

## Key Concepts

- Bots de software
- Automação de fluxo de trabalho
- Processamento baseado em regras
- Força de trabalho digital

## Use Cases

- Processamento de faturas
- Onboarding de clientes
- Tarefas de migração de dados
