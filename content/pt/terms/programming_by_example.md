## Definition

A Programação por Exemplo (PBE) é um paradigma na síntese de programas onde os desenvolvedores especificam o comportamento desejado por meio de pares concretos de entrada e saída, em vez de escrever código explícito. O sistema de IA analisa

### Summary

Uma técnica de IA na qual programas são sintetizados automaticamente inferindo regras a partir de exemplos de entrada e saída fornecidos pelos usuários.

## Key Concepts

- Síntese de Programas
- Exemplos de Entrada-Saída
- Inferência Indutiva
- Baixo Código/Sem Código

## Use Cases

- Geração de fórmulas do Excel a partir de dados de amostra
- Scripts de transformação de texto e limpeza de dados
- Automação de interações repetitivas na interface do usuário
