## Definition

O Uso de Ferramentas permite que modelos de linguagem interajam com ambientes de software externos chamando funções predefinidas, como calculadoras, mecanismos de busca ou consultas a bancos de dados. Essa abordagem estende as capacidades do modelo, permitindo ações concretas e acesso a informações em tempo real.

### Summary

Um paradigma no qual agentes de IA selecionam e executam funções externas ou APIs para realizar tarefas específicas além de suas capacidades nativas.

## Key Concepts

- Chamada de Função
- Arquitetura de Agente
- Integração de API
- Conhecimento Externo

## Use Cases

- Bots de suporte ao cliente automatizados
- Assistentes de análise de dados
- Automação residencial inteligente
