## Definition

O aprendizado de features, frequentemente associado ao deep learning, permite que os modelos aprendam representações hierárquicas diretamente dos dados de entrada brutos, em vez de depender da engenharia de features manual. Através de camadas sucessivas, o modelo extrai características cada vez mais abstratas e relevantes para a tarefa.

### Summary

Uma abordagem onde os algoritmos descobrem automaticamente os features necessários para detecção ou classificação a partir de dados brutos.

## Key Concepts

- Representação Automática
- Camadas Hierárquicas
- Deep Learning (Aprendizado Profundo)
- Evitação de Engenharia Manual

## Use Cases

- Reconhecimento de imagens e detecção de objetos
- Transcrição de fala para texto
- Tarefas de compreensão de linguagem natural
