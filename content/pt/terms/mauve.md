## Definition

O MAUVE é uma medida estatística projetada para avaliar quão de perto a saída de um modelo de linguagem generativa se assemelha ao uso da linguagem humana. Diferente das simples pontuações de perplexidade, o MAUVE usa embutimentos virtuais

### Summary

MAUVE (Medindo Alinhamento Usando Embutimentos Virtuais) é uma métrica usada no processamento de linguagem natural para avaliar o alinhamento entre distribuições de texto geradas e distribuições de texto escritas por humanos.

## Key Concepts

- Avaliação de Geração de Texto
- Correspondência de Distribuições
- Embutimentos Virtuais
- Naturalidade Linguística

## Use Cases

- Avaliação de saídas de modelos estilo GPT
- Ajuste fino de modelos de linguagem para texto semelhante ao humano
- Testes de desempenho de IA generativa
