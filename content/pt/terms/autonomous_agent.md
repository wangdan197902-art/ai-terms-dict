## Definition

Em inteligência artificial, um agente autônomo é uma entidade que opera de forma independente dentro de um ambiente. Ele usa sensores para perceber estados e atuadores para realizar ações, guiado por um intern

### Summary

Um agente autônomo é um sistema capaz de perceber seu ambiente e agir independentemente para alcançar objetivos específicos sem intervenção humana direta.

## Key Concepts

- Percepção
- Ação
- Comportamento orientado a objetivos
- Interação com o ambiente

## Use Cases

- Veículos autônomos
- Automação de processos robóticos (RPA)
- IA para jogos
