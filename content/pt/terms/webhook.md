## Definition

Um webhook é um mecanismo pelo qual um serviço fornece informações em tempo real a outro serviço quando um evento ocorre. Em vez de fazer polling (consulta periódica) para detectar mudanças, o sistema de origem envia uma requisição HTTP POST para um URL especificado.

### Summary

Um retorno de chamada HTTP definido pelo usuário, acionado por eventos específicos, permitindo que sistemas enviem notificações em tempo real para outros aplicativos.

## Key Concepts

- Orientado a eventos
- HTTP POST
- URL de retorno de chamada
- Notificação push

## Use Cases

- Implantações automatizadas de CI/CD
- Notificações de gateways de pagamento
- Integrações com bots do Slack
