## Definition

Esses bancos de dados permitem a modelagem dinâmica de dados ao não impor estruturas rígidas de tabelas ou definições de colunas antecipadamente. Essa flexibilidade permite que desenvolvedores armazenem dados não estruturados ou semiestruturados, atendendo

### Summary

Bancos de dados agnósticos ao esquema são sistemas de armazenamento que permitem estruturas de dados flexíveis sem exigir esquemas predefinidos, frequentemente usados em ambientes NoSQL.

## Key Concepts

- NoSQL
- Esquema dinâmico
- Flexibilidade
- Dados não estruturados

## Use Cases

- Armazenamento de conteúdo gerado por usuários
- Prototipagem para desenvolvimento rápido de aplicativos
- Manipulação de formatos de dados variados provenientes de dispositivos IoT
