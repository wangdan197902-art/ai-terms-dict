## Definition

O CAM gera mapas de calor sobrepostos às imagens de entrada para mostrar quais pixels contribuíram mais para a decisão do modelo para um rótulo de classe específico. Ele funciona aplicando o agrupamento médio global à última camada convolucional.

### Summary

O mapeamento de ativação de classe (CAM) é uma técnica de visualização que destaca as regiões em uma imagem de entrada mais responsáveis por uma classe prevista específica.

## Key Concepts

- Visualização de Mapa de Calor
- Interpretabilidade
- Importância dos Recursos
- Agrupamento Médio Global

## Use Cases

- Verificação de diagnóstico de imagens médicas
- Depuração de detecção autônoma de objetos
- Relatórios de IA Explicável
