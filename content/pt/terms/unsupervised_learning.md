## Definition

O aprendizado não supervisionado identifica estruturas ocultas, clusters ou distribuições dentro de dados brutos de forma autônoma. Métodos comuns incluem agrupamento, redução de dimensionalidade e modelagem generativa. É

### Summary

Uma técnica de aprendizado de máquina onde os modelos aprendem padrões a partir de dados não rotulados, sem orientação explícita ou respostas corretas.

## Key Concepts

- Agrupamento
- Redução de Dimensionalidade
- Reconhecimento de Padrões
- Ausência de Dados Rotulados

## Use Cases

- Segmentação de clientes
- Detecção de anomalias
- Modelagem de tópicos em PLN
