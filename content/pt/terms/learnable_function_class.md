## Definition

Na teoria do aprendizado estatístico, uma classe de funções aprendíveis representa o espaço de hipóteses disponível para um algoritmo. Ela define a gama de padrões ou mapeamentos que o modelo potencialmente pode capturar com base em sua capacidade.

### Summary

Uma classe de funções aprendíveis é um conjunto de funções matemáticas definidas por uma arquitetura de modelo específica e um espaço de parâmetros que um algoritmo de aprendizado pode otimizar.

## Key Concepts

- Espaço de Hipóteses
- Capacidade do Modelo
- Dimensão VC

## Use Cases

- Seleção de arquiteturas de modelo apropriadas
- Análise teórica dos limites de generalização
- Compreensão das limitações do modelo
