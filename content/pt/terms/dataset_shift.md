## Definition

O desvio de conjunto de dados ocorre quando a distribuição dos dados usada para treinar um modelo de aprendizado de máquina difere da distribuição dos dados encontrados durante a inferência. Essa discrepância pode levar a uma queda significativa no desempenho.

### Summary

O desvio de conjunto de dados refere-se ao fenômeno em que as propriedades estatísticas dos dados de entrada mudam entre o treinamento e a implantação.
