## Definition

No contexto de IA, 'fonte' geralmente denota a procedência de conjuntos de dados de treinamento, bibliotecas de código aberto ou pesos de modelos pré-treinados. Rastrear as fontes é crítico para reprodutibilidade, conformidade com licenças

### Summary

Refere-se à origem de dados, código ou modelos usados no desenvolvimento e implantação de IA.

## Key Concepts

- Procedência de Dados
- Código Aberto
- Fluxo de Entrada
- Reprodutibilidade

## Use Cases

- Auditoria de dados de treinamento para viés
- Gerenciamento de versionamento de modelos
- Citação de dependências de código aberto
