### Summary

A tarefa de PLN (Processamento de Linguagem Natural) de reescrever o texto para transmitir o mesmo significado usando palavras ou estruturas de frase diferentes.

## Key Concepts

- Preservação Semântica
- Reescrita de Texto
- Geração de Linguagem Natural
- Sinonímia

## Use Cases

- Ferramentas de detecção e prevenção de plágio
- Augmentation de dados para treinamento de modelos de PLN
- Melhoria da legibilidade de documentos técnicos
