## Definition

A IA Centrada em Dados representa uma mudança de paradigma no desenvolvimento de inteligência artificial, focando na melhoria sistemática dos dados usados para treinar modelos, em vez de otimizar exclusivamente os algoritmos ou hiperparâmetros.

### Summary

Uma abordagem para o desenvolvimento de IA que prioriza a melhoria da qualidade e da quantidade dos dados de treinamento em vez de modificar as arquiteturas dos modelos.

## Key Concepts

- Qualidade dos Dados
- Refinamento Iterativo
- Consistência dos Rótulos
- Curadoria de Conjuntos de Dados

## Use Cases

- Melhorar a precisão da visão computacional corrigindo imagens mal rotuladas
- Reduzir o viés em algoritmos de contratação por meio de conjuntos de dados equilibrados
- Aumentar o desempenho de modelos de PLN corrigindo erros gramaticais no texto de treinamento
