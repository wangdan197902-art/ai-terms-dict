## Definition

Uma API define um conjunto de protocolos e ferramentas para criar software e aplicativos. Em IA, as APIs permitem que os desenvolvedores acessem modelos poderosos, como LLMs ou geradores de imagens, sem precisar hospedá-los localmente.

### Summary

Uma Interface de Programação de Aplicativos que permite que diferentes sistemas de software se comuniquem e troquem dados.

## Key Concepts

- Endpoints
- REST
- Autenticação
- Payload

## Use Cases

- Integração de chatbots em sites
- Acesso a modelos de ML baseados na nuvem
- Recuperação de dados de serviços de IA

## Code Example

```python
import requests
response = requests.get('https://api.ai.com/v1/generate', headers={'Authorization': 'Bearer token'})
```
