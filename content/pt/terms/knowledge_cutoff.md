## Definition

A data de limite de conhecimento define o limite temporal dos dados de treinamento de um modelo de linguagem. Quaisquer informações, eventos ou desenvolvimentos que ocorreram após essa data geralmente são desconhecidos pelo modelo, a menos que sejam fornecidos por meio de ferramentas externas ou atualizações contínuas.

### Summary

O limite de conhecimento é a data específica após a qual um modelo de linguagem não foi treinado e, portanto, carece de informações sobre eventos ocorridos posteriormente.
