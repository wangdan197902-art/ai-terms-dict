## Definition

Na teoria computacional do aprendizado, a complexidade de amostragem quantifica a quantidade de dados necessária para treinar um modelo efetivamente. Ela equilibra a compensação entre a capacidade do modelo e a disponibilidade de dados, garantindo

### Summary

A complexidade de amostragem refere-se ao número de exemplos de treinamento necessários para que um algoritmo de aprendizado de máquina atinja um nível específico de desempenho com alta probabilidade.

## Key Concepts

- Erro de Generalização
- Tamanho dos dados de treinamento
- Convergência
- Prevenção de overfitting

## Use Cases

- Determinar requisitos de conjuntos de dados para novos modelos
- Comparar a eficiência de diferentes algoritmos de aprendizado
- Planejar esforços de coleta de dados
