## Definition

Esta abordagem emprega técnicas de satisfação de restrições dentro de uma base de conhecimento para garantir que os produtos montados atendam a todos os requisitos técnicos e do cliente. Ela evita combinações inválidas ao impor

### Summary

Um processo automatizado que utiliza bases de conhecimento específicas de domínio para gerar configurações válidas de produtos a partir de restrições do usuário.

## Key Concepts

- Satisfação de Restrições
- Modelagem de Produto
- Sistemas Baseados em Regras
- Raciocínio Automatizado

## Use Cases

- Montagem personalizada de hardware de computador
- Seleção de licenças de software e recursos
- Configuração de máquinas industriais
