## Definition

O aprendizado ativo reduz a quantidade de dados rotulados necessários permitindo que o modelo escolha as instâncias mais informativas para rotulação humana. Em vez de receber passivamente amostras aleatórias, o algoritmo seleciona ativamente os exemplos que mais contribuirão para a melhoria do modelo.

### Summary

Uma estratégia de aprendizado de máquina na qual o algoritmo consulta seletivamente um usuário ou oráculo para rotular novos pontos de dados, otimizando o processo de treinamento.

## Key Concepts

- Estratégia de consulta
- Eficiência de rotulação
- Amostragem por incerteza
- Humano no loop

## Use Cases

- Anotação de imagens médicas com radiologistas especialistas
- Rotulagem de análise de sentimento para dialetos raros
- Seleção de cenários para direção autônoma
