## Definition

A chamada de função permite que grandes modelos de linguagem interajam com ferramentas externas e APIs gerando saídas estruturadas, como objetos JSON, que especificam qual função executar e quais argumentos fornecer...

### Summary

Um mecanismo que permite que LLMs gerem dados estruturados que acionam funções específicas de software.

## Key Concepts

- Saída Estruturada
- Integração de API
- Uso de Ferramentas
- Schema JSON

## Use Cases

- Reserva de voos via APIs de viagem
- Execução de cálculos matemáticos complexos
- Consulta a registros de banco de dados
