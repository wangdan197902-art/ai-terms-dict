## Definition

No processamento de linguagem natural, o contexto é crucial para resolver ambiguidades, como entender pronomes ou idiomatismos com base em frases anteriores. Arquiteturas modernas, como os transformadores, utilizam mecanismos de atenção para capturar dependências de longo alcance no contexto.

### Summary

Contexto refere-se às informações ou ao ambiente circundante que ajudam um modelo de IA a interpretar os dados de entrada com precisão e gerar respostas relevantes.

## Key Concepts

- Mecanismo de Atenção
- Modelagem de Sequência
- Resolução de Ambiguidade
- Tamanho da Janela

## Use Cases

- Chatbots conversacionais
- Resumo de documentos
- Assistentes de conclusão de código
