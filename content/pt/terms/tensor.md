## Definition

Em ciência da computação e aprendizado profundo, um tensor é um objeto matemático que generaliza escalares, vetores e matrizes para dimensões superiores. Ele é caracterizado por sua ordem (número de dimensões) e...

### Summary

Uma matriz multidimensional que serve como estrutura de dados fundamental para frameworks de aprendizado profundo.

## Key Concepts

- Ordem (Rank)
- Formato (Shape)
- Dimensionalidade
- Broadcasting (transmissão)

## Use Cases

- Processamento de imagens (tensores 4D)
- Armazenamento de pesos de redes neurais
- Entrada de dados em lote

## Code Example

```python
import torch
t = torch.tensor([[1, 2], [3, 4]])
```
