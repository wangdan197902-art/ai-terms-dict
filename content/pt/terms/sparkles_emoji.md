## Definition

O emoji de brilhos é um ícone gráfico frequentemente empregado em interfaces de usuário para aplicações de IA, significando novidade, melhoria ou aprimoramento criativo. Ele serve como uma dica não verbal indicando que algo foi atualizado, otimizado ou que possui características especiais.

### Summary

Um símbolo visual frequentemente usado em interfaces de IA para denotar novos recursos, melhorias ou transformações mágicas no conteúdo gerado.

## Key Concepts

- Design UI/UX
- Pistas Visuais
- Destaque de Recursos
- Engajamento do Usuário

## Use Cases

- Indicando atualizações do modelo
- Destacando novos recursos
- Melhorando a apresentação de saídas criativas
