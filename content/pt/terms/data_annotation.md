## Definition

Esta etapa crítica envolve anexar metadados significativos aos pontos de dados brutos para que os algoritmos possam aprender a relação entre entrada e saída. Por exemplo, caixas delimitadoras ao redor de objetos em imagem

### Summary

A anotação de dados é o processo de rotular dados brutos, como imagens ou texto, para torná-los adequados para o treinamento de aprendizado de máquina supervisionado.

## Key Concepts

- Aprendizado Supervisionado
- Rotulagem
- Verdade Fundamental (Ground Truth)
- Metadados

## Use Cases

- Treinamento de modelos de detecção de objetos
- Construção de classificadores de análise de sentimento
- Criação de conjuntos de dados para reconhecimento de fala
