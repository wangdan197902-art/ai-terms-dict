## Definition

A tradução de Imagem para Imagem (I2I) envolve mapear pixels de um domínio de origem para um domínio de destino usando modelos de aprendizado profundo, como GANs ou modelos de difusão. Isso permite transferência de estilo, segmentação semântica

### Summary

A tradução de imagem para imagem (I2I) é uma técnica de visão computacional que transforma uma imagem de entrada em uma imagem de saída correspondente, preservando o conteúdo semântico.

## Key Concepts

- Redes Generativas Adversariais
- Segmentação Semântica
- Transferência de Estilo
- Espaço Latente

## Use Cases

- Transferência de estilo artístico
- Colorização de fotos
- Melhoria de imagens de satélite
