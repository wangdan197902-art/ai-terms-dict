## Definition

O CBR opera no princípio de que problemas semelhantes têm soluções semelhantes. O processo envolve recuperar o caso histórico mais semelhante de uma base de conhecimento, adaptar sua solução para se adequar à situação atual e reter a nova experiência para uso futuro.

### Summary

O raciocínio baseado em casos (CBR) é uma abordagem de resolução de problemas de IA que utiliza experiências passadas ou casos semelhantes para encontrar soluções para novos problemas.

## Key Concepts

- recuperação
- reutilização
- revisão
- retenção

## Use Cases

- Análise de precedentes legais
- Assistência ao diagnóstico médico
- Chatbots de suporte ao cliente
