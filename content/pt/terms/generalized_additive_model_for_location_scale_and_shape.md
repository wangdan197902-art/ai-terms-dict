## Definition

Ao contrário dos modelos de regressão tradicionais que se concentram apenas na média, o GAMLSS modela toda a distribuição, incluindo localização (média/mediana), escala (variância), assimetria e curtose. Ele usa generalizado

### Summary

O GAMLSS é uma estrutura estatística flexível que permite que todos os parâmetros de uma distribuição sejam modelados como funções de variáveis explicativas usando suavizadores aditivos.

## Key Concepts

- Modelagem de distribuição
- Suavização aditiva
- Modelagem de variância
- Flexibilidade estatística

## Use Cases

- Análise de curvas de crescimento
- Avaliação de riscos
- Ajuste de distribuições de dados complexos
