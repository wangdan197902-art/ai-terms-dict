## Definition

A Incorporação de Distribuições por Kernel permite que objetos probabilísticos sejam tratados como pontos em um espaço de características de alta dimensão chamado Espaço de Hilbert de Kernel Reprodutor (RKHS). Ao mapear distribuições para este espaço, operações algébricas lineares podem ser realizadas diretamente nas distribuições, facilitando tarefas como comparação e teste de hipóteses.

### Summary

Uma técnica que mapeia distribuições de probabilidade para um Espaço de Hilbert de Kernel Reprodutor (RKHS) para permitir comparação e manipulação por meio de operações vetoriais.

## Key Concepts

- Espaço de Hilbert de Kernel Reprodutor
- Incorporação Média
- Inferência Não Paramétrica
- Comparação de Distribuições

## Use Cases

- Testes de hipótese de duas amostras
- Descoberta causal a partir de dados observacionais
- Comparação de saídas de modelos generativos
