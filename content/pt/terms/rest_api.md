## Definition

As APIs REST permitem a comunicação entre clientes e servidores por meio de operações sem estado sobre protocolos HTTP, como GET, POST, PUT e DELETE. Elas estruturam recursos como URIs e utilizam formatos padrão...

### Summary

Uma API baseada no estilo arquitetural Representational State Transfer, projetada para aplicativos em rede que utiliza métodos HTTP padrão.

## Key Concepts

- Ausência de Estado (Statelessness)
- Métodos HTTP
- URIs Baseadas em Recursos
- Serialização JSON

## Use Cases

- Comunicação entre Microsserviços
- Integração de Backend para Aplicativos Móveis
- Acesso a Dados de Terceiros
