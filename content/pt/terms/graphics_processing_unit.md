## Definition

Uma GPU é um processador de alto desempenho desenvolvido originalmente para lidar com tarefas de renderização gráfica. Diferente das CPUs, que possuem poucos núcleos otimizados para processamento sequencial serial, as GPUs contêm milhares de núcleos menores projetados para executar múltiplas operações simultaneamente.

### Summary

Um circuito eletrônico especializado projetado inicialmente para manipulação e renderização rápidas de imagens, agora amplamente utilizado para computação paralela em IA.

## Key Concepts

- Processamento Paralelo
- CUDA Cores
- Multiplicação de Matrizes
- Aceleração de Hardware

## Use Cases

- Treinamento de grandes redes neurais
- Inferência em visão computacional em tempo real
- Renderização de gráficos 3D em simulações
