## Definition

Em inteligência artificial, geração refere-se à capacidade dos modelos, particularmente Redes Adversárias Generativas (GANs) e LLMs baseados em Transformers, de produzir conteúdo novo, como texto, imagens

### Summary

O processo pelo qual os modelos generativos criam novas instâncias de dados que se assemelham à distribuição de treinamento.

## Key Concepts

- Modelagem Probabilística
- Espaço Latente
- Previsão de Tokens
- Dados Sintéticos

## Use Cases

- Geração de Linguagem Natural
- Síntese de Imagens
- Autocompletar Código
