## Definition

Um sistema de descoberta é uma estrutura computacional voltada para acelerar avanços científicos ou analíticos automatizando a exploração de vastos espaços de dados. Diferente da otimização tradicional que se concentra em refinar parâmetros conhecidos, este sistema busca ativamente novas regiões no espaço de solução.

### Summary

Uma estrutura automatizada ou processo algorítmico projetado para identificar, explorar e avaliar soluções potenciais, padrões ou hipóteses dentro de um grande espaço de busca.

## Key Concepts

- Exploração Automatizada
- Geração de Hipóteses
- Aprendizado Ativo
- Navegação no Espaço de Busca

## Use Cases

- Identificação de novas estruturas moleculares para farmacêuticos
- Descoberta de materiais novos com propriedades específicas
- Encontrar correlações inesperadas em grandes conjuntos de dados
