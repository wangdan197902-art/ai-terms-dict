## Definition

No contexto de IA e ciência de dados, numérico refere-se a tipos de dados ou métodos que envolvem valores quantitativos, como inteiros, pontos flutuantes e decimais. Diferente de dados categóricos ou textuais, os dados numéricos

### Summary

Relativo a, envolvendo ou utilizando números ou cálculos matemáticos em vez de lógica simbólica ou texto.

## Key Concepts

- Pontos Flutuantes
- Inteiros
- Tipos de Dados
- Dados Quantitativos

## Use Cases

- Modelagem estatística
- Previsão financeira
- Processamento de dados de sensores

