## Definition

Az AutoML (Automated Machine Learning) egyszerűsíti a ML-modellek fejlesztését olyan feladatok automatizálásával, mint az adat-előfeldolgozás, a jellemzőmérnökség, a modellválasztás és a hiperparaméter-optimalizálás. Lehetővé teszi...

### Summary

Egy metodológia, amely automatizálja a gépi tanulás valós problémákra való alkalmazásának végpontig terjedő folyamatát, csökkentve a manuális munkát.

## Key Concepts

- Hiperparaméter-optimalizálás
- Jellemzőmérnökség
- Modellválasztás
- Demokratizáció

## Use Cases

- Gyors prototípuskészítés üzleti elemzők számára
- Nagy léptékű termelési folyamatok optimalizálása
- Több algoritmus automatikus összehasonlítása

## Code Example

```python
from auto_ml import Predictor
predictor = Predictor(type_of_estimator='classifier')
predictor.fit(dataframe, target='label')
```
