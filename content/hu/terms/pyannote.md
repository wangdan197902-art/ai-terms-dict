## Definition

A Pyannote egy nyílt forráskódú Python könyvtár, amelyet a pyannote.audio fejlesztett, és specializálódott a beszélődiarizációra, amelynek során meghatározzák, hogy 'ki beszélt mikor' egy adófolyamban. A mélytanulásra épít.

### Summary

A Pyannote egy Python könyvtár, amely állóhellyel rendelkező megoldásokat kínál a beszélődiarizációra és az audiószegmentálási feladatokra.

## Key Concepts

- Beszélődiarizáció
- Audiószegmentálás
- Hangtevékenység-észlelés
- Beágyazásos klaszterezés

## Use Cases

- Többbeszédős megbeszélések lefordítása
- Adózott rádiós tartalmak elemzése
- Beszélgetési dinamikák kutatása
