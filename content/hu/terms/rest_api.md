## Definition

A REST API-k lehetővé teszik a kommunikációt az ügyfelek és a szerverek között az állapotmentes műveletek HTTP protokollokon (például GET, POST, PUT és DELETE) keresztüli kihasználásával. Erőforrásokat URI-ként strukturálnak, és szabványos formátumokat használnak az adatcserére.

### Summary

A Representational State Transfer (REST) API egy olyan építészeti stílus a hálózati alkalmazások tervezéséhez, amely a szabványos HTTP módszerekre támaszkodik.

## Key Concepts

- Állapotmentesség
- HTTP módszerek
- Erőforráson alapuló URI-k
- JSON szerializálás

## Use Cases

- Mikroszolgáltatások közötti kommunikáció
- Mobilalkalmazások backend integrációja
- Harmadik féltől származó adatokhoz való hozzáférés
