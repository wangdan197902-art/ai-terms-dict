## Definition

Az AI és az adattudomány kontextusában a numerikus a kvantitatív értékeket (például egész számokat, lebegőpontos számokat és tizedestörteket) magában foglaló adattípusokra vagy módszerekre utal. A kategorikus vagy szöveges adatokkal ellentétben a numerikus

### Summary

Számokkal vagy matematikai számításokkal kapcsolatos, azokban résztvevő vagy azokat használó, nem pedig szimbolikus logikát vagy szöveget alkalmazó.

## Key Concepts

- Lebegőpontos számok (Floats)
- Egész számok
- Adattípusok
- Kvantitatív adatok

## Use Cases

- Statisztikai modellezés
- Pénzügyi előrejelzés
- Érzékelői adatok feldolgozása

## Code Example

```python
import numpy as np
arr = np.array([1.5, 2.7, 3.9])
mean_val = np.mean(arr)
print(mean_val)
```
