## Definition

Ez a gyakorlat nagy nyelvi modellek (LLM) csatlakoztatását jelenti olyan szoftverfejlesztői környezetekhez, mint a VS Code vagy az IntelliJ. Lehetővé teszi az intelligens kódkiegészítés, automatizált refaktorálás és egyéb fejlesztői funkciók használatát.

### Summary

Az IDE integráció az AI-alapú eszközök közvetlen beágyazását jelöli az integrált fejlesztői környezetekbe (IDE), hogy segítsék a programozókat a kódolási feladatokban.

## Key Concepts

- Automatikus kiegészítés
- Kódkiegészítés
- LLM (Nagy Nyelvi Modell)
- Fejlesztői élmény

## Use Cases

- Intelligens kódsugallatok
- Automatizált refaktorálás
- Természetes nyelvű lekérdezések végrehajtása
