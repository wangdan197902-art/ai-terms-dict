## Definition

A statisztikai tanuláselméletben egy tanulható függvényosztály az algoritmus számára elérhető hipotézistérre utal. Meghatározza azon mintázatok vagy leképezések körét, amelyeket a modell potenciálisan képes elsajátítani.

### Summary

Egy tanulható függvényosztály egy matematikai függvényekből álló halmaz, amelyet egy adott modellarchitektúra és paramétertér definiál, és amelyet egy tanító algoritmus optimalizálhat.

## Key Concepts

- Hipotézistér
- Modellkapacitás
- VC-dimenzió

## Use Cases

- Megfelelő modellarchitektúrák kiválasztása
- Az általánosítási határok elméleti elemzése
- A modell korlátainak megértése
