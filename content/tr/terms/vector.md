## Definition

Yapay zekada vektör, bilgileri sayısal olarak temsil etmek için kullanılan temel bir veri yapısıdır. Bir varlığın özelliklerini eşleyen, sıralı bir sayılar listesinden (öğelerden) oluşur.

### Summary

Hem büyüklük hem de yönü temsil eden, genellikle yüksek boyutlu uzayda veri noktalarını kodlamak için yapay zekada kullanılan matematiksel bir nesne.

## Key Concepts

- Boyutsallık
- Gömme (Embedding)
- Büyüklük
- Yön

## Use Cases

- Veritabanlarında anlamsal arama
- Doğal dil işleme gömme modelleri
- Görüntü tanıma özellik çıkarma

## Code Example

```python
import numpy as np
vec = np.array([1.0, 2.0, 3.0])
print(vec.shape)
```
