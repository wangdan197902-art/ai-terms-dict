## Definition

Sunucu Tarafından Gönderilen Olaylar (SSE), sunucunun istemciye sürekli veri akışı yapmasına olanak tanıyan, istemcinin tekrar tekrar istek göndermesini gerektirmeyen tek yönlü bir iletişim sağlar. Düz HTTP kullanır, bu da onu kurulumu kolay ve güvenilir kılar.

### Summary

Web sunucularının istemcilere tek bir HTTP bağlantısı üzerinden gerçek zamanlı güncellemeler göndermesine olanak tanıyan standart bir protokol.

## Key Concepts

- Olay akışı
- Tek yönlü iletişim
- Otomatik yeniden bağlanma
- HTTP tabanlı

## Use Cases

- Canlı hisse senedi fiyat akışları
- Gerçek zamanlı bildirimler
- İlerleme takibi güncellemeleri
