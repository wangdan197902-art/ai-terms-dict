## Definition

Azure'a dağıtım, makine öğrenmesi modellerini ölçeklenebilir şekilde sunmak için Azure Machine Learning, Azure Kubernetes Service (AKS) veya Azure Functions gibi bulut yerel araçların kullanılmasını içerir. Hesaplama kaynaklarının yönetimini ve model hizmet sunumunu kapsar.

### Summary

Makine öğrenmesi modellerinin Microsoft Azure bulut altyapısı üzerinde barındırılması ve çalıştırılması süreci.

## Key Concepts

- Bulut Altyapısı
- Model Sunumu
- Ölçeklenebilirlik
- CI/CD Süreçleri

## Use Cases

- Tahminler için gerçek zamanlı API uç noktaları
- Büyük veri setlerinin toplu puanlandırılması
- Kurumsal düzeyde model yönetimi
