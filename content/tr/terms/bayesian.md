## Definition

Yapay zekada Bayesçi yaklaşımlar, daha fazla kanıt mevcut olduğunda hipotezlerin olasılığını güncellemek için olasılık teorisini kullanır. Bu yöntem, modellerin belirsizliği nicelendirmesine ve dinamik olarak tahminlerini iyileştirmesine olanak tanır.

### Summary

Yeni kanıtlarla olasılıkları güncellemek için Bayes Teoremi'ne dayalı istatistiksel yöntemlerle ilgilidir.

## Key Concepts

- Bayes Teoremi
- Önceki olasılık
- Sonraki olasılık
- Belirsizliğin nicelendirilmesi

## Use Cases

- Spam e-posta filtreleme
- Tıbbi tanı sistemleri
- A/B testi analizi

## Code Example

```python
from sklearn.naive_bayes import GaussianNB
model = GaussianNB()
model.fit(X_train, y_train)
```
