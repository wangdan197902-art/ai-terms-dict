## Definition

Bu terim, önde gelen yapay zeka araştırma laboratuvarlarından biri olan OpenAI tarafından yaratılan ticari ve araştırma ürünlerini kapsar. Temel teklifler arasında Jeneratif Önceden Eğitilmiş Dönüştürücü (GPT) serisi, DALL-E görüntü üretimi ve çeşitli API hizmetleri yer alır.

### Summary

OpenAI tarafından geliştirilen GPT modelleri ve DALL-E dahil olmak üzere yapay zeka araçları, API'leri ve araştırma çıktılarına atıf.

## Key Concepts

- Büyük Dil Modelleri
- Jeneratif Yapay Zeka
- API Hizmetleri
- Ticari Dağıtım

## Use Cases

- GitHub Copilot üzerinden otomatik kodlama yardımı
- Pazarlama için yaratıcı içerik üretimi
- Müşteri desteği için konuşma tabanlı ajanlar
