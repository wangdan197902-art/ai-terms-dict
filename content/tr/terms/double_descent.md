## Definition

Çift iniş, yüksek derecede aşırı parametreli modellerin eğitim verilerini tam olarak ezberlemesine (interpolasyon) rağmen düşük test hatasına ulaşabildiğini göstererek geleneksel sapma-varyans ödünleşimini sorgular. Başlangıçta hata artar, ancak model yeterince büyük olduğunda hata tekrar düşer.

### Summary

Model karmaşıklığı interpolasyon eşiğini aşarken test hatasının önce azaldığı, sonra arttığı ve tekrar azaldığı bir olgu.

## Key Concepts

- İnterpolasyon Eşiği
- Aşırı Parametreleme
- Sapma-Varyans Ödünleşimi
- Test Hatası

## Use Cases

- Sinir ağları ölçekleme yasalarının analizi
- Derin öğrenmede genelleme davranışının anlaşılması
- Büyük ölçekli yapay zeka sistemlerinde model seçimi
