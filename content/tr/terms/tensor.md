## Definition

Bilgisayar bilimlerinde ve derin öğrenmede tensör, skalerleri, vektörleri ve matrisleri daha yüksek boyutlara genelleştiren matematiksel bir nesnedir. Sırası (boyut sayısı) ve şekli ile karakterize edilir.

### Summary

Derin öğrenme çerçevelerinin temel veri yapısı olarak hizmet veren çok boyutlu bir dizi.

## Key Concepts

- Sıra (Rank)
- Şekil (Shape)
- Boyutluluk
- Yayınlama (Broadcasting)

## Use Cases

- Görüntü işleme (4 boyutlu tensörler)
- Sinir ağı ağırlıklarının depolanması
- Toplu veri girişi

## Code Example

```python
import torch
t = torch.tensor([[1, 2], [3, 4]])
```
