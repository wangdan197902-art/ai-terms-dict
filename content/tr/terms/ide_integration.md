## Definition

Bu uygulama, büyük dil modelleri gibi yapay zeka modellerini VS Code veya IntelliJ gibi yazılım geliştirme ortamlarına bağlamayı içerir. Akıllı kod tamamlama, otomatik yeniden düzenleme ve doğal dil sorgulama yürütme gibi özellikleri sağlar.

### Summary

IDE Entegrasyonu, geliştiricilere kodlama görevlerinde yardımcı olmak için yapay zeka destekli araçların Doğrudan Geliştirme Ortamlarına (IDE) gömülmesini ifade eder.

## Key Concepts

- Otomatik Tamamlama
- Kod Tamamlama
- LLM (Büyük Dil Modeli)
- Geliştirici Deneyimi

## Use Cases

- Akıllı kod önerileri
- Otomatik yeniden düzenleme
- Doğal dil sorgusu yürütme
