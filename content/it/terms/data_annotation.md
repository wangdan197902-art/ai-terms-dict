## Definition

Questo passaggio critico consiste nell'attachare metadati significativi ai punti dati grezzi in modo che gli algoritmi possano apprendere la relazione tra input e output. Ad esempio, riquadri di delimitazione attorno agli oggetti nelle immagini

### Summary

L'annotazione dei dati è il processo di etichettatura di dati grezzi, come immagini o testo, per renderli adatti all'addestramento supervisionato dell'apprendimento automatico.

## Key Concepts

- Apprendimento Supervisionato
- Etichettatura
- Verità Fondamentale
- Metadati

## Use Cases

- Addestramento di modelli di rilevamento oggetti
- Costruzione di classificatori per l'analisi del sentiment
- Creazione di dataset per il riconoscimento vocale
