## Definition

L'ingegneria dei prompt implica la creazione di input specifici, noti come prompt, per ottenere risposte accurate, pertinenti e di alta qualità dai modelli di IA generativa. Richiede una comprensione di come i modelli interpretano il contesto e le istruzioni.

### Summary

La pratica di progettare e ottimizzare i testi di input per guidare i modelli linguistici di grandi dimensioni verso output desiderati.

## Key Concepts

- Inquadramento contestuale
- Apprendimento few-shot
- Ottimizzazione delle istruzioni
- Ottimizzazione dei token

## Use Cases

- Chatbot per l'assistenza clienti automatizzata
- Assistenti per la generazione di codice
- Strumenti di supporto alla scrittura creativa

## Code Example

```python
prompt = "Translate the following English text to French: 'Hello world'"
response = llm.generate(prompt)
```
