## Definition

Nel machine learning, un'istanza si riferisce a un esempio specifico proveniente dal dataset. È costituita da un insieme di funzionalità di input (attributi) e potenzialmente da un'etichetta di destinazione. Le istanze costituiscono l'unità fondamentale.

### Summary

Un singolo campione di dati o osservazione utilizzato nei compiti di machine learning, tipicamente rappresentato come un vettore di funzionalità.

## Key Concepts

- Campione di dati
- Vettore di funzionalità
- Dati di addestramento
- Osservazione

## Use Cases

- Addestramento per apprendimento supervisionato
- Divisione del dataset
- Esempi per il rilevamento di anomalie
