## Definition

Google Colaboratory, comunemente noto come Colab, è un servizio di notebook Jupyter ospitato che non richiede alcuna configurazione e fornisce accesso gratuito a risorse di calcolo, incluse le unità di elaborazione grafica (GPU).

### Summary

Un ambiente di notebook Jupyter basato su cloud che consente agli utenti di scrivere ed eseguire codice Python tramite un browser, con accesso gratuito alle GPU.

## Key Concepts

- Cloud Computing
- Notebook Jupyter
- Accelerazione GPU
- Programmazione Python

## Use Cases

- Prototipazione di Machine Learning
- Educazione alla Data Science
- Programmazione Collaborativa
