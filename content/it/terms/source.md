## Definition

Nei contesti dell'IA, 'fonte' denota tipicamente la provenienza dei dataset di addestramento, delle librerie open source o dei pesi dei modelli pre-addestrati. Tracciare le fonti è fondamentale per la riproducibilità, la conformità alle licenze...

### Summary

Si riferisce all'origine di dati, codice o modelli utilizzati nello sviluppo e nella distribuzione dell'IA.

## Key Concepts

- Provenienza dei Dati
- Open Source
- Flusso di Input
- Riproducibilità

## Use Cases

- Audit dei dati di addestramento per rilevare pregiudizi
- Gestione del versioning dei modelli
- Citazione delle dipendenze open source
