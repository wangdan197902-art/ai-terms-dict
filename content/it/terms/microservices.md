## Definition

Nel contesto dell'ingegneria dell'IA, i microservizi consentono a diversi componenti della pipeline dell'IA, come la pre-elaborazione dei dati, l'inferenza del modello e l'archiviazione dei risultati, di essere sviluppati, scalati e mantenuti in modo indipendente.

### Summary

I microservizi sono uno stile architettonico che struttura un'applicazione come una raccolta di servizi disaccoppiati e indipendentemente distribuibili, organizzati attorno alle capacità aziendali.

## Key Concepts

- Disaccoppiamento leggero
- Distribuzione indipendente
- Decomposizione dei servizi
- Gateway API

## Use Cases

- Endpoint di inferenza dell'IA scalabili
- Pipeline di machine learning modulari
- Sistemi di raccomandazione resilienti
