## Definition

A differenza della classificazione o regressione standard, il learning to rank si concentra sulla previsione di un ordinamento relativo degli elementi. Utilizza approcci pairwise, listwise o pointwise per minimizzare gli errori di ranking come la NDCG (Normalized Discounted Cumulative Gain).

### Summary

Il Learning to rank è una tecnica di machine learning supervisionato utilizzata per ordinare gli elementi in base alla loro rilevanza rispetto a una determinata query, comunemente usata nei motori di ricerca.

## Key Concepts

- NDCG
- Ranking pairwise
- Recupero delle informazioni

## Use Cases

- Ordinamento dei risultati nei motori di ricerca
- Sistemi di raccomandazione di prodotti
- Algoritmi di ranking pubblicitario
