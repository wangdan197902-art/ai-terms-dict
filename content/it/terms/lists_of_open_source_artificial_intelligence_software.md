## Definition

Questi si riferiscono a repository organizzati, come argomenti GitHub, liste 'Awesome' o wiki mantenuti dalla comunità, che aggregano software open source relativi all'intelligenza artificiale. Servono come risorse essenziali per scoprire e valutare le tecnologie disponibili.

### Summary

Collezioni curate o directory che catalogano strumenti, librerie e framework di IA pubblicamente disponibili.

- Curatela delle risorse
- Manutenzione comunitaria
- Scoperta del software

## Use Cases

- Selezione dello stack tecnologico per startup
- Raccolta di risorse educative
