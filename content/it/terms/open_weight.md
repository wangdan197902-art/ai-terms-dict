## Definition

I modelli open-weight differiscono dall'IA completamente open-source perché vengono rilasciati solo i parametri appresi finali, non necessariamente le infrastrutture o i dati utilizzati per crearli. Ciò consente agli utenti di eseguire inferenze

### Summary

Modelli di IA in cui i parametri addestrati (pesi) sono pubblicati, ma il codice di addestramento e il dataset potrebbero rimanere privati.

## Key Concepts

- Pesi del modello
- Inferenza
- Affinamento (fine-tuning)
- Trasparenza parziale

## Use Cases

- Deployment di LLM su hardware locale
- Personalizzazione di modelli pre-addestrati
- Riduzione dei costi API
