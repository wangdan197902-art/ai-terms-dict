## Definition

Nell'intelligenza artificiale, la robustezza si riferisce alla resilienza di un modello contro attacchi avversari, cambiamenti nella distribuzione dei dati o input rumorosi. Un algoritmo robusto continua a funzionare correttamente anche in presenza di queste perturbazioni.

### Summary

Descrive la capacità di un modello o sistema di IA di mantenere le prestazioni nonostante rumore, errori o input imprevisti.

## Key Concepts

- Resilienza Adversariale
- Generalizzazione
- Tolleranza al Rumore
- Stabilità

## Use Cases

- Guida autonoma in condizioni meteorologiche avverse
- Rilevamento delle frodi con dati rumorosi
- Diagnosi medica con cartelle cliniche incomplete
