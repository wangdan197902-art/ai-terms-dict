## Definition

Il Deploy Continuo è un'estensione del Continuous Delivery che automatizza l'intero processo di rilascio. Una volta che le modifiche al codice superano tutti i criteri di qualità, inclusi i test unitari, i test di integrazione e la sicurezza...

### Summary

Una pratica di ingegneria del software in cui ogni modifica al codice che supera i test automatizzati viene rilasciata automaticamente in produzione.

## Key Concepts

- Test Automatizzati
- Pipeline CI/CD
- Deploy a Zero Downtime
- Feature Flags

## Use Cases

- Piattaforme di trading ad alta frequenza
- Applicazioni web su larga scala
- Gestione dell'architettura a microservizi
