## Definition

L'Apprendimento della Parità è un problema di riferimento nella teoria dell'apprendimento automatico in cui l'obiettivo è prevedere la parità (somma XOR) di un insieme di variabili di input binarie. È notoriamente difficile per i feedforward standard, richiedendo capacità di modellazione delle dipendenze a lungo raggio.

### Summary

Un problema teorico di apprendimento automatico focalizzato sulla previsione della somma XOR di input binari, utilizzato per testare l'espressività dei modelli.

## Key Concepts

- Problema XOR
- Espressività del Modello
- Classificazione Binaria
- Dipendenze a Lungo Raggio

## Use Cases

- Valutazione della capacità delle reti neurali
- Test della robustezza degli algoritmi di ottimizzazione
- Ricerca sulle capacità delle reti neurali ricorrenti
