## Definition

Nell'apprendimento automatico, una feature è un attributo o variabile distinta che descrive un'istanza all'interno di un dataset. Le feature possono essere numeriche, categoriali o testuali e fungono da input fondamentali per i modelli.

### Summary

Una proprietà o caratteristica misurabile individuale di un fenomeno osservato, che funge da dati di input per i modelli di apprendimento automatico.

## Key Concepts

- Variabile di Input
- Attributo
- Colonna del Dataset
- Input ML

## Use Cases

- Input per modelli predittivi
- Analisi e visualizzazione dei dati
- Processi di selezione delle feature
