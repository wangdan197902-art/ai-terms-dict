## Definition

Il machine learning quantistico (QML) è un campo interdisciplinare emergente che integra le capacità del calcolo quantistico con le tecniche di machine learning. Mira a sfruttare fenomeni quantistici come l'entanglement

### Summary

Il machine learning quantistico combina il calcolo quantistico con gli algoritmi di machine learning per risolvere problemi più velocemente o in modo più efficiente.
