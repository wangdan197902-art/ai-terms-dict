## Definition

Il cloud computing fornisce infrastrutture scalabili per i carichi di lavoro di AI, consentendo agli sviluppatori di accedere a potenti GPU e spazio di archiviazione senza mantenere data center fisici. Supporta vari modelli di servizio come IaaS, PaaS e SaaS.

### Summary

Il cloud si riferisce a server remoti ospitati su Internet utilizzati per archiviare, gestire ed elaborare dati e modelli di intelligenza artificiale invece di hardware locale.

## Key Concepts

- Scalabilità
- Infrastruttura come Servizio
- Archiviazione Remota
- Elasticità

## Use Cases

- Addestramento di grandi modelli linguistici
- Hostaggio di API AI per applicazioni web
- Archiviazione di dataset massicci per l'analisi
