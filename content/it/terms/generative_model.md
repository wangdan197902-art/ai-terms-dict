## Definition

I modelli generativi sono algoritmi progettati per comprendere i modelli e le strutture all'interno di un insieme di dati dato, in modo da poter creare nuove istanze di dati che assomiglino all'originale. A differenza dei modelli discriminativi

### Summary

Un tipo di modello di apprendimento automatico che apprende la distribuzione sottostante dei dati per generare nuovi campioni sintetici simili ai dati di addestramento.

## Key Concepts

- Distribuzione di Probabilità
- Dati Sintetici
- Spazio Latente
- Obiettivo di Addestramento

## Use Cases

- Generazione e modifica di immagini
- Elaborazione del linguaggio naturale
- Aumento dei dati per piccoli set di dati
