## Definition

Originariamente noto come GPT Index, LlamaIndex è un potente framework di dati che consente agli LLM di ingerire e interagire con dati strutturati e non strutturati. Fornisce strumenti per indicizzare, interrogare e gestire dati personalizzati.

### Summary

LlamaIndex è un framework progettato per connettere fonti di dati personalizzate ai modelli linguistici di grandi dimensioni, facilitando la creazione di applicazioni di Generazione Aumentata dal Recupero (RAG).

## Key Concepts

- RAG (Retrieval-Augmented Generation)
- Indicizzazione dei dati
- Database vettoriali
- Iniezione del contesto

## Use Cases

- Creazione di chatbot con dati privati
- Sistemi di domanda e risposta su documenti
- Basi di conoscenza aziendali
