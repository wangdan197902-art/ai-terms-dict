## Definition

ROCm (Radeon Open Compute) è uno stack di driver e software sviluppato da AMD per abilitare il calcolo ad alte prestazioni sulle GPU AMD. Fornisce librerie, compilatori e strumenti necessari per lo sviluppo di applicazioni parallele e l'accelerazione hardware.

### Summary

Piattaforma software open source di AMD per il calcolo GPU, che fornisce alternative all'ecosistema CUDA di NVIDIA.

## Key Concepts

- Calcolo GPU
- Software Open Source
- Elaborazione Parallela
- Accelerazione Hardware

## Use Cases

- Addestramento di modelli AI su GPU AMD
- Simulazioni scientifiche
- Cluster di calcolo ad alte prestazioni
