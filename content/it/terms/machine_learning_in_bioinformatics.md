## Definition

Questo campo interdisciplinare utilizza l'apprendimento automatico per elaborare enormi quantità di dati biologici, consentendo ai ricercatori di prevedere le funzioni geniche, classificare le malattie e comprendere le interazioni molecolari complesse.

### Summary

L'applicazione di modelli computazionali per analizzare dati biologici, come sequenze genomiche e strutture proteiche, al fine di scoprire approfondimenti biologici.

## Key Concepts

- Analisi genomica
- Piegamento delle proteine
- Modellazione predittiva
- Dati ad alto rendimento

## Use Cases

- Previsione delle strutture proteiche dalle sequenze di amminoacidi
- Classificazione dei tipi di cancro basata sull'espressione genica
- Identificazione di potenziali bersagli farmacologici nei database molecolari
