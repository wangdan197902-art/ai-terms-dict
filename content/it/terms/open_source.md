## Definition

Open-source si riferisce a un modello di sviluppo in cui il codice sottostante di un progetto software è pubblicamente accessibile. Nell'IA, ciò consente ai ricercatori e agli sviluppatori di ispezionare, modificare e ridistribuire gli algoritmi

### Summary

Software il cui codice sorgente è reso liberamente disponibile per l'uso, la modifica e la distribuzione da chiunque.

## Key Concepts

- Trasparenza
- Collaborazione
- Accessibilità
- Guidato dalla comunità

## Use Cases

- Sviluppo del framework PyTorch
- Librerie di modelli di Hugging Face
- Riproducibilità della ricerca accademica
