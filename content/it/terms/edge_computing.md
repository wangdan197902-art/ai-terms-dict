## Definition

L'edge computing affronta i limiti di latenza e banda delle architetture basate sul cloud elaborando i dati vicino al punto in cui vengono generati, come dispositivi IoT, sensori o gateway locali. Nel contesto dell'intelligenza artificiale

### Summary

L'edge computing è un paradigma di calcolo distribuito che avvicina l'elaborazione e l'archiviazione dei dati alle fonti di generazione dei dati, invece di fare affidamento esclusivamente su server cloud centralizzati.

## Key Concepts

- Bassa Latenza
- Elaborazione Distribuita
- Integrazione IoT
- Compressione del Modello

## Use Cases

- Navigazione di veicoli autonomi
- Sorveglianza con telecamere intelligenti
- Manutenzione predittiva industriale
