## Definition

La Programmazione Induttiva, spesso riferita come Sintesi di Programmi, implica la creazione di codice software basandosi su specifiche fornite sotto forma di coppie input-output piuttosto che su istruzioni esplicite. Il sistema inferisce la logica o il codice necessario per mappare correttamente gli input agli output desiderati.

### Summary

Un campo della sintesi di programmi che genera automaticamente programmi informatici da esempi di input-output.

## Key Concepts

- Sintesi di Programmi
- Esempi Input-Output
- Codifica Automatizzata
- Inferenza Logica

## Use Cases

- Strumenti di completamento del codice
- Generazione di script dai dati
- Ausili per la programmazione educativa
