## Definition

Le API REST abilitano la comunicazione tra client e server utilizzando operazioni senza stato sui protocolli HTTP come GET, POST, PUT e DELETE. Strutturano le risorse come URI e utilizzano formati standard (come JSON) per lo scambio dei dati.

### Summary

Un'API REST (Representational State Transfer) è uno stile architettonico per la progettazione di applicazioni in rete che si basa sui metodi HTTP standard.

## Key Concepts

- Assenza di stato (Statelessness)
- Metodi HTTP
- URI basati sulle risorse
- Serializzazione JSON

## Use Cases

- Comunicazione tra microservizi
- Integrazione con backend per app mobili
- Accesso a dati di terze parti
