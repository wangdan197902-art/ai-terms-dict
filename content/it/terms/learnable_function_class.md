## Definition

Nella teoria statistica dell'apprendimento, una classe di funzioni apprendibili rappresenta lo spazio delle ipotesi disponibile per un algoritmo. Definisce l'intervallo di pattern o mappature che il modello può potenzialmente catturare, influenzando direttamente la capacità di generalizzazione.

### Summary

Una classe di funzioni apprendibili è un insieme di funzioni matematiche definite da una specifica architettura di modello e spazio dei parametri che un algoritmo di apprendimento può ottimizzare.

## Key Concepts

- Spazio delle ipotesi
- Capacità del modello
- Dimensione VC

## Use Cases

- Selezione di architetture di modello appropriate
- Analisi teorica dei limiti di generalizzazione
- Comprensione dei limiti del modello
