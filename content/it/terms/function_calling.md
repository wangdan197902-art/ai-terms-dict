## Definition

La chiamata di funzione permette ai modelli linguistici di grandi dimensioni di interagire con strumenti esterni e API generando output strutturati, come oggetti JSON, che specificano quale funzione eseguire e quali argomenti utilizzare.

### Summary

Un meccanismo che consente ai modelli linguistici di grandi dimensioni (LLM) di generare dati strutturati per attivare specifiche funzioni software.

## Key Concepts

- Output Strutturato
- Integrazione API
- Utilizzo di Strumenti
- Schema JSON

## Use Cases

- Prenotazione voli tramite API di viaggio
- Esecuzione di calcoli matematici complessi
- Interrogazione di record nel database
