## Definition

Un agente IA è un'entità software che opera in modo autonomo all'interno di un ambiente definito per raggiungere obiettivi predefiniti. Utilizza meccanismi di percezione per raccogliere dati ed elaborare queste informazioni.

### Summary

Un sistema autonomo progettato per percepire il proprio ambiente e intraprendere azioni per raggiungere obiettivi specifici.

## Key Concepts

- autonomia
- ciclo percezione-azione
- comportamento orientato agli obiettivi
- interazione con l'ambiente

## Use Cases

- Bot di assistenza clienti automatizzati
- Navigazione di veicoli autonomi
- Assistenti per la generazione di codice
