## Definition

Un agente intelligente è un sistema capace di percepire le proprie circostanze attraverso sensori o input di dati, elaborare queste informazioni utilizzando algoritmi di ragionamento e agire sull'ambiente tramite attuatori

### Summary

Un'entità software autonoma che percepisce il proprio ambiente, ragiona sulle azioni ed esegue compiti per raggiungere obiettivi specifici.

## Key Concepts

- Autonomia
- Ciclo percezione-azione
- Orientamento agli obiettivi
- Reattività

## Use Cases

- Bot di trading autonomi che eseguono strategie di mercato
- Automazione dei processi robotici che gestisce flussi di lavoro multistep
- Assistenti personali che gestiscono compiti di calendario ed email
