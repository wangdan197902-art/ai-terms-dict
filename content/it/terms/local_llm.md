## Definition

Eseguire un LLM Locale implica il deployment di modelli a peso aperto direttamente su hardware consumer come PC, Mac o server locali. Questo approccio elimina la dipendenza dai fornitori di API di terze parti, garantendo maggiore controllo sui dati.

### Summary

Un LLM Locale si riferisce all'esecuzione di modelli linguistici di grandi dimensioni su hardware personale anziché su servizi cloud, privilegiando la privacy dei dati e l'accessibilità offline.

## Key Concepts

- Inferenza su dispositivo
- Privacy dei dati
- Quantizzazione
- Requisiti hardware

## Use Cases

- Assistenti per la presa di appunti privati
- Strumenti di ricerca offline
- Elaborazione sicura di dati aziendali
