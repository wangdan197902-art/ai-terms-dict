## Definition

Gli approcci bayesiani nell'IA utilizzano la teoria della probabilità per aggiornare la verosimiglianza di ipotesi man mano che diventano disponibili ulteriori evidenze. Questo metodo consente ai modelli di quantificare l'incertezza e affinare le previsioni dinamicamente.

### Summary

Riguarda metodi statistici basati sul Teorema di Bayes per aggiornare le probabilità alla luce di nuove evidenze.

## Key Concepts

- Teorema di Bayes
- Probabilità a priori
- Probabilità a posteriori
- Quantificazione dell'incertezza

## Use Cases

- Filtraggio della posta indesiderata (spam)
- Sistemi diagnostici medici
- Analisi dei test A/B

## Code Example

```python
from sklearn.naive_bayes import GaussianNB
model = GaussianNB()
model.fit(X_train, y_train)
```
