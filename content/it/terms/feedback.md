## Definition

I meccanismi di feedback consentono ai sistemi AI di apprendere dalle loro interazioni con utenti o ambienti, affinando le previsioni o le azioni future. Questo include segnali di apprendimento per rinforzo, umano nel loop (human-in-the-loop), ecc.

### Summary

Il feedback implica l'utilizzo dei risultati di output per regolare e migliorare iterativamente le prestazioni di un modello o sistema AI.

## Key Concepts

- Apprendimento per rinforzo
- Umano nel loop
- Correzione degli errori
- Miglioramento iterativo

## Use Cases

- Regolazione dei sistemi di raccomandazione
- Raffinamento delle conversazioni dei chatbot
- Regolazione del controllo robotico
