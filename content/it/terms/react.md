## Definition

Il framework ReAct consente ai LLM di generare tracce di ragionamento e azioni specifiche per il compito in modo alternato. Simulando processi di pensiero simili a quelli umani, permette ai modelli di interagire con l'ambiente esterno e integrare le osservazioni per prendere decisioni migliori.

### Summary

ReAct è un paradigma di prompting che combina ragionamento e azione per migliorare le prestazioni dei modelli linguistici di grandi dimensioni (LLM) su compiti complessi.

## Key Concepts

- Ragionamento alternato
- Generazione di azioni
- Integrazione delle osservazioni
- Catena di pensiero (Chain-of-Thought)

## Use Cases

- Sistemi di domanda e risposta complessi
- Agenti autonomi
- Flussi di lavoro di verifica dei fatti
