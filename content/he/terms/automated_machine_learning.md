## Definition

AutoML מפשט את פיתוח מודלי ML על ידי אוטומציה של משימות כמו ניקוי נתונים, הנדסת תכונות, בחירת מודל וכיוונון פרמטרים. הוא מאפשר למפתחים ולאנליסטים ליצור מודלים מדויקים במהירות, תוך הפחתת חסמי הכניסה לתחום למידת המכונה.

### Summary

מתודולוגיה המאוטמטית את התהליך המלא של יישום למידת מכונה לבעיות בעולם האמיתי, תוך הפחתת הצורך במאמץ ידני.

## Key Concepts

- כיוונון פרמטרים היפר
- הנדסת תכונות
- בחירת מודל
- דמוקרטיזציה

## Use Cases

- פיתוח אבטיפוס מהיר עבור אנליסטים עסקיים
- אופטימיזציה של צינורות ייצור בקנה מידה גדול
- השוואה אוטומטית בין מספר אלגוריתמים

## Code Example

```python
from auto_ml import Predictor
predictor = Predictor(type_of_estimator='classifier')
predictor.fit(dataframe, target='label')
```
