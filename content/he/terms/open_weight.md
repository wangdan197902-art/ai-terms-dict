## Definition

מודלים עם משקולות פתוחות שונים מבינה מלאכותית בקוד פתוח מלא, שכן רק הפרמטרים הלמדניים הסופיים משוחררים, ולא בהכרח התשתית או הנתונים ששימשו ליצירתם. זה מאפשר למשתמשים להריץ הסקה

### Summary

מודלי בינה מלאכותית שבהם הפרמטרים המאומתים (המשקולות) פורסמו, אך קוד האימון וערכת הנתונים עשויים להישאר פרטיים.

## Key Concepts

- משקולות מודל
- הסקה (Inference)
- התאמה עדינה (Fine-tuning)
- שקיפות חלקית

## Use Cases

- הטמעת מודלי שפה גדולים (LLMs) בחומרה מקומית
- התאמה אישית של מודלים מאומנים מראש
- הפחתת עלויות ממשקי API
