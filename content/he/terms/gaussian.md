## Definition

מונח 'גאוסיאני' מתייחס להתפלגות נורמלית, התפלגות הסתברותית רציפה האופיינית לממוצע ולשונות שלה. בבינה מלאכותית, היא משמשת באופן נרחב במודלים הסתברותיים, הסקה בייסיאנית ו

### Summary

הקשור להתפלגות נורמלית, עקומה בצורת פעמון מהותית לסטטיסטיקה ולמודלי רעש בבינה מלאכותית.

## Key Concepts

- התפלגות נורמלית
- ממוצע
- שונות
- צפיפות הסתברות

## Use Cases

- מודלי רעש
- רשתות בייסיאניות
- אתחול משקולות

## Code Example

```python
import numpy as np
# Generate 1000 samples from a standard Gaussian distribution
samples = np.random.normal(loc=0.0, scale=1.0, size=1000)
```
