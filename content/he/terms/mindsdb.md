## Definition

MindsDB משמשת כגשר בין מסדי נתונים יחסיים מסורתיים לבין זרימות עבודה מודרניות של למידת מכונה. היא מאפשרת למשתמשים ליצור מודלים חיזויים באמצעות שאילתות SQL סטנדרטיות, ללא צורך בכלים חיצוניים מורכבים.

### Summary

MindsDB היא פלטפורמת קוד פתוח המאפשרת למפתחים לבנות, לאמן ולהטמיע מודלי למידת מכונה ישירות בתוך מסדי הנתונים הקיימים שלהם באמצעות SQL.

## Key Concepts

- למידת מכונה מבוססת SQL
- אינטגרציה למסדי נתונים
- AutoML
- מודלים חיזויים

## Use Cases

- זיהוי הונאות בזמן אמת במסדי נתונים
- חיזוי נטישת לקוחות
- דיווח אוטומטי עם תחזיות
