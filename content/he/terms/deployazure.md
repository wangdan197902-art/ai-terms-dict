## Definition

הטמעה ל-Azure כוללת שימוש בכלים ילידי ענן כמו Azure Machine Learning, Azure Kubernetes Service (AKS) או Azure Functions כדי להפעיל מודלי ML בקנה מידה גדול. התהליך כולל ניהול משאבי חישוב, אבטחה וסקאלביליות.

### Summary

תהליך האחסון והרצת מודלי למידת מכונה על תשתיות הענן של מיקרוסופט Azure.

## Key Concepts

- תשתית ענן
- הפעלת מודל (Model Serving)
- סקאלביליות
- צינורות CI/CD

## Use Cases

- נקודות קצה API בזמן אמת לאינפראנס
- ציון מקבצי נתונים גדולים (Batch scoring)
- ניהול מודלים ברמה ארגונית
