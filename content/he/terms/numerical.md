## Definition

בהקשר של בינה מלאכותית ומדעי הנתונים, מספרי מתייחס לסוגי נתונים או שיטות הכוללות ערכים כמותיים, כמו שלמים, עשרוניים ושברים. בניגוד לנתונים קטגוריאליים או טקסטואליים, נתונים מספריים ניתנים לפעולות מתמטיות.

### Summary

הקשור למספרים, מערב אותם או משתמש בהם, בניגוד ללוגיקה סמלית או טקסט.

## Key Concepts

- מספרים עשרוניים (Floats)
- מספרים שלמים (Integers)
- סוגי נתונים
- נתונים כמותיים

## Use Cases

- מודלים סטטיסטיים
- תחזיות פיננסיות
- עיבוד נתוני חיישנים

## Code Example

```python
import numpy as np
arr = np.array([1.5, 2.7, 3.9])
mean_val = np.mean(arr)
print(mean_val)
```
