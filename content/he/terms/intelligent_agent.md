## Definition

סוכן חכם הוא מערכת המסוגלת לתפוס את סביבתה באמצעות חיישנים או קלט נתונים, לעבד מידע זה באמצעות אלגוריתמי היגיון, ולפעול בסביבה דרך מנגנוני פעולה (Actuators)...

### Summary

ישות תוכנה אוטונומית התופסת את סביבתה, חושבת לגבי פעולות ומבצעת משימות כדי להשיג יעדים ספציפיים.

## Key Concepts

- אוטונומיה
- לולאת תפיסה-פעולה (Perception-Action Loop)
- כוונתיות למטרות (Goal Orientation)
- תגובתיות (Reactivity)

## Use Cases

- בוטים מסחר אוטונומיים המבצעים אסטרטגיות שוק
- אוטומציה תהליכית רובוטית (RPA) המטפלת בעבודות רב-שלבים
- עוזרים אישיים המנהלים יומן ודואר אלקטרוני
