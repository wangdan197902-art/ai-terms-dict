## Definition

הנדסת תכונות היא האמנות של ניצול מומחיות בתחום כדי להמיר נתונים גולמיים לתכונות המייצגות טוב יותר את התבניות הבסיסיות בפני אלגוריתמי למידת מכונה. תהליך זה כולל יצירה, סינון ועיצוב מחדש של נתונים.

### Summary

הפרקטיקה של שימוש בידע בתחום כדי ליצור תכונות חדשות או לשנות קיימות כדי לשפר את הביצועים של מודלי למידת מכונה.

## Key Concepts

- ידע בתחום
- המרת נתונים
- ביצועי מודל
- יצירת משתנים

## Use Cases

- שיפור דיוק מודלי רגרסיה
- חיזוק גבולות סיווג
- אופטימיזציה של תחזיות סדרות זמן

## Code Example

```python
df['new_feature'] = df['feature_a'] * df['feature_b']
```
