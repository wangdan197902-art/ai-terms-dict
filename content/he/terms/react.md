## Definition

מסגרת ReAct מאפשרת למודלי שפה גדולים (LLMs) ליצור גם עקבות חשיבה וגם פעולות ספציפיות למשימה בצורה מתחלפת. על ידי סימולציה של תהליכי חשיבה דמויי אדם, היא מאפשרת למודלים להגיב לסביבה ולבצע פעולות.

### Summary

ReAct הוא פרדיגמת הנחיה המשלבת חשיבה ופעולה כדי לשפר את הביצועים של מודלי שפה גדולים במשימות מורכבות.

## Key Concepts

- חשיבה מתחלפת
- יצירת פעולות
- שילוב תצפיות
- שרשרת מחשבה (Chain-of-Thought)

## Use Cases

- מערכות שאלות ותשובות מורכבות
- סוכנים אוטונומיים
- תהליכי אימות עובדות
