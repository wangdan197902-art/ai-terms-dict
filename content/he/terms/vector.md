## Definition

בבינה מלאכותית, וקטור הוא מבנה נתונים בסיסי המשמש לייצוג מידע בצורה מספרית. הוא מורכבריר רשימה מסודרת של מספרים (אלמנטים) המתארים תכונות של ישות בתוך מרחב וקטורי.

### Summary

אובייקט מתמטי המייצג כמות בעלת גודל וכיוון, המשמש לעיתים קרובות בבינה מלאכותית לקידוד נקודות נתונים במרחב רב-ממדי.

## Key Concepts

- ממדיות
- הטמעה (Embedding)
- גודל
- כיוון

## Use Cases

- חיפוש סמנטי במסדי נתונים
- הטמעות בעיבוד שפה טבעית
- חילוץ תכונות בזיהוי תמונות

## Code Example

```python
import numpy as np
vec = np.array([1.0, 2.0, 3.0])
print(vec.shape)
```
