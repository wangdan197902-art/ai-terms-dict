## Definition

קריאה לפונקציה מאפשרת למודלי שפה גדולים ליצור אינטראקציה עם כלים חיצוניים ו-APIs על ידי יצירת פלט מבני, כגון אובייקטי JSON, המפרטים איזו פונקציה לבצע ואילו פרמטרים להעביר.

### Summary

מנגנון המאפשר למודלי שפה גדולים (LLMs) להפיק נתונים מבניים המפעילים פונקציות תוכנה ספציפיות.

## Key Concepts

- פלט מבני
- אינטגרציה של API
- שימוש בכלים
- סכמת JSON

## Use Cases

- הזמנת טיסות דרך ממשקי API של חברות תעופה
- ביצוע חישובים מתמטיים מורכבים
- שאילתת רשומות במאגר נתונים
