## Definition

פרוטוקול הקשר מודלים (MCP) הוא תקן פתוח המאפשר ליישומי בינה מלאכותית להתחבר למגוון מקורות נתונים, כגון מסדי נתונים, ממשקי תכנות יישומים (APIs) ומערכות קבצים, בצורה אחידה. הוא מפשט את האינטגרציה ומאפשר גישה סטנדרטית למידע חיצוני.

### Summary

מסגרת תקנית המיועדת לאפשר תקשורת מאובטחת ויעילה בין מודלי בינה מלאכותית לבין מקורות נתונים חיצוניים או כלים.

## Key Concepts

- אינטראופרבליות (תאימות הדדית)
- סטנדרטיזציה
- אינטגרציית נתונים
- הפעלת כלים

## Use Cases

- חיבור מודלי שפה גדולים (LLMs) לנתוני ארגון
- בניית עוזרים דיגיטליים ניידים
- אינטגרציה מאובטחת של ממשקי API
