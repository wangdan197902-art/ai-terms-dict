## Definition

בבינה מלאכותית, סוכן עצמאי הוא ישות הפועלת באופן עצמאי בתוך סביבה מסוימת. היא משתמשת בחיישנים כדי לתפוס מצבים ומפעילים כדי לבצע פעולות, בהנחיית מודל פנימי או אלגוריתם קבלת החלטות.

### Summary

סוכן עצמאי הוא מערכת המסוגלת לתפוס את סביבתה ולפעול באופן עצמאי כדי להשיג יעדים ספציפיים ללא התערבות ישירה של אדם.

## Key Concepts

- תפיסה
- פעולה
- התנהגות מכוונת-יעד
- אינטראקציה עם הסביבה

## Use Cases

- רכבים אוטונומיים
- אוטומציה תהליכית רובוטית (RPA)
- בינה מלאכותית למשחקים
