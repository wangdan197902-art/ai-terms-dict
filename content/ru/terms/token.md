## Definition

Токены являются фундаментальными строительными блоками входных данных в NLP, обычно представляющими слова, подслова или символы. Большие языковые модели (LLM) обрабатывают текст, преобразуя его в токены, которые затем векторизуются для дальнейшей обработки.

### Summary

Токен — это дискретная единица текста или данных, служащая базовым элементом ввода для моделей обработки естественного языка (NLP).

## Key Concepts

- Дискретная единица
- Словарь
- Встраивание (Embedding)
- Контекстное окно

## Use Cases

- Генерация текста в больших языковых моделях
- Предварительная обработка для машинного перевода
- Форматирование ввода для анализа тональности
