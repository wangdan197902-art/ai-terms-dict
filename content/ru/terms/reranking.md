## Definition

Переранжирование — это стратегия, используемая в информационном поиске и рекомендательных системах для повышения точности. Сначала быстрая, но менее точная модель извлекает большой набор кандидатов. Затем более медленная, но сложная модель переранжирует этот отфильтрованный список.

### Summary

Двухэтапный процесс поиска, при котором первоначальное грубое ранжирование уточняется более вычислительно затратной моделью для повышения релевантности результатов.

## Key Concepts

- Двухуровневый поиск
- Генерация кандидатов
- Кросс-внимание
- Оптимизация точности

## Use Cases

- Поисковые системы
- Рекомендательные системы
- Генерация с дополнением извлечения (RAG)
