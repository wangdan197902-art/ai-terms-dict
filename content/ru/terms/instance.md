## Definition

В машинном обучении экземпляр относится к одному конкретному примеру из набора данных. Он состоит из набора входных признаков (атрибутов) и, возможно, целевой метки. Экземпляры являются фундаментальной единицей

### Summary

Один отдельный образец данных или наблюдение, используемое в задачах машинного обучения, обычно представленное в виде вектора признаков.

## Key Concepts

- Образец данных
- Вектор признаков
- Обучающие данные
- Наблюдение

## Use Cases

- Обучение с учителем
- Разделение наборов данных
- Примеры для обнаружения аномалий
