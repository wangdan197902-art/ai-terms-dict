## Definition

Ограничение частоты запросов защищает ИИ-сервисы и API от злоупотреблений, перегрузки и чрезмерного потребления ресурсов. Оно обеспечивает справедливое использование среди пользователей и поддерживает стабильность системы, ограничивая пропускную способность. Распространенные стратегии включают...

### Summary

Инженерный механизм контроля, ограничивающий количество запросов, которые клиент может отправить сервису в течение определенного временного окна.

## Key Concepts

- Защита API
- Контроль пропускной способности
- Политика справедливого использования
- Стабильность системы

## Use Cases

- Управление шлюзами API больших языковых моделей (LLM)
- Предотвращение DDoS-атак
- Управление затратами на облачные вычисления
