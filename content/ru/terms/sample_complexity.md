## Definition

В теории вычислительного обучения сложность выборки количественно оценивает объем данных, необходимых для эффективного обучения модели. Она балансирует компромисс между емкостью модели и доступностью данных, обеспечивая обобщающую способность.

### Summary

Сложность выборки — это количество примеров обучения, необходимых алгоритму машинного обучения для достижения определенного уровня производительности с высокой вероятностью.

## Key Concepts

- Ошибка обобщения
- Размер обучающих данных
- Сходимость
- Предотвращение переобучения

## Use Cases

- Определение требований к набору данных для новых моделей
- Сравнение эффективности различных алгоритмов обучения
- Планирование ресурсов для сбора данных
