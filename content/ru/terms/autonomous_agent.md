## Definition

В искусственном интеллекте автономный агент — это сущность, которая функционирует независимо в окружающей среде. Он использует датчики для восприятия состояния среды и исполнительные механизмы для выполнения действий, руководствуясь внутренними алгоритмами принятия решений.

### Summary

Автономный агент — это система, способная воспринимать окружающую среду и действовать независимо для достижения конкретных целей без прямого вмешательства человека.

## Key Concepts

- Восприятие
- Действие
- Целенаправленное поведение
- Взаимодействие со средой

## Use Cases

- Беспилотные транспортные средства
- Роботизированная автоматизация процессов
- Игровой ИИ
