## Definition

Эмпирическое динамическое моделирование (EDM) — это метод анализа нелинейных динамических систем с использованием наблюдательных данных без предположения о конкретной параметрической форме. Он опирается на метод вложения Такенса

### Summary

Непараметрический статистический подход, используемый для реконструкции динамики сложных систем на основе наблюдательных временных рядов.

## Key Concepts

- Реконструкция фазового пространства
- Теорема Такенса
- Нелинейность
- Причинно-следственный вывод

## Use Cases

- Прогнозирование численности популяций в экологии
- Анализ сигналов в нейронауке
- Предсказание волатильности финансовых рынков
