## Definition

Умные объекты являются компонентами Интернета вещей (IoT), обладающими уникальными идентификаторами и способностью передавать данные по сети без прямого взаимодействия человек-человек или человек-компьютер.

### Summary

Физический объект, оснащенный датчиками, исполнительными механизмами и средствами связи для сбора данных и интеллектуального взаимодействия с окружающей средой.

## Key Concepts

- Интернет вещей (IoT)
- Встроенные датчики
- Связь / Подключение
- Обработка данных

## Use Cases

- Системы автоматизации умного дома
- Прогнозирующее техническое обслуживание в промышленности
- Носимые устройства для мониторинга здоровья
