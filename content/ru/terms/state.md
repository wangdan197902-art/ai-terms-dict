## Definition

Состояние представляет собой всю релевантную информацию, необходимую для определения будущего поведения в таких системах, как марковские процессы принятия решений (MDP). В обучении с подкреплением состояние описывает среду.

### Summary

Полная конфигурация системы или агента в определенный момент времени.

## Key Concepts

- Конфигурация
- Шаг по времени
- Наблюдение

## Use Cases

- Агенты в обучении с подкреплением
- Скрытые марковские модели
- Мониторинг систем
