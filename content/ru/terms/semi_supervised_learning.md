## Definition

Полуобучаемое обучение — это гибридная парадигма обучения, которая использует небольшое количество размеченных данных вместе с большим объемом неразмеченных данных. Основная гипотеза заключается в том, что структура неразмеченных данных помогает улучшить качество модели.

### Summary

Подход к машинному обучению, использующий как размеченные, так и неразмеченные данные для повышения точности и обобщающей способности модели.

## Key Concepts

- Размеченные данные
- Неразмеченные данные
- Самообучение (self-training)
- Гипотеза многообразия

## Use Cases

- Классификация изображений при ограниченной аннотации
- Анализ тональности текста при разреженных метках
- Прогнозирование медицинских диагнозов при дефиците экспертных данных
