## Definition

Графический процессор (GPU) — это высокопроизводительный процессор, изначально разработанный для задач рендеринга графики. В отличие от центральных процессоров (CPU), которые имеют небольшое количество ядер, оптимизированных для последовательной обработки, графические процессоры содержат тысячи ядер, что позволяет им эффективно выполнять массово параллельные вычисления, необходимые для обучения нейронных сетей.

### Summary

Специализированная электронная схема, изначально разработанная для быстрой обработки и рендеринга изображений, в настоящее время широко используемая для параллельных вычислений в области искусственного интеллекта.

## Key Concepts

- Параллельная обработка
- Ядра CUDA
- Матричное умножение
- Аппаратное ускорение

## Use Cases

- Обучение крупных нейронных сетей
- Вывод моделей компьютерного зрения в реальном времени
- Рендеринг 3D-графики в симуляциях
