## Definition

В обработке естественного языка контекст имеет решающее значение для разрешения неоднозначности, например, для понимания местоимений или идиом на основе предыдущих предложений. Современные архитектуры, такие как трансформеры, используют механизмы внимания.

### Summary

Контекст относится к окружающей информации или среде, которая помогает модели ИИ точно интерпретировать входные данные и генерировать релевантные ответы.

## Key Concepts

- Механизм внимания
- Последовательностное моделирование
- Разрешение неоднозначности
- Размер окна

## Use Cases

- Разговорные чат-боты
- Суммаризация документов
- Ассистенты для автодополнения кода
