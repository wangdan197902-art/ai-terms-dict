## Definition

Бессерверная архитектура позволяет разработчикам создавать и запускать приложения без управления серверной инфраструктурой. Поставщик облачных услуг автоматически масштабирует ресурсы вверх или вниз в зависимости от спроса, взимая плату только за фактически использованные ресурсы, что снижает операционные издержки.

### Summary

Модель облачных вычислений, при которой поставщик динамически управляет распределением ресурсов вместо пользователя.

## Key Concepts

- Функция как услуга (FaaS)
- Автоматическое масштабирование
- Оплата по факту использования
- Событийно-ориентированная архитектура

## Use Cases

- Конвейеры обработки изображений
- Бэкенд для мобильных приложений
- Запланированные задачи обработки данных
