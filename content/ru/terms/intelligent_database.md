## Definition

Интеллектуальная база данных использует машинное обучение и ИИ для расширения стандартных функций баз данных за пределы простого хранения и извлечения данных. Она может автоматически оптимизировать производительность запросов, предсказывать использование ресурсов и выявлять аномалии.

### Summary

Система баз данных, включающая возможности ИИ для автоматизации управления данными, оптимизации запросов и генерации аналитических выводов.

## Key Concepts

- Оптимизация запросов
- Обнаружение аномалий
- Самонастройка
- Запросы на естественном языке

## Use Cases

- Автоматическое масштабирование ресурсов облачной базы данных в зависимости от нагрузки
- Обнаружение мошеннических транзакций в реальном времени в банковских данных
- Генерация SQL-запросов на основе вопросов, заданных на естественном языке
