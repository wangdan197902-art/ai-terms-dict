## Definition

Фреймворк ReAct позволяет LLM генерировать цепочки рассуждений и специфические для задачи действия в чередующемся порядке. Имитируя человеческий мыслительный процесс, он позволяет моделям взаимодействовать с внешней средой и получать обратную связь.

### Summary

ReAct — это парадигма промптинга, сочетающая рассуждение и действие для повышения эффективности больших языковых моделей (LLM) при решении сложных задач.

## Key Concepts

- Чередование рассуждений и действий
- Генерация действий
- Интеграция наблюдений
- Цепочка рассуждений (Chain-of-Thought)

## Use Cases

- Системы сложного ответа на вопросы
- Автономные агенты
- Рабочие процессы проверки фактов
