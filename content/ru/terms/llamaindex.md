## Definition

Изначально известная как GPT Index, LlamaIndex — мощный фреймворк для работы с данными, позволяющий LLM потреблять и взаимодействовать со структурированными и неструктурированными данными. Он предоставляет инструменты для индексации, запросов и управления контекстом.

### Summary

LlamaIndex — это фреймворк, предназначенный для подключения пользовательских источников данных к большим языковым моделям, что облегчает создание приложений с использованием генерации с дополнением извлечениями (RAG).

## Key Concepts

- RAG (генерация с дополнением извлечениями)
- Индексация данных
- Векторные базы данных
- Внедрение контекста

## Use Cases

- Создание чат-ботов с доступом к приватным данным
- Системы вопросов и ответов по документам
- Корпоративные базы знаний
