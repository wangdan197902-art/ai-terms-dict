## Definition

Данно-центричный ИИ представляет собой смену парадигмы в разработке искусственного интеллекта, фокусируясь на систематическом улучшении данных, используемых для обучения моделей, а не только на оптимизации алгоритмов или гиперпараметров. Этот подход предполагает итеративное улучшение качества разметки и курирование наборов данных.

### Summary

Подход к разработке ИИ, при котором приоритет отдается улучшению качества и количества обучающих данных, а не изменению архитектур моделей.

## Key Concepts

- Качество данных
- Итеративное уточнение
- Согласованность разметки
- Курирование наборов данных

## Use Cases

- Повышение точности компьютерного зрения путем исправления неправильно размеченных изображений
- Снижение предвзятости в алгоритмах найма через сбалансированные наборы данных
- Улучшение производительности NLP-моделей путем исправления грамматических ошибок в обучающем тексте
