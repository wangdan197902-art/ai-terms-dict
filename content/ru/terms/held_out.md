## Definition

Набор данных «held-out» (отложенный) состоит из примеров, намеренно исключенных из фазы обучения модели машинного обучения. Этот подмножество используется для оценки того, насколько хорошо модель обобщает знания на невиданных ранее данных, предоставляя unbiased (несмещенную) оценку во время настройки.

### Summary

Выборки данных, выделенные из обучающей выборки для оценки производительности модели и предотвращения переобучения в процессе разработки.

## Key Concepts

- Обобщение
- Переобучение
- Валидационная выборка
- Несмещенная оценка

## Use Cases

- Настройка гиперпараметров
- Сравнение различных архитектур моделей
- Окончательная оценка производительности перед запуском в продакшн
