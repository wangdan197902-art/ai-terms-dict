## Definition

Эта междисциплинарная область использует машинное обучение для обработки огромных объемов биологических данных, позволяя исследователям предсказывать функции генов, классифицировать заболевания и понимать молекулярные взаимодействия.

### Summary

Применение вычислительных моделей для анализа биологических данных, таких как геномные последовательности и структуры белков, с целью получения биологических инсайтов.

## Key Concepts

- Геномный анализ
- Сворачивание белков
- Предиктивное моделирование
- Данные высокого пропускного потока

## Use Cases

- Предсказание структур белков по аминокислотным последовательностям
- Классификация типов рака на основе экспрессии генов
- Выявление потенциальных мишеней для лекарств в молекулярных базах данных
