## Definition

В искусственном интеллекте фундаментальная модель — это крупномасштабная модель машинного обучения, обученная на широких данных в больших объемах, таких как изображения, текст или аудио. Эти модели предназначены для адаптации

### Summary

Предобученная модель, обученная на обширных наборах данных для использования в качестве основы для различных последующих задач.

## Key Concepts

- Предобучение
- Дообучение
- Крупномасштабные данные
- Обобщение

## Use Cases

- Обработка естественного языка
- Генерация изображений
- Автодополнение кода
