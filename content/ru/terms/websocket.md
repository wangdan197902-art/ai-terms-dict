## Definition

WebSocket — это протокол компьютерной связи, который обеспечивает постоянное двустороннее взаимодействие между клиентом и сервером. В отличие от HTTP, который требует нового соединения для каждого цикла запрос-ответ, WebSocket поддерживает открытое соединение.

### Summary

Протокол, обеспечивающий полнодуплексные каналы связи через одно TCP-соединение для обмена данными в реальном времени.

## Key Concepts

- Полнодуплексная связь
- Постоянное соединение
- Низкая задержка
- Протокол TCP

## Use Cases

- Приложения для чата в реальном времени
- Обновления спортивных результатов в прямом эфире
- Онлайн-игры для нескольких игроков
