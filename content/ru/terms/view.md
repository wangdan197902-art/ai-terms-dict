## Definition

В управлении базами данных представление действует как сохраненный SQL-запрос, который ведет себя как таблица, но само не содержит данных. Оно обеспечивает упрощенный или настроенный взгляд на исходные данные, повышая безопасность и удобство использования.

### Summary

Виртуальная таблица в базе данных, являющаяся результатом сохраненного запроса, которая отображает данные из одной или нескольких таблиц без их физического хранения.

## Key Concepts

- Виртуальная таблица
- SQL-запрос
- Абстракция данных
- Безопасность

## Use Cases

- Создание упрощенных отчетов для нетехнических пользователей
- Ограничение доступа к чувствительным столбцам в таблице
- Стандартизация сложной логики объединений (JOIN) между приложениями

## Code Example

```python
CREATE VIEW ActiveUsers AS SELECT id, name FROM users WHERE status = 'active';
```
