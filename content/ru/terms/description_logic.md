## Definition

Логика описаний (DL) — это разрешимые фрагменты логики первого порядка, которые составляют теоретическую основу для онтологий, особенно языка онтологий веб-семантики (OWL). Они позволяют точно определять концепции и отношения между ними.

### Summary

Семейство формальных языков представления знаний, используемых для представления и логического вывода о концептуальной структуре предметных областей.

## Key Concepts

- Онтология
- Логический вывод
- Подчинение (субсумпция)
- Иерархия концепций

## Use Cases

- Создание онтологий семантической паутины
- Графы знаний в медицине
- Автоматизированные системы классификации
