## Definition

Роботизация бизнес-процессов (RPA) использует программных роботов, часто усиленных технологиями ИИ, для имитации взаимодействия людей с цифровыми системами. Она применяется для оптимизации рабочих процессов, таких как ввод данных, обработка счетов

### Summary

Использование программного обеспечения (ботов) для автоматизации массовых, повторяющихся цифровых задач, которые традиционно выполняются людьми в рамках бизнес-процессов.

## Key Concepts

- Программные боты
- Автоматизация рабочих процессов
- Обработка на основе правил
- Цифровая рабочая сила

## Use Cases

- Обработка счетов-фактур
- Онбординг клиентов
- Задачи миграции данных
