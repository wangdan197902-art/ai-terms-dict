## Definition

Набор данных WikiHow состоит примерно из 60 000 статей-инструкций, собранных с веб-сайта WikiHow. Он широко используется в исследованиях обработки естественного языка для задач абстрактной суммаризации текста и генерации пошаговых инструкций.

### Summary

Крупномасштабный набор данных, состоящий из инструкций со сайта WikiHow, используемый преимущественно для задач суммаризации текста и генерации инструкций.

## Key Concepts

- Суммаризация текста
- Процедурный текст
- Извлечение инструкций

## Use Cases

- Обучение моделей абстрактной суммаризации
- Генерация пошаговых инструкций
- Анализ структур процедурного языка
