## Definition

S2ORC — это всеобъемлющий корпус научных статей, полученный из Semantic Scholar. Он включает полное текстовое содержание, метаданные и отношения цитирования для миллионов статей по различным научным дисциплинам.

### Summary

Корпус открытых исследований Semantic Scholar, крупномасштабный набор данных академических статей со структурированными метаданными и сетями цитирования.

## Key Concepts

- Академическая обработка естественного языка
- Сети цитирования
- Метаданные
- Научные данные

## Use Cases

- Построение систем рекомендаций цитируемых работ
- Классификация научных текстов
- Извлечение сущностей из исследовательских статей
