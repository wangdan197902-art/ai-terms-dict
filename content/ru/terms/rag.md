## Definition

Генерация с дополнением извлечения (RAG) объединяет сильные стороны систем, основанных на поиске, и систем, основанных на генерации текста. Вместо того чтобы полагаться исключительно на параметры предварительно обученной языковой модели, RAG сначала извлекает актуальные данные из внешней базы знаний, добавляет их в контекст запроса и затем использует эту информацию для формирования более точного и обоснованного ответа.

### Summary

RAG — это архитектура ИИ, которая улучшает генеративные модели, извлекая релевантную информацию из внешних баз знаний перед формированием ответа.

## Key Concepts

- Векторная база данных
- Векторные представления (эмбеддинги)
- Контекстное окно
- Семантический поиск

## Use Cases

- Корпоративные базы знаний
- Боты технической поддержки
- Научные ассистенты
