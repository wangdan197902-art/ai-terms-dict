## Definition

Развертывание в Azure включает использование облачно-нативных инструментов, таких как Azure Machine Learning, Azure Kubernetes Service (AKS) или Azure Functions, для масштабного обслуживания моделей ML. Это охватывает управление вычислительными ресурсами и обеспечение надежности сервисов.

### Summary

Процесс хостинга и запуска моделей машинного обучения на облачной инфраструктуре Microsoft Azure.

## Key Concepts

- Облачная инфраструктура
- Сервисная доставка моделей
- Масштабируемость
- Конвейеры CI/CD

## Use Cases

- Точки конечного доступа (API) для вывода моделей в реальном времени
- Пакетное оценивание больших наборов данных
- Управление моделями корпоративного уровня
