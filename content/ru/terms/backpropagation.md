## Definition

Обратное распространение ошибки (backpropagation) — это метод, используемый в искусственных нейронных сетях для вычисления градиента функции потерь по весам. Он работает путем распространения

### Summary

Широко используемый алгоритм обучения нейронных сетей путем вычисления градиентов для минимизации ошибки.

## Key Concepts

- Градиентный спуск
- Правило цепочки
- Функция потерь
- Обновление весов

## Use Cases

- Обучение глубоких нейронных сетей
- Задачи классификации изображений
- Системы распознавания речи
