## Definition

Набор данных PAQ (Pseudo-Answer Quality) содержит миллионы автоматически сгенерированных пар вопрос-ответ, извлеченных из Википедии. Он специально разработан для обучения плотных поисковых систем путем предоставления...

### Summary

Крупномасштабный набор данных пар вопрос-ответ, полученных из Википедии, предназначенный для обучения плотного поиска по отрывкам.

## Key Concepts

- Плотный поиск по отрывкам (Dense Passage Retrieval)
- Отрицательная выборка
- Вопросно-ответные системы открытого домена
- Извлечение данных из Википедии

## Use Cases

- Обучение плотных поисковых систем
- Ответы на вопросы в открытом домене
- Тестирование производительности поиска
