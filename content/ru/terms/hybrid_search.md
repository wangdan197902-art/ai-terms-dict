## Definition

Гибридный поиск объединяет два различных метода извлечения: плотный векторный поиск, который улавливает семантическое значение и контекст, и разреженный векторный (ключевой) поиск, который сопоставляет точные термины. За счет использования этих двух подходов достигается более высокое качество поиска.

### Summary

Стратегия извлечения информации, сочетающая семантический векторный поиск с традиционным индексированием на основе ключевых слов для повышения точности и релевантности результатов.

## Key Concepts

- Векторный поиск
- Сопоставление ключевых слов
- Переранжирование
- Слияние рангов по обратной формуле

## Use Cases

- Поиск документов в корпоративной среде
- Поиск товаров в электронной коммерции
- Продвинутые конвейеры RAG (генерация с дополнением контекста)
