## Definition

Динамическая эпистемическая логика (DEL) расширяет модальную логику для моделирования эволюции знаний, когда агенты получают новую информацию. Она предоставляет инструменты для анализа многоагентных систем, где убеждения изменяются из-за публичных или частных объявлений.

### Summary

Формальная рамка для рассуждений об изменениях состояний знаний агентов в результате обновления информации или событий.

## Key Concepts

- Обновление знаний
- Поток информации
- Многоагентные системы
- Пересмотр убеждений

## Use Cases

- Верификация протоколов безопасности
- Моделирование коммуникации в роях роботов
- Анализ сценариев теории игр
