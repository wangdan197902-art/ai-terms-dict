## Definition

Многочленная адаптивная регрессионная сплайн-модель (MARS) — это гибкий метод регрессии, который моделирует сложные нелинейные зависимости путем подбора кусочно-линейных базисных функций. Он автоматически выбирает оптимальные узлы аппроксимации.

### Summary

Непараметрический метод регрессии, который автоматически выбирает базисные функции для моделирования сложных зависимостей.
