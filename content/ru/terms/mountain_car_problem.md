## Definition

Задача горного автомобиля — это стандартный бенчмарк в исследованиях обучения с подкреплением. Цель состоит в том, чтобы управлять автомобилем с недостаточной мощностью так, чтобы он достиг вершины крутого холма. Поскольку автомобиль не может подняться на холм за один проход, агент должен научиться раскачивать его.

### Summary

Классическая задача обучения с подкреплением, где агент должен загнать автомобиль на крутой холм, используя только управление ускорением.

## Key Concepts

- Разреженные награды (Sparse Rewards)
- Отложенные последствия
- Непрерывное управление
- Бенчмаркинг

## Use Cases

- Тестирование новых алгоритмов обучения с подкреплением
- Демонстрация методов аппроксимации функции ценности
- Образовательные примеры для концепций RL
