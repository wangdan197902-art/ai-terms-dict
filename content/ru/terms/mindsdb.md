### Summary

MindsDB — это платформа с открытым исходным кодом, которая позволяет разработчикам создавать, обучать и развертывать модели машинного обучения непосредственно в существующих базах данных с помощью SQL.

## Key Concepts

- Машинное обучение на основе SQL (SQL-based ML)
- Интеграция с базами данных (Database integration)
- AutoML (Автоматизированное машинное обучение)
- Прогнозное моделирование (Predictive modeling)

## Use Cases

- Обнаружение мошенничества в реальном времени в базах данных
- Прогнозирование оттока клиентов (Customer churn prediction)
- Автоматизированная отчетность с прогнозами
