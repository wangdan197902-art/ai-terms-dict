## Definition

Этот подход использует методы удовлетворения ограничений внутри базы знаний, чтобы убедиться, что собранные продукты соответствуют всем техническим требованиям и требованиям заказчика. Он предотвращает создание недопустимых комбинаций компонентов.

### Summary

Автоматизированный процесс, использующий базы знаний предметной области для генерации корректных конфигураций продуктов на основе ограничений пользователя.

## Key Concepts

- Удовлетворение ограничений
- Моделирование продукта
- Системы на основе правил
- Автоматический логический вывод

## Use Cases

- Сборка пользовательского компьютерного оборудования
- Выбор лицензий и функций программного обеспечения
- Настройка промышленного оборудования
