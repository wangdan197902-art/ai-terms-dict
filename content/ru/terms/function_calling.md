## Definition

Вызов функций позволяет большим языковым моделям взаимодействовать с внешними инструментами и API путем генерации структурированных выходных данных, таких как объекты JSON, которые указывают, какую функцию необходимо выполнить и какие аргументы передать.

### Summary

Механизм, позволяющий большим языковым моделям (LLM) генерировать структурированные данные для запуска конкретных программных функций.

## Key Concepts

- Структурированный вывод
- Интеграция API
- Использование инструментов
- Схема JSON

## Use Cases

- Бронирование авиабилетов через туристические API
- Выполнение сложных математических вычислений
- Запрос записей из базы данных
