## Definition

В контексте ИИ и науки о данных термин «числовой» относится к типам данных или методам, включающим количественные значения, такие как целые числа, числа с плавающей запятой и десятичные дроби. В отличие от категориальных или текстовых данных, числовые данные поддаются математическим операциям.

### Summary

Относящийся к числам или математическим вычислениям, а не к символической логике или тексту.

## Key Concepts

- Числа с плавающей запятой
- Целые числа
- Типы данных
- Количественные данные

## Use Cases

- Статистическое моделирование
- Финансовое прогнозирование
- Обработка данных с датчиков

## Code Example

```python
import numpy as np
arr = np.array([1.5, 2.7, 3.9])
mean_val = np.mean(arr)
print(mean_val)
```
