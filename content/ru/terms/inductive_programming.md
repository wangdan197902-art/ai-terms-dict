## Definition

Индуктивное программирование, часто называемое синтезом программ, включает создание программного кода на основе спецификаций, представленных в виде пар входных и выходных данных, а не явных инструкций. Система выводит логику программы из примеров.

### Summary

Область синтеза программ, в которой компьютерные программы автоматически генерируются на основе примеров ввода и вывода.

## Key Concepts

- Синтез программ
- Примеры ввода-вывода
- Автоматическое кодирование
- Логический вывод

## Use Cases

- Инструменты автодополнения кода
- Генерация скриптов из данных
- Образовательные помощники по программированию
