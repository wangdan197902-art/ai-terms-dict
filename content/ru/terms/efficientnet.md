## Definition

Разработанная компанией Google, EfficientNet использует метод комплексного масштабирования для баланса глубины сети, её ширины и разрешения входного изображения. Этот подход позволяет модели достигать передовых показателей точности при сохранении эффективности вычислений.

### Summary

EfficientNet — это семейство архитектур сверточных нейронных сетей, которое равномерно масштабирует глубину, ширину и разрешение для достижения более высокой точности при меньшем количестве параметров.

## Key Concepts

- Комплексное масштабирование
- Блоки MobileNetV2
- Внимание SE (Squeeze-and-Excitation)
- Эффективность параметров

## Use Cases

- Классификация изображений на мобильных устройствах
- Обнаружение объектов в условиях ограниченных ресурсов
- Извлечение признаков для крупномасштабных визуальных наборов данных
