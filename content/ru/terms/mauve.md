## Definition

MAUVE — это статистическая мера, предназначенная для оценки того, насколько близко вывод генеративной языковой модели имитирует человеческое использование языка. В отличие от простых показателей перплексии, MAUVE использует виртуальные эмбеддинги

### Summary

MAUVE (Measuring Alignment Using Virtual Embeddings) — метрика, используемая в обработке естественного языка для оценки соответствия между распределениями сгенерированного текста и текстов, написанных людьми.

## Key Concepts

- Оценка генерации текста
- Согласование распределений
- Виртуальные эмбеддинги
- Лингвистическая естественность

## Use Cases

- Оценка выходных данных моделей типа GPT
- Тонкая настройка языковых моделей для создания текста, похожего на человеческий
- Бенчмаркинг производительности генеративного ИИ
