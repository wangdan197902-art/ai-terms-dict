## Definition

В физике машинное обучение помогает моделировать квантовую механику, анализировать данные о столкновениях высоких энергий и открывать новые материалы. Оно помогает физикам ориентироваться в многомерных пространствах параметров.

### Summary

Применение машинного обучения для решения сложных физических задач, симуляции квантовых систем и анализа экспериментальных данных с ускорителей частиц.

## Key Concepts

- Квантовое моделирование
- Физика элементарных частиц
- Открытие материалов
- Обнаружение симметрии

## Use Cases

- Анализ данных Большого адронного коллайдера
- Предсказание свойств новых кристаллических материалов
- Моделирование распределения темной материи в космологии
