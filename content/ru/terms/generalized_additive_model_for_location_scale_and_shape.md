## Definition

В отличие от традиционных регрессионных моделей, фокусирующихся только на среднем значении, GAMLSS моделирует всё распределение целиком, включая местоположение (среднее/медиана), масштаб (дисперсия), асимметрию и эксцесс. Она использует обобщенные линейные модели для каждого параметра.

### Summary

GAMLSS — это гибкая статистическая структура, позволяющая моделировать все параметры распределения как функции объясняющих переменных с использованием аддитивных сглаживаний.

## Key Concepts

- Моделирование распределений
- Аддитивное сглаживание
- Моделирование дисперсии
- Статистическая гибкость

## Use Cases

- Анализ кривых роста
- Оценка рисков
- Подгонка сложных распределений данных
