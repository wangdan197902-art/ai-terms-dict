## Definition

Суперсэмплинг в глубоком обучении (DLSS) — это технология, использующая нейронные сети для реконструкции изображений высокого разрешения из входных данных низкого разрешения. Анализируя временные данные и пространственную информацию, она восстанавливает детали.

### Summary

Техника, использующая ИИ для масштабирования изображений низкого разрешения до высокого с улучшением детализации и снижением артефактов.

## Key Concepts

- Масштабирование изображений
- Нейронная реконструкция
- Накопление кадров
- Оптимизация производительности

## Use Cases

- Трассировка лучей в реальном времени
- Улучшение графики в видеоиграх
- Воспроизведение видео высокого разрешения
