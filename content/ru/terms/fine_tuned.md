## Definition

Дообучение предполагает использование модели, уже обученной на большом общем наборе данных, и продолжение ее обучения на меньшем наборе данных, специфичном для задачи. Эта техника использует общие функ

### Summary

Процесс дополнительной тренировки предварительно обученной модели на конкретном наборе данных для адаптации к определенной задаче.

## Key Concepts

- перенос обучения
- обновление весов
- специфичность задачи
- предварительно обученная модель

## Use Cases

- Адаптация больших языковых моделей для обзора юридических документов
- Настройка моделей компьютерного зрения для обнаружения дефектов в промышленности
- Специализация распознавания речи для конкретных акцентов
