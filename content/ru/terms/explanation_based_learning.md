## Definition

EBL сочетает символические рассуждения с машинным обучением для ускорения процесса обучения. Вместо использования больших наборов данных он берет один положительный пример и использует заранее существующую теорию предметной области для вывода общих правил.

### Summary

Обучение на основе объяснений — это метод машинного обучения, использующий теорию предметной области и один пример обучения для генерации объяснения, которое затем обобщается в новое правило.

## Key Concepts

- Теория предметной области
- Операционализация
- Обучение по одному примеру
- Символические рассуждения

## Use Cases

- Приобретение навыков роботами при ограниченном количестве примеров
- Автоматическое доказательство теорем
- Диагностические системы в узкоспециализированных технических областях
