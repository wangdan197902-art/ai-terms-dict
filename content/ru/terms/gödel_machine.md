## Definition

Машина Гёделя — это гипотетический универсальный решатель задач, предложенный Юргеном Шмидхубером, основанный на формальной логике и теории вычислимости. Она работает путем непрерывного анализа собственного исходного кода и поиска способов его оптимизации для повышения производительности при сохранении неизменной целевой функции (utility function).

### Summary

Теоретическая самообучающаяся программа ИИ, способная переписывать свой собственный исходный код для повышения эффективности без изменения поставленных целей.

## Key Concepts

- Самомодификация
- Функция полезности
- Формальная верификация
- Вычислительная эффективность

## Use Cases

- Теоретические исследования в области искусственного общего интеллекта (AGI)
- Изучение пределов алгоритмического самоусовершенствования
- Исследование формальных методов в области безопасности ИИ
