## Definition

Дерево мыслей (ToT) расширяет традиционное стимулирование цепочки мыслей (chain-of-thought), позволяя модели исследовать несколько различных путей рассуждения на каждом шаге, формируя структуру дерева. Модель оценивает состояние каждого узла и выбирает направление, которое с наибольшей вероятностью приведет к решению.

### Summary

Дерево мыслей (Tree of Thoughts) — это框架 рассуждений, который исследует несколько возможных путей рассуждения одновременно, оценивая их для выбора наиболее перспективного следующего шага.

## Key Concepts

- Алгоритм поиска
- Обратный ход (Backtracking)
- Оценка состояния
- Пути рассуждения

## Use Cases

- Решение сложных математических задач
- Генерация творческого текста
- Стратегические игры
