## Definition

Минимизация данных — это ключевой принцип конфиденциальности, требующий от организаций ограничивать сбор данных тем, что является адекватным, релевантным и необходимым. В контексте ИИ это означает проектирование моделей, которые не хранят избыточную информацию.

### Summary

Принцип сбора и обработки только тех персональных данных, которые строго необходимы для конкретной цели.

## Key Concepts

- Ограничение целей
- Конфиденциальность по умолчанию (Privacy by Design)
- Сокращение объема данных
- Соответствие нормативным требованиям

## Use Cases

- Создание анонимизированных наборов данных для общественных исследований
- Реализация федеративного обучения для сохранения исходных данных локально
- Аудит систем ИИ на предмет чрезмерного хранения данных
