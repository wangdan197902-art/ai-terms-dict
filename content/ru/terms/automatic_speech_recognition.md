## Definition

Автоматическое распознавание речи (ASR), также известное как преобразование речи в текст, является подразделом обработки речи, который использует искусственный интеллект для транскрибации аудиосигналов в письменный текст. Современные системы ASR...

### Summary

Технология, преобразующая устную речь в текст с использованием моделей глубокого обучения.

## Key Concepts

- Акустическое моделирование
- Языковое моделирование
- Глубокое обучение
- Транскрибация

## Use Cases

- Голосовые помощники (Siri, Alexa)
- Субтитры в прямом эфире для видео
- Программное обеспечение для диктовки профессионалами
