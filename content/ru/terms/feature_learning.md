### Summary

Подход, при котором алгоритмы автоматически обнаруживают признаки, необходимые для обнаружения или классификации, из сырых данных.

## Key Concepts

- Автоматическое представление
- Иерархические слои
- Глубокое обучение
- Избегание ручной разработки

## Use Cases

- Распознавание изображений и обнаружение объектов
- Транскрибация речи в текст
- Задачи понимания естественного языка
