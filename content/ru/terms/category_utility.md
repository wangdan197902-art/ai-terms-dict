## Definition

Этот метрика количественно оценивает, насколько хорошо набор категорий позволяет предсказывать значения атрибутов внутри этих категорий. Она балансирует размер категорий против их однородности по содержанию.

### Summary

Полезность категории — это математическая мера, используемая для оценки эффективности схемы категоризации на основе выигрыша в информации, который она дает о значениях атрибутов.

## Key Concepts

- выигрыш в информации
- оценка кластеризации
- предиктивная точность
- обучение концепциям

## Use Cases

- Оценка качества кластеризации
- Исследования приобретения концепций
- Оптимизация отбора признаков
