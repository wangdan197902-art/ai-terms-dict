## Definition

Извлечение знаний из реляционных данных фокусируется на извлечении полезной информации из баз данных, где данные организованы в несколько связанных таблиц, а не в одной плоской таблице. Оно использует связи между различными объектами для выявления скрытых зависимостей.

### Summary

Процесс обнаружения закономерностей и получения знаний из данных, состоящих из нескольких взаимосвязанных таблиц или сущностей.

## Key Concepts

- Сущность-связь
- Мультиреляционное обучение
- Анализ графов
- Интеграция баз данных

## Use Cases

- Анализ социальных сетей
- Выявление мошенничества в финансовых транзакциях
- Открытие новых лекарств и биоинформатика
