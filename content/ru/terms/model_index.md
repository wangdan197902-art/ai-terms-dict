## Definition

Файл индекса, обычно называемый 'model_index.json', содержит структурированную информацию об архитектуре модели, включая тип конвейера, подмодели и пути к конфигурациям. Это позволяет платформе Hub объединять

### Summary

Индекс модели — это файл метаданных, используемый платформой Hugging Face Hub для описания и организации компонентов и конфигураций модели.

## Key Concepts

- Метаданные
- Конфигурация конвейера
- Загрузка модели
- Интероперабельность

## Use Cases

- Определение многокомпонентных конвейеров
- Загрузка сложных моделей на платформу Hub
- Автоматическое обнаружение и загрузка моделей
