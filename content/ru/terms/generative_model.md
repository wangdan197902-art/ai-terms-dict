## Definition

Генеративные модели — это алгоритмы, предназначенные для понимания закономерностей и структур в заданном наборе данных, чтобы создавать новые экземпляры данных, напоминающие оригинальные. В отличие от дискриминативных моделей...

### Summary

Тип модели машинного обучения, которая изучает лежащую в основе распределённость данных для генерации новых синтетических образцов, похожих на обучающие данные.

## Key Concepts

- Вероятностное распределение
- Синтетические данные
- Латентное пространство
- Цель обучения

## Use Cases

- Генерация и редактирование изображений
- Обработка естественного языка
- Аугментация данных для малых наборов данных
