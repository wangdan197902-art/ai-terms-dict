## Definition

Облачные вычисления обеспечивают масштабируемую инфраструктуру для задач ИИ, позволяя разработчикам получать доступ к мощным GPU и хранилищам без необходимости содержания физических дата-центров. Это поддерживает различные модели обслуживания.

### Summary

Облако относится к удаленным серверам, размещенным в интернете, которые используются для хранения, управления и обработки данных и моделей ИИ вместо локального оборудования.

## Key Concepts

- Масштабируемость
- Инфраструктура как сервис (IaaS)
- Удаленное хранение
- Эластичность

## Use Cases

- Обучение больших языковых моделей
- Размещение API для веб-приложений
- Хранение массивных наборов данных для аналитики
