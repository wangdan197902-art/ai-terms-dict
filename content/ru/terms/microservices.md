## Definition

В контексте инженерии ИИ микросервисы позволяют различным компонентам конвейера ИИ, таким как предварительная обработка данных, вывод модели и хранение результатов, разрабатываться, масштабироваться и поддерживаться...

### Summary

Микросервисы — это архитектурный стиль, который структурирует приложение как набор слабо связанных, независимо развертываемых сервисов, организованных вокруг бизнес-возможностей.

## Key Concepts

- Слабая связанность (Loose coupling)
- Независимое развертывание (Independent deployment)
- Декомпозиция сервисов (Service decomposition)
- Шлюзы API (API gateways)

## Use Cases

- Масштабируемые конечные точки для вывода ИИ (AI inference endpoints)
- Модульные конвейеры машинного обучения
- Отказоустойчивые рекомендательные системы
