## Definition

AutoML (Автоматизированное машинное обучение) оптимизирует разработку моделей МЛ путем автоматизации таких задач, как предварительная обработка данных, инженерия признаков, выбор модели и настройка гиперпараметров. Это делает технологии доступными для более широкой аудитории.

### Summary

Методология, автоматизирующая сквозной процесс применения машинного обучения к реальным проблемам, снижая объем ручной работы.

## Key Concepts

- Настройка гиперпараметров
- Инженерия признаков
- Выбор модели
- Демократизация

## Use Cases

- Быстрое прототипирование для бизнес-аналитиков
- Оптимизация крупномасштабных производственных конвейеров
- Автоматическое сравнение нескольких алгоритмов

## Code Example

```python
from auto_ml import Predictor
predictor = Predictor(type_of_estimator='classifier')
predictor.fit(dataframe, target='label')
```
