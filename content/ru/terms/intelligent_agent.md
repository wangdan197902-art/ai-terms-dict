## Definition

Интеллектуальный агент — это система, способная воспринимать свое окружение через датчики или входные данные, обрабатывать эту информацию с помощью алгоритмов рассуждения и воздействовать на среду посредством исполнительных механизмов (актуаторов) для достижения поставленных целей.

### Summary

Автономная программная сущность, которая воспринимает окружающую среду, анализирует возможные действия и выполняет задачи для достижения конкретных целей.

## Key Concepts

- Автономность
- Петля восприятие-действие
- Ориентация на цель
- Реактивность

## Use Cases

- Автоматические торговые боты, исполняющие рыночные стратегии
- Роботизированная автоматизация процессов (RPA), управляющая многоэтапными рабочими потоками
- Персональные ассистенты, управляющие календарем и электронной почтой
