## Definition

Роботы охватывают разнообразный класс машин, которые можно классифицировать по их мобильности, конструкции или области применения. В эту категорию входят промышленные манипуляторы, автономные мобильные роботы (AMR), дроны и другие устройства.

### Summary

Множественное число, обозначающее несколько программируемых машин, предназначенных для автономного выполнения задач.

## Key Concepts

- Мультиагентные системы
- Инженерия робототехники
- Классификация
- Развертывание

## Use Cases

- Флоты складской логистики
- Операции роя дронов
- Образовательные наборы
