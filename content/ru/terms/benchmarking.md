## Definition

Бенчмаркинг — это активная практика проведения экспериментов для измерения того, насколько хорошо модель ИИ справляется с конкретными задачами с использованием заранее определенных бенчмарков. Этот процесс включает запуск моделей через стандартные тестовые среды.

### Summary

Систематический процесс тестирования моделей ИИ на бенчмарках для количественной оценки их производительности и выявления областей для улучшения.

## Key Concepts

- Тестирование производительности
- Количественный анализ
- Оптимизация
- Валидация

## Use Cases

- Сравнение скорости вывода (инференса) различных аппаратных ускорителей.
- Измерение точности дообученной модели по сравнению с предварительно обученным базовым уровнем.
- Аудит справедливости модели и наличия смещений (bias) среди разнообразных демографических групп.
