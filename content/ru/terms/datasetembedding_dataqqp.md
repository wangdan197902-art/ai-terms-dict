## Definition

Quora Question Pairs (QQP) — это набор данных для бинарной классификации, содержащий более 400 000 пар вопросов с платформы Quora. Задача заключается в определении того, имеют ли два вопроса одинаковый смысл или...

### Summary

Набор данных Quora Question Pairs, используемый для обучения моделей обнаружения семантического сходства между вопросами.

## Key Concepts

- Семантическое текстовое сходство
- Бинарная классификация
- Сопоставление намерений
- Векторные представления предложений

## Use Cases

- Обнаружение дубликатов вопросов
- Дообучение трансформеров предложений
- Улучшение работы чат-ботов FAQ
