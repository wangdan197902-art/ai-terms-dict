## Definition

REST API обеспечивают взаимодействие между клиентами и серверами за счет использования безсостоятельных операций по протоколам HTTP, таким как GET, POST, PUT и DELETE. Они структурируют ресурсы в виде URI и используют стандартные форматы данных (например, JSON/XML).

### Summary

REST API — это архитектурный стиль для проектирования сетевых приложений, основанный на стандартных методах HTTP.

## Key Concepts

- Безсостоятельность
- Методы HTTP
- URI, основанные на ресурсах
- Сериализация JSON

## Use Cases

- Взаимодействие микросервисов
- Интеграция с бэкендом мобильных приложений
- Доступ к данным сторонних сервисов
