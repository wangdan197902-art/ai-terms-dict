## Definition

Эта практика предполагает подключение моделей ИИ, таких как большие языковые модели (LLM), к средам разработки программного обеспечения, таким как VS Code или IntelliJ. Это включает функции интеллектуального завершения кода, автоматического рефакторинга и выполнения запросов на естественном языке.

### Summary

Интеграция в IDE (Integrated Development Environment) означает внедрение инструментов на базе ИИ непосредственно в среды разработки для помощи программистам в выполнении задач кодирования.

## Key Concepts

- Автозавершение
- Завершение кода
- LLM (Большие языковые модели)
- Опыт разработчика (Developer Experience)

## Use Cases

- Интеллектуальные предложения кода
- Автоматический рефакторинг
- Выполнение запросов на естественном языке
