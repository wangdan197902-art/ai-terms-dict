## Definition

Многоагентные системы состоят из нескольких независимых агентов, каждый из которых может специализироваться на различных задачах или доменах. Эти агенты общаются и координируют свои действия для достижения общей цели...

### Summary

Архитектурный подход, при котором несколько автономных ИИ-агентов сотрудничают, конкурируют или координируют свои действия для решения сложных задач, превышающих возможности отдельных агентов.

## Key Concepts

- Сотрудничество агентов
- Специализация
- Координация
- Децентрализация

## Use Cases

- Автоматизированная разработка программного обеспечения
- Среды сложных симуляций
- Распределенное принятие решений
