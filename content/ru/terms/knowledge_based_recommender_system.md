## Definition

В отличие от коллаборативной фильтрации, опирающейся на прошлое поведение пользователей, KBRS использует явные знания об элементах и предпочтениях пользователей для формирования рекомендаций. Она особенно эффективна на рынках с разреженными данными.

### Summary

Система рекомендаций, генерирующая предложения путем логического вывода на основе явных знаний о предметной области и ограничений пользователя, а не исторических данных.

## Key Concepts

- Явные ограничения
- Логический вывод
- Объяснимость
- Работа с разреженными данными

## Use Cases

- Поиск недвижимости
- Сложные закупки оборудования для бизнеса (B2B)
- Планирование персонализированных путешествий
