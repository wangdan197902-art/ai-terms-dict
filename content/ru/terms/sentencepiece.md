## Definition

SentencePiece — популярная библиотека с открытым исходным кодом для нормализации и токенизации текста, широко используемая в современных конвейерах NLP. Она выполняет ненадзорное обучение совместного словаря слов-частей и подслов, обрабатывая текст как последовательность символов.

### Summary

Библиотека для ненадзорного токенизации и детокенизации текста, рассматривающая исходный текст как последовательность подслов для предобработки в NLP.

## Key Concepts

- Токенизация на основе подслов
- Обучение словаря
- Детокенизация
- Языковая независимость

## Use Cases

- Предобработка данных для моделей Трансформер
- Обработка многоязычных текстовых корпусов
- Сокращение размера словаря языковых моделей
