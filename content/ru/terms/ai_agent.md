## Definition

ИИ-агент — это программная сущность, которая автономно функционирует в определенной среде для достижения заранее определенных целей. Он использует механизмы восприятия для сбора данных, обрабатывает эту информацию и принимает решения на основе полученной информации.

### Summary

Автономная система, предназначенная для восприятия окружающей среды и выполнения действий для достижения конкретных целей.

## Key Concepts

- автономность
- контур восприятие-действие
- целеориентированное поведение
- взаимодействие со средой

## Use Cases

- Автоматизированные боты службы поддержки клиентов
- Навигация автономных транспортных средств
- Помощники для генерации кода
