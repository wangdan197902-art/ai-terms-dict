## Definition

Граф — это фундаментальная структура данных в ИИ, состоящая из вершин (узлов) и ребер (связей), обозначающих отношения. Графовые нейронные сети (GNN) используют эту структуру для выполнения обучения на неевклидовых данных.

### Summary

Математическая структура, состоящая из узлов, соединенных ребрами, используемая для представления отношений между сущностями.

## Key Concepts

- Узлы/Вершины
- Ребра
- Матрица смежности
- Топология

## Use Cases

- Графы знаний
- Анализ социальных сетей
- Предсказание свойств молекул
