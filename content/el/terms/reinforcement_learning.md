## Definition

Η Μάθηση με Ενίσχυση (RL) είναι ένας κλάδος της μηχανικής μάθησης που εστιάζει στο πώς οι έξυπνοι πράκτορες θα έπρεπε να ενεργούν σε ένα περιβάλλον για να μεγιστοποιήσουν τη συσσωρευμένη ανταμοιβή. Σε αντίθεση με την εποπτευόμενη μάθηση

### Summary

Η Μάθηση με Ενίσχυση είναι ένας τύπος μηχανικής μάθησης όπου οι πράκτορες μαθαίνουν να λαμβάνουν αποφάσεις αλληλεπιδρώντας με ένα περιβάλλον για τη μεγιστοποίηση των ανταμοιβών.

## Key Concepts

- Πράκτορας
- Περιβάλλον
- Πολιτική (Policy)
- Συνάρτηση Ανταμοιβής
- Χώρος Καταστάσεων

## Use Cases

- Αυτόνομη οδήγηση
- Τεχνητή Νοημοσύνη σε παιχνίδια (π.χ. AlphaGo)
- Συστήματα σύστασης
