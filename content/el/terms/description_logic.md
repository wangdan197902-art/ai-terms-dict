## Definition

Οι Λογικές Περιγραφής (DL) είναι αποφασίσιμα τμήματα της λογικής πρώτης τάξης που αποτελούν τη θεωρητική βάση για τις οντολογία, ιδιαίτερα για τη Γλώσσα Οντολογίας Web (OWL). Επιτρέπουν τον ακριβή ορισμό και τον συλλογισμό εννοιών και σχέσεων.

### Summary

Μια οικογένεια τυπικών γλωσσών αναπαράστασης γνώσης που χρησιμοποιούνται για την αναπαράσταση και τον συλλογισμό σχετικά με την εννοιολογική δομή πεδίων.

## Key Concepts

- Οντολογία
- Συλλογισμός
- Υπερκατάσταση
- Ιεραρχία Εννοιών

## Use Cases

- Κατασκευή οντολογιών για το ημιανθρώπινο διαδίκτυο
- Γράφοι γνώσης στον ιατρικό τομέα
- Συστήματα αυτοματοποιημένης ταξινόμησης
