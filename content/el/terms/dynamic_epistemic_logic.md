## Definition

Η Δυναμική Επιγνωστική Λογική (DEL) επεκτείνει τη μορφική λογική για να μοντελοποιήσει πώς εξελίσσεται η γνώση όταν οι πράκτορες λαμβάνουν νέες πληροφορίες. Παρέχει εργαλεία για την ανάλυση συστημάτων πολλών πρακτόρων όπου οι πεποιθήσεις αλλάζουν λόγω δημόσιων ανακοινώσεων ή ιδιωτικών επικοινωνιών.

### Summary

Ένας τυπικός πλαίσιο για τον συλλογισμό σχετικά με τις αλλαγές στις καταστάσεις γνώσης των πρακτόρων που προκύπτουν από ενημερώσεις πληροφοριών ή γεγονότα.

## Key Concepts

- Ενημέρωση Γνώσης
- Ροή Πληροφοριών
- Συστήματα Πολλών Πρακτόρων
- Επαναξιολόγηση Πεποιθήσεων

## Use Cases

- Επαλήθευση πρωτοκόλλων ασφαλείας
- Μοντελοποίηση επικοινωνίας σε σμήνη ρομποτικής
- Ανάλυση σεναρίων θεωρίας παιγνίων
