## Definition

Η ενεργητική μάθηση μειώνει την ποσότητα των ετικετοποιημένων δεδομένων που απαιτούνται, επιτρέποντας στο μοντέλο να επιλέγει τις πιο πληροφοριακές στιγμιότυπα για ανθρώπινη ετικετοποίηση. Αντί να λαμβάνει παθητικά τυχαία δείγματα, ο αλγόριθμος ενεργοποιείται για να καθορίσει ποια δεδομένα θα ετικετοποιηθούν επόμενα.

### Summary

Μια στρατηγική μηχανικής μάθησης όπου ο αλγόριθμος επιλεκτικά ζητά από έναν χρήστη ή ένα οράκιο να ετικετοποιήσει νέα σημεία δεδομένων, βελτιστοποιώντας τη διαδικασία εκπαίδευσης.

## Key Concepts

- Στρατηγική ερώτησης
- Απόδοση ετικετών
- Δειγματοληψία αβεβαιότητας
- Άνθρωπος στον βρόχο

## Use Cases

- Ετικετοποίηση ιατρικών απεικονίσεων από ειδικούς ακτινολόγους
- Ετικετοποίηση ανάλυσης συναισθήματος για σπάνιες διαλέκτους
- Επιλογή σεναίων για αυτοκίνητα αυτόνομης οδήγησης
