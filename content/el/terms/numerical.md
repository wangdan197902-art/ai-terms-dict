## Definition

Στο πλαίσιο της Τεχνητής Νοημοσύνης και της επιστήμης δεδομένων, ο όρος «αριθμητικός» αναφέρεται σε τύπους δεδομένων ή μεθόδους που περιλαμβάνουν ποσοτικές τιμές, όπως ακέραιοι, δεκαδικοί και ρεαλοί. Σε αντίθεση με τα κατηγορικά ή τα κειμενικά δεδομένα, τα αριθμητικά δεδομένα επιτρέπουν μαθηματικές πράξεις.

### Summary

Σχετικό με, που περιλαμβάνει ή χρησιμοποιεί αριθμούς ή μαθηματικούς υπολογισμούς αντί για συμβολική λογική ή κείμενο.

## Key Concepts

- Δεκαδικοί Αριθμοί (Floats)
- Ακέραιοι
- Τύποι Δεδομένων
- Ποσοτικά Δεδομένα

## Use Cases

- Στατιστική μοντελοποίηση
- Οικονομικές προβλέψεις
- Επεξεργασία δεδομένων αισθητήρων

## Code Example

```python
import numpy as np
arr = np.array([1.5, 2.7, 3.9])
mean_val = np.mean(arr)
print(mean_val)
```
