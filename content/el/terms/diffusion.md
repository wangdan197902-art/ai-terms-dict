## Definition

Τα μοντέλα διάχυσης είναι μια κατηγορία γεννητικής ΤΝ που μαθαίνουν να αντιστρέφουν μια στοχαστική διαδικασία προσθήκης θορύβου στα δεδομένα. Εκπαιδεύοντας ένα νευρωνικό δίκτυο να προβλέπει και να αφαιρεί αυτόν τον θόρυβο βήμα προς βήμα, μπορούν να γεννήσουν υψηλής ποιότητας δεδομένα.

### Summary

Μια τεχνική γεννητικού μοντελοποίησης που δημιουργεί δεδομένα αντιστρέφοντας μια σταδιακή διαδικασία θορύβου για την ανακατασκευή καθαρών δειγμάτων.

## Key Concepts

- Στοχαστική Διαδικασία
- Πρόγραμμα Θορύβου
- Αντίστροφη Διαδικασία
- Λανθάνων Χώρος

## Use Cases

- Γεννήτρια εικόνων υψηλής πιστότητας
- Σύνθεση ήχου
- Σχεδιασμός μορίων για την ανακάλυψη φαρμάκων
