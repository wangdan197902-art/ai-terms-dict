## Definition

Το πλαίσιο ReAct επιτρέπει στα LLM να δημιουργούν τόσο ιχνηλατήσεις σκέψης όσο και ενέργειες ειδικές για την εργασία με τρόπο εναλλασσόμενο. Προσομοιώνοντας ανθρώπινες διαδικασίες σκέψης, επιτρέπει στα μοντέλα να αλληλεπιδρούν με εξωτερικά περιβάλλοντα και εργαλεία για την επίλυση προβλημάτων.

### Summary

Το ReAct είναι ένα παράδειγμα prompting (εντοπισμού) που συνδυάζει τη λογική σκέψη και τη δράση για τη βελτίωση της απόδοσης των μεγάλων γλωσσικών μοντέλων σε σύνθετες εργασίες.

## Key Concepts

- Εναλλασσόμενη Λογική Σκέψη
- Δημιουργία Ενέργειας
- Ολοκλήρωση Παρατήρησης
- Αλυσίδα Σκέψης (Chain-of-Thought)

## Use Cases

- Συστήματα σύνθετης ερώτησης-απάντησης
- Αυτόνομοι πράκτορες
- Ροές εργασίας επαλήθευσης γεγονότων
