## Definition

Η μηχανική χαρακτηριστικών είναι η τέχνη της αξιοποίησης της εξειδίκευσης στον κλάδο για τον μετασχηματισμό των ωμών δεδομένων σε χαρακτηριστικά που αντιπροσωπεύουν καλύτερα τα υποκείμενα πρότυπα στα αλγόριθμα μηχανικής μάθησης. Αυτή η διαδικασία περιλαμβάνει τη δημιουργία νέων μεταβλητών.

### Summary

Η πρακτική της χρήσης γνώσης του πεδίου εφαρμογής για τη δημιουργία νέων χαρακτηριστικών ή την τροποποίηση υπαρχόντων, με στόχο τη βελτίωση της απόδοσης των μοντέλων μηχανικής μάθησης.

## Key Concepts

- Γνώση Πεδίου Εφαρμογής
- Μετασχηματισμός Δεδομένων
- Απόδοση Μοντέλου
- Δημιουργία Μεταβλητών

## Use Cases

- Βελτίωση ακρίβειας μοντέλων παλινδρόμησης
- Ενίσχυση ορίων ταξινόμησης
- Βελτιστοποίηση πρόβλεψης χρονοσειρών

## Code Example

```python
df['new_feature'] = df['feature_a'] * df['feature_b']
```
