## Definition

Τα REST APIs επιτρέπουν την επικοινωνία μεταξύ πελατών και διακομιστών χρησιμοποιώντας ατελείωτες λειτουργίες μέσω πρωτοκόλλων HTTP όπως GET, POST, PUT και DELETE. Δομούν τους πόρους ως URI και χρησιμοποιούν τυποποιημένα μορφοποιημένα δεδομένα (συνήθως JSON) για την ανταλλαγή πληροφοριών.

### Summary

Η REST API είναι ένα αρχιτεκτονικό στυλ για τον σχεδιασμό δικτυακών εφαρμογών που βασίζεται σε τυποποιημένες μεθόδους HTTP.

## Key Concepts

- Ατελείωτη φύση (Statelessness)
- Μέθοδοι HTTP
- URI Βασισμένα σε Πόρους
- Σειριοποίηση JSON

## Use Cases

- Επικοινωνία μικροϋπηρεσιών
- Ολοκλήρωση back-end για εφαρμογές κινητών
- Πρόσβαση σε δεδομένα τρίτων
