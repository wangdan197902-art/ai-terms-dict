## Definition

Η καθοδική κλίση είναι ένας αλγόριθμος βελτιστοποίησης πρώτης τάξης για την εύρεση ενός τοπικού ελαχίστου μιας παραγωγίσιμης συνάρτησης. Στην μηχανική μάθηση, ενημερώνει τα βάρη του μοντέλου στην αντίθετη κατεύθυνση της κλίσης.

### Summary

Επαναληπτικός αλγόριθμος βελτιστοποίησης που χρησιμοποιείται για τη μείωση μιας συνάρτησης σφάλματος προσαρμόζοντας τις παραμέτρους του μοντέλου.

## Key Concepts

- Συνάρτηση Σφάλματος
- Ρυθμά Εκμάθησης
- Βελτιστοποίηση
- Οπισθοδιάδοση

## Use Cases

- Εκπαίδευση βαθιών νευρωνικών δικτύων
- Βελτιστοποίηση παραμέτρων γραμμικής παλινδρόμησης
- Βελτιστοποίηση μοντέλων αναγνώρισης εικόνας
