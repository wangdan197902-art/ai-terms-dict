## Definition

Στα GANs, η mode collapse συμβαίνει όταν ο γεννήτριας μαθαίνει να εκμεταλλεύεται τις αδυναμίες του δικριμιναντέρ παράγοντας ένα στενό φάσμα πιθανών δειγμάτων, αγνοώντας άλλους τρόπους της κατανομής των δεδομένων. Αυτό

### Summary

Η Mode collapse (κατάρρευση τρόπων) είναι μια κατάσταση αποτυχίας στα Γενερατορικά Ανταγωνιστικά Δίκτυα όπου ο γεννήτριας παράγει περιορισμένες ποικιλίες εξόδων.

## Key Concepts

- Σταθερότητα GAN
- Ποικιλομορφία Κατανομής
- Αποτυχία Γεννήτριας
- Ανάδραση Δικριμιναντέρ

## Use Cases

- Διάγνωση αστάθειας στην εκπαίδευση GAN
- Βελτίωση της ποικιλομορφίας στη δημιουργία εικόνων
- Ανάλυση κάλυψης του κρυφού χώρου (latent space)
