## Definition

Τα πλαίσια συλλογισμού παρέχουν μια μαθηματική βάση για την αναπαράσταση επιχειρημάτων, επιθέσεων και άμυνας μεταξύ τους. Στην μηχανική ΤΝ, βοηθούν τα συστήματα να λαμβάνουν διαφανείς και αιτιολογημένες αποφάσεις ζυγίζοντας τα διαφορετικά επιχειρήματα.

### Summary

Μια τυπική δομή που χρησιμοποιείται για τη μοντελοποίηση και την επίλυση συγκρούσεων μεταξύ ανταγωνιστικών ισχυρισμών ή αποφάσεων σε συστήματα ΤΝ.

## Key Concepts

- Επίλυση Συγκρούσεων
- Εξηγήσιμη ΤΝ
- Λογικός Συλλογισμός
- Διαφάνεια Αποφάσεων

## Use Cases

- Συστήματα νομικού συλλογισμού
- Αιτιολόγηση ιατρικής διάγνωσης
- Πολιτικές διαχείρισης περιεχομένου
