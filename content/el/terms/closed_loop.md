## Definition

Τα συστήματα closed-loop στην ΤΝ αξιοποιούν την ανατροφοδότηση σε πραγματικό χρόνο από το περιβάλλον για να προσαρμόζουν δυναμικά τη συμπεριφορά ή τις παραμέτρους τους. Αυτό αντιπαραβάλλεται με τα συστήματα open-loop που εκτελούν προκαθορισμένες ακολουθίες χωρίς ανατροφοδότηση...

### Summary

Ένα σύστημα ελέγχου όπου η ανατροφοδότηση της εξόδου χρησιμοποιείται συνεχώς για τη ρύθμιση μελλοντικών ενεργειών.

## Key Concepts

- Βρόχος Ανατροφοδότησης
- Προσαρμογή σε Πραγματικό Χρόνο
- Αυτονομία
- Θεωρία Ελέγχου

## Use Cases

- Έλεγχος Αυτοκινήτων Αυτονομίας
- Ενισχυτική Μάθηση
- Προσαρμοστική Ρομποτική
