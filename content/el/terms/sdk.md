## Definition

Ένα SDK είναι ένα σύνολο εργαλείων ανάπτυξης λογισμικού που επιτρέπει στους προγραμματιστές να δημιουργούν εφαρμογές για συγκεκριμένες πλατφόρμες ή υπηρεσίες. Για την ΤΝ, τα SDK παρέχουν έτοιμες βιβλιοθήκες, API και βοηθητικά προγράμματα για...

### Summary

Ένα Πακέτο Ανάπτυξης Λογισμικού που παρέχει εργαλεία, βιβλιοθήκες και τεκμηρίωση για την κατασκευή εφαρμογών.
