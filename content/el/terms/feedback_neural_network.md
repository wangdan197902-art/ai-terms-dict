## Definition

Τα δίκτυα νευρωνικής ανατροφοδότησης, γνωστά επίσης ως επαναλαμβανόμενα νευρωνικά δίκτυα (RNNs), περιέχουν βρόχους που επιτρέπουν στα σήματα να διαδίδονται πίσω σε προηγούμενες στρώσεις. Αυτή η επανάληψη επιτρέπει στο δίκτυο να διατηρεί μια εσωτερική κατάσταση.

### Summary

Μια αρχιτεκτονική νευρωνικού δικτύου όπου οι συνδέσεις σχηματίζουν κατευθυντικούς κύκλους, επιτρέποντας στην πληροφορία να διατηρείται στον χρόνο.

## Key Concepts

- Επανάληψη
- Εσωτερική Κατάσταση
- Χρονική Δυναμική
- Μνήμη

## Use Cases

- Αναγνώριση ομιλίας
- Πρόβλεψη χρονοσειρών
- Επεξεργασία φυσικής γλώσσας
