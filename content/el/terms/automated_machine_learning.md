## Definition

Το AutoML (Automated Machine Learning) απλοποιεί την ανάπτυξη μοντέλων ML αυτοματοποιώντας εργασίες όπως την προεπεξεργασία δεδομένων, τη μηχανική χαρακτηριστικών, την επιλογή μοντέλου και τη βελτιστοποίηση υπερπαραμέτρων. Δυνατοποιεί τους μη ειδικούς.

### Summary

Μια μεθοδολογία που αυτοματοποιεί την ολοκληρωμένη διαδικασία εφαρμογής της μηχανικής μάθησης σε πραγματικά προβλήματα, μειώνοντας την χειροκίνητη προσπάθεια.

## Key Concepts

- Βελτιστοποίηση Υπερπαραμέτρων
- Μηχανική Χαρακτηριστικών
- Επιλογή Μοντέλου
- Δημοκρατικοποίηση

## Use Cases

- Γρήγορη δημιουργία πρωτοτύπων για επιχειρηματικούς αναλυτές
- Βελτιστοποίηση μεγάλων κλίμακων γραμμών παραγωγής
- Αυτόματη σύγκριση πολλαπλών αλγορίθμων

## Code Example

```python
from auto_ml import Predictor
predictor = Predictor(type_of_estimator='classifier')
predictor.fit(dataframe, target='label')
```
