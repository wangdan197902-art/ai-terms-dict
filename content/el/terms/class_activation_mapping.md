## Definition

Η CAM δημιουργεί χάρτες θερμότητας που επικαλύπτουν τις εικόνες εισόδου, δείχνοντας ποια pixels συνέβαλαν περισσότερο στην απόφαση του μοντέλου για μια συγκεκριμένη ετικέτα κλάσης. Λειτουργεί εφαρμόζοντας παγκόσμια μέση συσσώρευση (global average pooling) στα τελικά επίπεδα χαρακτηριστικών του δικτύου.

### Summary

Η χαρτογράφηση ενεργοποίησης κλάσης (CAM) είναι μια τεχνική οπτικοποίησης που τονίζει τις περιοχές σε μια εικόνα εισόδου που ευθύνονται περισσότερο για μια συγκεκριμένη προβλεπόμενη κλάση.

## Key Concepts

- Οπτικοποίηση Χάρτη Θερμότητας
- Εξηγησιμότητα
- Σημασία Χαρακτηριστικών
- Παγκόσμια Μέση Συσσώρευση

## Use Cases

- Επαλήθευση διάγνωσης ιατρικών απεικονίσεων
- Αποσφαλμάτωση αυτόνομης ανίχνευσης αντικειμένων
- Αναφορές Εξηγήσιμης ΤΝ
