## Definition

Τα γεννητικά μοντέλα βάσει ροής κατασκευάζουν πολύπλοκες πιθανοτικές κατανομές εφαρμόζοντας μια σειρά από αντιστρέψιμους, διαφορίσιμους μετασχηματισμούς σε μια απλή βασική κατανομή, όπως μια Γκαουσιανή. Επειδή οι μετασχηματισμοί είναι αντιστρέψιμοι, επιτρέπουν τον ακριβή υπολογισμό.

### Summary

Μια κατηγορία γεννητικών μοντέλων που χρησιμοποιούν αντιστρέψιμες μετασχηματισμούς για να χαρτογραφήσουν απλές κατανομές σε πολύπλοκες κατανομές δεδομένων.

## Key Concepts

- Αντιστρέψιμοι Μετασχηματισμοί
- Ακριβής Πιθανοφάνεια
- Ροές Κανονικοποίησης
- Αλλαγή Μεταβλητών

## Use Cases

- Γένεση εικόνων
- Εκτίμηση πυκνότητας
- Ανίχνευση ανωμαλιών
