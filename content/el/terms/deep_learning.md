## Definition

Οι αλγόριθμοι βαθιάς μάθησης επιχειρούν να μιμηθούν τις αναλυτικές και μαθησιακές διαδικασίες του ανθρώπινου εγκεφάλου. Στοιβάζοντας πολλαπλά στρώματα διασυνδεδεμένων κόμβων, αυτά τα μοντέλα μπορούν να μάθουν ιεραρχικά χαρακτηριστικά από μεγάλα σύνολα δεδομένων, αποκαλύπτοντας πολύπλοκες σχέσεις.

### Summary

Ένα υποσύνολο της μηχανικής μάθησης που χρησιμοποιεί πολυεπίπεδα τεχνητά νευρωνικά δίκτυα για τη μοντελοποίηση πολύπλοκων μοτίβων και αναπαραστάσεων στα δεδομένα.

## Key Concepts

- Νευρωνικά Δίκτυα
- Ιεραρχικά Χαρακτηριστικά
- Μη Γραμμική Μοντελοποίηση
- Μάθηση Αναπαράστασης

## Use Cases

- Μετατροπή ομιλίας σε κείμενο
- Συστήματα σύστασης
- Αντίληψη σε αυτονομική οδήγηση
