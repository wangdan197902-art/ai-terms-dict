## Definition

Οι προσαρμογείς είναι μια τεχνική λεπτομερούς προσαρμογής με αποδοτικότητα παραμέτρων, που χρησιμοποιείται κυρίως σε μεγάλα γλωσσικά μοντέλα και μετασχηματιστές. Αντί να ενημερώνονται όλα τα βάρη του μοντέλου, κάτι που είναι υπολογιστικά απαιτητικό, οι προσαρμογείς εισάγουν μόνο έναν μικρό αριθμό νέων παραμέτρων.

### Summary

Ένα ελαφρύ μοντέλο που παρεμβάλλεται σε προεκπαιδευμένα μοντέλα για να επιτρέψει την αποδοτική λεπτομερή προσαρμογή για συγκεκριμένες εργασίες.

## Key Concepts

- Λεπτομερής Προσαρμογή με Αποδοτικότητα Παραμέτρων
- Μεταφορά Μάθησης
- Μοντουλιακή Αρχιτεκτονική
- Καταστροφλή Λήθη

## Use Cases

- Λεπτομερής προσαρμογή LLMs για τσάμποτ εξυπηρέτησης πελατών
- Προσαρμογή οπτικών μοντέλων για ανάλυση ιατρικών απεικονίσεων
- Τοποθέτηση πολλαπλών μοντέλων ειδικών για κάθε τομέα με αποδοτικότητα
