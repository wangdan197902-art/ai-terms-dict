## Definition

Σε αντίθεση με τη συνεργατική φιλτραρίσματα, η οποία βασίζεται στη συμπεριφορά των χρηστών στο παρελθόν, τα συστήματα αυτά χρησιμοποιούν ρητή γνώση για τα αντικείμενα και τις προτιμήσεις των χρηστών για να εξαγάγουν συστάσεις. Είναι ιδιαίτερα αποτελεσματικά σε αγορές όπου τα δεδομένα είναι αραιά ή τα προϊόντα είναι πολύπλοκα και δύσκολο να συγκριθούν.

### Summary

Ένας μηχανισμός σύστασης που παράγει προτάσεις σκεπτόμενος πάνω σε ρητή γνώση τομέα και περιορισμούς χρήστη, αντί για ιστορικά δεδομένα.

## Key Concepts

- Ρητοί Περιορισμοί
- Συλλογισμός
- Εξηγησιμότητα
- Χειρισμός Αραιών Δεδομένων

## Use Cases

- Αναζήτηση ακινήτων
- Σύνθετη προμήθεια εξοπλισμού B2B
- Προσωποποιημένος σχεδιασμός ταξιδιωτικών δρομολογίων
