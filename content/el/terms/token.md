## Definition

Τα tokens είναι τα θεμελιώδη δομικά στοιχεία των δεδομένων εισόδου στην Επεξεργασία Φυσικής Γλώσσας (NLP), αντιπροσωπεύοντας συνήθως λέξεις, υπολέξεις ή χαρακτήρες. Τα Μεγάλα Γλωσσικά Μοντέλα (LLMs) επεξεργάζονται το κείμενο μετατρέποντάς το σε tokens, τα οποία στη συνέχεια αντιστοιχίζονται σε διανύσματα (embeddings) για περαιτέρω ανάλυση.

### Summary

Ένα token είναι μια διακριτή μονάδα κειμένου ή δεδομένων που λειτουργεί ως το βασικό στοιχείο εισόδου για τα μοντέλα επεξεργασίας φυσικής γλώσσας.

## Key Concepts

- Διακριτή Μονάδα
- Λεξιλόγιο
- Embedding
- Παράθυρο Συγκειμένου

## Use Cases

- Γενέση κειμένου σε LLMs
- Προεπεξεργασία μηχανικής μετάφρασης
- Μορφοποίηση εισόδου για ανάλυση συναισθήματος
