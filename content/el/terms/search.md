## Definition

Η αναζήτηση είναι ένα θεμελιώδες παράδειγμα στην ΤΝ που χρησιμοποιείται για την πλοήγηση σε πολύπλοκους χώρους προβλημάτων, όπως στο παιχνίδι ή στον σχεδιασμό διαδρομών. Αλγόριθμοι όπως ο A*, Minimax ή η Αναζήτηση Δέντρου Monte Carlo αξιολογούν τις πιθανές κινήσεις

### Summary

Οι αλγόριθμοι αναζήτησης εξερευνούν συστηματικά τους χώρους λύσεων για να βέλτιστες ή ικανοποιητικές εξόδους στις εργασίες ΤΝ.

## Key Concepts

- Χώρος Καταστάσεων
- Ευριστικές Μέθοδοι
- Βελτιστοποίηση
- Εύρεση Διαδρομής

## Use Cases

- Στρατηγικές ΤΝ σε παιχνίδια
- Σχεδιασμός διαδρομής στη ρομποτική
- Προβλήματα κορεσμού περιορισμών
