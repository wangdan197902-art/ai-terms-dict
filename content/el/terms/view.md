## Definition

Στη διαχείριση βάσεων δεδομένων, μια προβολή λειτουργεί ως μια αποθηκευμένη ερώτηση SQL που συμπεριφέρεται σαν πίνακας αλλά δεν περιέχει δεδομένα από μόνη της. Παρέχει μια απλοποιημένη ή εξατομικευμένη προοπτική των υποκείμενων δεδομένων, ενισχύοντας την ασφάλεια.

### Summary

Μια εικονική πίνακας σε μια βάση δεδομένων που προκύπτει από μια αποθηκευμένη ερώτηση, παρουσιάζοντας δεδομένα από έναν ή περισσότερους πίνακες χωρίς φυσική αποθήκευση.

## Key Concepts

- Εικονικός Πίνακας
- Ερώτηση SQL
- Αφαίρεση Δεδομένων
- Ασφάλεια

## Use Cases

- Δημιουργία απλοποιημένων αναφορών για μη τεχνικούς χρήστες
- Περιορισμός πρόσβασης σε ευαίσθητες στήλες σε έναν πίνακα
- Τυποποίηση πολύπλοκης λογικής συγχώνευσης (join) μεταξύ εφαρμογών

## Code Example

```python
CREATE VIEW ActiveUsers AS SELECT id, name FROM users WHERE status = 'active';
```
