## Definition

Ο υπολογισμός στο Cloud παρέχει κλιμακώσιμη υποδομή για φορτία εργασίας ΤΝ, επιτρέποντας στους προγραμματιστές να έχουν πρόσβαση σε ισχυρές κάρτες GPU και αποθηκευτικό χώρο χωρίς τη συντήρηση φυσικών κέντρων δεδομένων. Υποστηρίζει διάφορα μοντέλα παροχής υπηρεσιών όπως IaaS, PaaS και SaaS.

### Summary

Το Cloud αναφέρεται σε απομακρυσμένους διακομιστές που φιλοξενούνται στο διαδίκτυο και χρησιμοποιούνται για την αποθήκευση, διαχείριση και επεξεργασία δεδομένων και μοντέλων ΤΝ, αντί για τοπικό υλικό.

## Key Concepts

- Κλιμακωτότητα
- Υποδοχή ως Υπηρεσία (IaaS)
- Απομακρυσμένη Αποθήκευση
- Ελαστικότητα

## Use Cases

- Εκπαίδευση μεγάλων γλωσσικών μοντέλων
- Φιλοξενία API ΤΝ για εφαρμογές ιστού
- Αποθήκευση τεράστιων συνόλων δεδομένων για αναλυτικούς σκοπούς
