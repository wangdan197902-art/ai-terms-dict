## Definition

Τα αυτογνωστικά αναφέρονται στους μηχανισμούς αυτοπαρακολούθησης και αυτοεπισκευής που είναι ενσωματωμένοι σε έξυπνα συστήματα. Επιτρέπει στους πράκτορες ΤΝ να ανιχνεύουν ανωμαλίες, να διαγιγνώσκουν τις αιτίες βλαβών και, ενδεχομένως, να συνεργάζονται για την αποκατάστασή τους.

### Summary

Η ικανότητα ενός συστήματος ΤΝ να αυτοδιαγιγνώσκει την εσωτερική του κατάσταση, τα προβλήματα απόδοσης ή τα σφάλματα χωρίς ανθρώπινη παρέμβαση.

## Key Concepts

- Αυτοπαρακολούθηση
- Διάγνωση Βλαβών
- Αυτονομία
- Υγεία Συστήματος

## Use Cases

- Υποδομές cloud με αυτοθεραπεία
- Ρομποτικά συστήματα που ανιχνεύουν φθορά μηχανικών εξαρτημάτων
- Μοντέλα ΤΝ που εντοπίζουν εσωτερικά μεταβολή στην κατανομή δεδομένων (data drift)
