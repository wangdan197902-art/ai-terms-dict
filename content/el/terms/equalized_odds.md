## Definition

Οι ισοσταθμισμένες πιθανότητες είναι ένας περιορισμός στατιστικής ισότητας που χρησιμοποιείται στην αλγοριθμική δικαιοσύνη για να διασφαλιστεί ότι ένα μοντέλο λειτουργεί рівσο καλά για όλες τις προστατευόμενες ομάδες. Συγκεκριμένα, απαιτεί η πιθανότητα ενός θετικού αποτελέσματος (για πραγματικά θετικά) και ενός ψευδώς θετικού αποτελέσματος να είναι ανεξάρτητη από την ανήκοντα στην ομάδα.

### Summary

Ένα μέτρο δικαιοσύνης που απαιτεί οι ρυθμοί θετικών και ψευδώς θετικών αποτελεσμάτων να είναι ίσοι μεταξύ διαφορετικών δημογραφικών ομάδων.

## Key Concepts

- Αλγοριθμική Δικαιοσύνη
- Ρυθμός Θετικών Αποτελεσμάτων
- Ρυθμός Ψευδώς Θετικών Αποτελεσμάτων
- Μείωση Μεροληψίας

## Use Cases

- Αλγόριθμοι δανεισμού με δικαιοσύνη
- Συστήματα σύστασης για προσλήψεις
- Αξιολόγηση κινδύνου στο ποινικό δίκαιο
