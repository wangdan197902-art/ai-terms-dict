## Definition

Το Code Search Net είναι ένα ολοκληρωμένο σύνολο δεδομένων που δημιουργήθηκε για να προωθήσει την έρευνα στην ανάκτηση κώδικα. Περιέχει εκατομμύρια ζεύγη περιγραφών φυσικής γλώσσας και αντίστοιχων αποσπασμάτων κώδικα Java.

### Summary

Ένα σύνολο δεδομένων αναφοράς σχεδιασμένο για την αξιολόγηση συστημάτων αναζήτησης κώδικα, συνδυάζοντας ερωτήματα φυσικής γλώσσας με σχετικά αποσπάσματα κώδικα.

## Key Concepts

- Ανάκτηση Κώδικα (Code Retrieval)
- Σημασιολογική Αναζήτηση (Semantic Search)
- Κώδικας Java (Java Code)
- Επεξεργασία Φυσικής Γλώσσας (Natural Language Processing)

## Use Cases

- Κατασκευή πρόσθετων για IDE
- Έρευνα στην κατανόηση κώδικα
- Αυτόματη τεκμηρίωση κώδικα
