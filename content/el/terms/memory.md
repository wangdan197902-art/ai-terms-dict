## Definition

Στην τεχνητή νοημοσύνη, η μνήμη αναφέρεται στους μηχανισμούς που επιτρέπουν στα μοντέλα να διατηρούν πληροφορίες πέρα από ένα μόνο βήμα συμπερασmatismou. Αυτό περιλαμβάνει τη βραχυπρόθεσμη λειτουργική μνήμη για το άμεσο πλαίσιο εντός μιας συνομιλίας ή εργασίας.

### Summary

Η ικανότητα ενός συστήματος Τεχνητής Νοημοσύνης να αποθηκεύει, να ανακτά και να χρησιμοποιεί πληροφορίες σε όλες τις αλληλεπιδράσεις, διατηρώντας το πλαίσιο και βελτιώνοντας την απόδοση με τον χρόνο.

## Key Concepts

- Βραχυπρόθεσμη μνήμη
- Μακροπρόθεσμη μνήμη
- Ανάκτηση διανυσμάτων
- Παράθυρο πλαισίου

## Use Cases

- Εξατομικευμένα chatbots
- Αυτόνομοι πράκτορες
- Συστήματα συνεχούς μάθησης
