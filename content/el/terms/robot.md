## Definition

Ένα ρομπότ είναι ένα αυτονομικό ή ημι-αυτόνομο μηχανολογικό σύστημα σχεδιασμένο για την εκτέλεση εργασιών είτε ανεξάρτητα είτε υπό απομακρυσμένο έλεγχο. Συνήθως αποτελείται από αισθητήρες για τη περιβαλλοντική αντίληψη, ενεργοποιητές (actuators) και μονάδες επεξεργασίας δεδομένων.

### Summary

Μια προγραμματιζόμενη μηχανή ικανή να εκτελεί σύνθετες ενέργειες αυτόματα.

## Key Concepts

- Αυτοματοποίηση
- Ενεργοποιητές
- Αισθητήρες
- Αυτονομία

## Use Cases

- Βιομηχανικές γραμμές συναρμολόγησης
- Χειρουργική υποστήριξη
- Οικιακή καθαριότητα
