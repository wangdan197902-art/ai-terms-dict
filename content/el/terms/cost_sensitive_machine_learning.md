## Definition

Η μηχανική μάθηση ευαισθησία στο κόστος επεκτείνει την παραδοσιακή εποπτευόμενη μάθηση αναθέτοντας διαφορετικές ποινές σε διαφορετικούς τύπους σφαλμάτων. Σε πραγματικά σενάρια, τα ψευδώς θετικά και τα ψευδώς αρνητικά αποτελέσματα συχνά έχουν διαφορετικό κόστος.

### Summary

Ένα παράδειγμα μηχανικής μάθησης που ενσωματώνει το κόστος των λανθασμένων ταξινομήσεων στη διαδικασία εκπαίδευσης, βελτιστοποιώντας για οικονομικό αντίκτυπο αντί μόνο για ακρίβεια.

## Key Concepts

- Τροποποίηση Συνάρτειας Απώλειας
- Ανισορροπία Τάξεων
- Κόστος Λανθασμένης Ταξινόμησης
- Στόχος Βελτιστοποίησης

## Use Cases

- Ανίχνευση απάτης στις τράπεζες
- Ιατρικός έλεγχος ασθενειών
- Φιλτράρισμα spam με υψηλό κόστος ψευδώς θετικών αποτελεσμάτων
