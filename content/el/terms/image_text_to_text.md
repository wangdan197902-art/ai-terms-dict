## Definition

Η Αντιστοίχιση Κειμένου Εικόνας σε Κείμενο αναφέρεται σε μοντέλα που επεξεργάζονται οπτικές εισόδους μαζί με λεκτικές ερωτήσεις για να παράγουν συνεκτικές εξόδους φυσικής γλώσσας. Αυτά τα συστήματα, συχνά γνωστά ως Μοντέλα Γλώσσας-Οράσεως (VLMs), συνδυάζουν την κατανόηση της εικόνας με την επεξεργασία γλώσσας.

### Summary

Μια πολυτροπική ικανότητα Τεχνητής Νοημοσύνης που παράγει λεκτικές περιγραφές ή απαντήσεις με βάση εικόνες εισόδου και προαιρετικά κείμενα εντολών.

## Key Concepts

- Πολυτροπική Μάθηση
- Απάντηση σε Οπτικές Ερωτήσεις
- Υποτιτλισμός
- Δια-τροπική Προσοχή

## Use Cases

- Αυτόματος υποτιτλισμός εικόνων για προσβασιμότητα
- Συστήματα απάντησης σε οπτικές ερωτήσεις
- Ανάλυση διαχείρισης περιεχομένου
