## Definition

Αυτή η πρακτική περιλαμβάνει την ανάπτυξη εκπαιδευμένων μοντέλων τεχνητής νοημοσύνης απευθείας σε υλικό, όπως smartphones, αισθητήρες IoT ή ενσωματωμένα συστήματα. Επεξεργαζόμενα τα δεδομένα τοπικά, η εκτίμηση στο Edge μειώνει σημαντικά την καθυστέρηση

### Summary

Η εκτίμηση στο Edge είναι η διαδικασία εκτέλεσης μοντέλων μηχανικής μάθησης τοπικά σε συσκευές τελικών χρηστών αντί σε κεντρικούς διακομιστές cloud.

## Key Concepts

- Μείωση καθυστέρησης
- Διατήρηση απορρήτου
- Ποσοτικοποίηση μοντέλου
- Τοπική επεξεργασία

## Use Cases

- Ανίχνευση αντικειμένων σε πραγματικό χρόνο σε αυτονομά drones
- Αναγνώριση φωνής σε έξυπνες ηχεία
- Προληπτική συντήρηση σε βιομηχανικούς αισθητήρες IoT
