## Definition

Τα μοντέλα open-weight διαφέρουν από την πλήρη open-source ΤΝ επειδή μόνο οι τελικές μαθημένες παράμετροι απελευθερώνονται, όχι απαραίτητα υποδομή ή δεδομένα που χρησιμοποιήθηκαν για τη δημιουργία τους. Αυτό επιτρέπει στους χρήστες να εκτελούν inference

### Summary

Μοντέλα ΤΝ όπου οι εκπαιδευμένες παραμέτροι (βάρη) δημοσιεύονται, αλλά ο κώδικας εκπαίδευσης και το σύνολο δεδομένων μπορεί να παραμείνουν ιδιωτικά.

## Key Concepts

- Βάρη μοντέλου
- Εξαγωγή συμπερασμάτων (Inference)
- Λεπτομερής προσαρμογή (Fine-tuning)
- Μερική διαφάνεια

## Use Cases

- Τοποθέτηση LLMs σε τοπικό υλικό
- Εξατομίκευση προκαταρκτικά εκπαιδευμένων μοντέλων
- Μείωση κόστους API
