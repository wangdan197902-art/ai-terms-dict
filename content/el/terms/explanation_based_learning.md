## Definition

Η EBL συνδυάζει συμβολικό συλλογισμό με τη μηχανική μάθηση για να επιταχύνει τη διαδικασία μάθησης. Αντί να βασίζεται σε μεγάλα σύνολα δεδομένων, λαμβάνει ένα μόνο θετικό παράδειγμα και χρησιμοποιεί μια προϋπάρχουσα θεωρία πεδίου.

### Summary

Η μάθηση βασισμένη σε εξηγήσεις είναι μια μέθοδος μηχανικής μάθησης που χρησιμοποιεί θεωρία πεδίου και ένα μόνο παράδειγμα εκπαίδευσης για να δημιουργήσει μια εξήγηση, η οποία στη συνέχεια γενικεύεται σε έναν νέο κανόνα.

## Key Concepts

- Θεωρία Πεδίου
- Λειτουργικοποίηση
- Μάθηση με Ένα Παράδειγμα
- Συμβολικός Συλλογισμός

## Use Cases

- Απόκτηση δεξιοτήτων στη ρομποτική με περιορισμένα παραδείγματα
- Αυτόματη απόδειξη θεωρημάτων
- Συστήματα διάγνωσης σε εξειδικευμένα τεχνικά πεδία
