## Definition

Στη στατιστική θεωρία μάθησης, μια κλάση μανθάνουσων συναρτήσεων αντιπροσωπεύει τον χώρο υποθέσεων που διατίθεται σε έναν αλγόριθμο. Ορίζει το εύρος των μοτίβων ή απεικονίσεων που το μοντέλο μπορεί δυνητικά να αποκτήσει.

### Summary

Μια κλάση μανθάνουσων συναρτήσεων είναι ένα σύνολο μαθηματικών συναρτήσεων που ορίζονται από μια συγκεκριμένη αρχιτεκτονική μοντέλου και χώρο παραμέτρων, τα οποία ένας αλγόριθμος μάθησης μπορεί να βελτιστοποιήσει.

## Key Concepts

- Χώρος Υποθέσεων
- Χωρητικότητα Μοντέλου
- Διάσταση VC

## Use Cases

- Επιλογή κατάλληλων αρχιτεκτονικών μοντέλων
- Θεωρητική ανάλυση ορίων γενίκευσης
- Κατανόηση των ορίων του μοντέλου
