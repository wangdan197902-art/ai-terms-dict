## Definition

Ένας μηχανισμός προσοχής επιτρέπει σε ένα μοντέλο να ζυγίζει δυναμικά τη σημασία των διαφορετικών στοιχείων μέσα σε μια ακολουθία εισόδου. Αντί να αντιμετωπίζει όλα τα δεδομένα εισόδου ισότιμα, αναθέτει διαφορετικά επίπεδα σημαντικότητας

### Summary

Μια τεχνική που επιτρέπει στα νευρωνικά δίκτυα να εστιάζουν σε συγκεκριμένα μέρη των δεδομένων εισόδου κατά την παραγωγή εξόδων.

## Key Concepts

- Εσωτερική Προσοχή (Self-Attention)
- Διανύσματα Πλαισίου (Context Vectors)
- Βαρυθμημένο Άθροισμα (Weighted Sum)
- Αρχιτεκτονική Transformer

## Use Cases

- Μοντέλα μηχανικής μετάφρασης
- Δημιουργία λεζάντων σε εικόνες
- Σύνοψη κειμένου
