## Definition

Αναπτυγμένο από τη Facebook, το GraphQL παρέχει μια πλήρη και κατανοητή περιγραφή των δεδομένων στο API σας, δίνοντας δύναμη στους πελάτες να ζητούν ακριβώς αυτό που χρειάζονται και τίποτα περισσότερο. Αντικαθιστά πολλά HTTP αιτήματα με ένα ενιαίο σημείο πρόσβασης.

### Summary

Το GraphQL είναι μια γλώσσα ερωτημάτων για APIs και ένας χρόνος εκτέλεσης για την εκπλήρωση αυτών των ερωτημάτων με υπάρχοντα δεδομένα, επιτρέποντας στους πελάτες να ζητούν ακριβώς αυτό που χρειάζονται.

## Key Concepts

- Ορισμός Σχήματος (Schema)
- Γλώσσα Ερωτημάτων
- Ενιαίο Σημείο Πρόσβασης
- Τυπική Ασφάλεια

## Use Cases

- Οπισθοχώρηση σύγχρονων εφαρμογών ιστού
- Λήψη δεδομένων για εφαρμογές κινητών
- Ορχήστρωση μικροϋπηρεσιών
