## Definition

Τα συστήματα πολυ-πρακτόρων αποτελούνται από αρκετούς ανεξάρτητους πράκτορες, ο καθένας από τους οποίους μπορεί να ειδικεύεται σε διαφορετικές εργασίες ή τομείς. Αυτοί οι πράκτορες επικοινωνούν και συντονίζουν τις ενέργειές τους για να επιτύχουν έναν κοινό στόχο.

### Summary

Μια αρχιτεκτονική προσέγγιση όπου πολλοί αυτόνομοι πράκτορες ΤΝ συνεργάζονται, ανταγωνίζονται ή συντονίζονται για την επίλυση πολύπλοκων προβλημάτων που ξεπερνούν τις ατομικές ικανότητες.

## Key Concepts

- Συνεργασία Πρακτόρων
- Εξειδίκευση
- Συντονισμός
- Αποκέντρωση

## Use Cases

- Αυτοματοποιημένη ανάπτυξη λογισμικού
- Περιβάλλοντα σύνθετης προσομοίωσης
- Κατανομημένη λήψη αποφάσεων
