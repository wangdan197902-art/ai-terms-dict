## Definition

Αυτός ο κλάδος περιλαμβάνει τόσο επιθετικές τεχνικές για τη διάσπαση μοντέλων όσο και αμυντικές στρατηγικές για την ενίσχυσή τους. Αφορά την εκπαίδευση μοντέλων σε εχθρικά παραδείγματα για τη βελτίωση της ανθεκτικότητάς τους, μια διαδικασία γνωστή ως εχθρική εκπαίδευση.

### Summary

Η εχθρική μηχανική μάθηση είναι ένας κλάδος που μελετά πώς να καταστεί τα μοντέλα μηχανικής μάθησης ανθεκτικά σε κακόβουλες εισόδους σχεδιασμένες να τα εξαπατούν.

## Key Concepts

- Εχθρική Εκπαίδευση
- Ανθεκτικότητα
- Χάσμα Γενίκευσης
- Ασφάλεια

## Use Cases

- Βελτίωση της ανθεκτικότητας των μοντέλων
- Έλεγχος ασφαλείας συστημάτων Τεχνητής Νοημοσύνης
- Ανάπτυξη μηχανισμών άμυνας
