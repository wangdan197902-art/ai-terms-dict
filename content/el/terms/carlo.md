## Definition

Οι μέθοδοι Μόντε Κάρλο είναι απαραίτητες τεχνικές στην ΤΝ και στα στατιστικά για την προσέγγιση σύνθετων μαθηματικών προβλημάτων που είναι δύσκολο να λυθούν αναλυτικά. Δημιουργώντας χιλιάδες ή εκατομμύρια τυχαίες δειγματοληψίες, επιτρέπουν την εκτίμηση πιθανοτήτων και την προσομοίωση πολύπλοκων συστημάτων.

### Summary

Αναφέρεται στις μεθόδους Μόντε Κάρλο, μια κατηγορία υπολογιστικών αλγορίθμων που βασίζονται σε επαναλαμβανόμενο τυχαίο δειγματοληπτικό για την επίτευξη αριθμητικών αποτελεσμάτων.

## Key Concepts

- Τυχαίο Δειγματοληπτικό
- Στατιστική Προσέγγιση
- Προσομοίωση
- Εκτίμηση Πιθανότητας

## Use Cases

- Εκτίμηση της τιμής μιας κατάστασης στην ενισχυτική μάθηση μέσω προσομοίωσης.
- Διεξαγωγή βαυσιανής επαγωγής οπίσθιας κατανομής χρησιμοποιώντας Αλυσίδα Μάρκοβ Μόντε Κάρλο (MCMC).
- Υπολογισμός ολοκληρωμάτων σε χώρους υψηλής διάστασης για πιθανοτικά μοντέλα.

## Code Example

```python
import numpy as np
# Monte Carlo estimation of Pi
def estimate_pi(samples):
    points = np.random.uniform(-1, 1, size=(samples, 2))
    inside = np.sum(points[:, 0]**2 + points[:, 1]**2 <= 1)
    return 4 * inside / samples
```
