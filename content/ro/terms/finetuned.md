## Definition

Finetunarea se referă la tehnica de a lua un model deja antrenat pe un set de date mare și generalist și de a continua antrenarea acestuia pe un set de date mai mic, specific unui domeniu. Acest lucru permite modelului să se adapteze.

### Summary

Procesul de antrenare suplimentară a unui model pre-antrenat pe un set de date specific pentru a-l adapta la o sarcină particulară.

## Key Concepts

- Învățare prin transfer
- Ajustarea parametrilor
- Adaptarea la domeniu
- Modele pre-antrenate

## Use Cases

- Clasificarea personalizată a imaginilor
- Analiza sentimentelor specializată
- Asistență în diagnosticul medical
