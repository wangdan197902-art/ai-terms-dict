## Definition

Un agent inteligent este un sistem capabil să perceapă împrejurimile prin senzori sau intrări de date, să proceseze aceste informații folosind algoritmi de raționament și să acționeze asupra mediului prin actuatori

### Summary

O entitate software autonomă care percepe mediul înconjurător, raționează asupra acțiunilor și execută sarcini pentru a atinge obiective specifice.

## Key Concepts

- Autonomie
- Bucla percepție-acțiune
- Orientare către obiective
- Reactivitate

## Use Cases

- Boți de tranzacționare autonomi care execută strategii de piață
- Automatizarea proceselor robotice care gestionează fluxuri de lucru complexe
- Asistenți personali care gestionează calendarul și sarcinile de e-mail
