## Definition

Setul de date Yahoo Answers Topics este un subset al arhivei mai mari Yahoo Answers, concentrându-se pe întrebări și răspunsuri organizate în categorii distincte de subiecte. Este utilizat frecvent pentru clasificarea textului și evaluarea similarității semantice.

### Summary

Un set de date derivat din Yahoo Answers, conținând întrebări și răspunsuri generate de utilizatori, categorizate în subiecte specifice pentru potrivirea semantică și clasificare.

## Key Concepts

- Clasificarea textului
- Similaritate semantică
- Conținut generat de utilizatori

## Use Cases

- Antrenarea modelelor de clasificare a întrebărilor
- Benchmarking-ul similarității textuale semantice
- Analiza patternurilor de limbaj informal
