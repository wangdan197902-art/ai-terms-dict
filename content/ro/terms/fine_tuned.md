## Definition

Ajustarea fină implică preluarea unui model care a fost deja antrenat pe un set de date mare și general și continuarea antrenamentului acestuia pe un set de date mai mic, specific sarcinii. Această tehnică exploatează caracteristicile generale

### Summary

Procesul de antrenare suplimentară a unui model pre-antrenat pe un set de date specific pentru a-l adapta la o anumită sarcină downstream.

## Key Concepts

- învățare prin transfer
- actualizarea greutăților
- specific sarcinii

## Use Cases

- Adaptarea LLM-urilor pentru revizuirea documentelor juridice
- Personalizarea modelelor de viziune pentru detectarea defectelor industriale
- Specializarea recunoașterii vocale pentru accente specifice
