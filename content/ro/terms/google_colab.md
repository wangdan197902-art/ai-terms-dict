## Definition

Google Colaboratory, cunoscut comun sub numele de Colab, este un serviciu găzduit de notebook Jupyter care nu necesită configurare și oferă acces gratuit la resurse de calcul, inclusiv procesoare grafice (GPU-uri) și alte hardware-uri accelerate.

### Summary

Un mediu de notebook Jupyter bazat pe cloud care permite utilizatorilor să scrie și să execute cod Python prin intermediul unui browser, oferind acces gratuit la GPU-uri.

## Key Concepts

- Cloud Computing
- Notebook Jupyter
- Accelerare GPU
- Programare Python

## Use Cases

- Prototipare în Învățarea Automată
- Educație în Știința Datelor
- Programare Colaborativă
