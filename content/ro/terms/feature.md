## Definition

În învățarea automată, o caracteristică este un atribut sau o variabilă distinctă care descrie un exemplu într-un set de date. Caracteristicile pot fi numerice, categorice sau textuale și servesc ca intrări fundamentale.

### Summary

O proprietate sau o caracteristică măsurabilă individuală a unui fenomen observat, servind ca date de intrare pentru modelele de învățare automată.

## Key Concepts

- Variabilă de Intrare
- Atribut
- Coloană Set de Date
- Intrare ML

## Use Cases

- Intrări pentru modelare predictivă
- Analiza și vizualizarea datelor
- Procese de selecție a caracteristicilor
