## Definition

API-urile REST permit comunicarea între clienți și servere prin utilizarea operațiilor fără stare peste protocoale HTTP precum GET, POST, PUT și DELETE. Ele structurează resursele ca URI-uri și utilizează formate standard de serializare, cum ar fi JSON, pentru schimbul de date.

### Summary

Un API de Transfer de Stare Reprezentatională este un stil arhitectural pentru proiectarea aplicațiilor în rețea care se bazează pe metode HTTP standard.

## Key Concepts

- Fără stare (Statelessness)
- Metode HTTP
- URI-uri bazate pe resurse
- Serializare JSON

## Use Cases

- Comunicare între microservicii
- Integrarea backend-ului pentru aplicații mobile
- Acces la date terțe părți
