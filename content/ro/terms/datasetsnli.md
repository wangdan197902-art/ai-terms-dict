## Definition

SNLI este un set de date de referință care conține peste 500.000 de perechi de propoziții etichetate, annotate cu trei clase: implicare, contradicție și neutru. A fost creat pentru a avansa cercetarea în inferența lingvistică naturală.

### Summary

Corpusul Stanford Natural Language Inference, un set de date mare de propoziții în engleză, asociate cu etichete de implicare textuală scrise de oameni.

## Key Concepts

- Inferența lingvistică naturală
- Implicare textuală
- Perechi de propoziții
- Set de date de referință

## Use Cases

- Evaluarea modelelor de similitudine semantică
- Antrenarea clasificatoarelor NLI
- Cercetarea raționamentului logic în procesarea limbajului natural
