## Definition

Un benchmark servește ca punct de referință standardizat pentru compararea capacităților diferitelor modele sau algoritmi de IA. Implică de obicei un set de date curat și metrici de evaluare specifice, cum ar fi precizia sau recall-ul.

### Summary

Prescurtare pentru benchmark, un set standard de teste sau o metrică utilizată pentru a evalua performanța modelelor IA.

## Key Concepts

- Metrici de evaluare
- Testare standardizată
- Compararea performanței
- Curatarea seturilor de date

## Use Cases

- Benchmark GLUE pentru PNL
- ImageNet pentru vizionare pe calculator
- Selecția modelelor în producție
