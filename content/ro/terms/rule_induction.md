## Definition

Inducerea regulilor este o metodă de învățare automată simbolică care derivă reguli de tipul „dacă-atunci” direct din date. Spre deosebire de rețelele neuronale, care produc ponderi opace, inducerea regulilor oferă modele interpretabile.

### Summary

O tehnică de învățare automată care generează automat un set de reguli logice din date de antrenament etichetate pentru a clasifica noi instanțe.

## Key Concepts

- Reguli de tipul dacă-atunci
- Liste de decizie
- Interpretabilitate
- Învățare simbolică

## Use Cases

- Ghiduri de diagnostic medical
- Criterii de aprobare a creditelor
- Detectarea defectelor în manufactură
