## Definition

ELI5 (Explain Like I'm Five) este un set de date derivat din comunitatea Reddit cu același nume. Este format din întrebări trimise de utilizatori, împreună cu răspunsuri detaliate și simplificate oferite de comunitate, menite să explice concepte complexe într-un mod accesibil.

### Summary

Un set de date la scară largă de întrebări și răspunsuri formatate ca 'Explică-mi ca și cum aș avea cinci ani', axat pe explicații detaliate.

## Key Concepts

- Răspuns la întrebări
- Generare pe termen lung
- Date Reddit
- Generarea explicațiilor

## Use Cases

- Antrenarea roboților de răspuns la întrebări (QA bots)
- Generarea de conținut educațional
- Studiul dinamicii conversaționale
