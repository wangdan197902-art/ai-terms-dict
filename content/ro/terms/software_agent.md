## Definition

Un agent software este o entitate autonomă capabilă să perceapă mediul său, să raționeze și să acționeze pentru a atinge obiective specifice. Acești agenți pot opera independent, se pot adapta la schimbări și pot colabora

### Summary

Un program de calculator care execută sarcini în numele utilizatorilor sau al altor programe, cu un anumit grad de autonomie.

## Key Concepts

- Autonomie
- Ciclul Percepție-Acțiune
- Comportament orientat către obiective
- Reactivitate

- Boți de serviciu clienți automatizați
