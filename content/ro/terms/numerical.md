## Definition

În contextul inteligenței artificiale și al științei datelor, numeric se referă la tipuri de date sau metode care implică valori cantitative, cum ar fi numere întregi, numere în virgulă mobilă și zecimale. Spre deosebire de datele categorice sau textuale, datele numerice permit operațiuni matematice directe.

### Summary

Referitor la, implicând sau utilizând numere sau calcule matematice, în loc de logică simbolică sau text.

## Key Concepts

- Numere în virgulă mobilă
- Numere întregi
- Tipuri de date
- Date cantitative

## Use Cases

- Modelare statistică
- Prognozarea financiară
- Procesarea datelor de la senzori

## Code Example

```python
import numpy as np
arr = np.array([1.5, 2.7, 3.9])
mean_val = np.mean(arr)
print(mean_val)
```
