## Definition

Mamba reprezintă o avansare semnificativă în modelarea secvențelor prin introducerea unui model de spațiu de stare selectiv, conștient de hardware (SSM). Spre deosebire de transformatoarele tradiționale, care scalază patrat cu lungimea secvenței, Mamba permite procesarea eficientă a datelor pe termen lung.

### Summary

Mamba este un model de secvențe bazat pe spații de stare care oferă inferență în timp liniar, menținând totodată performanța transformatoarelor în sarcinile cu context lung.

## Key Concepts

- Modele de Spațiu de Stare Selective
- Complexitate Liniară
- Arhitectură Conștientă de Hardware
- Procesarea Contextului Lung

## Use Cases

- Sumarizarea documentelor lungi
- Analiza secvențelor genomice
- Previziunea seriei temporale cu frecvență ridicată
