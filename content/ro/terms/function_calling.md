## Definition

Apelarea de funcții permite modelelor lingvistice mari să interacționeze cu instrumente externe și API-uri generând ieșiri structurate, cum ar fi obiecte JSON, care specifică ce funcție trebuie executată și ce argumente trebuie transmise.

### Summary

Un mecanism care permite modelelor lingvistice mari (LLM) să genereze date structurate care declanșează funcții software specifice.

## Key Concepts

- Ieșire structurată
- Integrare API
- Utilizarea instrumentelor
- Schema JSON

## Use Cases

- Rezervarea zborurilor prin intermediul API-urilor de călătorii
- Executarea unor calcule matematice complexe
- Interogarea înregistrărilor din baze de date
