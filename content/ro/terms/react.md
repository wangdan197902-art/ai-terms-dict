## Definition

Framework-ul ReAct permite modelelor lingvistice mari (LLM) să genereze atât urme de raționament, cât și acțiuni specifice sarcinii într-o manieră intercalată. Prin simularea proceselor de gândire similare celor umane, acesta permite modelelor să interacționeze cu mediile externe și să corecteze erorile pe măsură ce avansează.

### Summary

ReAct este un paradigmă de promptare care combină raționamentul și acțiunea pentru a îmbunătăți performanța modelelor lingvistice mari pe sarcini complexe.

## Key Concepts

- Raționament intercalat
- Generarea acțiunilor
- Integrarea observațiilor
- Lanțul de gândire (Chain-of-Thought)

## Use Cases

- Sisteme complexe de întrebări și răspunsuri
- Agenți autonomi
- Fluxuri de lucru de verificare a faptelor
