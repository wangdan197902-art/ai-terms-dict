## Definition

Acest metric cuantifică cât de bine un set de categorii permite predicția valorilor atributelor din interiorul acestora. Echilibrează dimensiunea categoriilor cu omogenitatea conținutului lor, oferind o metrică obiectivă pentru calitatea împărțirii datelor.

### Summary

Utilitatea categoriei este o măsurătoare matematică utilizată pentru a evalua eficacitatea unui scheme de categorisire, bazată pe câștigul de informație pe care îl oferă privind valorile atributelor.

## Key Concepts

- câștig de informație
- evaluarea clusterizării
- acuratețe predictivă
- învățarea conceptelor

## Use Cases

- Evaluarea calității clusterizării
- Studii de achiziție a conceptelor
- Optimizarea selecției caracteristicilor
