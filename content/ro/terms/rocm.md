## Definition

ROCm (Radeon Open Compute) este un stack de drivere și software dezvoltat de AMD pentru a permite calculul de înaltă performanță pe GPU-urile AMD. Oferă biblioteci, compilatoare și instrumente necesare dezvoltării aplicațiilor paralele.

### Summary

Platforma software open-source a AMD pentru calculul GPU, oferind alternative la ecosistemul CUDA al NVIDIA.

## Key Concepts

- Calcul GPU
- Software Open Source
- Procesare Paralelă
- Accelerare Hardware

## Use Cases

- Antrenarea modelelor AI pe GPU-uri AMD
- Simulări științifice
- Cluster de calcul de înaltă performanță
