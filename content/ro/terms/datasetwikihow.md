## Definition

Setul de date WikiHow constă în aproximativ 60.000 de articole de tip „cum se face” colectate de pe site-ul web WikiHow. Este utilizat pe scară largă în cercetarea procesării limbajului natural pentru sarcini precum rezumarea abstractivă a textului și extragerea pașilor procedurali.

### Summary

Un set de date la scară largă compus din articole de tip „cum se face” de pe WikiHow, utilizat în principal pentru sarcini de rezumare a textului și generare de instrucțiuni.

## Key Concepts

- Rezumarea textului
- Text procedural
- Extragerea instrucțiunilor

## Use Cases

- Antrenarea modelelor de rezumare abstractivă
- Generarea instrucțiunilor pas cu pas
- Înțelegerea structurilor limbajului procedural
