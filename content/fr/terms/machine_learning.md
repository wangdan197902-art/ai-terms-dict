## Definition

L'apprentissage automatique (ML) permet aux ordinateurs d'apprendre des motifs à partir de données historiques et de prendre des décisions ou de faire des prédictions sur de nouvelles données non vues. Il englobe diverses techniques, notamment l'apprentissage supervisé, l'apprentissage non

### Summary

Un sous-ensemble de l'IA axé sur la création de systèmes qui apprennent à partir de données pour améliorer leurs performances sans programmation explicite.

## Key Concepts

- Apprentissage supervisé
- Apprentissage non supervisé
- Entraînement du modèle
- Prédiction

## Use Cases

- Détection de fraude
- Moteurs de recommandation
- Reconnaissance vocale
