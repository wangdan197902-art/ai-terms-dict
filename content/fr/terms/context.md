## Definition

En traitement automatique du langage naturel, le contexte est crucial pour résoudre les ambiguïtés, comme comprendre les pronoms ou les idiomes en fonction des phrases précédentes. Les architectures modernes comme les transformateurs utilisent des mécanismes d'attention pour capturer ce contexte.

### Summary

Le contexte désigne les informations environnantes ou l'environnement qui aident un modèle d'IA à interpréter précisément les données d'entrée et à générer des réponses pertinentes.

## Key Concepts

- Mécanisme d'attention
- Modélisation de séquence
- Résolution d'ambiguïté
- Taille de fenêtre

## Use Cases

- Chatbots conversationnels
- Résumé de documents
- Assistants de complétion de code
