## Definition

Le Protocole de Contexte de Modèle (MCP) est une norme ouverte qui permet aux applications d'IA de se connecter à diverses sources de données, telles que des bases de données, des API et des systèmes de fichiers, de manière uniforme. Il abstrait la complexité sous-jacente.

### Summary

Un cadre normalisé conçu pour faciliter une communication sécurisée et efficace entre les modèles d'IA et des sources de données ou outils externes.

## Key Concepts

- Interopérabilité
- Normalisation
- Intégration des données
- Invocation d'outils

## Use Cases

- Connecter les LLM aux données d'entreprise
- Construire des assistants IA portables
- Intégration sécurisée d'API
