## Definition

Ces systèmes, y compris les grands modèles de langage et les modèles de diffusion, ne se contentent pas de récupérer des informations existantes mais synthétisent des sorties nouvelles. Ils apprennent la structure sous-jacente et le style de leurs données d'entraînement...

### Summary

L'IA générative est un type d'intelligence artificielle capable de créer de nouveaux contenus, tels que du texte, des images, de l'audio et du code, sur la base de motifs appris à partir de données d'entraînement.

## Key Concepts

- Synthèse de contenu
- Apprentissage de motifs
- LLM (Modèles de langage de grande taille)
- Modèles de diffusion

## Use Cases

- Génération de texte
- Création d'images
- Complétion de code
