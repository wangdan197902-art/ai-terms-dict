## Definition

Le terme « driven » est couramment utilisé comme suffixe pour indiquer la force ou le mécanisme principal derrière une approche IA. Par exemple, « data-driven » (piloté par les données) implique que les décisions sont prises sur la base de modèles statistiques.

### Summary

Le terme « piloté par » décrit les systèmes ou méthodologies IA où la prise de décision ou l'architecture du modèle est principalement guidée par un type d'entrée spécifique, tel que les données ou les objectifs.

## Key Concepts

- Approche centrée sur les données
- Optimisation des objectifs
- Identification du paradigme
- Apprentissage par renforcement

## Use Cases

- Description des flux de travail d'IA centrés sur les données
- Définition des agents d'apprentissage par renforcement comme étant orientés vers un objectif
- Classification des systèmes hybrides comme étant pilotés par les connaissances ou les données
