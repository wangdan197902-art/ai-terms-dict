## Definition

Le terme « généré » décrit la sortie produite par des modèles d'IA générative, tels que du texte, des images, de l'audio ou du code. Contrairement aux systèmes basés sur la récupération qui extraient des données existantes, les modèles génératifs synthétisent de nouveaux

### Summary

Contenu créé par un modèle d'IA plutôt que récupéré directement à partir d'un ensemble de données statique.

## Key Concepts

- IA générative
- Synthèse
- Création
- Sortie

## Use Cases

- Résumé de texte
- Synthèse d'images
- Génération de code
