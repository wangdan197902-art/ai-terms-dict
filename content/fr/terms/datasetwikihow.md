## Definition

Le jeu de données WikiHow comprend environ 60 000 articles explicatifs collectés depuis le site Web WikiHow. Il est largement utilisé dans la recherche en traitement automatique du langage naturel (NLP) pour des tâches telles que la résumation abstraite de texte.

### Summary

Un jeu de données à grande échelle composé d'articles explicatifs de WikiHow, utilisé principalement pour la résumation de texte et la génération d'instructions.

## Key Concepts

- Résumation de texte
- Texte procédural
- Extraction d'instructions

## Use Cases

- Entraînement de modèles de résumation abstraite
- Génération d'instructions étape par étape
- Compréhension des structures linguistiques procédurales
