## Definition

Code Search Net est un jeu de données complet créé pour faire avancer la recherche dans le domaine de la récupération de code. Il contient des millions de paires de descriptions en langage naturel et d'extraits de code Java correspondants.

### Summary

Un jeu de données de référence conçu pour évaluer les systèmes de recherche de code en associant des requêtes en langage naturel à des extraits de code pertinents.

## Key Concepts

- Récupération de code
- Recherche sémantique
- Code Java
- Traitement du langage naturel

## Use Cases

- Développement de plugins IDE
- Recherche sur la compréhension du code
- Documentation automatique du code
