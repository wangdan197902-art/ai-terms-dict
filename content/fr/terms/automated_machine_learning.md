## Definition

L'AutoML (Apprentissage Automatique Automatisé) rationalise le développement de modèles ML en automatisant des tâches telles que le prétraitement des données, l'ingénierie des fonctionnalités, la sélection de modèles et le réglage des hyperparamètres. Il permet...

### Summary

Une méthodologie qui automatise le processus complet d'application de l'apprentissage automatique à des problèmes réels, réduisant ainsi l'effort manuel.

## Key Concepts

- Réglage des hyperparamètres
- Ingénierie des fonctionnalités
- Sélection de modèles
- Démocratisation

## Use Cases

- Prototypage rapide pour les analystes d'affaires
- Optimisation des pipelines de production à grande échelle
- Comparaison automatique de plusieurs algorithmes

## Code Example

```python
from auto_ml import Predictor
predictor = Predictor(type_of_estimator='classifier')
predictor.fit(dataframe, target='label')
```
