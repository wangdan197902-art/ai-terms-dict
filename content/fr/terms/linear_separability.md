## Definition

La séparabilité linéaire désigne la condition géométrique selon laquelle les points de données appartenant à différentes classes peuvent être entièrement séparés par une frontière linéaire, telle qu'une ligne dans un espace en 2D ou un hyperplan dans un espace de dimension supérieure.

### Summary

La propriété d'un ensemble de données où deux classes peuvent être parfaitement divisées par une seule ligne droite ou un hyperplan.
