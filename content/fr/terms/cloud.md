## Definition

Le cloud computing fournit une infrastructure évolutive pour les charges de travail de l'IA, permettant aux développeurs d'accéder à des GPU puissants et à du stockage sans maintenir de centres de données physiques. Il prend en charge divers modèles de services tels que l'IaaS, le PaaS et le SaaS.

### Summary

Le cloud désigne des serveurs distants hébergés sur Internet utilisés pour stocker, gérer et traiter des données et des modèles d'IA, plutôt que d'utiliser du matériel local.

## Key Concepts

- Évolutivité
- Infrastructure as a Service
- Stockage distant
- Élasticité

## Use Cases

- Entraînement de grands modèles de langage
- Hébergement d'APIs IA pour les applications web
- Stockage de vastes ensembles de données pour l'analyse
