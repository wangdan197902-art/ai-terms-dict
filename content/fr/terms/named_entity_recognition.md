## Definition

La reconnaissance d'entités nommées (NER) est une sous-tâche de l'extraction d'informations qui localise et classe les entités nommées dans le texte selon des catégories prédéfinies telles que les noms de personnes, les organisations, les lieux, les dates, etc.

### Summary

Une tâche de traitement du langage naturel qui identifie et classe les entités d'information clés dans des catégories prédéfinies.

## Key Concepts

- Typage d'entité
- Classification de jetons
- Étiquetage de séquence
- Architectures BiLSTM-CRF

## Use Cases

- Analyse de CV
- Détection d'intention dans le support client
- Analyse de dossiers médicaux
