## Definition

Publiée en août 2024, Llama 3.1 élargit la famille Llama pour inclure un modèle massif de 405 milliards de paramètres aux côtés des variantes plus petites de 8 et 70 milliards. Une fonctionnalité majeure est la fenêtre de contexte étendue.

### Summary

Llama 3.1 est la mise à jour d'août 2024 de Meta, comprenant des modèles de 8, 70 et 405 milliards de paramètres, une fenêtre de contexte étendue de 128k tokens et des capacités natives d'utilisation d'outils.

## Key Concepts

- Fenêtre de contexte longue
- Appel de fonctions
- Utilisation d'outils
- 405 milliards de paramètres

## Use Cases

- Analyse de longs documents
- Agents intégrés via API
- Synthèse de code avancée
