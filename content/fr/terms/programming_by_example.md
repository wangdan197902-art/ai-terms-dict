## Definition

La Programmation par Exemple (PBE) est un paradigme de synthèse de programme où les développeurs spécifient le comportement souhaité par des paires concrètes entrée-sortie plutôt qu'en écrivant du code explicite. Le système d'IA analyse ces exemples pour inférer la logique sous-jacente et générer un programme fonctionnel capable de reproduire cette logique sur de nouvelles données.

### Summary

Une technique d'IA où les programmes sont synthétisés automatiquement en inférant des règles à partir d'exemples entrée-sortie fournis par les utilisateurs.

## Key Concepts

- Synthèse de programme
- Exemples entrée-sortie
- Inférence inductive
- Low-code/No-code

## Use Cases

- Génération de formules Excel à partir de données d'échantillon
- Scripts de transformation de texte et de nettoyage de données
- Automatisation des interactions répétitives avec l'interface utilisateur
