## Definition

Un référentiel sert de point de référence standardisé pour comparer les capacités de différents modèles ou algorithmes d'IA. Il implique généralement un ensemble de données soigneusement sélectionné et des métriques d'évaluation spécifiques telles que la précision ou le rappel.

### Summary

Abréviation de benchmark, un ensemble de tests standard ou une métrique utilisée pour évaluer les performances des modèles d'IA.

## Key Concepts

- Métriques d'évaluation
- Tests standardisés
- Comparaison des performances
- Sélection d'ensembles de données

## Use Cases

- Référentiel GLUE pour le TALN
- ImageNet pour la vision par ordinateur
- Sélection de modèles en production
