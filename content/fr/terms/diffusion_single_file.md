## Definition

Le Fichier Unique de Diffusion fait référence à une stratégie d'emballage pour les modèles d'apprentissage automatique, en particulier les modèles de diffusion, où tout l'artefact du modèle — y compris les poids binaires, les hyperparamètres et l'architecture du modèle — est consolidé.

### Summary

Un format de distribution pour les modèles de diffusion où tous les poids du modèle, les configurations et parfois même le code d'inférence sont regroupés dans un seul fichier pour faciliter la portabilité.

## Key Concepts

- Portabilité des Modèles
- Distribution en Fichier Unique
- Sérialisation des Poids
- Simplification du Déploiement

## Use Cases

- Partage de modèles sur des plateformes communautaires comme Civitai
- Déploiement d'applications légères sans dépendances complexes
- Archivage des versions de modèles pour la reproductibilité
