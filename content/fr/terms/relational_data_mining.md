## Definition

La fouille de données relationnelles se concentre sur l'extraction d'informations utiles à partir de bases de données où les données sont organisées en plusieurs tables liées, plutôt qu'en une seule table plate. Elle exploite les relations entre ces différentes entités.

### Summary

Le processus de découverte de motifs et de connaissances à partir de données composées de plusieurs tables ou entités interdépendantes.

## Key Concepts

- Entité-relation
- Apprentissage multi-relationnel
- Fouille de graphes
- Intégration de bases de données

## Use Cases

- Analyse des réseaux sociaux
- Détection de fraude dans les transactions financières
- Découverte de médicaments et bio-informatique
