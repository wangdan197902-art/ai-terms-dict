## Definition

En IA, « de haut niveau » désigne les abstractions qui simplifient les processus complexes. Les langages de haut niveau (comme Python) ou les API permettent aux développeurs de construire des modèles sans gérer manuellement la mémoire ou les spécificités matérielles. Cela augmente la productivité et réduit la courbe d'apprentissage, au prix d'une abstraction des détails d'exécution fine.

### Summary

Désigne des représentations abstraites ou des interfaces de programmation qui masquent les détails d'implémentation de bas niveau à l'utilisateur.

## Key Concepts

- Abstraction
- API (Interfaces de programmation)
- Masquage de la complexité
- Productivité

## Use Cases

- Utilisation des API de haut niveau de PyTorch/TensorFlow
- Ingénierie des caractéristiques dans les pipelines ML
- Langages spécifiques au domaine pour l'IA
