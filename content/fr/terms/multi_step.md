## Definition

Les méthodes multi-étapes consistent à décomposer une requête ou une tâche complexe en étapes plus petites et exécutables. Cette approche est cruciale pour les tâches de raisonnement, telles que la résolution de problèmes mathématiques ou la génération de code.

### Summary

Une stratégie de résolution de problèmes nécessitant la réalisation d'une séquence d'opérations logiques ou de calculs pour parvenir à une solution finale.

## Key Concepts

- raisonnement
- étapes intermédiaires
- chaîne de pensée
- décomposition

## Use Cases

- Problèmes mathématiques complexes
- Génération de code avec dépendances
- Tâches de déduction logique
