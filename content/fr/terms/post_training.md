## Definition

Le post-entraînement est une étape critique du cycle de vie de l'apprentissage automatique qui intervient après l'entraînement initial (pré-entraînement) du modèle sur des données générales à grande échelle. Au cours de cette phase, le modèle subit un ajustement

### Summary

Le post-entraînement désigne la phase de raffinement d'un modèle pré-entraîné sur des ensembles de données spécifiques pour l'adapter à des tâches particulières ou optimiser ses performances.

## Key Concepts

- Affinage (Fine-tuning)
- RLHF
- Quantification
- Adaptation

## Use Cases

- Alignement des LLM avec les préférences humaines
- Optimisation de la taille du modèle pour les appareils edge
- Spécialisation des modèles pour le diagnostic médical
