## Definition

Dans le contexte de l'IA, 'source' désigne généralement la provenance des ensembles de données d'entraînement, des bibliothèques open source ou des poids de modèles pré-entraînés. Le suivi des sources est crucial pour la reproductibilité et la conformité aux licences.

### Summary

Désigne l'origine des données, du code ou des modèles utilisés dans le développement et le déploiement de l'IA.

## Key Concepts

- Provenance des données
- Open Source (Logiciel libre)
- Flux d'entrée
- Reproductibilité

## Use Cases

- Audit des données d'entraînement pour détecter les biais
- Gestion du versionnage des modèles
- Citation des dépendances open source
