## Definition

En apprentissage automatique, une fonctionnalité (ou caractéristique) est un attribut ou une variable distincte qui décrit une instance au sein d'un jeu de données. Les fonctionnalités peuvent être numériques, catégorielles ou textuelles, et elles constituent les entrées fondamentales.

### Summary

Une propriété ou caractéristique mesurable individuelle d'un phénomène observé, servant de données d'entrée pour les modèles d'apprentissage automatique.

## Key Concepts

- Variable d'Entrée
- Attribut
- Colonne de Jeu de Données
- Entrée ML

## Use Cases

- Entrées pour la modélisation prédictive
- Analyse et visualisation de données
- Processus de sélection de fonctionnalités
