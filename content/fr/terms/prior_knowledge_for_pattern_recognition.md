## Definition

Les connaissances antérieures désignent les insights spécifiques au domaine, les contraintes ou les données historiques incorporées dans les algorithmes avant le début de l'entraînement. Cela aide à orienter le modèle vers des solutions plausibles, réduisant ainsi le risque de surapprentissage.

### Summary

Informations ou hypothèses existantes intégrées dans les modèles d'apprentissage automatique pour améliorer la précision de l'identification des motifs.

## Key Concepts

- Biais inductif
- Connaissance du domaine
- Régularisation
- Généralisation

## Use Cases

- Diagnostic d'images médicales avec contraintes anatomiques
- Reconnaissance vocale avec règles linguistiques
- Détection d'objets avec a priori sur la forme
