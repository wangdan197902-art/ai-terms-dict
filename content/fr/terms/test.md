## Definition

L'ensemble de test est une partie des données conservée pendant le processus d'entraînement pour évaluer la capacité de généralisation finale du modèle. Contrairement aux ensembles de validation utilisés pour le réglage des hyperparamètres, l'ensemble de test fournit une estimation non biaisée des performances du modèle sur de nouvelles données,

### Summary

Le test fait référence à la phase d'évaluation où un modèle d'IA entraîné est évalué sur des données invisibles pour mesurer ses performances.

## Key Concepts

- Généralisation
- Données invisibles
- Évaluation du modèle
- Prévention du surapprentissage

## Use Cases

- Mesure de la précision d'un modèle de classification
- Comparaison de différentes versions d'algorithmes
- Validation finale avant le déploiement
