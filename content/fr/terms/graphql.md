## Definition

Développé par Facebook, GraphQL fournit une description complète et compréhensible des données de votre API, donnant aux clients le pouvoir de demander exactement ce dont ils ont besoin et rien de plus. Il remplace les multiples appels d'API traditionnels par un point d'accès unique.

### Summary

GraphQL est un langage de requête pour les API et un temps d'exécution pour satisfaire ces requêtes avec des données existantes, permettant aux clients de demander exactement ce dont ils ont besoin.

## Key Concepts

- Définition du schéma
- Langage de requête
- Point d'accès unique
- Sécurité des types

## Use Cases

- Backends d'applications web modernes
- Récupération de données pour applications mobiles
- Orchestration de microservices
