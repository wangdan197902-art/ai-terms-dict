## Definition

Elle étend la logique traditionnelle pour tenir compte de l'agence, permettant aux systèmes de représenter les croyances, les désirs et les intentions (modèles BDI). Cette logique permet aux agents de planifier des actions dynamiquement en fonction des changements de l'environnement.

### Summary

La logique agentive fait référence aux cadres de raisonnement formel utilisés pour modéliser les intentions, les objectifs et les processus de prise de décision des agents autonomes.

## Key Concepts

- Modèle BDI
- Comportement orienté vers les objectifs
- Intentionnalité
- Raisonnement dans l'incertitude

## Use Cases

- Conception de robots autonomes
- Création d'assistants intelligents
- Simulation d'interactions sociales
