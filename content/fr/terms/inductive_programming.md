## Definition

La Programmation Inductive, souvent appelée Synthèse de Programmes, consiste à créer du code logiciel basé sur des spécifications fournies sous forme de paires entrée-sortie plutôt que d'instructions explicites. Le système infère la logique sous-jacente nécessaire pour transformer les entrées en sorties souhaitées.

### Summary

Un domaine de la synthèse de programmes qui génère automatiquement des programmes informatiques à partir d'exemples entrée-sortie.

## Key Concepts

- Synthèse de programmes
- Exemples entrée-sortie
- Codage automatisé
- Inférence logique

## Use Cases

- Outils d'achèvement de code
- Génération de scripts à partir de données
- Aides à la programmation éducative
