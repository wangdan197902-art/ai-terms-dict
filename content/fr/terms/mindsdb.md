## Definition

MindsDB agit comme un pont entre les bases de données relationnelles traditionnelles et les workflows modernes d'apprentissage automatique. Il permet aux utilisateurs de créer des modèles prédictifs à l'aide de requêtes SQL standard, éliminant ainsi le besoin de déplacer les données vers des environnements d'IA distincts.

### Summary

MindsDB est une plateforme open source qui permet aux développeurs de construire, d'entraîner et de déployer des modèles d'apprentissage automatique directement dans leurs bases de données existantes en utilisant SQL.

## Key Concepts

- ML basé sur SQL
- Intégration de base de données
- AutoML
- Modélisation prédictive

## Use Cases

- Détection de fraude en temps réel dans les bases de données
- Prédiction de l'attrition client
- Rapports automatisés avec prévisions
