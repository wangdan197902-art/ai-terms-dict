## Definition

L'apprentissage semi-supervisé est un paradigme d'entraînement hybride qui utilise une petite quantité de données étiquetées aux côtés d'un grand volume de données non étiquetées. L'hypothèse centrale est que la structure des données non étiquetées permet d'améliorer l'apprentissage.

### Summary

Une approche d'apprentissage automatique qui exploite à la fois des données étiquetées et non étiquetées pour améliorer la précision et la généralisation du modèle.

## Key Concepts

- Données étiquetées
- Données non étiquetées
- Auto-entraînement
- Hypothèse de variété

## Use Cases

- Classification d'images avec annotations limitées
- Analyse de sentiment textuel avec étiquettes rares
- Prédiction de diagnostics médicaux avec des données expertes limitées
