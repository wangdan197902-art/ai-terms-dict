## Definition

Cette approche va au-delà du simple étiquetage humain dans la boucle. Elle implique un transfert de connaissances bidirectionnel : les humains corrigent les erreurs du modèle tandis que le modèle assiste les humains dans l'identification de motifs ou l'automatisation de tâches répétitives, créant ainsi une synergie d'amélioration continue.

### Summary

Un paradigme d'apprentissage collaboratif où les humains et les machines s'enseignent et apprennent continuellement mutuellement pour améliorer leurs performances.

## Key Concepts

- Apprentissage bidirectionnel
- Collaboration Humain-IA
- Systèmes adaptatifs
- Amélioration continue

## Use Cases

- Outils d'annotation interactive
- Ajustement des recommandations personnalisées
- Diagnostic médical avec expert dans la boucle
