## Definition

Contrairement aux modèles génératifs standards axés sur la fluidité, les modèles de raisonnement privilégient la précision dans les tâches multi-étapes telles que les mathématiques, la programmation et les énigmes logiques. Ils emploient souvent des techniques comme le Chain-of-Thought (CoT) pour structurer leur processus de décision.

### Summary

Un modèle d'IA spécifiquement optimisé pour effectuer des déductions logiques complexes, la résolution de problèmes étape par étape et le traitement en chaîne de pensée.

## Key Concepts

- Chaîne de pensée (Chain-of-Thought)
- Déduction logique
- Raisonnement étape par étape
- Résolution de problèmes complexes

## Use Cases

- Génération de preuves mathématiques
- Débogage de code complexe
- Assistance à la planification stratégique
