## Definition

Le déploiement continu est une extension de la livraison continue qui automatise l'intégralité du processus de publication. Une fois que les modifications de code passent toutes les barrières de qualité, y compris les tests unitaires, les tests d'intégration et la sécurité, elles sont déployées automatiquement.

### Summary

Une pratique d'ingénierie logicielle où chaque modification de code passant les tests automatisés est automatiquement mise en production.

## Key Concepts

- Tests automatisés
- Pipelines CI/CD
- Déploiement sans temps d'arrêt
- Drapeaux fonctionnels (Feature Flags)

## Use Cases

- Plateformes de trading haute fréquence
- Applications web à grande échelle
- Gestion d'architectures microservices
