## Definition

Bien que signifiant traditionnellement un moyen de transport, dans la terminologie de l'IA, « véhicule » peut décrire métaphoriquement le mécanisme de livraison des services intelligents, tels que les applications mobiles, les interfaces web ou les systèmes embarqués.

### Summary

Dans le contexte de l'IA, un véhicule désigne souvent la plateforme ou le support à travers lequel les modèles d'IA sont déployés ou livrés aux utilisateurs finaux.

## Key Concepts

- Plateforme de déploiement
- Interface
- Intégration matérielle
- Mécanisme de livraison

## Use Cases

- Décrire l'IA dans les voitures autonomes
- Identifier les applications mobiles comme véhicules de services IA
- Discuter des dispositifs IoT comme porteurs d'IA en périphérie
