## Definition

Ces bases de données permettent un modélisation dynamique des données en n'imposant pas de structures rigides de tables ou de définitions de colonnes à l'avance. Cette flexibilité permet aux développeurs de stocker des données non structurées ou semi-structurées, s'adaptant ainsi aux changements fréquents.

### Summary

Les bases de données agnostiques au schéma sont des systèmes de stockage qui permettent des structures de données flexibles sans exiger de schémas prédéfinis, souvent utilisées dans les environnements NoSQL.

## Key Concepts

- NoSQL
- Schéma dynamique
- Flexibilité
- Données non structurées

## Use Cases

- Stockage de contenu généré par les utilisateurs
- Prototypage pour le développement rapide d'applications
- Gestion de formats de données variés provenant d'appareils IoT
