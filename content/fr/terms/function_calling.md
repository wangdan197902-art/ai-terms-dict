## Definition

L'appel de fonction permet aux grands modèles de langage d'interagir avec des outils externes et des API en générant des sorties structurées, telles que des objets JSON, qui spécifient quelle fonction exécuter et quels arguments lui transmettre.

### Summary

Un mécanisme permettant aux LLM de générer des données structurées qui déclenchent l'exécution de fonctions logicielles spécifiques.

## Key Concepts

- Sortie structurée
- Intégration d'API
- Utilisation d'outils
- Schéma JSON

## Use Cases

- Réservation de vols via des API de voyage
- Exécution de calculs mathématiques complexes
- Interrogation de bases de données
