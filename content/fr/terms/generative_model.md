## Definition

Les modèles génératifs sont des algorithmes conçus pour comprendre les motifs et les structures au sein d'un ensemble de données donné afin de créer de nouvelles instances de données ressemblant à l'original. Contrairement aux modèles discriminatifs...

### Summary

Un type de modèle d'apprentissage automatique qui apprend la distribution sous-jacente des données pour générer de nouveaux échantillons synthétiques similaires aux données d'entraînement.

## Key Concepts

- Distribution de probabilité
- Données synthétiques
- Espace latent
- Objectif d'entraînement

## Use Cases

- Génération et édition d'images
- Traitement du langage naturel
- Augmentation de données pour petits ensembles de données
