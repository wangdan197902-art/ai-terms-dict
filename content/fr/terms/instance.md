## Definition

En apprentissage automatique, une instance désigne un exemple spécifique issu du jeu de données. Elle se compose d'un ensemble de caractéristiques d'entrée (attributs) et potentiellement d'une étiquette cible. Les instances constituent l'unité fondamentale.

### Summary

Un seul échantillon de données ou observation utilisé dans les tâches d'apprentissage automatique, généralement représenté par un vecteur de caractéristiques.

## Key Concepts

- Échantillon de données
- Vecteur de caractéristiques
- Données d'entraînement
- Observation

## Use Cases

- Entraînement en apprentissage supervisé
- Fractionnement des jeux de données
- Exemples pour la détection d'anomalies
