## Definition

Développé dans le cadre de l'initiative MLCommons, Croissant utilise le format JSON-LD pour fournir une manière standardisée de décrire les ensembles de données, y compris leur structure, leurs citations et leurs licences. Il vise à résoudre les problèmes de fragmentation et de découverte des données.

### Summary

Croissant est un format de métadonnées open source pour décrire des ensembles de données, facilitant leur découverte et leur interopérabilité dans le domaine de l'IA.

## Key Concepts

- Standard de métadonnées
- JSON-LD
- Découverte d'ensembles de données
- Interopérabilité

## Use Cases

- Pipelines d'ingestion automatisée de données
- Catalogage de données et moteurs de recherche
- Documentation de recherche reproductible
