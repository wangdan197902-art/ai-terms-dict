## Definition

Un modèle piloté par les données est un type de système d'intelligence artificielle où le comportement et les prédictions émergent des motifs identifiés dans les données historiques, plutôt que d'être définis par des règles codées en dur ou des équations physiques explicites.

### Summary

Un modèle computationnel dont les paramètres et la structure sont appris directement à partir de données empiriques plutôt que par programmation explicite.

## Key Concepts

- Apprentissage empirique
- Reconnaissance de motifs
- Inférence statistique
- Modèles boîte noire

## Use Cases

- Prédire les tendances du marché boursier à l'aide de données de prix historiques
- Poser des diagnostics médicaux à partir d'analyses d'imagerie médicale
- Prévoir les modèles météorologiques à l'aide d'observations satellitaires
