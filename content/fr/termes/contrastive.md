## Definition

Cette méthode incite le modèle à rapprocher les plongements vectoriels des paires positives (éléments similaires) tout en éloignant les paires négatives (éléments dissemblables) dans l'espace latent. Elle est largement utilisée dans la représentation visuelle et le traitement du langage.

### Summary

L'apprentissage contrastif est une technique auto-supervisée qui entraîne les modèles à distinguer les paires de données similaires des paires dissemblables.

## Key Concepts

- Plongements vectoriels
- Paires positives/négatives
- Fonction de perte
- Auto-supervision

## Use Cases

- Systèmes de recherche d'images
- Moteurs de recherche sémantique
- Vérification par reconnaissance faciale
