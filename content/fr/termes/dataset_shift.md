## Definition

Le changement de jeu de données se produit lorsque la distribution des données utilisées pour entraîner un modèle d'apprentissage automatique diffère de celle des données rencontrées lors de l'inférence. Cette divergence peut entraîner une baisse significative des performances du modèle.

### Summary

Le changement de jeu de données désigne le phénomène où les propriétés statistiques des données d'entrée évoluent entre l'entraînement et le déploiement.

## Key Concepts

- Changement de covariable
- Dérive conceptuelle
- Adaptation de domaine
- Généralisation

- Surveillance des modèles ML en production
- Stratégies de réentraînement
- Tests de robustesse
