## Definition

Dans l'apprentissage hâtif, le système construit une fonction cible ou un modèle général basé sur les données d'entraînement avant de rencontrer de nouvelles instances. Cela contraste avec l'apprentissage paresseux, qui reporte la généralisation

### Summary

L'apprentissage hâtif est une approche d'apprentissage automatique où la fonction de généralisation est apprise pendant la phase d'entraînement, permettant des temps de prédiction rapides une fois l'entraînement terminé.

## Key Concepts

- Généralisation lors de la phase d'entraînement
- Inférence rapide
- Complexité du modèle
- Risque de surapprentissage

## Use Cases

- Classification d'images en temps réel
- Systèmes de détection de fraude
- Moteurs de recommandation
