## Definition

L'apprentissage par apprentissage assisté, également connu sous le nom d'apprentissage par renforcement inverse à partir de démonstrations, permet aux agents d'acquérir des compétences en observant le comportement d'un expert plutôt que de s'appuyer uniquement sur des fonctions de récompense.

### Summary

Une méthode d'apprentissage par renforcement où un agent apprend une politique en imitant les démonstrations d'un expert.

## Key Concepts

- Apprentissage par renforcement inverse
- Démonstrations d'expert
- Imitation de politique
- Inférence de récompense

## Use Cases

- Formation à la conduite autonome
- Tâches de manipulation robotique
- Jeux vidéo joués par IA
