## Definition

L'open-source désigne un modèle de développement où le code sous-jacent d'un projet logiciel est accessible au public. Dans le domaine de l'IA, cela permet aux chercheurs et développeurs d'inspecter, modifier et redistribuer les algorithmes

### Summary

Logiciel dont le code source est mis gratuitement à disposition pour être utilisé, modifié et distribué par quiconque.

## Key Concepts

- Transparence
- Collaboration
- Accessibilité
- Piloté par la communauté

## Use Cases

- Développement du framework PyTorch
- Bibliothèques de modèles Hugging Face
- Reproductibilité de la recherche académique
