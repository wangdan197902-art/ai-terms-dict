### Summary

Une structure de programmation qui répète un bloc de code plusieurs fois jusqu'à ce qu'une condition soit remplie.

## Key Concepts

- Itération
- Contrôle de flux
- Époques
- Traitement par lots

## Use Cases

- Entraînement de réseaux neuronaux sur plusieurs époques
- Itération sur les échantillons d'un ensemble de données
- Exécution étape par étape en apprentissage par renforcement

## Code Example

```python
for epoch in range(10):
    for batch in dataloader:
        train_step(batch)
```
