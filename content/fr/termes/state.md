## Definition

Un état représente toutes les informations pertinentes nécessaires pour déterminer le comportement futur dans des systèmes tels que les Processus de Décision Markoviens (MDP). En apprentissage par renforcement, l'état encapsule la situation actuelle de l'environnement.

### Summary

La configuration complète d'un système ou d'un agent à un moment donné.

## Key Concepts

- Configuration
- Pas de temps
- Observation

## Use Cases

- Agents d'apprentissage par renforcement
- Modèles de Markov cachés
- Surveillance des systèmes
