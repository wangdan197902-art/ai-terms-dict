## Definition

Une interface en ligne de commande (CLI) permet aux utilisateurs de contrôler un logiciel en saisissant des commandes textuelles plutôt qu'en utilisant des éléments graphiques. Dans le développement de l'IA, les CLI sont essentielles pour exécuter des scripts, gérer des modèles et automatiser des tâches.

### Summary

L'interface en ligne de commande (CLI) est une méthode basée sur le texte pour interagir avec des programmes logiciels en tapant des commandes dans un terminal.

## Key Concepts

- Saisie de texte
- Émulation de terminal
- Scripting
- Automatisation

## Use Cases

- Exécution de scripts d'inférence de modèles
- Gestion des ressources IA dans le cloud
- Automatisation des pipelines de prétraitement des données
