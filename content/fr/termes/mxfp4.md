## Definition

MXFP4 (Mixed eXtended Floating Point 4-bit) est un format de type de données spécialisé introduit pour optimiser les performances et réduire l'utilisation de la bande passante mémoire dans les charges de travail d'IA. En permettant des opérations de précision mixte

### Summary

MXFP4 est un format à virgule flottante de précision mixte optimisé pour la multiplication matricielle efficace dans les accélérateurs matériels d'IA.

## Key Concepts

- Quantification
- Précision mixte
- Accélération matérielle
- Multiplication matricielle

## Use Cases

- Optimisation de l'inférence des LLM sur les appareils edge
- Réduction de l'empreinte mémoire lors de l'entraînement à grande échelle
- Accélération du déploiement de modèles basés sur des transformateurs
