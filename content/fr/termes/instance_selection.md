## Definition

La sélection d'instances vise à améliorer l'efficacité computationnelle et les performances du modèle en supprimant les points de données redondants ou bruités. Contrairement à la sélection de caractéristiques, elle opère sur les lignes du jeu de données.

### Summary

Une technique de prétraitement qui réduit la taille d'un jeu de données en sélectionnant un sous-ensemble d'instances représentatives.

## Key Concepts

- Réduction de données
- Élimination du bruit
- Sous-ensemble représentatif
- Efficacité computationnelle

## Use Cases

- Prétraitement de grands jeux de données
- Accélération des recherches de plus proches voisins
- Nettoyage de jeux de données déséquilibrés
