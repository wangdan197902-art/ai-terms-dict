## Definition

Server-sända händelser (SSE) möjliggör enkelriktad kommunikation från servern till klienten, där servern kan strömma data kontinuerligt utan att klienten behöver polla upprepade gånger. Det använder vanlig HTTP, vilket gör det enkelt att implementera.

### Summary

Ett standardprotokoll som gör det möjligt för webbserver att skicka realtidsuppdateringar till klienter över en enda HTTP-anslutning.

## Key Concepts

- Händelseström
- Enkelriktad kommunikation
- Automatisk återanslutning
- HTTP-baserat

## Use Cases

- Direkta aktiekursflöden
- Realtidsnotifikationer
- Uppdateringar för spårning av framsteg
