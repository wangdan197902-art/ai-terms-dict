## Definition

Ett tillstånd representerar all relevant information som krävs för att bestämma framtida beteende i system som Markovbeslutsprocesser (MDP). Inom förstärkningsinlärning sammanfattar tillståndet omgivningens aktuella läge.

### Summary

Den fullständiga konfigurationen hos ett system eller agens vid en specifik tidpunkt.

- Konfiguration
- Tidssteg
- Observation

## Use Cases

- Förstärkningsinlärningsagens
- Dolda Markov-modeller
- Systemövervakning
