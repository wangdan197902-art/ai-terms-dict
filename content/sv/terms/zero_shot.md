## Definition

Nollskottslärande gör det möjligt för modeller att generalisera till nya kategorier eller uppgifter för vilka ingen märkt träningsdata tillhandahölls under den inledande träningsfasen. Detta uppnås vanligtvis genom att utnyttja semantiska embeddingar eller förtränade representationer som kopplar ihop olika domäner.

### Summary

Förmågan att utföra uppgifter på osedda klasser utan tidigare träningsexempel.

## Key Concepts

- Generalisering
- Semantisk embedding
- Inga märkta data
- Överföringslärande

## Use Cases

- Klassificering av nya produktkategorier
- Att svara på frågor om osedda ämnen
- Överspråklig textklassificering
