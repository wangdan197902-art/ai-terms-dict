## Definition

Robotiserad processautomatisering (RPA) använder mjukvarubots, ofta förstärkta med AI, för att imitera mänskliga interaktioner med digitala system. Det används för att effektivisera arbetsflöden såsom datainmatning, fakturahantering

### Summary

Användningen av mjukvarubots för att automatisera högvolym, repetitiva digitala uppgifter som traditionellt utförs av människor i affärsprocesser.

## Key Concepts

- Mjukvarubots
- Arbetsflödesautomatisering
- Regelbaserad bearbetning
- Digital arbetskraft

## Use Cases

- Fakturahantering
- Kundonboarding
- Dataflyttningsuppgifter
