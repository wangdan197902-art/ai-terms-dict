## Definition

Detta fält kombinerar teknik inom maskininlärning med naturlig språkbehandling och datamining för att omvandla rådata till handlingsbar kunskap. Det innebär att träna modeller för att känna igen enheter, relationer och andra semantiska element i text och data.

### Summary

Processen att använda maskininlärningsalgoritmer för automatiskt att identifiera mönster och härleda strukturerad information från stora, ostrukturerade datamängder.

## Key Concepts

- Mönsterigenkänning
- Naturlig språkbehandling
- Datamining
- Funktionsteknik

## Use Cases

- Att extrahera kundkänslor från inlägg på sociala medier
- Att identifiera medicinska tillstånd från kliniska anteckningar
- Att automatisera dokumentklassificering i advokatbyråer
