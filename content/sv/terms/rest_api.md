## Definition

REST-APIer möjliggör kommunikation mellan klienter och servrar genom att utnyttja tillståndslösa operationer över HTTP-protokoll såsom GET, POST, PUT och DELETE. De strukturerar resurser som URI:er och använder standardiserade format (oftast JSON) för datautbyte.

### Summary

En Representational State Transfer-API är en arkitektonisk stil för att designa nätverksbaserade applikationer som förlitar sig på standardiserade HTTP-metoder.

## Key Concepts

- Tillståndslöshet
- HTTP-metoder
- Resursbaserade URI:er
- JSON-serierisering

## Use Cases

- Kommunikation mellan mikrotjänster
- Backend-integration för mobilappar
- Åtkomst till tredjepartsdata
