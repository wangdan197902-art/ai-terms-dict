## Definition

Bradley-Terry-modellen är en probabilistisk modell som används flitigt inom psykometri och maskininlärning för att hantera parvisa jämförelser. Den tilldelar varje objekt ett dolt värde och beräknar sannolikheten...

### Summary

En statistisk modell som används för att analysera data från parvisa jämförelser och uppskatta sannolikheten att ett objekt föredras framför ett annat.

## Key Concepts

- Parvisa jämförelser
- Rankningssystem
- Logistisk regression
- Förståelselärande (Preference Learning)

## Use Cases

- Algoritmer för sportrankning
- Analys av A/B-testning
- Styrning av förstärkningsinlärning från mänsklig feedback (RLHF)
