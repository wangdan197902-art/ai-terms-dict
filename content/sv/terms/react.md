## Definition

Ramverket ReAct gör det möjligt för stora språkmodeller (LLM:er) att generera både resonemangsspår och uppgiftsspecifika åtgärder på ett vävande sätt. Genom att simulera mänskligt tänkande möjliggör det att modeller interagerar med externa miljöer och verktyg för att lösa problem stegvis.

### Summary

ReAct är ett promptningsparadigm som kombinerar resonemang och handling för att förbättra prestandan hos stora språkmodeller på komplexa uppgifter.

## Key Concepts

- Vävande resonemang
- Åtgärdsgenerering
- Integration av observationer
- Tankekedja (Chain-of-Thought)

## Use Cases

- Komplexa frågesystem
- Autonoma agenter
- Arbetsflöden för faktagranskning
