## Definition

Ljugarens utdelning syftar på den samhällsrisk som avancerad generativ AI, särskilt deepfakes, utgör. När syntetiskt media blir okänslbart från verkligheten kan skadliga individer hävda att äkta bevis på felaktigt beteende är tillverkade, vilket underminerar förtroendet för visuell och auditiv information.

### Summary

Det fenomen där existensen av deepfakes och AI-genererat media gör att dåliga aktörer kan avfärda äkta bevis för sitt felprestera som falskt.

## Key Concepts

- Deepfakes
- Desinformation
- Plausibel förnekelse
- Mediekunnighet

## Use Cases

- Analys av politiska desinformationskampanjer
- Utveckling av faktagränningsprotokoll
- Juridiska försvarstrategier som involverar digitala bevis
