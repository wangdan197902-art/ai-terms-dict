## Definition

Inom artificiell intelligens avser kontroll de mekanismer och algoritmer som används för att styra ett systems handlingar baserat på nuvarande tillstånd och mål. Det involverar återkopplingsloopar där utdata övervakas och justeras kontinuerligt.

### Summary

Processen att hantera, styra eller reglera ett systems beteende och tillstånd för att uppnå önskade resultat.

## Key Concepts

- Återkopplingsloop
- Tillståndshantering
- Aktuering
- Reglering

## Use Cases

- Navigering för autonoma fordon
- Manipulering med robotarmar
- Industriella automatiseringsprocesser
