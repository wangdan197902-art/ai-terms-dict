## Definition

En Automatiserad Matematiker använder maskininlärning och symboliskt resonemang för att utforska matematiska rum bortom mänsklig intuition. Dessa system kan generera hypoteser, verifiera bevis och hitta mönster (att använda datorer för att göra matematiska upptäckter).

### Summary

Ett AI-system designat för att upptäcka nya matematiska satser, konjekturer eller bevis genom beräkningsmässig sökning och resonemang.

## Key Concepts

- Satsbevisning
- Symboliskt resonemang
- Hypotesgenerering
- Formell verifiering

## Use Cases

- Upptäckt av nya egenskaper inom grafteori
- Automatisering av bevisverifiering inom formella metoder
- Hitta motexempel till konjekturer
