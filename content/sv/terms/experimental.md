## Definition

Experimentell betecknar AI-komponenter som för närvarande testas, forskas på eller prototypas innan de uppnår stabilitet eller bred användning. Dessa system använder ofta nya arkitekturer eller oprövade metoder som kan vara mindre pålitliga i kritiska miljöer.

### Summary

Avser AI-teknologier, modeller eller metoder som befinner sig i ett tidigt utvecklingsstadium och ännu inte har validerats fullt ut för produktion.

## Key Concepts

- Prototyputveckling
- Fas för forskning
- Ovaliderad
- Nyskapande

## Use Cases

- Testning av nya transformer-arkitekturer
- Pilotprogram för generativa AI-funktioner
- Akademisk forskning om nya optimeringstekniker
