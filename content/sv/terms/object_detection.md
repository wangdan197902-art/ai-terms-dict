## Definition

Objektdetektering utvidgar bildklassificering genom att inte bara fastställa vilka objekt som finns närvarande, utan också var de befinner sig. Den returnerar koordinater för ramar runt detekterade föremål tillsammans med deras klassificeringslabel.

### Summary

En teknik inom datorsektion som identifierar och lokalerar objekt i bilder eller videoflöden med hjälp av ramar (bounding boxes).

## Key Concepts

- Ramar (Bounding Boxes)
- Klasslabel
- YOLO
- Faster R-CNN

## Use Cases

- Navigering för autonoma fordon
- Lagerhantering inom detaljhandel
- Övervakning via säkerhetskameror
