## Definition

Programmering genom exempel (PBE) är en paradigm inom programsyntes där utvecklare specificerar önskat beteende genom konkreta input-output-par istället för att skriva explicit kod. AI-systemet analyserar exemplen för att hitta en generaliserbar regel eller programkod som reproducerar resultaten för nya inputs.

### Summary

En AI-teknik där program automatiskt syntetiseras genom att inferera regler från input-output-exempel som tillhandahålls av användaren.

## Key Concepts

- Programsyntes
- Input-output-exempel
- Induktiv inferens
- Låg-kod/Ingen-kod

## Use Cases

- Generering av Excel-formler från exempeldata
- Texttransformation och skript för datarensning
- Automatisering av repetitiva UI-interaktioner
