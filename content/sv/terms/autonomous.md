## Definition

Autonomi inom AI avser förmågan hos ett system att uppfatta sin omgivning, fatta beslut och utföra handlingar utan direkt mänsklig kontroll. Till skillnad från enkel automation anpassar sig autonoma system till förändringar.

### Summary

Beskriver system som kan fatta beslut och agera oberoende i dynamiska miljöer.

## Key Concepts

- Beslutsfattande
- Omuppfattning
- Anpassningsförmåga
- Oberoende

## Use Cases

- Självkörande fordon
- Autonoma drönare
- Hantering av smarta elnät
