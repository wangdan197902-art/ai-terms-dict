## Definition

Detta kritiska steg innebär att fästa meningsfull metadata till rådatapunkter så att algoritmer kan lära sig relationen mellan indata och utdata. Till exempel, bounding boxes runt objekt i bild

### Summary

Dataannotering är processen att märka rådata, såsom bilder eller text, för att göra den lämplig för träning av övervakad maskininlärning.

## Key Concepts

- Övervakad inlärning
- Märkning
- Grundsanning
- Metadata

## Use Cases

- Träna objektidentifieringsmodeller
- Bygga klassificerare för sentimentanalys
- Skapa dataset för taligenkänning
