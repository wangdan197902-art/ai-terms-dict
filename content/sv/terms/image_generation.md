## Definition

Denna paradigm använder modeller som Stable Diffusion eller DALL-E för att producera högkvalitativa bilder baserat på textprompter eller andra indata. Det involverar att lära sig komplexa datafördelningar för att syntetisera realistiska bilder.

### Summary

Bildgenerering är processen att skapa ny visuell innehåll från grunden eller modifiera befintliga bilder med hjälp av generativa AI-modeller.

## Key Concepts

- Text-till-bild
- Diffusionsmodeller
- Latent diffusion
- Syntetisk data

## Use Cases

- Skapande av digital konst
- Design av marknadsföringsmaterial
- Dataaugmentering för träning
