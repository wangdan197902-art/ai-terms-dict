## Definition

Kunskapsbaserade system (KBS) är en gren av artificiell intelligens som integrerar specifik domänkunskap i ett datasystem för att utföra uppgifter som vanligtvis kräver mänsklig expertis. De utnyttjar strukturerad kunskap för att dra slutsatser och lösa problem genom logisk resonemangsföring snarare än enbart statistiska mönster.

### Summary

AI-system som använder explicita kunskapsbasers och inferensmotorer för att lösa komplexa problem som kräver mänsklig expertis.

## Key Concepts

- Kunskapsbas
- Inferensmotor
- Expertssystem
- Regelbaserad resonemangsföring

## Use Cases

- Stöd vid medicinsk diagnostik
- Felsökning inom industrin
- Bedömning av finansiell risk
