## Definition

Detta koncept fokuserar på manipulation av text där den grundläggande beräkningsenheten är ett enskilt tecken. Det används ofta i uppgifter som kräver finmaskig textanalys, såsom stavkontroll

### Summary

Teckenberäkning innebär bearbetning, generering eller analys av text på individnivå snarare än på ord- eller meningsnivå.

## Key Concepts

- Tokenisering
- Textgenerering
- Finmaskig analys
- Delordsenheter

## Use Cases

- Handskriftsigenkänning (OCR)
- Språkmodellering för språk med begränsade resurser
- Analys av krypterad text
