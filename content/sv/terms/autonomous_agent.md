## Definition

Inom artificiell intelligens är en autonom agent en entitet som verkar oberoende i en omgivning. Den använder sensorer för att uppfatta tillstånd och aktuatorer för att utföra handlingar, styrda av interna beslutsmekanismer.

### Summary

En autonom agent är ett system som kan uppfatta sin omgivning och agera oberoende för att uppnå specifika mål utan direkt mänsklig inblandning.

## Key Concepts

- Uppfattning
- Handling
- Målorienterat beteende
- Interaktion med omgivningen
