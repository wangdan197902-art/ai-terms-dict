## Definition

Inom AI syftar grounding på att länka symboliska representationer eller genererad text till konkreta enheter, data eller sensoriska upplevelser i verkligheten. För språkmodeller innebär detta ofta användning av Retrieval-Augmented Generation (RAG).

### Summary

Processen att koppla abstrakta AI-utfall till fakta, datakällor eller fysiska sammanhang i verkliga världen för att säkerställa noggrannhet och relevans.

## Key Concepts

- Retrieval-Augmented Generation
- Kontextuell noggrannhet
- Fakticitet
- Sensorimotorisk integration

## Use Cases

- Att minska hallucinationer i stora språkmodeller
- Att koppla NLP till robotstyrning
- Att verifiera påståenden mot kunskapsbanker
