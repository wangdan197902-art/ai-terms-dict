## Definition

Ett konnektionistiskt expertssystem integrerar mönsterigenkänning och inlärningsstyrkor hos neuronnätverk (konnektionism) med explicit kunskapsrepresentation och logisk resonemangsförmåga från traditionella expertssystem.

### Summary

Ett intelligentsystem som kombinerar neuronnätverkets inlärningsförmåga med symboliska resonemangsstrukturer.

## Key Concepts

- Hybrid AI
- Neural-symboolisk integration
- Kunskapsrepresentation
- Subsymbolisk inlärning

## Use Cases

- Medicinska diagnosystem
- Komplex industriell felupptäckt
- Naturlig språkförståelse med resonemang
