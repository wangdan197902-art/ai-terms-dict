## Definition

Code Search Net är ett omfattande dataset skapat för att främja forskning inom kodåtervinning. Det innehåller miljoner par av naturliga språkbeteckningar och motsvarande Java-kodsnuttar. Denna data möjliggör träning av modeller som kan förstå semantiken i kod.

### Summary

Ett benchmark-datasett utformat för att utvärdera kodsökmotorer genom att para ihop naturliga språkfrågor med relevanta kodsnuttar.
