## Definition

Inom artificiell intelligens avser generering modellens förmåga, särskilt hos Generativa Adversariella Nätverk (GANs) och Transformer-baserade stora språkmodeller (LLMs), att producera nytt innehåll såsom text eller bilder.

### Summary

Processen genom vilken generativa modeller skapar nya datainstanser som liknar träningsfördelningen.

## Key Concepts

- Probabilistisk modellering
- Latent rum
- Tokenförutsägelse
- Syntetiska data

## Use Cases

- Generering av naturligt språk
- Bildsynthes
- Kodautokomplettering
