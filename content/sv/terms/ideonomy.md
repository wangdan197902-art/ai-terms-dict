## Definition

Detta fält studerar processerna bakom hur idéer formas, kombineras och utvecklas. Det tillämpar strukturerade tekniker för att förbättra kreativitet och problemlösningsförmåga. Inom AI-kontext kan ideonomi användas för automatiserad brainstorming.

### Summary

Ideonomi är idégenereringens vetenskap, med fokus på systematiska metoder för att skapa och organisera nya koncept och innovationer.

## Key Concepts

- Kreativitetsalgoritmer
- Konceptuell blandning
- Innovationsteori
- Systematisk kreativitet

## Use Cases

- Verktyg för automatiserad brainstorming
- System för patentgenerering
- Assistenter för kreativt skrivande
