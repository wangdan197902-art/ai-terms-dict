## Definition

Inom AI och datavetenskap syftar numerisk på datatyper eller metoder som involverar kvantitativa värden, såsom heltal, flyttal och decimaler. Till skillnad från kategorisk eller textuell data hanterar numerisk data mätbara storheter.

### Summary

Som rör, involverar eller använder siffror eller matematiska beräkningar snarare än symbolisk logik eller text.

## Key Concepts

- Flyttal
- Heltal
- Datatyper
- Kvantitativ data

## Use Cases

- Statistisk modellering
- Finansiell prognos
- Bearbetning av sensordata

## Code Example

```python
import numpy as np
arr = np.array([1.5, 2.7, 3.9])
mean_val = np.mean(arr)
print(mean_val)
```
