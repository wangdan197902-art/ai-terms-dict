## Definition

Begreppet 'drivet' används ofta som ett suffix för att indikera den primära kraften eller mekanismen bakom ett AI-tillvägagångssätt. Till exempel innebär 'datadrivet' att beslut fattas baserat på statistiska mönster i data, medan 'måldrivet' fokuserar på optimering mot specifika belöningar eller mål.

### Summary

Drivet beskriver AI-system eller metoder där beslutsfattande eller modellarkitektur främst styrs av en specifik inmatningstyp, såsom data eller mål.

## Key Concepts

- Datcentrerat tillvägagångssätt
- Måloptimering
- Identifikation av paradigm
- Styrkinlärning

## Use Cases

- Att beskriva datacentrerade AI-arbetsflöden
- Att definiera styrkinlärningsagenter som målstyrda
- Att klassificera hybrida system som kunskapsdrivna eller datadrivna
