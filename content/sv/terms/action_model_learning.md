## Definition

Lärande av handlingsmodeller innebär att en agent konstruerar en intern representation av hur dess handlingar överför omgivningen från ett tillstånd till ett annat. Till skillnad från passiv observation utnyttjar denna metod agentens förmåga att aktivt testa och lära sig miljöns regler.

### Summary

En teknik inom förstärkningsinlärning där en agent lär sig omgivningsdynamiken genom att observera effekterna av sina egna handlingar.

## Key Concepts

- Tillståndsövergångar
- Omgivningsdynamik
- Modellbaserad RL
- Prediktiv modellering

## Use Cases

- Robotnavigation i okända terränger
- Spel-AI som lär sig fysikmotorer
- Kontrollsystem för industriell automation
