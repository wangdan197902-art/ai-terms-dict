## Definition

Denna praxis innebär att ansluta AI-modeller, såsom stora språkmodeller, till mjukvaruutvecklingsmiljöer som VS Code eller IntelliJ. Det möjliggör funktioner som intelligent kodkomplettering och automatisk refaktorering.

### Summary

IDE-integration avser att integrera AI-drivna verktyg direkt i integrerade utvecklingsmiljöer för att assistera programmerare med kodningsuppgifter.

## Key Concepts

- Autokomplettering
- Kodkomplettering
- LLM (Stor språkmodell)
- Utvecklarupplevelse

## Use Cases

- Intelligenta kodförslag
- Automatisk refaktorering
- Utförande av frågor på naturligt språk
