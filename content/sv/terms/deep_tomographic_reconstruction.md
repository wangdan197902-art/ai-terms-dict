## Definition

Djup tomografisk rekonstruktion representerar ett betydande framsteg jämfört med traditionella algebraiska eller analytiska metoder som filtrerad bakprojektion. Genom att utnyttja konvolutionella neurala nätverk (CNNs) kan tekniken effektivt hantera komplexa inversa problem och producera bilder med hög kvalitet även vid låg stråldos eller bristfällig datainsamling.

### Summary

En beräkningsmässig avbildningsteknik som använder djupa neurala nätverk för att rekonstruera högupplösta tvärsnittsbilder från sparsam eller brusig projektionsdata.

## Key Concepts

- Neurana nätverk
- Medicinsk avbildning
- Inversa problem
- Sinogrambehandling
- Artefaktreduktion

## Use Cases

- Rekonstruktion av lågdosade CT-skanningar
- Snabb MR-acquisition
- Industriell icke-destruktiv provning
