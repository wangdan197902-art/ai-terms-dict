## Definition

Prompt engineering innebär att utforma specifika inmatningar, så kallade prompts, för att få generativa AI-modeller att ge korrekta, relevanta och högkvalitativa svar. Det kräver en förståelse för hur modeller tolkar instruktioner.

### Summary

Praktiken att designa och optimera inmatningstexter för att styra stora språkmodeller mot önskade resultat.

## Key Concepts

- Kontextuell ramning
- Few-shot learning
- Instruktionsträning
- Tokenoptimering

## Use Cases

- Automatiserade kundsupportchattbottar
- Assistenter för kodgenerering
- Hjälpmedel för kreativt skrivande

```python
prompt = "Translate the following English text to French: 'Hello world'"
response = llm.generate(prompt)
```
