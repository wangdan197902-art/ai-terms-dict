## Definition

AIにおいて、「高水準」とは複雑なプロセスを簡素化する抽象化を指します。Pythonなどの高水準言語やAPIを使用することで、開発者はメモリ管理やハードウェアの詳細を意識せずにモデルを構築できます。これは開発生産性を高め、アクセシビリティを向上させるために重要です。

### Summary

低レベルの実装詳細をユーザーから隠蔽し、抽象的な表現やプログラミングインターフェースを指します。

## Key Concepts

- 抽象化
- API
- 複雑さの隠蔽
- 生産性

## Use Cases

- PyTorch/TensorFlowの高水準APIの使用
- MLパイプラインにおける特徴量エンジニアリング
- AI用のドメイン固有言語
