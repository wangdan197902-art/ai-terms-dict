## Definition

行動モデル学習では、エージェントは自身の行動が環境をある状態から別の状態へ遷移させる方法を内部的に表現します。この手法は受動的な観察とは異なり、環境の動態を構築するために活用されます。

### Summary

エージェントが自身の行動の影響を観察することで環境の動態を学習する強化学習手法。

## Key Concepts

- 状態遷移
- 環境動態
- モデルベース強化学習
- 予測モデリング

## Use Cases

- 未知の地形におけるロボティクス航法
- ゲームAIによる物理エンジン学習
- 産業用自動化制御システム
