## Definition

この概念は、計算の基本的な単位が単一の文字であるテキスト操作に焦点を当てています。スペルチェックなど、細粒度のテキスト分析が必要なタスクでよく使用されます。

### Summary

文字計算とは、単語や文レベルではなく、個々の文字レベルでテキストの処理、生成、分析を行うことを指します。

## Key Concepts

- トークン化
- テキスト生成
- 細粒度分析
- サブワードユニット

## Use Cases

- 手書き認識（OCR）
- 低資源言語モデリング
- 暗号文分析
