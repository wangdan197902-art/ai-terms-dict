## Definition

関数呼び出しにより、大規模言語モデルはJSONオブジェクトなどの構造化出力を生成することで外部ツールやAPIと連携し、実行すべき関数とその引数を指定できます。

### Summary

LLMが特定のソフトウェア関数をトリガーする構造化データを出力できるメカニズム。

## Key Concepts

- 構造化出力
- API統合
- ツール使用
- JSONスキーマ

## Use Cases

- 旅行API経由でのフライト予約
- 複雑な数学計算の実行
- データベースレコードの照会
