## Definition

在人工智能中，自主智能体是在环境中独立运作的实体。它利用传感器感知状态，并通过执行器执行动作，由内部模型指导其行为。

### Summary

自主智能体是一种能够感知环境并独立采取行动以实现特定目标，而无需直接人类干预的系统。

## Key Concepts

- 感知
- 行动
- 目标导向行为
- 环境交互

## Use Cases

- 自动驾驶汽车
- 机器人流程自动化
- 博弈AI
