## Definition

ReAct 框架使大语言模型能够以交错的方式生成推理轨迹和特定于任务的行动。通过模拟类人的思维过程，它允许模型与外部环境进行交互（原文截断，意为 interaction with external environment）。

### Summary

ReAct 是一种提示范式，通过结合推理和行动来提升大型语言模型在复杂任务上的表现。

## Key Concepts

- 交错推理
- 行动生成
- 观察整合
- 思维链

## Use Cases

- 自主智能体
- 事实核查工作流
