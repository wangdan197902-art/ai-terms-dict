## Definition

该术语指代 Qwen 家族中的一种专用架构，可能利用了混合专家（Mixture of Experts, MoE）设计。在此类模型中，仅激活神经网络参数（即“专家”）的一个子集，从而在保持性能的同时显著降低计算开销和内存需求。

### Summary

一种假设或未来的稀疏混合专家（MoE）变体，属于 Qwen3 系列，专为高效率设计。

## Key Concepts

- 混合专家
- 稀疏激活
- 推理效率
- 大语言模型

## Use Cases

- 高吞吐量 API 服务
- 边缘设备部署
- 具有成本效益的扩展
