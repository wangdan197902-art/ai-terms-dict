## Definition

机器学习控制将自适应算法与传统控制系统集成，以处理非线性或不确定环境。与静态控制器不同，这些系统能够从运行数据中学习并调整策略，从而在动态变化的环境中保持最优性能。

### Summary

一种控制理论方法，其中机器学习算法自适应地管理系统动态，以实时优化性能。

## Key Concepts

- 强化学习
- 自适应控制
- 实时优化
- 系统动力学

## Use Cases

- 强风条件下自主无人机的导航
- 智能电网中的能耗优化
- 制造过程中机械臂的精度控制
