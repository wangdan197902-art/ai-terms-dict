## Definition

向 Azure 部署涉及利用云原生工具，如 Azure Machine Learning、Azure Kubernetes Service (AKS) 或 Azure Functions，以大规模提供机器学习模型服务。它包括管理计算资源、监控模型性能以及确保可扩展性。

### Summary

在微软 Azure 云基础设施服务上托管和运行机器学习模型的过程。

## Key Concepts

- 云基础设施
- 模型服务化
- 可扩展性
- 持续集成/持续交付管道

## Use Cases

- 用于推理的实时 API 端点
- 大型数据集的批量评分
- 企业级模型管理
