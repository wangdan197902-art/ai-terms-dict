## Definition

函数调用使大型语言模型能够通过生成指定执行哪个函数及其参数的结构化输出（如 JSON 对象）来与外部工具和 API 进行交互。

### Summary

一种允许大语言模型输出结构化数据以触发特定软件功能的机制。

## Key Concepts

- 结构化输出
- API 集成
- 工具使用
- JSON 模式

## Use Cases

- 通过旅游 API 预订航班
- 执行复杂的数学计算
- 查询数据库记录
