## Definition

索引文件（通常名为 'model_index.json'）包含有关模型架构的结构化信息，包括管道类型、子模型和配置路径。它使 Hub 能够正确协调多组件模型的加载。

### Summary

模型索引是 Hugging Face Hub 用于描述和组织模型组件及配置的元数据文件。

## Key Concepts

- 元数据
- 管道配置
- 模型加载
- 互操作性

## Use Cases

- 定义多组件管道
- 上传复杂模型到 Hub
- 自动化模型发现与加载
