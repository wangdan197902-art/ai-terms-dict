## Definition

在人工智能中，“高层”指简化复杂过程的抽象层。高级语言（如Python）或API允许开发人员在不管理内存或硬件细节的情况下构建模型。高层抽象提高了开发效率和代码可读性，使研究者能专注于算法逻辑而非底层优化。

### Summary

指抽象表示或编程接口，对用户隐藏了底层实现细节。

## Key Concepts

- 抽象
- API
- 复杂性隐藏
- 生产力

## Use Cases

- 使用PyTorch/TensorFlow高层API
- 机器学习管道中的特征工程
- AI专用领域特定语言
