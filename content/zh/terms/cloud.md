## Definition

云计算为AI工作负载提供了可扩展的基础设施，使开发人员无需维护物理数据中心即可访问强大的GPU和存储空间。它支持各种服务模型，如基础设施即服务（IaaS）和平台即服务（PaaS）。

### Summary

云是指通过互联网托管的远程服务器，用于存储、管理和处理数据及AI模型，而非依赖本地硬件。

## Key Concepts

- 可扩展性
- 基础设施即服务
- 远程存储
- 弹性

## Use Cases

- 训练大型语言模型
- 为Web应用托管AI API
- 存储用于分析的海量数据集
