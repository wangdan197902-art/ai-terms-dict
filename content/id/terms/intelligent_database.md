## Definition

Basis data cerdas memanfaatkan pembelajaran mesin dan AI untuk meningkatkan fungsionalitas basis data standar melampaui penyimpanan dan pengambilan sederhana. Basis data ini dapat secara otomatis mengoptimalkan kinerja kueri, memprediksi penggunaan

### Summary

Sistem basis data yang menggabungkan kemampuan AI untuk mengelola data secara otomatis, mengoptimalkan kueri, dan menghasilkan wawasan.

## Key Concepts

- Optimasi Kueri
- Deteksi Anomali
- Penyetelan Mandiri
- Kueri Bahasa Natural

## Use Cases

- Skala otomatis sumber daya basis data awan berdasarkan beban
- Mendeteksi transaksi penipuan dalam data perbankan secara waktu nyata
- Menghasilkan kueri SQL dari pertanyaan bahasa natural
