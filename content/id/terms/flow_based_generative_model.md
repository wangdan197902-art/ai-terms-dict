## Definition

Model generatif berbasis aliran membangun distribusi probabilitas kompleks dengan menerapkan serangkaian transformasi inversibel dan dapat diturunkan pada distribusi dasar yang sederhana, seperti Gaussian. Karena transformasinya inversibel, model ini memungkinkan perhitungan likelihood eksak.

### Summary

Kelas model generatif yang menggunakan transformasi inversibel untuk memetakan distribusi sederhana ke distribusi data yang kompleks.

## Key Concepts

- Transformasi Inversibel
- Likelihood Eksak
- Normalizing Flows (Aliran Normalisasi)
- Perubahan Variabel

## Use Cases

- Generasi gambar
- Estimasi densitas
- Deteksi anomali
