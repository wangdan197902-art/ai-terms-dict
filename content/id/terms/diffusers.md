## Definition

Hugging Face Diffusers adalah toolkit modular yang dirancang untuk menyederhanakan penggunaan model difusi. Pustaka ini menawarkan pipa pra-dilatih untuk tugas-tugas seperti generasi teks-ke-gambar, inpainting gambar, dan super-resolusi.

### Summary

Sebuah pustaka dalam ekosistem Hugging Face yang menyediakan implementasi mutakhir dari model difusi untuk generasi gambar, audio, dan teks.

## Key Concepts

- Abstraksi Pipa
- Model Pra-dilatih
- Proses Denoising
- Hub Komunitas

## Use Cases

- Membangun aplikasi seni AI kreatif
- Mengimplementasikan alat pengeditan gambar
- Prototipe cepat fitur generatif
