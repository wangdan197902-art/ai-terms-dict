## Definition

Pembelajaran few-shot bertujuan memungkinkan model untuk menggeneralisasi dari hanya beberapa contoh, meniru efisiensi pembelajaran manusia. Hal ini biasanya mengandalkan strategi meta-pembelajaran, di mana model dilatih pada berbagai tugas sehingga dapat beradaptasi dengan cepat dengan tugas baru menggunakan sedikit data.

### Summary

Pembelajaran few-shot adalah paradigma pembelajaran mesin di mana model mempelajari konsep baru dari data pelatihan berlabel yang sangat terbatas.

## Key Concepts

- Meta-pembelajaran
- Efisiensi data
- Generalisasi
- Adaptasi tugas

## Use Cases

- Identifikasi penyakit langka
- Kategorisasi produk baru
- Deteksi objek khusus
