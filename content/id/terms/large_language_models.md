## Definition

Istilah ini merujuk pada paradigma aplikasi yang lebih luas di mana model dengan miliaran parameter dimanfaatkan untuk pembelajaran nol-sampel atau sedikit-sampel di berbagai tugas linguistik. Berbeda dengan model khusus

### Summary

Paradigma penggunaan jaringan saraf yang diskalakan untuk tugas pemahaman dan generasi bahasa alami spektrum luas.

## Key Concepts

- Pembelajaran Nol-Sampel
- Rekayasa Prompt
- Generalisasi
- Skala Parameter

## Use Cases

- Sistem Dialog Multi-giliran
- Pencarian Semantik
- Ekstraksi Data
