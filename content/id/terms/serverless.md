## Definition

Arsitektur Serverless memungkinkan pengembang membangun dan menjalankan aplikasi tanpa mengelola infrastruktur server. Penyedia awan secara otomatis menyesuaikan skala sumber daya naik atau turun berdasarkan permintaan, dan menagih pengguna hanya untuk sumber daya yang benar-benar digunakan.

### Summary

Model eksekusi komputasi awan di mana penyedia mengelola alokasi sumber daya mesin secara dinamis alih-alih pengguna.

## Key Concepts

- Fungsi sebagai Layanan (FaaS)
- Skala Otomatis
- Bayar per Penggunaan
- Berbasis Peristiwa

## Use Cases

- Pipeline pemrosesan gambar
- Backend untuk aplikasi seluler
- Tugas data terjadwal
