## Definition

Pembelajaran federasi memungkinkan organisasi untuk secara kolaboratif melatih model AI tanpa membagikan data mentah yang sensitif. Alih-alih memusatkan informasi, model dikirim ke perangkat lokal di mana ia belajar, dan hanya pembaruan parameter yang dikirim kembali ke server pusat untuk agregasi.

### Summary

Pembelajaran federasi adalah pendekatan pembelajaran mesin terdistribusi yang melatih model di perangkat yang terdesentralisasi sambil menjaga data tetap lokal.

## Key Concepts

- Privasi data
- Pelatihan terdistribusi
- Agregasi model
- Komputasi tepi

## Use Cases

- Keyboard teks prediktif
- Kolaborasi diagnosis medis
- Deteksi penipuan keuangan
