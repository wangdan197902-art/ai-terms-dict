## Definition

Gudang Fitur bertindak sebagai jembatan antara tim rekayasa data dan tim pembelajaran mesin, menyediakan pandangan terpadu tentang fitur untuk pelatihan batch dan inferensi waktu nyata. Hal ini memastikan konsistensi dengan mencegah kesenjangan antara data pelatihan dan data layanan (training-serving skew).

### Summary

Repository terpusat yang dirancang untuk mengelola, membagikan, dan menyajikan fitur secara konsisten di seluruh pelatihan dan inferensi pembelajaran mesin.

## Key Concepts

- Kesenjangan Pelatihan-Layanan
- Manajemen Terpusat
- Versi Fitur
- Infrastruktur MLOps

## Use Cases

- Platform ML perusahaan
- Sistem deteksi penipuan waktu nyata
- Tim pengembangan model kolaboratif
