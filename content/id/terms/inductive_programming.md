## Definition

Pemrograman Induktif, sering disebut sebagai Sintesis Program, melibatkan pembuatan kode perangkat lunak berdasarkan spesifikasi yang diberikan sebagai pasangan input-output, bukan instruksi eksplisit. Sistem menyimpulkan logika program dari contoh-contoh tersebut.

### Summary

Bidang sintesis program yang secara otomatis menghasilkan program komputer dari contoh input-output.

## Key Concepts

- Sintesis Program
- Contoh Input-Output
- Koding Otomatis
- Inferensi Logika

## Use Cases

- Alat penyelesaian kode
- Menghasilkan skrip dari data
- Bantuan pemrograman edukatif
