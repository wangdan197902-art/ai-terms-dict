## Definition

Dikembangkan oleh Facebook, GraphQL menyediakan deskripsi lengkap dan mudah dipahami tentang data dalam API Anda, memberi klien kekuatan untuk meminta apa yang mereka butuhkan saja tanpa lebih. GraphQL menggantikan penggunaan banyak endpoint REST dengan satu titik akhir.

### Summary

GraphQL adalah bahasa kueri untuk API dan runtime untuk memenuhi kueri tersebut dengan data yang ada, memungkinkan klien meminta tepat apa yang mereka butuhkan.

- Definisi Skema
- Bahasa Kueri
- Satu Titik Akhir
- Keselamatan Tipe

- Pengambilan data aplikasi seluler
