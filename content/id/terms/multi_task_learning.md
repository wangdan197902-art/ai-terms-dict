Multi-task learning adalah paradigma pembelajaran mesin di mana sebuah model dilatih pada beberapa tugas terkait secara bersamaan untuk meningkatkan kemampuan generalisasi.

## Key Concepts

- Representasi Bersama
- Bias Induktif
- Interferensi Tugas
- Generalisasi

## Use Cases

- Pemrosesan bahasa alami (misalnya, pengenalan entitas bernama dan pelabelan part-of-speech)
- Penglihatan komputer (misalnya, deteksi dan segmentasi)
- Pengenalan dan sintesis ucapan
