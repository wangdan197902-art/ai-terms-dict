## Definition

Pendekatan ini melampaui pelabelan sederhana berbasis manusia dalam loop. Ini melibatkan transfer pengetahuan dua arah: manusia memperbaiki kesalahan model sementara model membantu manusia dalam mengidentifikasi pola atau mengotomatisasi tugas tertentu, menciptakan siklus peningkatan berkelanjutan.

### Summary

Paradigma pembelajaran kolaboratif di mana manusia dan mesin terus-menerus mengajar dan belajar satu sama lain untuk meningkatkan kinerja.

## Key Concepts

- Pembelajaran dua arah
- Kolaborasi Manusia-AI
- Sistem adaptif
- Peningkatan berkelanjutan

## Use Cases

- Alat anotasi interaktif
- Penyetelan rekomendasi personalisasi
- Diagnosis medis dengan ahli dalam loop
