## Definition

Metode ini mendorong model untuk mendekatkan embedding pasangan positif (item serupa) dan menjauhkan pasangan negatif (item berbeda) dalam ruang laten. Teknik ini banyak digunakan dalam representasi visual dan pemrosesan bahasa alami untuk pembelajaran fitur tanpa label.

### Summary

Pembelajaran kontrastif adalah teknik pembelajaran mandiri yang melatih model untuk membedakan antara pasangan data yang mirip dan tidak mirip.

## Key Concepts

- Embedding
- Pasangan Positif/Negatif
- Fungsi Kerugian
- Pembelajaran Mandiri

## Use Cases

- Sistem pencarian gambar
- Mesin pencari semantik
- Verifikasi pengenalan wajah
