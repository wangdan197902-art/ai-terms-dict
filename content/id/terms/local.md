## Definition

Dalam kecerdasan buatan, 'lokal' biasanya merujuk pada operasi yang dilakukan langsung pada perangkat keras pengguna, seperti laptop atau ponsel pintar, tanpa bergantung pada server jarak jauh. Pendekatan ini meningkatkan privasi data

### Summary

Mengacu pada pemrosesan atau penyimpanan data pada perangkat tertentu, bukan di lingkungan cloud terpusat.

## Key Concepts

- Komputasi Tepi
- Privasi Data
- Pemrosesan Perangkat
- Pengurangan Latensi

## Use Cases

- Asisten suara yang sensitif terhadap privasi
- Deteksi objek secara real-time pada perangkat seluler
- Aplikasi AI offline
