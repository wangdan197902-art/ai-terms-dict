## Definition

Desen gradien adalah algoritma optimisasi iteratif orde pertama untuk menemukan minimum lokal dari fungsi yang dapat diturunkan. Dalam pembelajaran mesin, algoritma ini memperbarui bobot model ke arah berlawanan dari gradien fungsi kerugian.

### Summary

Algoritma optimisasi iteratif yang digunakan untuk meminimalkan fungsi kerugian dengan menyesuaikan parameter model.

## Key Concepts

- Fungsi Kerugian
- Laju Pembelajaran
- Optimisasi
- Propagasi Balik

## Use Cases

- Melatih jaringan saraf dalam
- Penyetelan parameter regresi linier
- Optimisasi model pengenalan gambar
