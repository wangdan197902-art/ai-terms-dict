## Definition

Langkah kritis ini melibatkan pemberian metadata yang bermakna pada titik data mentah sehingga algoritma dapat mempelajari hubungan antara input dan output. Misalnya, kotak pembatas di sekitar objek dalam gambar

### Summary

Anotasi data adalah proses memberi label pada data mentah, seperti gambar atau teks, agar sesuai untuk pelatihan pembelajaran mesin terawasi.

## Key Concepts

- Pembelajaran Terawasi
- Pemberian Label
- Ground Truth (Kebenaran Dasar)
- Metadata

## Use Cases

- Melatih model deteksi objek
- Membangun klasifikator analisis sentimen
- Menciptakan dataset pengenalan suara
