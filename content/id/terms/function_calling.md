## Definition

Pemanggilan fungsi memungkinkan model bahasa besar berinteraksi dengan alat eksternal dan API dengan menghasilkan output terstruktur, seperti objek JSON, yang menentukan fungsi mana yang akan dieksekusi dan argumen apa yang diperlukan.

### Summary

Mekanisme yang memungkinkan LLM menghasilkan data terstruktur untuk memicu fungsi perangkat lunak tertentu.

## Key Concepts

- Output Terstruktur
- Integrasi API
- Penggunaan Alat
- Skema JSON

## Use Cases

- Pemesanan tiket pesawat melalui API perjalanan
- Melakukan perhitungan matematika kompleks
- Mengakses catatan basis data
