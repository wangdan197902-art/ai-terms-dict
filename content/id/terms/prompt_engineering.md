## Definition

Rekayasa prompt melibatkan pembuatan masukan spesifik, yang dikenal sebagai prompt, untuk memicu respons yang akurat, relevan, dan berkualitas tinggi dari model AI generatif. Hal ini memerlukan pemahaman tentang bagaimana model menginterpretasi konteks dan instruksi.

### Summary

Praktik merancang dan mengoptimalkan teks masukan untuk membimbing model bahasa besar menuju keluaran yang diinginkan.

## Key Concepts

- Pembingkai kontekstual
- Pembelajaran few-shot
- Penyetelan instruksi
- Optimasi token

## Use Cases

- Chatbot dukungan pelanggan otomatis
- Asisten generasi kode
- Alat bantu penulisan kreatif

## Code Example

```python
prompt = "Translate the following English text to French: 'Hello world'"
response = llm.generate(prompt)
```
