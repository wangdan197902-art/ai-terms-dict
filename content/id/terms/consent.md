## Definition

Dalam etika AI, persetujuan merujuk pada izin sukarela dan sadar yang diberikan oleh pengguna atau subjek sebelum data pribadi mereka dikumpulkan, disimpan, atau dimanfaatkan dalam model pembelajaran mesin. Hal ini memerlukan transparansi mengenai bagaimana data akan digunakan.

### Summary

Persetujuan eksplisit dan sadar dari individu untuk membiarkan data mereka dikumpulkan dan diproses untuk keperluan AI.

## Key Concepts

- Persetujuan Berinformasi
- Privasi Data
- Agensi Pengguna
- Kepatuhan GDPR

## Use Cases

- Perjanjian penggunaan data kesehatan
- Kebijakan pengumpulan data media sosial
- Pendaftaran opt-in untuk autentikasi biometrik
