## Definition

Pemodelan Dinamis Empiris (EDM) adalah kerangka kerja untuk menganalisis sistem dinamis nonlinier menggunakan data observasional tanpa mengasumsikan bentuk parametrik tertentu. Metode ini mengandalkan metode embedding Takens

### Summary

Pendekatan statistik non-parametrik yang digunakan untuk merekonstruksi dinamika sistem kompleks dari data deret waktu observasional.

## Key Concepts

- Rekonstruksi ruang keadaan
- Teorema Takens
- Nonlinearitas
- Inferensi kausal

## Use Cases

- Perkiraan populasi ekologis
- Analisis sinyal neurosains
- Prediksi volatilitas pasar keuangan
