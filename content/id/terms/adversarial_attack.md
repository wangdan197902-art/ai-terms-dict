### Summary

Serangan adversarial adalah teknik di mana gangguan kecil, seringkali tak terlihat, ditambahkan ke data input untuk menipu model pembelajaran mesin agar membuat prediksi yang salah.

## Key Concepts

- Gangguan (Perturbation)
- Ketahanan Model (Model Robustness)
- Kotak Putih vs Kotak Hitam (White-box vs Black-box)
- Serangan Penghindaran (Evasion Attacks)

## Use Cases

- Menguji keamanan model
- Menghasilkan contoh adversarial untuk pelatihan
- Menganalisis kerentanan pada sistem penglihatan komputer
