## Definition

Dalam konteks AI dan ilmu data, numerik merujuk pada tipe data atau metode yang melibatkan nilai kuantitatif, seperti bilangan bulat, desimal, dan pecahan. Berbeda dengan data kategorikal atau tekstual, data numerik memungkinkan operasi matematika langsung.

### Summary

Berhubungan dengan, melibatkan, atau menggunakan angka atau perhitungan matematika, bukan logika simbolik atau teks.

## Key Concepts

- Desimal (Floats)
- Bilangan Bulat (Integers)
- Tipe Data
- Data Kuantitatif

## Use Cases

- Pemodelan statistik
- Peramalan keuangan
- Pemrosesan data sensor

## Code Example

```python
import numpy as np
arr = np.array([1.5, 2.7, 3.9])
mean_val = np.mean(arr)
print(mean_val)
```
