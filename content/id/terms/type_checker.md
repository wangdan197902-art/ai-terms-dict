## Definition

Dalam rekayasa perangkat lunak, khususnya untuk pustaka AI yang ditulis dalam Python, C++, atau Rust, pemeriksa tipe memastikan kebenaran kode dengan memvalidasi bahwa operasi dilakukan pada tipe data yang kompatibel. Ini membantu mencegah kesalahan runtime dan meningkatkan kualitas kode sebelum dieksekusi.

### Summary

Pemeriksa tipe adalah alat analisis statis yang memverifikasi apakah tipe variabel dan ekspresi dalam kode sesuai dengan aturan sistem tipe bahasa pemrograman.

## Key Concepts

- Analisis Statis
- Keselamatan Tipe
- Pencegahan Kesalahan
- Kualitas Kode

## Use Cases

- Memvalidasi bentuk tensor di PyTorch
- Memastikan kepatuhan kontrak API
- Refaktor basis kode AI yang besar
