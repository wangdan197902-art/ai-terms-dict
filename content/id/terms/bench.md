## Definition

Sebuah benchmark berfungsi sebagai titik referensi standar untuk membandingkan kemampuan berbagai model atau algoritma AI. Hal ini biasanya melibatkan dataset yang dikurasi dan metrik evaluasi spesifik seperti akurasi atau presisi.

### Summary

Singkatan dari benchmark, yaitu seperangkat tes standar atau metrik yang digunakan untuk mengevaluasi kinerja model AI.

## Key Concepts

- Metrik evaluasi
- Pengujian terstandarisasi
- Perbandingan kinerja
- Kurasi dataset

## Use Cases

- Benchmark GLUE untuk NLP
- ImageNet untuk visi komputer
- Seleksi model dalam produksi
