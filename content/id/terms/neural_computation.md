## Definition

Komputasi saraf merujuk pada operasi matematika yang dilakukan oleh neuron tiruan untuk mengubah sinyal input menjadi respons output. Proses ini melibatkan penjumlahan berbobot, fungsi aktivasi, dan propagasi balik (backpropagation).

### Summary

Proses pemrosesan informasi dalam jaringan saraf tiruan yang terinspirasi oleh neuron biologis.

## Key Concepts

- Fungsi aktivasi
- Penjumlahan berbobot
- Propagasi balik
- Representasi terdistribusi

## Use Cases

- Sistem pengenalan gambar
- Model pemrosesan bahasa alami
- Analitik prediktif
