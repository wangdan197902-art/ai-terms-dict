## Definition

Paradigma ini memanfaatkan model seperti Stable Diffusion atau DALL-E untuk menghasilkan gambar berkualitas tinggi berdasarkan prompt teks atau masukan lainnya. Proses ini melibatkan pembelajaran distribusi data yang kompleks untuk mensintesis konten visual yang realistis.

### Summary

Generasi gambar adalah proses membuat konten visual baru dari awal atau memodifikasi gambar yang ada menggunakan model AI generatif.

## Key Concepts

- Teks-ke-Gambar
- Model Difusi
- Difusi Laten
- Data Sintetis

## Use Cases

- Pembuatan seni digital
- Desain materi pemasaran
- Augmentasi data untuk pelatihan
