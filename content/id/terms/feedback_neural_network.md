## Definition

Jaringan saraf umpan balik, juga dikenal sebagai jaringan saraf berulang (RNN), mengandung loop yang memungkinkan sinyal merambat kembali ke lapisan sebelumnya. Rekurensi ini memungkinkan jaringan untuk mempertahankan keadaan internalnya sepanjang waktu.

### Summary

Arsitektur jaringan saraf di mana koneksi membentuk siklus berarah, memungkinkan informasi bertahan seiring waktu.

## Key Concepts

- Rekurensi
- Keadaan Internal
- Dinamika Temporal
- Memori

## Use Cases

- Pengenalan suara
- Prediksi deret waktu
- Pemrosesan bahasa alami
