## Definition

Metode ini memanfaatkan beberapa set fitur (tampilan) yang berbeda dari titik data yang sama. Awalnya, dua klasifikasi dilatih pada dataset berlabel kecil dari setiap tampilan. Mereka kemudian memprediksi label untuk data tak berlabel dan saling mengajar.

### Summary

Pelatihan bersama adalah algoritma pembelajaran semi-pengawasan di mana dua tampilan data digunakan untuk melatih klasifikasi terpisah yang secara iteratif memberi label pada data tak berlabel satu sama lain.

## Key Concepts

- Pembelajaran Semi-Pengawasan
- Beberapa Tampilan
- Pemberian Label Iteratif
- Seleksi Kepercayaan Tinggi

## Use Cases

- Klasifikasi teks dengan banyak fitur
- Kategorisasi halaman web
- Augmentasi data dengan label terbatas
