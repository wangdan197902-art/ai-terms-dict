## Definition

Masalah Mobil Gunung adalah standar benchmark dalam penelitian pembelajaran penguatan. Tujuannya adalah mengendalikan mobil bertenaga rendah untuk mencapai puncak bukit curam. Karena mobil tidak dapat mendaki bukit secara langsung

### Summary

Tugas pembelajaran penguatan klasik di mana agen harus mengemudikan mobil menanjak bukit curam hanya menggunakan kontrol akselerasi.

## Key Concepts

- Reward Jarang
- Konsekuensi Tertunda
- Kontrol Kontinu
- Benchmarking

## Use Cases

- Menguji algoritma pembelajaran penguatan baru
- Mendemonstrasikan teknik aproksimasi fungsi nilai
- Contoh edukatif untuk konsep RL
