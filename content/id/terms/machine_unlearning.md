### Summary

Penghapusan pembelajaran mesin adalah proses menghapus titik data spesifik atau pengaruhnya dari model yang telah dilatih tanpa melatih ulang model tersebut dari awal.

## Key Concepts

- Hak untuk Dilupakan
- Pendekatan Pelatihan Ulang Model
- Privasi Data
- Pembaruan Gradien

## Use Cases

- Mematuhi permintaan penghapusan data
- Menghapus data yang bias atau salah dari model
- Memitigasi serangan racun data (data poisoning)
