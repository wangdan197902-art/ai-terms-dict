## Definition

Logika Epistemik Dinamis (DEL) memperluas logika modal untuk memodelkan bagaimana pengetahuan berevolusi ketika agen menerima informasi baru. Logika ini menyediakan alat untuk menganalisis sistem multi-agen di mana keyakinan berubah karena publikasi informasi...

### Summary

Kerangka kerja formal untuk bernalar tentang perubahan keadaan pengetahuan agen akibat pembaruan informasi atau peristiwa.

## Key Concepts

- Pembaruan Pengetahuan
- Aliran Informasi
- Sistem Multi-Agen
- Revisi Keyakinan

## Use Cases

- Memverifikasi protokol keamanan
- Pemodelan komunikasi dalam kawanan robot
- Menganalisis skenario teori permainan
