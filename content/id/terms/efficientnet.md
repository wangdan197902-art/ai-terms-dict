## Definition

Dikembangkan oleh Google, EfficientNet menggunakan metode penskalaan kompaun untuk menyeimbangkan kedalaman jaringan, lebar, dan resolusi gambar input. Pendekatan ini memungkinkan model mencapai akurasi terbaik sambil

### Summary

EfficientNet adalah keluarga arsitektur jaringan saraf konvolusional yang menskalakan kedalaman, lebar, dan resolusi secara seragam untuk mencapai akurasi lebih tinggi dengan parameter lebih sedikit.

## Key Concepts

- Penskalaan kompaun
- Blok MobileNetV2
- Perhatian SE (Squeeze-and-Excitation)
- Efisiensi parameter

## Use Cases

- Klasifikasi gambar pada perangkat seluler
- Deteksi objek di lingkungan dengan sumber daya terbatas
- Ekstraksi fitur untuk dataset visual berskala besar
