## Definition

API REST memungkinkan komunikasi antara klien dan server dengan memanfaatkan operasi tanpa keadaan (stateless) melalui protokol HTTP seperti GET, POST, PUT, dan DELETE. API ini menyusun sumber daya sebagai URI dan menggunakan format standar f

### Summary

Representational State Transfer (REST) API adalah gaya arsitektur untuk merancang aplikasi jaringan yang mengandalkan metode HTTP standar.

## Key Concepts

- Tanpa Keadaan (Statelessness)
- Metode HTTP
- URI Berbasis Sumber Daya
- Serialisasi JSON

## Use Cases

- Komunikasi antar mikro layanan
- Integrasi backend aplikasi seluler
- Akses data pihak ketiga
