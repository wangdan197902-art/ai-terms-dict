## Definition

MXFP4 (Mixed eXtended Floating Point 4-bit) adalah format tipe data khusus yang diperkenalkan untuk mengoptimalkan kinerja dan mengurangi penggunaan bandwidth memori dalam beban kerja AI. Dengan memungkinkan operasi presisi campuran, format ini menyeimbangkan akurasi komputasi dengan efisiensi energi dan kecepatan pemrosesan pada perangkat keras khusus.

### Summary

MXFP4 adalah format titik mengambang presisi campuran yang dioptimalkan untuk perkalian matriks yang efisien di akselerator perangkat keras AI.

## Key Concepts

- Kuantisasi
- Presisi Campuran
- Akselerasi Perangkat Keras
- Perkalian Matriks

## Use Cases

- Mengoptimalkan inferensi LLM pada perangkat tepi
- Mengurangi jejak memori dalam pelatihan skala besar
- Mempercepat penyebaran model berbasis transformator
