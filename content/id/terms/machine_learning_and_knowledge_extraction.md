## Definition

Bidang ini menggabungkan teknik pembelajaran mesin dengan pemrosesan bahasa alami dan penambangan data untuk mengubah data mentah menjadi pengetahuan yang dapat ditindaklanjuti. Hal ini melibatkan pelatihan model untuk mengenali entitas, relasi, dan konsep dari teks atau data lainnya.

### Summary

Proses menggunakan algoritma pembelajaran mesin untuk secara otomatis mengidentifikasi pola dan menurunkan informasi terstruktur dari dataset besar yang tidak terstruktur.

## Key Concepts

- Pengenalan Pola
- Pemrosesan Bahasa Alami
- Penambangan Data
- Rekayasa Fitur

## Use Cases

- Mengekstrak sentimen pelanggan dari postingan media sosial
- Mengidentifikasi kondisi medis dari catatan klinis
- Mengotomatisasi klasifikasi dokumen di firma hukum
