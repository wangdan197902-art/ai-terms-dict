## Definition

Dalam AI, 'level tinggi' menunjukkan abstraksi yang menyederhanakan proses kompleks. Bahasa pemrograman level tinggi (seperti Python) atau API memungkinkan pengembang membangun model tanpa perlu mengelola memori atau spesifikasi perangkat keras secara manual, sehingga meningkatkan produktivitas dan kemudahan penggunaan.

### Summary

Mengacu pada representasi abstrak atau antarmuka pemrograman yang menyembunyikan detail implementasi tingkat rendah dari pengguna.

## Key Concepts

- Abstraksi
- API
- Penyembunyian kompleksitas
- Produktivitas

## Use Cases

- Menggunakan API level tinggi PyTorch/TensorFlow
- Rekayasa fitur dalam pipa ML
- Bahasa khusus domain untuk AI
