## Definition

Istilah ini menggambarkan klasifikasi struktural pembelajaran mesin menjadi pembelajaran terawasi, tidak terawasi, semi-terawasi, dan penguatan. Ini mencakup keluarga algoritma inti seperti regresi linier, pohon keputusan, dan mesin vektor dukungan, serta metode evaluasi kinerja model.

### Summary

Klasifikasi komprehensif mengenai paradigma, algoritma, dan metrik evaluasi dalam pembelajaran mesin.

## Key Concepts

- Pembelajaran Terawasi
- Pembelajaran Tidak Terawasi
- Pembelajaran Penguatan
- Evaluasi Model

## Use Cases

- Panduan pemilihan algoritma
- Perencanaan alur kerja ilmu data
- Pembuatan taksonomi akademik
