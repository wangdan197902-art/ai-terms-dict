## Definition

Model berbasis data adalah jenis sistem kecerdasan buatan di mana perilaku dan prediksi muncul dari pola yang diidentifikasi dalam data historis, bukan didefinisikan oleh aturan yang dikodekan secara keras atau model fisik eksplisit. Model ini mengandalkan inferensi statistik dan pembelajaran mesin untuk menemukan hubungan kompleks dalam data.

### Summary

Model komputasional yang parameter dan strukturnya dipelajari langsung dari data empiris daripada pemrograman eksplisit.

## Key Concepts

- Pembelajaran Empiris
- Pengenalan Pola
- Inferensi Statistik
- Model Kotak Hitam

## Use Cases

- Memprediksi tren pasar saham menggunakan data harga historis
- Mendiagnosis penyakit dari hasil pencitraan medis
- Meramalkan pola cuaca menggunakan observasi satelit
