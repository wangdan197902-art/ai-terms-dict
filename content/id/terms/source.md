## Definition

Dalam konteks AI, 'sumber' biasanya menunjukkan asal-usul dataset pelatihan, pustaka sumber terbuka, atau bobot model yang telah dilatih sebelumnya. Melacak sumber sangat penting untuk reproduktibilitas, kepatuhan lisensi...

### Summary

Merujuk pada asal data, kode, atau model yang digunakan dalam pengembangan dan penyebaran AI.

## Key Concepts

- Asal Usul Data
- Sumber Terbuka
- Aliran Input
- Reproduktibilitas

## Use Cases

- Audit data pelatihan untuk bias
- Mengelola versi model
- Mengutip ketergantungan sumber terbuka
