## Definition

Ilmu data melibatkan proses interdisipliner untuk mengekstrak pengetahuan dari data terstruktur dan tidak terstruktur, sementara analitik prediktif secara khusus berfokus pada penggunaan data historis untuk memprediksi

### Summary

Bidang ini menggabungkan analisis statistik dan pembelajaran mesin untuk mengekstrak wawasan dari data dan meramalkan peristiwa atau perilaku di masa depan.

## Key Concepts

- Pemodelan Statistik
- Peramalan
- Pengenalan Pola
- Dukungan Keputusan

## Use Cases

- Prediksi churn pelanggan
- Penilaian risiko keuangan
- Peramalan permintaan dalam rantai pasok
