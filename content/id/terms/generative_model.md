## Definition

Model generatif adalah algoritma yang dirancang untuk memahami pola dan struktur dalam kumpulan data tertentu agar dapat membuat instance data baru yang menyerupai data asli. Tidak seperti model diskriminatif...

### Summary

Jenis model pembelajaran mesin yang mempelajari distribusi mendasar dari data untuk menghasilkan sampel baru yang sintetis, mirip dengan data pelatihan.

## Key Concepts

- Distribusi Probabilitas
- Data Sintetis
- Ruang Laten
- Tujuan Pelatihan

## Use Cases

- Pembuatan dan pengeditan gambar
- Pemrosesan bahasa alami
- Augmentasi data untuk dataset kecil
