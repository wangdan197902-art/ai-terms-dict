## Definition

Secara umum, kurva pembelajaran menampilkan skor pelatihan dan validasi pada sumbu y terhadap jumlah sampel pelatihan atau iterasi pada sumbu x. Kurva ini membantu mendiagnosis apakah model mengalami underfitting (kurang belajar) atau overfitting (terlalu spesifik pada data latih).

### Summary

Kurva pembelajaran memplot metrik kinerja model terhadap jumlah data pelatihan atau epoch pelatihan untuk memvisualisasikan kemajuan pembelajaran.

## Key Concepts

- Skor pelatihan
- Skor validasi
- Underfitting vs Overfitting

## Use Cases

- Mendiagnosis masalah kinerja model
- Menentukan kebutuhan ukuran sampel
- Memantau konvergensi pelatihan
