## Definition

H2O adalah platform memori-in (in-memory) open-source yang banyak digunakan untuk pembelajaran mesin terdistribusi, skalabel, dan analitik prediktif. Awalnya dikembangkan oleh dua mahasiswa doktoral Harvard, platform ini menyediakan kerangka kerja terpadu untuk membangun model machine learning dan deep learning dengan cepat.

### Summary

Platform pembelajaran mesin open-source dan terdistribusi yang mendukung berbagai algoritma untuk analitik prediktif dan pembelajaran mendalam.

## Key Concepts

- Komputasi In-Memory
- AutoML
- Deep Learning (Pembelajaran Mendalam)
- Pemrosesan Terdistribusi

## Use Cases

- Membangun model prediktif untuk penilaian risiko keuangan
- Rekayasa fitur otomatis dan pemilihan model
- Menyebarkan model deep learning pada dataset berukuran besar
