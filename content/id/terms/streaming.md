## Definition

Streaming merujuk pada ingestion dan pemrosesan data secara kontinu secara waktu nyata atau hampir waktu nyata saat data tersebut dihasilkan. Berbeda dengan pemrosesan batch yang menangani dataset tetap, sistem streaming mengelola

### Summary

Paradigma pemrosesan data di mana informasi diproses secara terus-menerus dalam potongan kecil saat tiba, bukan dalam batch statis yang besar.

## Key Concepts

- Pemrosesan waktu nyata
- Pembaruan inkremental
- Latensi rendah
- Data tak terbatas

## Use Cases

- Deteksi penipuan waktu nyata dalam transaksi keuangan
- Pemantauan data sensor langsung dalam sistem IoT
- Umpan rekomendasi konten dinamis
