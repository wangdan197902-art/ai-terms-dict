## Definition

Pendekatan Bayesian dalam AI menggunakan teori probabilitas untuk memperbarui kemungkinan hipotesis seiring tersedianya lebih banyak bukti. Metode ini memungkinkan model untuk mengukur ketidakpastian dan menyempurnakan prediksi secara dinamis.

### Summary

Berhubungan dengan metode statistik berdasarkan Teorema Bayes untuk memperbarui probabilitas dengan bukti baru.

## Key Concepts

- Teorema Bayes
- Probabilitas prior
- Probabilitas posterior
- Kuantifikasi ketidakpastian

## Use Cases

- Filtering email spam
- Sistem diagnostik medis
- Analisis pengujian A/B

## Code Example

```python
from sklearn.naive_bayes import GaussianNB
model = GaussianNB()
model.fit(X_train, y_train)
```
