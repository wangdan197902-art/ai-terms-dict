## Definition

Pemrograman, juga dikenal sebagai coding, melibatkan penerjemahan logika dan persyaratan manusia ke dalam format yang dapat dieksekusi oleh komputer. Proses ini menggunakan sintaks dan semantik spesifik yang didefinisikan oleh bahasa pemrograman tertentu.

### Summary

Pemrograman adalah proses menulis instruksi untuk komputer menggunakan bahasa pemrograman guna menciptakan perangkat lunak, aplikasi, atau skrip.

## Key Concepts

- Bahasa Pemrograman
- Sintaks
- Algoritma
- Pengembangan Perangkat Lunak

## Use Cases

- Membangun aplikasi web
- Menciptakan aplikasi seluler
- Mengembangkan model AI
