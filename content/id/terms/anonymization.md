## Definition

Anonimisasi melibatkan modifikasi data sehingga data tersebut tidak lagi dapat dikaitkan dengan individu tertentu tanpa informasi tambahan. Teknik ini sangat kritis dalam pembelajaran mesin saat menangani data sensitif.

### Summary

Proses menghapus informasi pengenal pribadi dari kumpulan data untuk melindungi privasi individu.
