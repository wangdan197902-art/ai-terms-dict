## Definition

Penskalaan adalah metodologi aktif untuk memperluas sistem AI dengan menambahkan lebih banyak lapisan, neuron, atau contoh pelatihan. Ini mencakup teknik seperti pelatihan terdistribusi di beberapa GPU untuk menangani peningkatan

### Summary

Penskalaan adalah proses menyesuaikan ukuran model atau volume data untuk meningkatkan kemampuan belajar dan kinerja.

## Key Concepts

- Pelatihan Terdistribusi
- Kompleksitas Model
- Overfitting (Kelebihan Pasang)
- Akselerasi Perangkat Keras

## Use Cases

- Pembelajaran mendalam terdistribusi
- Meningkatkan kedalaman jaringan saraf
- Memperluas korpus pelatihan
