## Definition

AutoML (Automated Machine Learning) menyederhanakan pengembangan model ML dengan mengotomatisasi tugas-tugas seperti pra-pemrosesan data, rekayasa fitur, pemilihan model, dan penyetelan hiperparameter. Hal ini memungkinkan...

### Summary

Sebuah metodologi yang mengotomatisasi proses end-to-end penerapan pembelajaran mesin pada masalah dunia nyata, mengurangi upaya manual.

## Key Concepts

- Penyetelan Hiperparameter
- Rekayasa Fitur
- Pemilihan Model
- Demokratisasi

## Use Cases

- Prototipe cepat untuk analis bisnis
- Mengoptimalkan pipa produksi skala besar
- Membandingkan beberapa algoritma secara otomatis

## Code Example

```python
from auto_ml import Predictor
predictor = Predictor(type_of_estimator='classifier')
predictor.fit(dataframe, target='label')
```
