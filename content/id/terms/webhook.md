## Definition

Webhook adalah mekanisme bagi satu layanan untuk memberikan informasi waktu nyata kepada layanan lain ketika suatu peristiwa terjadi. Alih-alih melakukan polling (pengecekan berkala) untuk perubahan, sistem sumber mengirim permintaan HTTP POST ke URL yang ditentukan oleh penerima.

### Summary

Kembali panggilan HTTP yang ditentukan pengguna yang dipicu oleh peristiwa tertentu, memungkinkan sistem mengirimkan notifikasi waktu nyata ke aplikasi lain.

## Key Concepts

- Berpacu peristiwa
- HTTP POST
- URL kembali panggilan
- Notifikasi dorong

## Use Cases

- Deployment CI/CD otomatis
- Notifikasi gerbang pembayaran
- Integrasi bot Slack
