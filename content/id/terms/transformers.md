## Definition

Istilah 'Transformers' sering kali merujuk pada pustaka Python yang banyak digunakan dan dikelola oleh Hugging Face. Pustaka ini menyediakan antarmuka yang mudah digunakan untuk mengunduh, melatih, dan menerapkan model yang telah dilatih sebelumnya berdasarkan

### Summary

Dalam konteks ini, merujuk pada pustaka Hugging Face Transformers, sebuah toolkit sumber terbuka populer untuk model NLP dan multimodal terkini.

## Key Concepts

- Hugging Face Hub
- API Pipeline
- Kartu Model (Model Cards)
- Integrasi Tokenizer

## Use Cases

- Prototipe cepat aplikasi NLP
- Mengakses model yang telah dilatih komunitas
- Menstandarkan alur kerja penerapan model

```python
from transformers import pipeline
classifier = pipeline('sentiment-analysis')
```
