## Definition

AI berbasis data mewakili pergeseran paradigma dalam pengembangan kecerdasan buatan, dengan fokus pada perbaikan sistematis terhadap data yang digunakan untuk melatih model, bukan hanya mengoptimalkan algoritma atau hiperparameter. Pendekatan ini menekankan bahwa kualitas data memiliki dampak lebih besar pada kinerja model daripada kompleksitas arsitektur itu sendiri.

### Summary

Pendekatan pengembangan AI yang mengutamakan peningkatan kualitas dan kuantitas data pelatihan dibandingkan dengan memodifikasi arsitektur model.

## Key Concepts

- Kualitas Data
- Penyempurnaan Iteratif
- Konsistensi Label
- Kurasi Dataset

## Use Cases

- Meningkatkan akurasi visi komputer dengan memperbaiki gambar yang salah label
- Mengurangi bias dalam algoritma perekrutan melalui dataset yang seimbang
- Meningkatkan kinerja model NLP dengan memperbaiki kesalahan tata bahasa dalam teks pelatihan
