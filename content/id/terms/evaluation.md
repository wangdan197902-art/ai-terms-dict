## Definition

Evaluasi melibatkan pengukuran sistematis terhadap seberapa baik model AI melakukan tugas tertentu menggunakan metrik kuantitatif (misalnya, akurasi, skor F1, BLEU) dan penilaian kualitatif. Hal ini termasuk validasi silang dan pengujian pada set data yang belum pernah dilihat.

### Summary

Evaluasi adalah proses menilai kinerja, akurasi, dan ketahanan model AI terhadap metrik dan dataset yang telah ditentukan sebelumnya.

## Key Concepts

- Metrik
- Set Validasi
- Generalisasi
- Penilaian Acuan

## Use Cases

- Membandingkan versi model selama penyetelan hiperparameter
- Mengevaluasi model untuk keadilan dan bias
- Sertifikasi sistem AI untuk kepatuhan regulasi
