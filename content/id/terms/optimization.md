## Definition

Dalam pembelajaran mesin, optimasi merujuk pada algoritma yang digunakan untuk menyesuaikan parameter model guna meminimalkan fungsi kerugian (loss function), sehingga meningkatkan kinerja model. Metode umum termasuk Gradient Descent dan variannya.

### Summary

Proses matematika untuk meminimalkan atau memaksimalkan fungsi objektif guna menemukan parameter solusi terbaik.

## Key Concepts

- Minimisasi fungsi kerugian
- Gradient descent
- Learning rate (laju pembelajaran)
- Konvergensi

## Use Cases

- Pelatihan jaringan saraf
- Penyetelan hiperparameter
- Masalah alokasi sumber daya
