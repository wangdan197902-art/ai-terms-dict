## Definition

Dataset WikiHow terdiri dari sekitar 60.000 artikel 'cara melakukan' yang dikumpulkan dari situs web WikiHow. Dataset ini banyak digunakan dalam penelitian pemrosesan bahasa alami untuk tugas-tugas seperti ringkasan teks abstraktif dan ekstraksi langkah-langkah instruksional.

### Summary

Sebuah dataset berskala besar yang terdiri dari artikel 'cara melakukan' dari WikiHow, terutama digunakan untuk tugas ringkasan teks dan generasi instruksi.

## Key Concepts

- Ringkasan Teks
- Teks Prosedural
- Ekstraksi Instruksi

## Use Cases

- Pelatihan model ringkasan abstraktif
- Generasi instruksi langkah demi langkah
- Memahami struktur bahasa prosedural
