## Definition

Rekayasa fitur adalah seni memanfaatkan keahlian domain untuk mentransformasi data mentah menjadi fitur yang lebih baik mewakili pola mendasar bagi algoritma pembelajaran mesin. Proses ini mencakup penciptaan variabel baru.

### Summary

Praktik menggunakan pengetahuan domain untuk membuat fitur baru atau memodifikasi fitur yang ada guna meningkatkan kinerja model pembelajaran mesin.

## Key Concepts

- Pengetahuan Domain
- Transformasi Data
- Kinerja Model
- Penciptaan Variabel

## Use Cases

- Meningkatkan akurasi model regresi
- Memperkuat batas klasifikasi
- Mengoptimalkan peramalan deret waktu

## Code Example

```python
df['new_feature'] = df['feature_a'] * df['feature_b']
```
