## Definition

Tento kritický krok zahrnuje přidávání smysluplných metadat k surovým datovým bodům, aby algoritmy mohly naučit vztah mezi vstupem a výstupem. Například ohraničující boxy kolem objektů na obrázcích

### Summary

Data annotation je proces označování surových dat, jako jsou obrázky nebo text, aby byla vhodná pro trénink supervizovaného strojového učení.

## Key Concepts

- Supervizované učení
- Označování
- Základní pravda
- Metadata

## Use Cases

- Trénink modelů detekce objektů
- Vytváření klasifikátorů pro analýzu sentimentu
- Vytváření datasetů pro rozpoznávání řeči
