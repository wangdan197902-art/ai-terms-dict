## Definition

Cloud computing poskytuje škálovatelnou infrastrukturu pro zátěž AI, umožňuje vývojářům přístup k výkonným GPU a úložišti bez nutnosti udržovat fyzická data centra. Podporuje různé modely služeb, jako je IaaS, PaaS a SaaS.

### Summary

Cloud označuje vzdálené servery hostované na internetu, které slouží k ukládání, správě a zpracování dat a modelů AI místo lokálního hardwaru.

## Key Concepts

- Škálovatelnost
- Infrastruktura jako služba
- Vzdálené úložiště
- Elastičita

## Use Cases

- Trénink velkých jazykových modelů
- Hostování API pro webové aplikace
- Ukládání rozsáhlých datasetů pro analýzu
