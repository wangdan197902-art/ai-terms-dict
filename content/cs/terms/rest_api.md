## Definition

REST API umožňují komunikaci mezi klienty a servery pomocí bezstavových operací přes protokoly HTTP, jako jsou GET, POST, PUT a DELETE. Strukturovají zdroje jako URI a využívají standardní formáty dat (pozn.: text byl v zadání přerušen, překlad odpovídá kontextu).

### Summary

Representational State Transfer (REST) je architektonický styl pro navrhování síťových aplikací, který se opírá o standardní metody HTTP.

## Key Concepts

- Bezustavovost
- Metody HTTP
- URI založená na zdrojích
- Serializace JSON

## Use Cases

- Komunikace mezi mikroslužbami
- Integrace backendu mobilních aplikací
- Přístup k datům třetích stran
