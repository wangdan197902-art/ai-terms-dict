## Definition

Popisová logika (DL) jsou rozhodnutelné fragmenty prvního řádu, které tvoří teoretický základ pro ontologie, zejména jazyk Web Ontology Language (OWL). Umožňují přesnou definici pojmů a jejich vztahů.

### Summary

Rodina formálních jazyků pro reprezentaci znalostí používaných k reprezentaci a odvozování o konceptuální struktuře domén.

## Key Concepts

- Ontologie
- Odvozování
- Subsumpce
- Hierarchie pojmů

## Use Cases

- Vytváření ontologií pro sémantický web
- Medicínské znalostní grafy
- Automatizované klasifikační systémy
