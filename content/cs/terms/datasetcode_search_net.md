## Definition

Code Search Net je komplexní datová sada vytvořená k podpoře výzkumu v oblasti retrievu kódu. Obsahuje miliony párů popisů v přirozeném jazyce a odpovídajících úryvků kódu v jazyce Java. Tato data umožňují trénovat modely, které chápou sémantiku kódu.

### Summary

Benchmarková datová sada navržená k hodnocení systémů vyhledávání v kódu prostřednictvím párování přirodnějazykových dotazů s příslušnými úryvky kódu.

## Key Concepts

- Retrieval kódu
- Sémantické vyhledávání
- Java kód
- Zpracování přirozeného jazyka

## Use Cases

- Vývoj pluginů pro IDE
- Výzkum porozumění kódu
- Automatická dokumentace kódu
