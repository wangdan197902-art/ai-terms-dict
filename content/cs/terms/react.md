## Definition

Rámec ReAct umožňuje velkým jazykovým modelům generovat stopy uvažování i akce specifické pro úkol střídavým způsobem. Simulací lidského myšlenkového procesu umožňuje modelům interagovat s externím prostředím (pozn.: text byl v zadání přerušen, překlad odpovídá kontextu).

### Summary

ReAct je paradigma promptingu, které kombinuje uvažování a jednání za účelem zlepšení výkonu velkých jazykových modelů na složitých úkolech.

## Key Concepts

- Střídavé uvažování
- Generování akcí
- Integrace pozorování
- Řetězec myšlenek (Chain-of-Thought)

## Use Cases

- Systémy komplexních otázek a odpovědí
- Autonomní agenti
- Pracovní postupy ověřování faktů
