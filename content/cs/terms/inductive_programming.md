## Definition

Inductive Programming, často označovaná jako Program Synthesis, zahrnuje vytváření softwarového kódu na základě specifikací poskytnutých jako páry vstup-výstup, nikoliv explicitních instrukcí. Systém odvozuje pravidla z dat.

### Summary

Obor syntézy programů, který automaticky generuje počítačové programy z příkladů vstup-výstup.

## Key Concepts

- Syntéza programů
- Příklady vstup-výstup
- Automatizované kódování
- Logická inferenční metoda

## Use Cases

- Nástroje pro dokončování kódu
- Generování skriptů z dat
- Vzdělávací pomůcky pro programování
