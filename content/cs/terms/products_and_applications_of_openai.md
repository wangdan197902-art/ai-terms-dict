## Definition

Tento termín zahrnuje komerční a výzkumné produkty vytvořené společností OpenAI, přední laboratoří pro výzkum umělé inteligence. Mezi klíčové nabídky patří Generativní předtrénovaný transformátor (GPT), modely pro generování obrázků (DALL-E) a další služby založené na velkých jazykových modelech.

### Summary

Odkaz na sadu nástrojů AI, API a výzkumných výstupů vyvinutých společností OpenAI, včetně modelů GPT a DALL-E.

## Key Concepts

- Velké jazykové modely
- Generativní AI
- Služby API
- Komerční nasazení

## Use Cases

- Automatizovaná pomoc při kódování přes GitHub Copilot
- Generování kreativního obsahu pro marketing
- Konverzační agenti pro zákaznickou podporu
