## Definition

V strojovém učení se instance týká jednoho konkrétního příkladu z datové sady. Skládá se ze sady vstupních znaků (atributů) a případně cílového popisku. Instance jsou základní jednotkou

### Summary

Jednotlivý vzorek dat nebo pozorování používaný v úlohách strojového učení, obvykle reprezentovaný jako vektor znaků.

## Key Concepts

- Datový vzorek
- Vektor znaků
- Tréninková data
- Pozorování

## Use Cases

- Trénink supervidovaného učení
- Rozdělování datových sad
- Příklady detekce anomálií
