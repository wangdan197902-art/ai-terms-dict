## Definition

Serverem vysílané události (SSE) umožňují jednosměrnou komunikaci od serveru ke klientovi, kde server může streamovat data kontinuálně bez toho, aby klient musel opakovaně provádět dotazy. Využívá běžné HTTP, což jej činí snadno implementovatelným.

### Summary

Standardní protokol umožňující webovým serverům odesílat aktualizace v reálném čase klientům prostřednictvím jednoho HTTP spojení.

## Key Concepts

- Proud událostí
- Jednosměrná komunikace
- Automatické znovu připojení
- Založeno na HTTP

## Use Cases

- Živé kurzy akcií
- Oznámení v reálném čase
- Aktualizace sledování postupu
