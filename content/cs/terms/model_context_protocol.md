## Definition

Protokol kontextu modelu (MCP) je otevřený standard, který umožňuje aplikacím AI připojovat se k různým zdrojům dat, jako jsou databáze, API a souborové systémy, jednotným způsobem. Abstrahuje komplexitu

### Summary

Standardizovaný rámec navržený k usnadnění bezpečné a efektivní komunikace mezi modely AI a externími zdroji dat nebo nástroji.

## Key Concepts

- Interoperabilita
- Standardizace
- Integrace dat
- Volání nástrojů

## Use Cases

- Připojení LLM k podnikovým datům
- Vývoj přenosných AI asistentů
- Bezpečná integrace API
