## Definition

Programování, také známé jako psaní kódu, zahrnuje převod lidské logiky a požadavků do formátu, který mohou počítače vykonat. Používá specifickou syntaxi a sémantiku definovanou programovacími jazyky, jako jsou Python, Java nebo C++.

### Summary

Programování je proces zápisu instrukcí pro počítače pomocí programovacích jazyků za účelem vytváření softwaru, aplikací nebo skriptů.

## Key Concepts

- Programovací jazyky
- Syntaxe
- Algoritmy
- Vývoj softwaru

## Use Cases

- Vývoj webových aplikací
- Tvorba mobilních aplikací
- Vývoj modelů umělé inteligence
