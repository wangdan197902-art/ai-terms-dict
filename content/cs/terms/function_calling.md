## Definition

Volání funkcí umožňuje velkým jazykovým modelům interagovat s externími nástroji a API prostřednictvím generování strukturovaných výstupů, jako jsou objekty JSON, které specifikují, kterou funkci má provést a s jakými argumenty.

### Summary

Mechanismus umožňující velkým jazykovým modelům (LLM) generovat strukturovaná data, která spouštějí konkrétní softwarové funkce.

## Key Concepts

- Strukturovaný výstup
- Integrace API
- Použití nástrojů
- JSON schéma

## Use Cases

- Rezervace letenek přes cestovní API
- Provádění složitých matematických výpočtů
- Dotazování se na záznamy v databázi
