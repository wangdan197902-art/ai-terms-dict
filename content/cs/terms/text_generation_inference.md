## Definition

Inference generování textu (TGI) je specializovaný softwarový rámec určený pro obsluhu velkých jazykových modelů (LLM) s nízkou latencí a vysokou propustností. Optimalizuje proces inference pro generování textu pomocí technik jako kontinuální batchování a paralelizace tenzorů.

### Summary

Vysoce výkonný engine pro obsluhu modelů optimalizovaný specificky pro nasazení velkých jazykových modelů s cílem efektivně generovat text ve velkém měřítku.

## Key Concepts

- Kontinuální batchování
- Paralelizace tenzorů
- Obsluha s nízkou latencí
- Nasadění LLM

## Use Cases

- Produkční API pro chatboty
- Služby pro generování obsahu v reálném čase
- Platformy pro analýzu textu s vysokou propustností
