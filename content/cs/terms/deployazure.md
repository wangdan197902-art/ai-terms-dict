## Definition

Nasazování na Azure zahrnuje využívání cloudově nativních nástrojů, jako jsou Azure Machine Learning, Azure Kubernetes Service (AKS) nebo Azure Functions, pro nasazení ML modelů ve velkém měřítku. Zahrnuje správu výpočetních zdrojů a škálovatelnost.

### Summary

Proces hostování a spouštění strojových modelů na infrastruktuře cloudu Microsoft Azure.

## Key Concepts

- Cloudová infrastruktura
- Služba modelů
- Škálovatelnost
- CI/CD pipeline

## Use Cases

- Koncové body API pro inferenci v reálném čase
- Dávkové skórování velkých datových sad
- Správa modelů podnikové třídy
