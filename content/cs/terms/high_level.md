## Definition

V AI 'vysokoúrovňový' označuje abstrakce, které zjednodušují složité procesy. Vysokoúrovňové jazyky (jako Python) nebo API umožňují vývojářům budovat modely bez nutnosti spravovat paměť nebo specifiky hardwaru. Naproti tomu nízkourovňové rozhraní vyžaduje hlubší pochopení architektury a manuální správu zdrojů.

### Summary

Odkazuje na abstraktní reprezentace nebo programovací rozhraní, která skrývají nízkourovňové implementační detaily od uživatele.

## Key Concepts

- Abstrakce
- Rozhraní API (APIs)
- Skrývání složitosti
- Produktivita

## Use Cases

- Používání vysokoúrovňových API frameworků PyTorch/TensorFlow
- Inženýrství funkcí v pipelinech strojového učení
- Doménově specifické jazyky pro AI
