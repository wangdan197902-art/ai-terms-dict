## Definition

Geometrisk funksjonslæring fokuserer på behandling av data som har ikke-euklidiske strukturer, som sosiale nettverk, molekylære grafer eller 3D-meshes. Teknikker som graf-nevrale nettverk (GNNs) og mangfoldighetslæring brukes for å bevare romlige relasjoner og symmetrier som er viktige for nøyaktig analyse.

### Summary

En teknikk innen dyp læring som ekstraherer funksjoner fra data med iboende geometriske strukturer, som grafer eller mangfoldigheter, ved hjelp av spesialiserte nevrale nettverk.

## Key Concepts

- Graf-nevrale nettverk
- Mangfoldighetslæring
- Bevaring av symmetri
- Topologisk dataanalyse

## Use Cases

- Medisinsk oppdagelse og molekylær modellering
- Analyse av sosiale nettverk
- Gjenkjenning av 3D-objekter
