## Definition

Multi-agent-systemer består av flere uavhengige agenter, hvor hver enkelt kan være spesialisert på ulike oppgaver eller domener. Disse agentene kommuniserer og samordner handlingene sine for å oppnå et felles mål, ofte gjennom desentralisert styring.

### Summary

En arkitektonisk tilnærming der flere autonome AI-agenter samarbeider, konkurrerer eller samordner seg for å løse komplekse problemer som overstiger individuelle evner.

## Key Concepts

- Agentsamarbeid
- Spesialisering
- Samordning
- Desentralisering

## Use Cases

- Automatisert programvareutvikling
- Komplekse simuleringmiljøer
- Distribuert beslutningstaking
