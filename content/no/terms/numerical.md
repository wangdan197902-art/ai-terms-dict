## Definition

I sammenheng med kunstig intelligens og datavitenskap refererer numerisk til datatyper eller metoder som involverer kvantitative verdier, som heltall, flyttall og desimaltall. I motsetning til kategoriske eller tekstuelle data.

### Summary

Relatert til, involverer eller bruker tall eller matematiske beregninger i stedet for symbolisk logikk eller tekst.

## Key Concepts

- Flyttall
- Heltall
- Datatyper
- Kvantitativ data

## Use Cases

- Statistisk modellering
- Finansiell prognose
- Behandling av sensor data

## Code Example

```python
import numpy as np
arr = np.array([1.5, 2.7, 3.9])
mean_val = np.mean(arr)
print(mean_val)
```
