## Definition

Dette feltet binder sammen nevrovitenskap og robotikk ved å implementere nevronale nettverksmodeller i robotstyringssystemer. Det lar forskere teste hypoteser om motorisk kontroll, sansebehandling og kognitiv funksjon.

### Summary

Neurorobotikk er studiet av hvordan biologiske neurale systemer kan informere designet av autonome roboter og hvordan roboter kan tjene som modeller for å forstå hjernefunksjon.

## Key Concepts

- Forankret kognisjon
- Motorisk kontroll
- Sanse-motorisk integrasjon
- Biologisk modellering

## Use Cases

- Styring av proteser
- Autonome navigasjonsagenter
- Simuleringer for nevrovitenskapelig forskning
