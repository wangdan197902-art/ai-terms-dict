## Definition

En intelligent agent er et system som er i stand til å oppfatte omgivelsene sine gjennom sensorer eller datainnganger, behandle denne informasjonen ved hjelp av resonneringsalgoritmer og handle på miljøet via aktuatorer

### Summary

En autonom programvareenhet som oppfatter sitt miljø, resonerer om handlinger og utfører oppgaver for å nå spesifikke mål.

## Key Concepts

- Autonomi
- Oppfatnings-handlingsløkke
- Målrettethet
- Reaktivitet

## Use Cases

- Autonome handelsboter som utfører markedstrategier
- Robotprosessautomatisering som håndterer flertrinnede arbeidsprosesser
- Personlige assistenter som administrerer kalender- og e-postoppgaver
