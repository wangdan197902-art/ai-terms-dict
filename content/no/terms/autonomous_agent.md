## Definition

Innen kunstig intelligens er en autonom agent en enhet som opererer uavhengig i et miljø. Den bruker sensorer til å oppfatte tilstander og aktuatorer til å utføre handlinger, styrt av interne prosesser.

### Summary

En autonom agent er et system som kan oppfatte miljøet sitt og handle uavhengig for å oppnå spesifikke mål uten direkte menneskelig innblanding.

## Key Concepts

- Oppfatning
- Handling
- Målrettet atferd
- Interaksjon med miljøet

- Robotprosessautomatisering
- Spillende AI
