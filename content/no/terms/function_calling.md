## Definition

Funksjonskalling gjør det mulig for store språkmodeller å samhandle med eksterne verktøy og API-er ved å generere strukturerte utdata, som JSON-objekter, som spesifiserer hvilken funksjon som skal kjøres og hvilke argumenter som skal brukes.

### Summary

En mekanisme som lar store språkmodeller (LLM) produsere strukturert data som utløser spesifikke programvarefunksjoner.

## Key Concepts

- Strukturert utdata
- API-integrasjon
- Verktøybruk
- JSON-skjema

## Use Cases

- Bestilling av flybilletter via reise-API-er
- Utføre komplekse matematiske beregninger
- Spørre etter poster i databaser
