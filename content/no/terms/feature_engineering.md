## Definition

Egenskapsingeniørfag er kunsten å utnytte fagekspertise til å transformere rådata til egenskaper som bedre representerer de underliggende mønstrene for maskinlæringsalgoritmer. Denne prosessen inkluderer opprettelse av nye variabler.

### Summary

Praksisen med å bruke fagkunnskap til å lage nye egenskaper eller modifisere eksisterende for å forbedre ytelsen til maskinlæringsmodeller.

## Key Concepts

- Fagkunnskap
- Datatransformasjon
- Modellytelse
- Variabelopprettelse

## Use Cases

- Forbedring av nøyaktigheten til regresjonsmodeller
- Forsterkning av klassifiseringsgrenser
- Optimalisering av tidsserieforspåinger

## Code Example

```python
df['new_feature'] = df['feature_a'] * df['feature_b']
```
