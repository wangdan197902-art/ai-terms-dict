## Definition

Skykomputasjon gir skalerbar infrastruktur for AI-arbeidsbelastninger, og lar utviklere få tilgang til kraftige GPU-er og lagringskapasitet uten å vedlikeholde fysiske datasentre. Det støtter ulike tjenestemodeller som IaaS, PaaS og SaaS, noe som muliggjør fleksibel ressursbruk.

### Summary

Skyen refererer til eksterne servere hostet på internett som brukes til å lagre, administrere og behandle data og AI-modeller i stedet for lokal maskinvare.

## Key Concepts

- Skalerbarhet
- Infrastruktur som en tjeneste
- Ekstern lagring
- Elastisitet

## Use Cases

- Trening av store språkmodeller
- Vertskap for AI-API-er til webapplikasjoner
- Lagring av enorme datasett for analyse
