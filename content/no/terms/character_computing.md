## Definition

Dette konseptet fokuserer på manipulering av tekst der den grunnleggjande eininga for berekning er eit enkelt teikn. Det blir ofte brukt i oppgåver som krev finmaska tekstanalyse, slik som stavkontroll.

### Summary

Teiknberegning involverer prosessering, generering eller analyse av tekst på nivået enkeltteikn, i staden for ord- eller setningsnivå.

## Key Concepts

- Tokenisering
- Tekstgenerering
- Finmaska analyse
- Delord-einingar

## Use Cases

- Handskriftgjenkjenning (OCR)
- Språkmodellering for språk med lite ressursar
- Analyse av kryptert tekst
