## Definition

AutoML (Automated Machine Learning) strammer opp utviklingen av ML-modeller ved å automatisere oppgaver som dataforhåndsbehandling, funksjonsutvinning, modellvalg og hyperparameteroptimalisering. Det muliggjør bredere tilgang.

### Summary

En metodikk som automatiserer den komplette prosessen med å anvende maskinlæring på virkelige problemer, noe som reduserer manuelt arbeid.

## Key Concepts

- Hyperparameteroptimalisering
- Funksjonsutvinning
- Modellvalg
- Demokratisering

## Use Cases

- Rapid prototyping for forretningsanalytikere
- Optimering av produksjonspipelines i stor skala
- Sammenligning av flere algoritmer automatisk

## Code Example

```python
from auto_ml import Predictor
predictor = Predictor(type_of_estimator='classifier')
predictor.fit(dataframe, target='label')
```
