## Definition

REST--APIer muliggjør kommunikasjon mellom klienter og servere ved å bruke tilstandsløse operasjoner over HTTP-protokoller som GET, POST, PUT og DELETE. De strukturerer ressurser som URI-er og bruker standard format for datautveksling.

### Summary

Representational State Transfer (REST) er en arkitektonisk stil for design av nettbaserte applikasjoner som stoler på standard HTTP-metoder.

## Key Concepts

- Tilstandsløshet
- HTTP-metoder
- Ressursbaserte URI-er
- JSON-serialisering

## Use Cases

- Kommunikasjon mellom mikrotjenester
- Backend-integrasjon for mobilapper
- Tilgang til tredjepartsdata
