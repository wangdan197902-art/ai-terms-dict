## Definition

Promptingeniørfag involverer utforming av spesifikke inndata, kjent som prompts, for å fremkalle nøyaktige, relevante og høykvalitets svar fra generative AI-modeller. Det krever forståelse av hvordan modeller tolker kontekst og instruksjoner for å oppnå ønsket resultat.

### Summary

Praksisen med å designe og optimalisere inndata for å veilede store språkmodeller mot ønskede utdata.

## Key Concepts

- Kontekstuell rammesetting
- Læring med få eksempler
- Instruksjonstilpasning
- Optimalisering av tokens

## Use Cases

- Automatiserte kundeserviceroboter
- Assistentprogrammer for kodegenerering
- Hjelpemidler for kreativ skriving

## Code Example

```python
prompt = "Translate the following English text to French: 'Hello world'"
response = llm.generate(prompt)
```
