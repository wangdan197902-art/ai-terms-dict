## Definition

Webintelligens involverer bruk av datagruvedrift, maskinlæring og semantiske teknologier for å behandle den enorme mengden ustrukturerte data som finnes på internett. Målet er å omdanne rå webdata til nyttig kunnskap og innsikt.

### Summary

Bruken av intelligente teknikker for å hente ut, administrere og analysere informasjon fra Verdensveven.

## Key Concepts

- Datagruvedrift
- Semantisk web
- Lenkeanalyse
- Innholds klassifisering

## Use Cases

- Konkurransedyktig markedsanalyse
- Overvåking av stemninger på sosiale medier
- Søkemotoroptimalisering (SEO)
