## Definition

Model Context Protocol (MCP) er en åpen standard som gjør det mulig for AI-applikasjoner å koble seg til ulike datakilder, som databaser, API-er og filsystemer, på en enhetlig måte. Det abstraherer kompleksiteten ved integrasjon og gir en felles protokoll for kontekstutveksling.

### Summary

En standardisert rammeverk designet for å muliggjøre sikker og effektiv kommunikasjon mellom AI-modeller og eksterne datakilder eller verktøy.

## Key Concepts

- Interoperabilitet
- Standardisering
- Dataintegrering
- Verktøykalling

## Use Cases

- Å koble store språkmodeller (LLMs) til bedriftsdata
- Bygge bærbar AI-assistent
- Sikker API-integrasjon
