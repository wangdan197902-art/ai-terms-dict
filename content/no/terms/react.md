## Definition

ReAct-rammeverket lar store språkmodeller generere både resonneringstrinn og oppgavespesifikke handlinger i en vevd sekvens. Ved å simulere menneskelignende tenkeprosesser, muliggjør det at modellene interagerer med eksterne systemer.

### Summary

ReAct er et prompt-paradigme som kombinerer resonnering og handling for å forbedre ytelsen til store språkmodeller på komplekse oppgaver.

## Key Concepts

- Vevd resonnering
- Handlingsgenerering
- Integrasjon av observasjoner
- Tenkekjede

## Use Cases

- Komplekse spørsmål-og-svar-systemer
- Autonome agenter
- Arbeidsflyter for faktagransking
