## Definition

Dette kritiske steget innebærer å knytte meningsfull metadata til rådatapunkter slik at algoritmer kan lære sammenhengen mellom input og output. For eksempel, grenser rundt objekter i bilder

### Summary

Dataannotering er prosessen med å merke rådata, som bilder eller tekst, for å gjøre dem egnet for overvåket maskinlæringstrening.

## Key Concepts

- Overvåket læring
- Merking
- Sanntidsdata (Ground Truth)
- Metadata

## Use Cases

- Trening av objektdeteksjonsmodeller
- Bygging av klassifiserere for sentimentanalyse
- Opprettelse av datasett for talegjenkjenning
