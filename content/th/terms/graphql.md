## Definition

พัฒนาโดย Facebook GraphQL ให้คำอธิบายข้อมูลที่สมบูรณ์และเข้าใจง่ายของ API ช่วยให้ผู้ใช้งานสามารถขอข้อมูลที่ต้องการเท่านั้น ไม่มีมากหรือน้อยเกินไป ซึ่งช่วยแก้ปัญหา Over-fetching และ Under-fetching ที่พบใน REST API แบบดั้งเดิม

### Summary

GraphQL คือภาษาสำหรับเขียน Query ของ API และ Runtime สำหรับดำเนินการตาม Query นั้นกับข้อมูลที่มีอยู่ ช่วยให้ไคลเอนต์สามารถขอข้อมูลเฉพาะที่ต้องการได้อย่างแม่นยำ

## Key Concepts

- การกำหนด Schema
- ภาษา Query
- Single Endpoint (จุดปลายเดียว)
- ความปลอดภัยของประเภทข้อมูล (Type Safety)

## Use Cases

- Backend ของเว็บแอปพลิเคชันสมัยใหม่
- การดึงข้อมูลสำหรับแอปพลิเคชันมือถือ
- การประสานงานระหว่าง Microservices
