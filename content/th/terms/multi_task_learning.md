## Definition

เทคนิคนี้ใช้ประโยชน์จากอคติเชิงอุปนัย (inductive bias) ที่ร่วมกันระหว่างงานที่เกี่ยวข้องกันเพื่อเพิ่มประสิทธิภาพและประสิทธิภาพของการเรียนรู้ โดยการฝึกโมเดลเดียวให้ทำงานหลายอย่างพร้อมกัน โมเดลจะเรียนรู้การแสดงแทนร่วม

### Summary

การเรียนรู้หลายงานคือกระบวนทัศน์แมชชีนเลิร์นนิงที่ฝึกโมเดลบนงานที่เกี่ยวข้องกันหลายงานพร้อมกัน เพื่อปรับปรุงความสามารถในการสรุปผล (generalization)

## Key Concepts

- การแสดงแทนร่วม (Shared Representation)
- อคติเชิงอุปนัย (Inductive Bias)
- การรบกวนระหว่างงาน (Task Interference)
- ความสามารถในการสรุปผล (Generalization)

## Use Cases

- การประมวลผลภาษาธรรมชาติ (เช่น การระบุชื่อเอนทิตีและการทำเครื่องหมายส่วนพูด)
- การมองเห็นด้วยคอมพิวเตอร์ (เช่น การตรวจจับและการแบ่งส่วนภาพ)
- การรู้จำเสียงพูดและการสังเคราะห์เสียง
