## Definition

Flickr30K Captions เป็นชุดข้อมูลมาตรฐานที่ใช้กันอย่างแพร่หลาย ประกอบด้วยรูปภาพ 31,783 รูป แต่ละรูปมีคำบรรยายภาษาอังกฤษ 5 ประโยคที่อธิบายเนื้อหาภาพ ชุดข้อมูลนี้เป็นทรัพยากรพื้นฐานสำคัญสำหรับการเรียนรู้ข้ามรูปแบบระหว่างภาพและข้อความ

### Summary

ชุดข้อมูลหลายรูปแบบ (Multimodal) ที่เชื่อมโยงรูปภาพ 31,000 รูปกับคำบรรยายที่มนุษย์เขียน เพื่อใช้ฝึกโมเดล Embedding ข้ามรูปแบบ (Cross-modal)

## Key Concepts

- การเรียนรู้หลายรูปแบบ (Multimodal Learning)
- การจัดตำแหน่งภาพและข้อความ (Image-Text Alignment)
- การค้นหาข้ามรูปแบบ (Cross-Modal Retrieval)
- การระบุตำแหน่งวัตถุในภาพ (Visual Grounding)

## Use Cases

- สร้างระบบค้นหาภาพ
- สร้างคำบรรยายภาพ (Image Description Generation)
- ฝึกฝนโมเดลภาษาและภาพ (Vision-Language Models)
