## Definition

กราฟเป็นโครงสร้างข้อมูลพื้นฐานในปัญญาประดิษฐ์ ประกอบด้วยจุดยอด (โหนด) และเส้นเชื่อม (ขอบ) ที่แสดงถึงความสัมพันธ์ เครือข่ายประสาทเทียมแบบกราฟ (GNNs) ใช้โครงสร้างนี้เพื่อการเรียนรู้บนข้อมูลที่ไม่ใช่แบบตาราง (non-Euclidean data)

### Summary

โครงสร้างทางคณิตศาสตร์ที่ประกอบด้วยโหนดซึ่งเชื่อมต่อกันด้วยเส้นขอบ ใช้เพื่อแสดงความสัมพันธ์ระหว่างสิ่งต่างๆ

## Key Concepts

- โหนด/จุดยอด (Nodes/Vertices)
- เส้นเชื่อม (Edges)
- เมทริกซ์ประชิด (Adjacency Matrix)
- โทโพโลยี (Topology)

## Use Cases

- กราฟความรู้ (Knowledge Graphs)
- การวิเคราะห์เครือข่ายสังคม (Social Network Analysis)
- การทำนายคุณสมบัติของโมเลกุล (Molecular Property Prediction)
