## Definition

กรอบงาน ReAct ช่วยให้ LLM สามารถสร้างร่องรอยการให้เหตุผลและการกระทำเฉพาะงานแบบสลับกันไปมา โดยการจำลองกระบวนการคิดแบบมนุษย์ ทำให้โมเดลสามารถโต้ตอบกับสิ่งแวดล้อมภายนอกเพื่อแก้ปัญหาได้อย่างมีประสิทธิภาพ

### Summary

ReAct คือรูปแบบการตั้งคำสั่ง (prompting paradigm) ที่ผสมผสานระหว่างการให้เหตุผลและการกระทำ เพื่อปรับปรุงประสิทธิภาพของโมเดลภาษาขนาดใหญ่ (LLM) ในงานที่ซับซ้อน

## Key Concepts

- การให้เหตุผลแบบสลับกัน (Interleaved Reasoning)
- การสร้างการกระทำ (Action Generation)
- การบูรณาการข้อมูลสังเกตการณ์ (Observation Integration)
- โซ่แห่งความคิด (Chain-of-Thought)

## Use Cases

- ระบบตอบคำถามที่ซับซ้อน (Complex QA systems)
- เอเจนต์อัตโนมัติ (Autonomous agents)
- เวิร์กโฟลว์การตรวจสอบข้อเท็จจริง (Fact-checking workflows)
