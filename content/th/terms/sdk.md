## Definition

SDK คือชุดเครื่องมือสำหรับการพัฒนาซอฟต์แวร์ที่ช่วยให้ผู้พัฒนาสามารถสร้างแอปพลิเคชันสำหรับแพลตฟอร์มหรือบริการเฉพาะได้ สำหรับงานด้าน AI SDK จะมอบไลบรารีสำเร็จรูป API และยูทิลิตี้ต่างๆ เพื่อให้การพัฒนาโมเดลและการบูรณาการเป็นไปอย่างสะดวกและรวดเร็ว

### Summary

ชุดเครื่องมือพัฒนาซอฟต์แวร์ที่ประกอบด้วยไลบรารี เอกสารประกอบ และเครื่องมืออื่นๆ สำหรับการสร้างแอปพลิเคชัน

## Key Concepts

- การบูรณาการ API
- ไลบรารี
- เครื่องมือพัฒนาซอฟต์แวร์
- การนามธรรม (Abstraction)

## Use Cases

- การนำโมเดลภาษาขนาดใหญ่ (LLMs) ไปรวมเข้ากับแอปพลิเคชัน
- การสร้างสายการผลิตสำหรับคอมพิวเตอร์วิทัศน์
- การนำโมเดลแมชชีนเลิร์นนิงไปใช้งานจริง (Production)
