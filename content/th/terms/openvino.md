## Definition

พัฒนาโดย Intel OpenVINO (Open Visual Inference and Neural network Optimization) ช่วยให้นักพัฒนาสามารถนำโมเดลดีปเลิร์นนิงที่ผ่านการฝึกฝนแล้ว ไปใช้งานอย่างมีประสิทธิภาพบนฮาร์ดแวร์ของ Intel โดยประกอบด้วยเครื่องมือและไลบรารีที่ช่วยเพิ่มความเร็วและลดการใช้ทรัพยากรในการอนุมานผล (inference)

### Summary

OpenVINO คือ ชุดเครื่องมือโอเพนซอร์สโดย Intel สำหรับปรับแต่งและนำโมเดลดีปเลิร์นนิงไปใช้งานบนแพลตฟอร์มฮาร์ดแวร์หลากหลายชนิดอย่างมีประสิทธิภาพ

## Key Concepts

- การปรับแต่งโมเดล (Model Optimization)
- การเร่งความเร็วด้วยฮาร์ดแวร์ (Hardware Acceleration)
- เครื่องอนุมานผล (Inference Engine)
- การประมวลผลที่ขอบ (Edge Computing)
- การรองรับข้ามเฟรมเวิร์ก (Cross-Framework Support)

## Use Cases

- การนำโมเดลตรวจจับวัตถุไปใช้งานบนซีพียู/จีพียูของ Intel (Deploying object detection on Intel CPUs/GPUs)
- การปรับปรุงสายพานการผลิตสำหรับการวิเคราะห์วิดีโอ (Optimizing video analytics pipelines)
- การรันการอนุมานผลแบบเรียลไทม์บนอุปกรณ์ขอบ (Running real-time inference on edge devices)
