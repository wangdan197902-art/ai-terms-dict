## Definition

ชุดข้อมูล WikiHow ประกอบด้วยบทความวิธีการทำสิ่งต่างๆ ประมาณ 60,000 บทความที่รวบรวมจากเว็บไซต์ WikiHow ชุดข้อมูลนี้ถูกใช้อย่างกว้างขวางในงานวิจัยด้านการประมวลผลภาษาธรรมชาติ (NLP) สำหรับงานต่างๆ เช่น การสรุปความแบบนามธรรม (abstractive text summarization) และการสร้างลำดับขั้นตอนการทำงาน

### Summary

ชุดข้อมูลขนาดใหญ่ที่ประกอบด้วยบทความวิธีการทำสิ่งต่างๆ จากเว็บไซต์ WikiHow ซึ่งใช้หลักๆ ในงานสรุปความและการสร้างคำแนะนำ

## Key Concepts

- การสรุปความข้อความ (Text summarization)
- ข้อความเชิงขั้นตอน (Procedural text)
- การสกัดคำแนะนำ (Instruction extraction)

## Use Cases

- การฝึกโมเดลสรุปความแบบนามธรรม (Abstractive summarization model training)
- การสร้างคำแนะนำแบบทีละขั้นตอน (Step-by-step instruction generation)
- การทำความเข้าใจโครงสร้างภาษาเชิงกระบวนการ (Understanding procedural language structures)
