## Definition

Code Search Net เป็นชุดข้อมูลอันสมบูรณ์ที่สร้างขึ้นเพื่อส่งเสริมการวิจัยด้านการดึงคืนโค้ด ประกอบด้วยคู่อธิบายภาษาธรรมชาติและท่อนโค้ด Java ที่สอดคล้องกันจำนวนหลายล้านคู่ ข้อมูลนี้มีวัตถุประสงค์เพื่อ...

### Summary

ชุดข้อมูลมาตรฐานที่ออกแบบมาเพื่อประเมินระบบค้นหาโค้ด โดยการจับคู่คำถามด้วยภาษาธรรมชาติกับท่อนโค้ดที่เกี่ยวข้อง

## Key Concepts

- การดึงคืนโค้ด (Code Retrieval)
- การค้นหาเชิงความหมาย (Semantic Search)
- โค้ด Java (Java Code)
- การประมวลผลภาษาธรรมชาติ (Natural Language Processing)

## Use Cases

- การสร้างปลั๊กอินสำหรับ IDE
- การวิจัยความเข้าใจในโค้ด
- การสร้างเอกสารประกอบโค้ดอัตโนมัติ
