## Definition

ฐานข้อมูลอัจฉริยะใช้การเรียนรู้ของเครื่องและ AI เพื่อเสริมฟังก์ชันการทำงานมาตรฐานของฐานข้อมูลให้เกินกว่าการจัดเก็บและการเรียกค้นข้อมูลทั่วไป สามารถปรับปรุงประสิทธิภาพการสอบถามโดยอัตโนมัติ คาดการณ์การใช้งาน

### Summary

ระบบฐานข้อมูลที่รวมความสามารถของ AI เข้ามาเพื่อจัดการข้อมูลอัตโนมัติ ปรับปรุงประสิทธิภาพการสอบถาม และสร้างข้อมูลเชิงลึก

## Key Concepts

- การเพิ่มประสิทธิภาพการสอบถาม (Query Optimization)
- การตรวจจับความผิดปกติ (Anomaly Detection)
- การปรับแต่งตนเอง (Self-Tuning)
- การสอบถามด้วยภาษาธรรมชาติ (Natural Language Querying)

## Use Cases

- การปรับขนาดทรัพยากรฐานข้อมูลบนคลาวด์อัตโนมัติตามภาระงาน
- การตรวจจับธุรกรรมฉ้อโกงในระบบธนาคารแบบเรียลไทม์
- การสร้างคิวรี SQL จากคำถามด้วยภาษาธรรมชาติ
