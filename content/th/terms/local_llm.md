## Definition

การรัน Local LLM เกี่ยวข้องกับการติดตั้งแบบจำลองน้ำหนักเปิดโดยตรงบนฮาร์ดแวร์ระดับผู้บริโภค เช่น พีซี แมค หรือเซิร์ฟเวอร์ภายในองค์กร วิธีนี้ช่วยลดการพึ่งพาผู้ให้บริการ API ของบุคคลที่สาม

### Summary

Local LLM หมายถึงการรันแบบจำลองภาษาขนาดใหญ่บนฮาร์ดแวร์ส่วนบุคคลแทนที่จะใช้บริการบนคลาวด์ โดยให้ความสำคัญกับความเป็นส่วนตัวของข้อมูลและการเข้าถึงแบบออฟไลน์

## Key Concepts

- การอนุมานบนอุปกรณ์ (On-device inference)
- ความเป็นส่วนตัวของข้อมูล (Data privacy)
- การลดทอนขนาดโมเดล (Quantization)
- ข้อกำหนดด้านฮาร์ดแวร์ (Hardware requirements)

## Use Cases

- ผู้ช่วยบันทึกโน้ตแบบส่วนตัว
- เครื่องมือวิจัยแบบออฟไลน์
- การประมวลผลข้อมูลองค์กรอย่างปลอดภัย
