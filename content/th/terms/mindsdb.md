## Definition

MindsDB ทำหน้าที่เป็นสะพานเชื่อมระหว่างฐานข้อมูลเชิงสัมพันธ์แบบดั้งเดิมกับเวิร์กโฟลว์การเรียนรู้ของเครื่องสมัยใหม่ ช่วยให้ผู้ใช้งานสร้างโมเดลการทำนายโดยใช้คำสั่ง SQL มาตรฐาน ลดความจำเป็นในการย้ายข้อมูลหรือใช้เครื่องมือภายนอก

### Summary

MindsDB เป็นแพลตฟอร์มโอเพนซอร์สที่ช่วยให้นักพัฒนาสร้าง ฝึกฝน และปรับใช้โมเดลการเรียนรู้ของเครื่องโดยตรงภายในฐานข้อมูลที่มีอยู่ โดยใช้ภาษา SQL

## Key Concepts

- การเรียนรู้ของเครื่องผ่าน SQL (SQL-based ML)
- การผสานรวมฐานข้อมูล (Database integration)
- AutoML (การเรียนรู้ของเครื่องอัตโนมัติ)
- การสร้างโมเดลทำนาย (Predictive modeling)

## Use Cases

- การตรวจจับการฉ้อโกงแบบเรียลไทม์ในฐานข้อมูล
- การทำนายอัตราการยกเลิกบริการลูกค้า (Churn prediction)
- รายงานอัตโนมัติพร้อมการพยากรณ์แนวโน้ม
