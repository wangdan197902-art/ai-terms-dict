## Definition

การให้บริการโมเดลเกี่ยวข้องกับการนำโมเดลที่ผ่านการฝึกฝนแล้วซึ่งอยู่ในสถานะคงที่ มาบรรจุไว้ในโครงสร้างพื้นฐานที่สามารถขยายขนาดได้ เพื่อจัดการกับคำขอที่เข้ามา ดำเนินการทำนาย (inference) และส่งกลับผลลัพธ์ ความท้าทายหลัก ได้แก่ การจัดการทรัพยากรและการลดเวลาแฝง

### Summary

กระบวนการนำโมเดลแมชชีนเลิร์นนิงที่ได้รับการฝึกฝนแล้วไปใช้งานจริงในสภาพแวดล้อมการผลิต เพื่อทำการทำนายหรือสร้างผลลัพธ์สำหรับผู้ใช้งานปลายทาง

## Key Concepts

- การทำนาย (Inference)
- การเพิ่มประสิทธิภาพเวลาแฝง
- ความสามารถในการขยายขนาด
- การปรับใช้

## Use Cases

- ระบบแนะนำสินค้าแบบเรียลไทม์
- API สำหรับการจัดประเภทภาพ
- บริการประมวลผลภาษาธรรมชาติ
