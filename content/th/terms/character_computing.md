## Definition

แนวคิดนี้มุ่งเน้นไปที่การจัดการข้อความโดยที่หน่วยพื้นฐานของการคำนวณคือตัวอักษรเดียว มักใช้ในงานที่ต้องการการวิเคราะห์ข้อความอย่างละเอียด เช่น การตรวจสอบการสะกดคำ

### Summary

การประมวลผล สร้าง หรือวิเคราะห์ข้อความในระดับตัวอักษรแต่ละตัว แทนที่จะเป็นระดับคำหรือประโยค

## Key Concepts

- การแบ่ง Token (Tokenization)
- การสร้างข้อความ (Text Generation)
- การวิเคราะห์อย่างละเอียด (Fine-grained Analysis)
- หน่วยย่อยของคำ (Subword Units)

## Use Cases

- การรู้จำลายมือเขียน (OCR)
- การสร้างแบบจำลองภาษาสำหรับภาษาที่มีทรัพยากรน้อย
- การวิเคราะห์ข้อความเข้ารหัส
