## Definition

การใช้เครื่องมือช่วยให้โมเดลภาษาสามารถโต้ตอบกับสภาพแวดล้อมซอฟต์แวร์ภายนอกได้โดยการเรียกใช้ฟังก์ชันที่กำหนดไว้ล่วงหน้า เช่น เครื่องคิดเลข เครื่องมือค้นหา หรือการสอบถามฐานข้อมูล แนวทางนี้ขยายขีดความสามารถของโมเดล

### Summary

รูปแบบการทำงานที่เอเจนต์ AI เลือกและเรียกใช้ฟังก์ชันภายนอกหรือ API เพื่อทำงานเฉพาะด้านที่เกินขีดความสามารถพื้นฐานของโมเดล

## Key Concepts

- การเรียกใช้ฟังก์ชัน
- สถาปัตยกรรมเอเจนต์
- การผสานรวม API
- ความรู้ภายนอก

## Use Cases

- บอทบริการลูกค้าอัตโนมัติ
- ผู้ช่วยวิเคราะห์ข้อมูล
- ระบบบ้านอัจฉริยะ
