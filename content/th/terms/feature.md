## Definition

ในการเรียนรู้ของเครื่อง Feature คือคุณลักษณะหรือตัวแปรที่แตกต่างกันซึ่งอธิบายกรณีศึกษาหนึ่งๆ ภายในชุดข้อมูล Feature สามารถเป็นได้ทั้งตัวเลข หมวดหมู่ หรือข้อความ และทำหน้าที่เป็นข้อมูลพื้นฐานที่สำคัญ

### Summary

คุณสมบัติหรือลักษณะเฉพาะที่วัดค่าได้ของปรากฏการณ์ที่กำลังสังเกต ซึ่งทำหน้าที่เป็นข้อมูลป้อนเข้าสำหรับโมเดลการเรียนรู้ของเครื่อง

## Key Concepts

- ตัวแปรป้อนเข้า
- คุณลักษณะ
- คอลัมน์ในชุดข้อมูล
- ข้อมูลป้อนเข้า ML

## Use Cases

- ข้อมูลป้อนเข้าสำหรับการสร้างแบบจำลองทำนาย
- การวิเคราะห์และแสดงภาพข้อมูล
- กระบวนการเลือก Feature
