## Definition

ตรรกะคำอธิบาย (Description Logics - DL) เป็นส่วนย่อยของตรรกะอันดับหนึ่งที่สามารถตัดสินใจได้ (decidable) ซึ่งเป็นรากฐานทางทฤษฎีสำหรับออนโทโลยี โดยเฉพาะภาษาเว็บออนโทโลยี (OWL) พวกมันอนุญาตให้กำหนดนิยามอย่างแม่นยำเกี่ยวกับแนวคิดและความสัมพันธ์...

### Summary

ตระกูลของภาษาการแสดงความรู้เชิงรูปนัยที่ใช้แทนและให้เหตุผลเกี่ยวกับโครงสร้างแนวคิดของโดเมนต่างๆ

## Key Concepts

- Ontology (ออนโทโลยี)
- Reasoning (การให้เหตุผล)
- Subsumption (การครอบคลุม)
- Concept Hierarchy (ลำดับชั้นแนวคิด)

## Use Cases

- การสร้างออนโทโลยีสำหรับเว็บเชิงความหมาย
- กราฟความรู้ทางการแพทย์
- ระบบการจัดประเภทอัตโนมัติ
