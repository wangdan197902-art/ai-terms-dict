## Definition

Google Colaboratory หรือที่เรียกกันทั่วไปว่า Colab คือบริการ Jupyter Notebook แบบโฮสต์ที่ไม่ต้องการการติดตั้งใดๆ และให้การเข้าถึงทรัพยากรการคำนวณฟรี รวมถึงหน่วยประมวลผลกราฟิก (GPUs) ด้วย

### Summary

สภาพแวดล้อม Jupyter Notebook บนคลาวด์ที่อนุญาตให้ผู้ใช้เขียนและเรียกใช้โค้ด Python ผ่านเว็บเบราว์เซอร์ พร้อมเข้าถึง GPU ได้ฟรี

## Key Concepts

- Cloud Computing
- Jupyter Notebook
- GPU Acceleration
- Python Programming

## Use Cases

- Machine Learning Prototyping
- Data Science Education
- Collaborative Coding
