## Definition

ฟาย (Phi) ย่อมาจาก 'Foundation models based on Teaching-Learning Paradigm' เป็นตระกูลของโมเดลภาษาขนาดใหญ่ที่มีขนาดกะทัดรัดซึ่งสร้างโดย Microsoft ต่างจากโมเดลภาษาขนาดใหญ่แบบดั้งเดิมที่ฝึกฝนด้วยข้อมูลมหาศาลจากเว็บ ฟายมุ่งเน้นไปที่ข้อมูลคุณภาพสูงที่สังเคราะห์ขึ้นเพื่อเพิ่มประสิทธิภาพ

### Summary

ชุดโมเดลภาษาขนาดใหญ่ขนาดเล็กแต่มีประสิทธิภาพสูง ซึ่งพัฒนาโดย Microsoft Research โดยเน้นที่ความหนาแน่นของความรู้และความสามารถในการให้เหตุผล

## Key Concepts

- โมเดลภาษาขนาดเล็ก (Small Language Models)
- ข้อมูลสังเคราะห์ (Synthetic Data)
- การให้เหตุผล (Reasoning)
- ความหนาแน่นของความรู้ (Knowledge Density)

## Use Cases

- การนำ AI ไปใช้งานบนอุปกรณ์ขอบ (Edge devices)
- งานที่ต้องใช้การให้เหตุผลเชิงตรรกะที่ซับซ้อน
- บริการ API ที่มีต้นทุนประหยัด
