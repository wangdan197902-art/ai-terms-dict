## Definition

ในบริบทของปัญญาประดิษฐ์และวิทยาศาสตร์ข้อมูล คำว่าเชิงตัวเลขหมายถึงประเภทของข้อมูลหรือวิธีการที่เกี่ยวข้องกับค่าปริมาณ เช่น จำนวนเต็ม ทศนิยม และจำนวนลอย ซึ่งแตกต่างจากข้อมูลเชิงหมวดหมู่หรือข้อความ

### Summary

เกี่ยวข้องกับ ใช้ หรืออ้างอิงถึงตัวเลขหรือการคำนวณทางคณิตศาสตร์ แทนที่จะเป็นตรรกะเชิงสัญลักษณ์หรือข้อความ

## Key Concepts

- จำนวนลอย (Floats)
- จำนวนเต็ม (Integers)
- ประเภทข้อมูล (Data Types)
- ข้อมูลเชิงปริมาณ (Quantitative Data)

## Use Cases

- การสร้างแบบจำลองทางสถิติ
- การพยากรณ์ทางการเงิน
- การประมวลผลข้อมูลจากเซนเซอร์

## Code Example

```python
import numpy as np
arr = np.array([1.5, 2.7, 3.9])
mean_val = np.mean(arr)
print(mean_val)
```
