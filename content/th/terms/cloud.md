## Definition

การคำนวณแบบคลาวด์มอบโครงสร้างพื้นฐานที่ปรับขนาดได้สำหรับงานปัญญาประดิษฐ์ ช่วยให้开发人员สามารถเข้าถึง GPU ที่มีประสิทธิภาพและพื้นที่จัดเก็บข้อมูลโดยไม่ต้องดูแลศูนย์ข้อมูลทางกายภาพ รองรับรูปแบบบริการต่างๆ เช่น IaaS, PaaS และ SaaS

### Summary

คลาวด์หมายถึงเซิร์ฟเวอร์ระยะไกลที่โฮสต์บนอินเทอร์เน็ต ใช้สำหรับจัดเก็บ จัดการ และประมวลผลข้อมูลและโมเดลปัญญาประดิษฐ์ แทนการใช้ฮาร์ดแวร์ในท้องถิ่น

## Key Concepts

- ความสามารถในการขยายขนาด
- โครงสร้างพื้นฐานเป็นบริการ (IaaS)
- พื้นที่จัดเก็บข้อมูลระยะไกล
- ความยืดหยุ่น

## Use Cases

- การฝึกโมเดลภาษาขนาดใหญ่
- การโฮสต์ API สำหรับแอปพลิเคชันเว็บ
- การจัดเก็บชุดข้อมูลขนาดใหญ่วิเคราะห์
