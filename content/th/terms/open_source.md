## Definition

โอเพนซอร์สหมายถึงรูปแบบการพัฒนาที่โค้ดพื้นฐานของโปรเจกต์ซอฟต์แวร์สามารถเข้าถึงได้โดยสาธารณะ ในด้าน AI สิ่งนี้ทำให้ผู้วิจัยและนักพัฒนาสามารถตรวจสอบ แก้ไข และกระจายอัลกอริทึมใหม่ได้

### Summary

ซอฟต์แวร์whose source code ถูกเปิดเผยให้ใช้งาน แก้ไข และแจกจ่ายได้อย่างอิสระโดยทุกคน

## Key Concepts

- ความโปร่งใส
- ความร่วมมือ
- การเข้าถึงได้
- ขับเคลื่อนโดยชุมชน

## Use Cases

- การพัฒนาเฟรมเวิร์ก PyTorch
- ไลบรารีโมเดลของ Hugging Face
- ความสามารถในการทำซ้ำในงานวิจัยทางวิชาการ
