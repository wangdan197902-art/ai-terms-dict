## Definition

การปรับใช้บน Azure เกี่ยวข้องกับการใช้เครื่องมือที่เป็นเจ้าของคลาวด์ (cloud-native) เช่น Azure Machine Learning, Azure Kubernetes Service (AKS) หรือ Azure Functions เพื่อให้บริการโมเดล ML ในระดับใหญ่ ครอบคลุมการจัดการทรัพยากรการคำนวณและการปรับขนาด...

### Summary

กระบวนการโฮสต์และรันโมเดลแมชชีนเลิร์นนิงบนโครงสร้างพื้นฐานบริการคลาวด์ของ Microsoft Azure

## Key Concepts

- Cloud Infrastructure (โครงสร้างพื้นฐานคลาวด์)
- Model Serving (การให้บริการโมเดล)
- Scalability (ความสามารถในการขยายขนาด)
- CI/CD Pipelines (สายพานการผลิต CI/CD)

## Use Cases

- จุดปลาย API แบบเรียลไทม์สำหรับการอนุมาน
- การให้คะแนนแบบแบตช์สำหรับชุดข้อมูลขนาดใหญ่
- การจัดการโมเดลระดับองค์กร
