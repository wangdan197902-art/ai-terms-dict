## Definition

Llama 3.1 ซึ่งเผยแพร่ในเดือนสิงหาคม 2024 ขยายตระกูล Llama ให้รวมถึงโมเดลขนาดมหึวา 405 พันล้านพารามิเตอร์ alongside กับเวอร์ชันขนาดเล็กกว่าคือ 8B และ 70B จุดเด่นคือหน้าต่างบริบทที่ขยายใหญ่ขึ้น

### Summary

Llama 3.1 คือการอัปเดตของ Meta ในเดือนสิงหาคม 2024 ซึ่งมีโมเดลขนาด 8B, 70B และ 405B พร้อมหน้าต่างบริบทที่ขยายใหญ่ขึ้นถึง 128k และความสามารถในการใช้งานเครื่องมือ (tool-use) โดยตรง

## Key Concepts

- หน้าต่างบริบทยาว (Long context window)
- การเรียกใช้ฟังก์ชัน (Function calling)
- การใช้งานเครื่องมือ (Tool use)
- พารามิเตอร์ 405B

## Use Cases

- การวิเคราะห์เอกสารความยาวมาก
- เอเจนต์สำหรับการผสานรวม API
- การสังเคราะห์โค้ดขั้นสูง
