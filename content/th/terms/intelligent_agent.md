## Definition

เอเจนต์อัจฉริยะ คือ ระบบที่มีความสามารถในการรับรู้สภาพแวดล้อมรอบข้างผ่านเซ็นเซอร์หรือข้อมูลป้อนเข้า ประมวลผลข้อมูลนี้โดยใช้อัลกอริทึมการให้เหตุผล และกระทำต่อสภาพแวดล้อมผ่านแอคชูเอเตอร์ (actuators)

### Summary

เอนทิตีซอฟต์แวร์ที่มีอิสระ สามารถรับรู้สภาพแวดล้อม ให้เหตุผลเกี่ยวกับขั้นตอนการดำเนินการ และทำงานต่างๆ เพื่อให้บรรลุเป้าหมายเฉพาะ

## Key Concepts

- ความเป็นอิสระ (Autonomy)
- วงจรการรับรู้และการกระทำ (Perception-Action Loop)
- การมุ่งเน้นเป้าหมาย (Goal Orientation)
- การตอบสนอง (Reactivity)

## Use Cases

- บอทซื้อขายอัตโนมัติที่ดำเนินการกลยุทธ์ตลาด
- การทำให้กระบวนการหุ่นยนต์เป็นอัตโนมัติ (RPA) จัดการเวิร์กโฟลว์หลายขั้นตอน
- ผู้ช่วยส่วนตัวจัดการปฏิทินและงานอีเมล
