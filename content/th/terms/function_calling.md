## Definition

การเรียกใช้ฟังก์ชันช่วยให้โมเดลภาษาขนาดใหญ่สามารถโต้ตอบกับเครื่องมือภายนอกและ API ได้โดยการสร้างเอาต์พุตที่มีโครงสร้าง เช่น วัตถุ JSON ซึ่งระบุว่าจะต้องเรียกใช้ฟังก์ชันใดและส่งอาร์กิวเมนต์อะไรไป

### Summary

กลไกที่อนุญาตให้โมเดลภาษาขนาดใหญ่ (LLMs) ส่งออกข้อมูลที่มีโครงสร้างเพื่อกระตุ้นการทำงานของซอฟต์แวร์เฉพาะ

## Key Concepts

- เอาต์พุตที่มีโครงสร้าง
- การผสานรวม API
- การใช้งานเครื่องมือ
- สคีมา JSON

## Use Cases

- การจองตั๋วเครื่องบินผ่าน API การเดินทาง
- การคำนวณทางคณิตศาสตร์ที่ซับซ้อน
- การสืบค้นข้อมูลในฐานข้อมูล
