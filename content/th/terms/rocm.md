## Definition

ROCm (Radeon Open Compute) เป็นไดรเวอร์และชุดซอฟต์แวร์ที่พัฒนาโดย AMD เพื่อเปิดใช้งานการคำนวณประสิทธิภาพสูงบน GPU ของ AMD มันมอบไลบรารี คอมไพเลอร์ และเครื่องมือที่จำเป็นสำหรับการพัฒนาโปรแกรมขนาน

### Summary

แพลตฟอร์มซอฟต์แวร์โอเพนซอร์สของ AMD สำหรับการคำนวณด้วย GPU ซึ่งนำเสนอทางเลือกแทนระบบนิเวศ CUDA ของ NVIDIA

## Key Concepts

- GPU Computing
- Open Source Software
- Parallel Processing
- Hardware Acceleration

## Use Cases

- การฝึกโมเดล AI บน GPU ของ AMD
- การจำลองทางวิทยาศาสตร์
- คลัสเตอร์การคำนวณประสิทธิภาพสูง
