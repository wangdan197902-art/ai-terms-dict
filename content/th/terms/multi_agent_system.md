## Definition

ระบบหลายเอเจนต์ประกอบด้วยเอเจนต์อิสระหลายตัว ซึ่งแต่ละตัวอาจมีความเชี่ยวชาญในงานหรือโดเมนที่แตกต่างกัน เอเจนต์เหล่านี้สื่อสารและประสานงานการกระทำของตนเพื่อบรรลุเป้าหมายร่วมกัน

### Summary

แนวทางสถาปัตยกรรมที่มีเอเจนต์ AI อิสระหลายตัวทำงานร่วมกัน แข่งขันกัน หรือประสานงานกัน เพื่อแก้ปัญหาที่ซับซ้อนซึ่งเกินความสามารถของเอเจนต์เดียว

## Key Concepts

- ความร่วมมือของเอเจนต์
- ความเชี่ยวชาญเฉพาะด้าน
- การประสานงาน
- การกระจายอำนาจ

## Use Cases

- การพัฒนาซอฟต์แวร์อัตโนมัติ
- สภาพแวดล้อมการจำลองที่ซับซ้อน
- การตัดสินใจแบบกระจายศูนย์
