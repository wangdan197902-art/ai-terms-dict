## Definition

การเรียนรู้ของเครื่อง (ML) ช่วยให้คอมพิวเตอร์เรียนรู้รูปแบบจากข้อมูลในอดีตและทำการตัดสินใจหรือทำนายข้อมูลใหม่ที่ไม่เคยเห็นมาก่อน ML ครอบคลุมเทคนิคต่างๆ รวมถึงการเรียนรู้ภายใต้การดูแล (supervised learning) และการเรียนรู้โดยไม่มีการดูแล (unsupervised learning)

### Summary

ส่วนหนึ่งของ AI ที่มุ่งเน้นการสร้างระบบที่สามารถเรียนรู้จากข้อมูลเพื่อปรับปรุงประสิทธิภาพโดยไม่ต้องเขียนโปรแกรมอย่างชัดเจน

## Key Concepts

- การเรียนรู้ภายใต้การดูแล (Supervised Learning)
- การเรียนรู้โดยไม่มีการดูแล (Unsupervised Learning)
- การฝึกโมเดล (Model Training)
- การทำนาย (Prediction)

## Use Cases

- การตรวจจับการฉ้อโกง
- ระบบแนะนำสินค้า
- การรู้จำเสียงพูด
