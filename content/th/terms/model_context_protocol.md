## Definition

โปรโตคอลบริบทของโมเดล (MCP) เป็นมาตรฐานเปิดที่อนุญาตให้แอปพลิเคชัน AI เชื่อมต่อกับแหล่งข้อมูลต่างๆ เช่น ฐานข้อมูล API และระบบไฟล์ ในรูปแบบที่เป็นมาตรฐานเดียวกัน โดยทำหน้าที่นามธรรม化处理ส่วนเชื่อมต่อที่ซับซ้อน

### Summary

กรอบมาตรฐานที่ออกแบบมาเพื่ออำนวยความสะดวกในการสื่อสารอย่างปลอดภัยและมีประสิทธิภาพระหว่างโมเดล AI กับแหล่งข้อมูลหรือเครื่องมือภายนอก

## Key Concepts

- การทำงานร่วมกันได้
- มาตรฐานกลาง
- การบูรณาการข้อมูล
- การเรียกใช้เครื่องมือ

## Use Cases

- การเชื่อมต่อกับข้อมูลระดับองค์กรสำหรับ LLM
- การสร้างผู้ช่วย AI ที่พกพาได้
- การผสานรวม API อย่างปลอดภัย
