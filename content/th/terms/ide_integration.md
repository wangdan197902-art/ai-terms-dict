## Definition

แนวทางนี้เกี่ยวข้องกับการเชื่อมต่อโมเดล AI เช่น โมเดลภาษาขนาดใหญ่ (LLMs) กับสภาพแวดล้อมการพัฒนาซอฟต์แวร์อย่าง VS Code หรือ IntelliJ ทำให้เกิดฟีเจอร์ต่างๆ เช่น การเติมโค้ดอัจฉริยะและการทำซ้ำอัตโนมัติ

### Summary

การผสานรวม IDE หมายถึงการฝังเครื่องมือที่ขับเคลื่อนด้วย AI เข้าโดยตรงในสภาพแวดล้อมการพัฒนาแบบบูรณาการ เพื่อช่วยนักโปรแกรมเมอร์ในงานเขียนโค้ด

## Key Concepts

- การเติมอัตโนมัติ (Autocomplete)
- การเติมโค้ด (Code Completion)
- LLM (โมเดลภาษาขนาดใหญ่)
- ประสบการณ์ของผู้พัฒนา (Developer Experience)

## Use Cases

- คำแนะนำโค้ดอัจฉริยะ
- การปรับโครงสร้างโค้ดอัตโนมัติ
- การดำเนินการสอบถามด้วยภาษาธรรมชาติ
