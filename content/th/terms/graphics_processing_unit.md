## Definition

GPU คือโปรเซสเซอร์ประสิทธิภาพสูงที่พัฒนาขึ้นเดิมทีเพื่อจัดการงานเรนเดอร์กราฟิก ต่างจาก CPU ซึ่งมีจำนวนคอร์น้อยแต่เหมาะสำหรับการประมวลผลแบบลำดับ (sequential serial processing) GPU มีคอร์จำนวนมาก (หลายพันคอร์) ที่ออกแบบมาเพื่อทำงานพร้อมกัน ทำให้มีประสิทธิภาพสูงในการประมวลผลข้อมูลขนาดใหญ่และการคำนวณเชิงเมทริกซ์

### Summary

วงจรอิเล็กทรอนิกส์เฉพาะทางที่ออกแบบมาเพื่อจัดการและเรนเดอร์ภาพอย่างรวดเร็ว ในปัจจุบันนิยมใช้สำหรับการคำนวณแบบขนานในงานปัญญาประดิษฐ์

## Key Concepts

- การประมวลผลแบบขนาน
- CUDA Cores
- การคูณเมทริกซ์
- การเร่งความเร็วด้วยฮาร์ดแวร์

## Use Cases

- การฝึกโมเดลเครือข่ายประสาทเทียมขนาดใหญ่
- การอนุมาน (inference) ด้านคอมพิวเตอร์วิทัศน์แบบเรียลไทม์
- การเรนเดอร์กราฟิกสามมิติในสถานการณ์จำลอง
