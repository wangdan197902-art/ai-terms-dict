## Definition

เว็บบีตเป็นโปรโตคอลการสื่อสารคอมพิวเตอร์ที่เปิดใช้งานการสื่อสารสองทางอย่างต่อเนื่องระหว่างไคลเอนต์และเซิร์ฟเวอร์ ซึ่งแตกต่างจาก HTTP ที่ต้องสร้างการเชื่อมต่อใหม่สำหรับแต่ละรอบของการขอและตอบกลับ ทำให้เว็บบีตมีประสิทธิภาพสูงกว่าสำหรับการส่งข้อมูลแบบทันที

### Summary

โปรโตคอลที่ให้การสื่อสารสองทางเต็มรูปแบบผ่านช่องทางการเชื่อมต่อ TCP เดียว สำหรับการแลกเปลี่ยนข้อมูลแบบเรียลไทม์

## Key Concepts

- การสื่อสารสองทางเต็มรูปแบบ
- การเชื่อมต่อที่ยั่งยืน
- เวลาแฝงต่ำ
- โปรโตคอล TCP

## Use Cases

- แอปพลิเคชันแชทแบบเรียลไทม์
- การอัปเดตคะแนนกีฬาสด
- เกมออนไลน์หลายผู้เล่น
