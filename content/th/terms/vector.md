## Definition

ในปัญญาประดิษฐ์ เวกเตอร์เป็นโครงสร้างข้อมูลพื้นฐานที่ใช้แทนข้อมูลในรูปแบบตัวเลข ประกอบด้วยรายการตัวเลขที่มีลำดับ (องค์ประกอบ) ซึ่งแมปคุณลักษณะของสิ่งหนึ่งๆ เข้ากับ

### Summary

วัตถุทางคณิตศาสตร์ที่แสดงปริมาณทั้งขนาดและทิศทาง มักใช้ใน AI เพื่อเข้ารหัสจุดข้อมูลในปริภูมิหลายมิติ

## Key Concepts

- มิติ
- การฝังตัว (Embedding)
- ขนาด
- ทิศทาง

## Use Cases

- การค้นหาเชิงความหมายในฐานข้อมูล
- การฝังตัวสำหรับการประมวลผลภาษาธรรมชาติ
- การสกัดคุณลักษณะสำหรับการจดจำภาพ

## Code Example

```python
import numpy as np
vec = np.array([1.0, 2.0, 3.0])
print(vec.shape)
```
