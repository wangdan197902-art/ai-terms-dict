## Definition

MS MARCO (Microsoft Machine Reading Comprehension) เป็นชุดข้อมูลที่นิยมใช้อย่างกว้างขวางในด้านการประมวลผลภาษาธรรมชาติ โดยเฉพาะอย่างยิ่งสำหรับงานการค้นหาข้อมูลและการตอบคำถาม ประกอบด้วยคำค้นหาที่...

### Summary

ชุดข้อมูล Microsoft Machine Reading Comprehension ซึ่งเป็นคอลเลกชันขนาดใหญ่ของคำค้นหาจริงและข้อความจากเอกสารที่เกี่ยวข้อง ใช้สำหรับการฝึกฝนระบบการค้นหาข้อมูล

## Key Concepts

- การค้นหาข้อมูล (Information Retrieval)
- การจัดอันดับข้อความ (Passage Ranking)
- คำค้นหา (Search Queries)
- ความเข้าใจในการอ่านของเครื่อง (Machine Reading Comprehension)

## Use Cases

- การฝึกฝนเครื่องมือค้นหา
- การพัฒนา سیستمตอบคำถาม
- การทดสอบมาตรฐานโมเดลการค้นหา
