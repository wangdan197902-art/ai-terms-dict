## Definition

ไฟล์ดัชนี (มักชื่อ 'model_index.json') มีข้อมูลโครงสร้างเกี่ยวกับสถาปัตยกรรมของโมเดล รวมถึงประเภทของพิปไลน์ โมเดลย่อย และเส้นทางของการกำหนดค่า ช่วยให้ Hub สามารถรวบรวม

### Summary

Model Index คือไฟล์เมตาดาต้าที่ใช้โดย Hugging Face Hub เพื่ออธิบายและจัดระเบียบส่วนประกอบและการกำหนดค่าของโมเดล

## Key Concepts

- เมตาดาต้า
- การกำหนดค่าพิปไลน์
- การโหลดโมเดล
- การทำงานร่วมกันได้

## Use Cases

- การกำหนดค่าพิปไลน์หลายส่วนประกอบ
- การอัปโหลดโมเดลที่ซับซ้อนไปยัง Hub
- การค้นพบและการโหลดโมเดลอัตโนมัติ
