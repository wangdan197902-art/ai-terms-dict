## Definition

โมเดลแบบโอเพนเวตแตกต่างจาก AI แบบโอเพนซอร์สเต็มตัว เพราะมีการเผยแพร่เฉพาะพารามิเตอร์สุดท้ายที่เรียนรู้เท่านั้น ไม่ได้รวมถึงโครงสร้างพื้นฐานหรือข้อมูลที่ใช้ในการสร้างเสมอไป สิ่งนี้ช่วยให้ผู้ใช้สามารถรันการอนุมาน (inference)

### Summary

โมเดล AI ที่เผยแพร่พารามิเตอร์ที่ผ่านการฝึกแล้ว (น้ำหนัก) แต่โค้ดการฝึกและชุดข้อมูลอาจยังคงเป็นความลับ

## Key Concepts

- น้ำหนักโมเดล
- การอนุมาน
- การปรับแต่งละเอียด
- ความโปร่งใสบางส่วน

## Use Cases

- การปรับใช้ LLM บนฮาร์ดแวร์ท้องถิ่น
- การปรับแต่งโมเดลที่ผ่านการฝึกไว้แล้ว
- การลดต้นทุน API
