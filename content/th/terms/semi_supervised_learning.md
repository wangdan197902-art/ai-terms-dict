## Definition

การเรียนรู้แบบกึ่งควบคุมเป็นรูปแบบการฝึกฝนแบบไฮบริดที่ใช้ข้อมูลที่มีป้ายกำกับจำนวนน้อย ร่วมกับข้อมูลไม่มีป้ายกำกับปริมาณมาก สมมติฐานหลักคือโครงสร้างของข้อมูลไม่มีป้ายกำกับสามารถช่วยในการเรียนรู้รูปแบบได้

### Summary

แนวทางmachine learning ที่ใช้ทั้งข้อมูลที่มีป้ายกำกับและไม่มีป้ายกำกับ เพื่อปรับปรุงความแม่นยำและการทั่วไปของโมเดล

## Key Concepts

- ข้อมูลที่มีป้ายกำกับ (Labeled data)
- ข้อมูลไม่มีป้ายกำกับ (Unlabeled data)
- การฝึกด้วยตนเอง (Self-training)
- สมมติฐานแมนิโฟลด์ (Manifold assumption)

## Use Cases

- การจัดประเภทภาพที่มีการติดป้ายกำกับจำกัด
- การวิเคราะห์ความรู้สึกจากข้อความที่มีป้ายกำกับเบาบาง
- การทำนายการวินิจฉัยทางการแพทย์ที่มีข้อมูลจากผู้เชี่ยวชาญหายาก
