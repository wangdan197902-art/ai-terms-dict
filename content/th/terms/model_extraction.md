## Definition

การสกัดโมเดลเกี่ยวข้องกับการสอบถาม API ของโมเดลแมชชีนเลิร์นนิงเป้าหมายเพื่ออนุมานโครงสร้างภายใน น้ำหนัก หรือขอบเขตการตัดสินใจของผู้โจมตีใช้คำสอบถามเหล่านี้เพื่อสร้างโมเดลจำลองที่...

### Summary

การโจมตีที่ผู้ไม่หวังดีทำการสอบถามโมเดลเพื่อสร้างพารามิเตอร์ใหม่หรือสร้างสำเนาจำลอง

## Key Concepts

- การสร้างโมเดลจำลอง (Surrogate Modeling)
- การสอบถาม API (API Querying)
- การขโมยทรัพย์สินทางปัญญา (Intellectual Property Theft)
- การโจมตีแบบปฏิปักษ์ (Adversarial Attack)

## Use Cases

- การตรวจสอบความปลอดภัยของ API ปัญญาประดิษฐ์เชิงพาณิชย์
- การปกป้องอัลกอริทึมที่เป็นกรรมสิทธิ์จากการคัดลอก
- การวิจัยกลไกการป้องกันต่อการสกัดโมเดล
