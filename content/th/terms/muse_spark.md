## Definition

Muse Spark เป็นเฟรมเวิร์กการเรียนรู้เชิงลึกโอเพนซอร์สที่ออกแบบมาเพื่อทำงานอย่างมีประสิทธิภาพบน Apache Spark ช่วยให้开发人员สามารถฝึกเครือข่ายประสาทเทียมที่ซับซ้อนผ่านคลัสเตอร์แบบกระจายได้โดยการนำ

### Summary

เฟรมเวิร์กการเรียนรู้เชิงลึกแบบกระจายที่สร้างบน Apache Spark ซึ่งช่วยให้สามารถฝึกโมเดลได้อย่างมีสเกลผ่านคลัสเตอร์ขนาดใหญ่

## Key Concepts

- การคำนวณแบบกระจาย (Distributed Computing)
- Apache Spark
- การเรียนรู้เชิงลึก (Deep Learning)
- ความสามารถในการขยายขนาด (Scalability)

## Use Cases

- การฝึกเครือข่ายประสาทเทียมขนาดใหญ่อบนชุดข้อมูลระดับเพตะไบต์
- การรวมเส้นทางการเรียนรู้ของเครื่อง (ML pipelines) เข้ากับคลัสเตอร์ Hadoop/Spark ที่มีอยู่
- การอนุมานแบบกระจายแบบเรียลไทม์ในระดับใหญ่
