## Definition

สิ่งเหล่านี้หมายถึงคลังข้อมูลที่มีระเบียบ เช่น หัวข้อ GitHub รายการ Awesome หรือวิกิที่ดูแลโดยชุมชน ซึ่งรวบรวมซอฟต์แวร์โอเพนซอร์สที่เกี่ยวข้องกับปัญญาประดิษฐ์ไว้ด้วยกัน ทำหน้าที่เป็นแหล่งทรัพยากรสำคัญสำหรับการค้นหาและอ้างอิง

### Summary

คอลเลกชันหรือแคตตาล็อกที่คัดสรรแล้วสำหรับจัดหมวดหมู่เครื่องมือ ไลบรารี และเฟรมเวิร์ก AI ที่เปิดเผยต่อสาธารณะ

## Key Concepts

- โอเพนซอร์ส
- การจัดสรรทรัพยากร
- การบำรุงรักษาโดยชุมชน
- การค้นพบซอฟต์แวร์

## Use Cases

- การค้นหาเครื่องมือสำหรับการวิจัย
- การเลือกเทคโนโลยีสตาร์ทอัพ
- การรวบรวมทรัพยากรเพื่อการศึกษา
