## Definition

ตัวแทนทางการสอน (Pedagogical agent) เป็นส่วนประกอบของซอฟต์แวร์ ซึ่งมักปรากฏเป็นตัวละครเสมือนจริง ทำหน้าที่เป็นผู้สอนหรือติวเตอร์ในสภาพแวดล้อมทางการศึกษา เอเจนต์เหล่านี้ใช้การประมวลผลภาษาธรรมชาติเพื่

### Summary

เอเจนต์ปัญญาประดิษฐ์ที่ออกแบบมาเพื่ออำนวยความสะดวกในการเรียนรู้โดยการให้คำแนะนำ ข้อเสนอแนะ และการชี้นำ

## Key Concepts

- การสอนอัจฉริยะ (Intelligent tutoring)
- มนุษย์เสมือน (Virtual human)
- การเรียนรู้แบบปรับตัว (Adaptive learning)
- ปฏิสัมพันธ์ระหว่างมนุษย์กับคอมพิวเตอร์ (Human-computer interaction)

## Use Cases

- ผู้ช่วยหลักสูตรออนไลน์ (Online course assistants)
- แอปพลิเคชันการเรียนรู้ภาษา (Language learning apps)
- การจำลองการฝึกอบรมในองค์กร (Corporate training simulations)
