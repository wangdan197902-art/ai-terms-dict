## Definition

W sztucznej inteligencji wektor to podstawowa struktura danych służąca do numerycznego reprezentowania informacji. Składa się z uporządkowanej listy liczb (elementów), które odwzorowują cechy encji w przestrzeni wektorowej.

### Summary

Obiekt matematyczny reprezentujący wielkość o określonej wartości i kierunku, często używany w sztucznej inteligencji do kodowania punktów danych w przestrzeni wielowymiarowej.

## Key Concepts

- Wymiarowość
- Osadzenie (Embedding)
- Moduł (Wielkość)
- Kierunek

## Use Cases

- Wyszukiwanie semantyczne w bazach danych
- Osadzenia w przetwarzaniu języka naturalnego
- Ekstrakcja cech w rozpoznawaniu obrazów

## Code Example

```python
import numpy as np
vec = np.array([1.0, 2.0, 3.0])
print(vec.shape)
```
