## Definition

REST API umożliwiają komunikację między klientami a serwerami poprzez wykorzystanie bezstanowych operacji w ramach protokołu HTTP, takich jak GET, POST, PUT i DELETE. Zasoby są strukturyzowane jako URI, a dane przesyłane są za pomocą standardowych formatów serializacji, najczęściej JSON.

### Summary

REST API to styl architektoniczny służący do projektowania aplikacji sieciowych, oparty na standardowych metodach HTTP.

## Key Concepts

- Bezstanowość
- Metody HTTP
- URI oparte na zasobach
- Serializacja JSON

## Use Cases

- Komunikacja między mikroserwisami
- Integracja z backendem aplikacji mobilnych
- Dostęp do danych zewnętrznych usług
