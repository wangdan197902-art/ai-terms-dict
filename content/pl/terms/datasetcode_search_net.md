## Definition

Code Search Net to kompleksowy zbiór danych stworzony w celu zaawansowania badań w zakresie wyszukiwania kodu. Zawiera miliony par opisów w języku naturalnym odpowiadających im fragmentom kodu Java. Dane te umożliwiają efektywne trenowanie modeli rozumiejących strukturę i semantykę kodu.

### Summary

Zestaw danych służący jako benchmark do oceny systemów wyszukiwania kodu, łączący zapytania w języku naturalnym z odpowiednimi fragmentami kodu.

## Key Concepts

- Pobieranie kodu (Code Retrieval)
- Wyszukiwanie semantyczne (Semantic Search)
- Kod Java (Java Code)
- Przetwarzanie języka naturalnego (Natural Language Processing)

## Use Cases

- Tworzenie wtyczek do środowisk IDE
- Badania nad rozumieniem kodu
- Automatyczna dokumentacja kodu
