## Definition

Wielowymiarowa adaptacyjna regresja splajnowa (MARS) to elastyczna metoda regresji modelująca złożone nieliniowe zależności poprzez dopasowanie kawałkami liniowych funkcji bazowych. Automatycznie wybiera lokalizacje węzłów, aby zoptymalizować dopasowanie modelu do danych.

### Summary

Nieliniowa metoda regresji, która automatycznie wybiera funkcje bazowe do modelowania złożonych zależności.

## Key Concepts

- Funkcje kawałkami liniowe
- Funkcje bazowe
- Selekcja węzłów
- Regresja nieparametryczna

## Use Cases

- Prognozowanie finansowe
- Modelowanie środowiskowe
- Przewidywanie wyników medycznych
