## Definition

Ta praktyka obejmuje łączenie modeli AI, takich jak duże modele językowe (LLM), ze środowiskami rozwoju oprogramowania, takimi jak VS Code czy IntelliJ. Umożliwia to funkcje inteligentnego uzupełniania kodu, automatyzacji refaktoryzacji i innych ułatwień dla deweloperów.

### Summary

Integracja z IDE odnosi się do wbudowania narzędzi wspieranych przez sztuczną inteligencję bezpośrednio w środowiska programistyczne, aby pomóc programistom w zadaniach kodowania.

## Key Concepts

- Autouzupełnianie
- Uzupełnianie kodu
- LLM (Duże Modele Językowe)
- Doświadczenie dewelopera (DX)

## Use Cases

- Inteligentne sugestie kodu
- Zautomatyzowana refaktoryzacja
- Wykonywanie zapytań w języku naturalnym
