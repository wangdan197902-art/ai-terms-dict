## Definition

Wywoływanie funkcji umożliwia dużym modelom językowym interakcję z zewnętrznymi narzędziami i interfejsami API poprzez generowanie danych wyjściowych o strukturze sztywnej, takich jak obiekty JSON, które określają, którą funkcję należy wykonać oraz jakie argumenty przekazać.

### Summary

Mechanizm pozwalający dużym modelom językowym (LLM) na generowanie danych strukturalnych, które inicjują wykonanie określonych funkcji programistycznych.

## Key Concepts

- Dane strukturalne
- Integracja z API
- Używanie narzędzi
- Schemat JSON

## Use Cases

- Rezerwacja lotów poprzez API turystyczne
- Wykonywanie złożonych obliczeń matematycznych
- Wyszukiwanie rekordów w bazie danych
