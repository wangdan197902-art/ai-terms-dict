## Definition

Biblioteka Diffusers to narzędzie open-source od Hugging Face, zaprojektowane w celu uproszczenia korzystania z wstępnie wytrenowanych modeli dyfuzyjnych, w szczególności Stable Diffusion. Oferuje ona modularne potoki przetwarzania, które obsługują etapy takie jak kodowanie tekstu, proces dyfuzyjny w przestrzeni ukrytej oraz dekodowanie obrazu, ułatwiając zarówno inferencję, jak i fine-tuning modeli.

### Summary

Biblioteka Hugging Face zapewniająca standaryzowane potoki (pipelines) do uruchamiania i dopracowywania (fine-tuning) modeli Stable Diffusion.

## Key Concepts

- Hugging Face
- Abstrakcja potoku (Pipeline abstraction)
- Dopracowywanie modelu (Fine-tuning)
- Interoperacyjność modeli

## Use Cases

- Szybkie prototypowanie generatorów obrazów
- Dopracowywanie własnych modeli
- Prezentacje edukacyjne procesów dyfuzyjnych
