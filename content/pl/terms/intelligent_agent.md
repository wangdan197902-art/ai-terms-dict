## Definition

Agent inteligentny to system zdolny do postrzegania otoczenia za pomocą czujników lub danych wejściowych, przetwarzania tej informacji za pomocą algorytmów wnioskowania oraz działania na środowisku poprzez aktuator...

### Summary

Autonomiczna jednostka oprogramowania, która postrzega swoje środowisko, rozważa działania i wykonuje zadania w celu osiągnięcia określonych celów.

## Key Concepts

- Autonomia
- Pętla Postrzeganie-Działanie
- Orientacja Celowa
- Reaktywność

## Use Cases

- Autonomiczne boty tradingowe realizujące strategie rynkowe
- Automatyzacja procesów roboczych (RPA) obsługująca wieloetapowe przepływy pracy
- Asystenci osobowi zarządzający kalendarzem i zadaniami e-mail
