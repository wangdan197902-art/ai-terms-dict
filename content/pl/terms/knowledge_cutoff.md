## Definition

Data ograniczenia wiedzy definiuje czasowy zakres danych treningowych modelu językowego. Jakiekolwiek informacje, wydarzenia lub rozwój zdarzeń mających miejsce po tej dacie są generalnie nieznane modelowi.

### Summary

Ograniczenie wiedzy to konkretna data, po której model językowy nie był trenowany i dlatego nie posiada informacji o wydarzeniach mających miejsce później.

## Key Concepts

- Dane treningowe
- Ograniczenie czasowe
- Statyczna wiedza
- Świeżość danych

## Use Cases

- Ustalanie oczekiwań użytkowników
- Ocena aktualności modelu
- Integracja z narzędziami wyszukiwania
