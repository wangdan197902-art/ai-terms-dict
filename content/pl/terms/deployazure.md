## Definition

Wdrażanie modeli na platformie Azure obejmuje wykorzystanie natywnych dla chmury narzędzi, takich jak Azure Machine Learning, Azure Kubernetes Service (AKS) lub Azure Functions, w celu udostępniania modeli ML w skali produkcyjnej. Proces ten obejmuje zarządzanie zasobami obliczeniowymi, skalowaniem oraz monitorowaniem usług.

### Summary

Proces hostowania i uruchamiania modeli uczenia maszynowego na infrastrukturze chmurowej Microsoft Azure.

## Key Concepts

- Infrastruktura chmurowa
- Udostępnianie modelu (Model Serving)
- Skalowalność
- Potoki CI/CD

## Use Cases

- Punkty końcowe API w czasie rzeczywistym do wnioskowania
- Ocena wsadowa dużych zbiorów danych
- Zarządzanie modelami na poziomie enterprise
