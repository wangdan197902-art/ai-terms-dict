## Definition

SDK to zbiór narzędzi do tworzenia oprogramowania, który pozwala programistom na tworzenie aplikacji dla określonych platform lub usług. W przypadku AI, SDK dostarczają gotowe biblioteki, interfejsy API i utility do...

### Summary

Zestaw narzędzi programistycznych dostarczający narzędzia, biblioteki i dokumentację do tworzenia aplikacji.

- Integracja API
- Biblioteki
- Narzędzia deweloperskie

- Integracja dużych modeli językowych (LLM) w aplikacjach
- Budowanie potoków przetwarzania obrazu (computer vision)
