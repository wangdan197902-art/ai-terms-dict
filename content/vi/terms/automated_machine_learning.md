## Definition

AutoML (Học máy tự động) tinh gọn quá trình phát triển các mô hình ML bằng cách tự động hóa các nhiệm vụ như tiền xử lý dữ liệu, kỹ thuật đặc trưng, lựa chọn mô hình và tinh chỉnh siêu tham số. Nó cho phép...

### Summary

Một phương pháp luận tự động hóa quy trình từ đầu đến cuối khi áp dụng học máy vào các vấn đề thực tế, giảm bớt nỗ lực thủ công.

## Key Concepts

- Tinh chỉnh siêu tham số
- Kỹ thuật đặc trưng
- Lựa chọn mô hình
- Dân chủ hóa

## Use Cases

- Tạo mẫu nhanh cho các nhà phân tích kinh doanh
- Tối ưu hóa các quy trình sản xuất quy mô lớn
- So sánh tự động nhiều thuật toán khác nhau

## Code Example

```python
from auto_ml import Predictor
predictor = Predictor(type_of_estimator='classifier')
predictor.fit(dataframe, target='label')
```
