## Definition

Trong bối cảnh trí tuệ nhân tạo và khoa học dữ liệu, 'numerical' đề cập đến các loại dữ liệu hoặc phương pháp liên quan đến các giá trị định lượng, chẳng hạn như số nguyên, số thập phân (float) và số thập phân. Khác với dữ liệu phân loại hoặc văn bản, dữ liệu số cho phép thực hiện các phép toán toán học trực tiếp.

### Summary

Liên quan đến, sử dụng hoặc liên quan đến các con số hoặc phép tính toán học thay vì logic ký hiệu hay văn bản.

## Key Concepts

- Số thập phân (Floats)
- Số nguyên (Integers)
- Kiểu dữ liệu (Data Types)
- Dữ liệu định lượng (Quantitative Data)

## Use Cases

- Mô hình hóa thống kê
- Dự báo tài chính
- Xử lý dữ liệu cảm biến

## Code Example

```python
import numpy as np
arr = np.array([1.5, 2.7, 3.9])
mean_val = np.mean(arr)
print(mean_val)
```
