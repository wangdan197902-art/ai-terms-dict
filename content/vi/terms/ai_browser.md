## Definition

Trình duyệt AI là một ứng dụng duyệt web tích hợp trực tiếp các tính năng trí tuệ nhân tạo vào giao diện người dùng. Các tính năng này thường bao gồm tìm kiếm bằng ngôn ngữ tự nhiên, tự động tóm tắt nội dung trang web và hỗ trợ phân tích thông tin phức tạp.

### Summary

Trình duyệt web tích hợp các khả năng AI để hỗ trợ tìm kiếm, tóm tắt và phân tích nội dung.

## Key Concepts

- LLM tích hợp
- tìm kiếm thông minh
- tóm tắt nội dung
- giao diện người dùng hội thoại

## Use Cases

- Hỗ trợ nghiên cứu
- Tổng hợp thông tin nhanh chóng
- Cải thiện khả năng tiếp cận
