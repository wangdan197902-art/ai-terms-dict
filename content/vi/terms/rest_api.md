## Definition

REST API cho phép giao tiếp giữa máy khách và máy chủ bằng cách sử dụng các thao tác không trạng thái qua các giao thức HTTP như GET, POST, PUT và DELETE. Chúng cấu trúc tài nguyên dưới dạng URI và sử dụng các định dạng dữ liệu tiêu chuẩn như JSON.

### Summary

REST API là một phong cách kiến trúc để thiết kế các ứng dụng mạng, dựa trên các phương thức HTTP tiêu chuẩn.

## Key Concepts

- Tính vô trạng thái
- Phương thức HTTP
- URI dựa trên tài nguyên
- Chuẩn hóa JSON

## Use Cases

- Giao tiếp giữa các vi dịch vụ
- Tích hợp backend cho ứng dụng di động
- Truy cập dữ liệu bên thứ ba
