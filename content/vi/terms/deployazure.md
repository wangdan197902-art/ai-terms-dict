## Definition

Việc triển khai lên Azure bao gồm việc sử dụng các công cụ bản địa trên đám mây như Azure Machine Learning, Azure Kubernetes Service (AKS) hoặc Azure Functions để cung cấp các mô hình ML ở quy mô lớn. Nó bao gồm việc quản lý tài nguyên tính toán...

### Summary

Quy trình lưu trữ và chạy các mô hình học máy trên cơ sở hạ tầng đám mây của Microsoft Azure.

## Key Concepts

- Cơ sở hạ tầng đám mây (Cloud Infrastructure)
- Phục vụ mô hình (Model Serving)
- Khả năng mở rộng (Scalability)
- Quy trình CI/CD (Continuous Integration/Continuous Deployment)

## Use Cases

- Điểm cuối API thời gian thực cho suy luận (inference)
- Đánh giá hàng loạt (batch scoring) trên các tập dữ liệu lớn
- Quản lý mô hình cấp doanh nghiệp
