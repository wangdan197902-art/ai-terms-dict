## Definition

Logic Mô tả (DL) là các phân đoạn có thể quyết định được (decidable fragments) của logic bậc nhất, tạo nền tảng lý thuyết cho các ontology, đặc biệt là Ngôn ngữ Ontology Web (OWL). Chúng cho phép xác định chính xác...

### Summary

Một họ các ngôn ngữ biểu diễn tri thức chính thức được sử dụng để biểu diễn và suy luận về cấu trúc khái niệm của các miền.

## Key Concepts

- Ontology (Bản thể học)
- Suy luận (Reasoning)
- Phép bao hàm (Subsumption)
- Hệ thống phân cấp khái niệm (Concept Hierarchy)

## Use Cases

- Xây dựng ontology cho Web ngữ nghĩa
- Biểu đồ tri thức y tế
- Hệ thống phân loại tự động
