## Definition

SDK là một bộ sưu tập các công cụ phát triển phần mềm cho phép nhà phát triển tạo ra các ứng dụng cho các nền tảng hoặc dịch vụ cụ thể. Đối với AI, SDK cung cấp các thư viện, API và tiện ích được xây dựng sẵn để giúp tích hợp các mô hình AI vào ứng dụng một cách dễ dàng hơn.

### Summary

Một Bộ công cụ Phát triển Phần mềm cung cấp các công cụ, thư viện và tài liệu để xây dựng ứng dụng.

## Key Concepts

- Tích hợp API (API Integration)
- Thư viện (Libraries)
- Công cụ Phát triển (Development Tools)
- Trừu tượng hóa (Abstraction)

## Use Cases

- Tích hợp LLM vào ứng dụng
- Xây dựng quy trình thị giác máy tính
- Triển khai mô hình ML vào sản xuất
