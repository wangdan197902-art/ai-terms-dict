## Definition

Sử dụng Công cụ cho phép các mô hình ngôn ngữ tương tác với môi trường phần mềm bên ngoài bằng cách gọi các hàm đã xác định trước, như máy tính công cụ, công cụ tìm kiếm hoặc truy vấn cơ sở dữ liệu. Cách tiếp cận này mở rộng phạm vi

### Summary

Một mô hình trong đó các tác nhân AI chọn và thực thi các hàm hoặc API bên ngoài để thực hiện các nhiệm vụ cụ thể vượt quá khả năng gốc của chúng.

## Key Concepts

- Gọi Hàm (Function Calling)
- Kiến trúc Tác nhân
- Tích hợp API
- Kiến thức Bên ngoài

## Use Cases

- Bot hỗ trợ khách hàng tự động
- Trợ lý phân tích dữ liệu
- Tự động hóa nhà thông minh
