## Definition

I konteksten af kunstig intelligens og datavidenskab refererer numerisk til datatyper eller metoder, der involverer kvantitative værdier, såsom heltal, flydende komma-tal og decimaltal. I modsætning til kategoriske eller tekstuelle data.

### Summary

Relateret til, involverer eller bruger tal eller matematiske beregninger snarere end symbolsk logik eller tekst.

## Key Concepts

- Flydende komma-tal
- Heltal
- Datatyper
- Kvantitative data

## Use Cases

- Statistisk modellering
- Finansiel prognose
- Behandling af sensorsdata

## Code Example

```python
import numpy as np
arr = np.array([1.5, 2.7, 3.9])
mean_val = np.mean(arr)
print(mean_val)
```
