## Definition

Cloud computing giver skalerbar infrastruktur til AI-arbejdsbyrder og gør det muligt for udviklere at få adgang til kraftige GPU'er og lagerplads uden at vedligeholde fysiske datacentre. Det understøtter forskellige servicemodeller som IaaS, PaaS og SaaS.

### Summary

Cloud refererer til eksterne servere, der hostes på internettet og bruges til at lagre, administrere og behandle data og AI-modeller i stedet for lokal hardware.

## Key Concepts

- Skalerbarhed
- Infrastructure as a Service (IaaS)
- Eksternt lager
- Elasticitet

## Use Cases

- Træning af store sprogmodeller
- Hosting af AI-API'er til webapplikationer
- Lagring af enorme datasæt til analyse
