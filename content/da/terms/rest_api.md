## Definition

REST-API'er muliggør kommunikation mellem klienter og servere ved at udnytte tilstandsløse operationer over HTTP-protokoller som GET, POST, PUT og DELETE. De strukturerer ressourcer som URI'er og bruger standardiserede dataformater.

### Summary

En Representational State Transfer (REST) API er en arkitektonisk stil til design af netværksapplikationer, der stole på standard HTTP-metoder.

## Key Concepts

- Tilstandsløshed
- HTTP-metoder
- Ressourcebaserede URI'er
- JSON-serialisering

## Use Cases

- Kommunikation mellem mikrotjenester
- Backend-integration til mobilapps
- Adgang til tredjepartsdata
