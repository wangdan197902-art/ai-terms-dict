## Definition

Prompt engineering indebærer at udforme specifikke input, kendt som prompts, for at fremkalde nøjagtige, relevante og kvalitetsfyldte svar fra generative AI-modeller. Det kræver en forståelse af, hvordan modellerne fortolker konteksten.

### Summary

Praksisen med at designe og optimere indtastningstekster for at vejlede store sprogmodeller mod ønskede output.

## Key Concepts

- Kontekstuel ramme
- Few-shot learning
- Instruktionstilpasning
- Token-optimering

## Use Cases

- Automatiserede kundesupport-chatbots
- Assistenter til kodegenerering
- Hjælpemidler til kreativ skrivning

## Code Example

```python
prompt = "Translate the following English text to French: 'Hello world'"
response = llm.generate(prompt)
```
