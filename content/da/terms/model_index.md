## Definition

Indexfilen, typisk navngivet 'model_index.json', indeholder strukturerede oplysninger om en models arkitektur, herunder pipeline-type, undermodeller og konfigurationsstier. Den gør det muligt for Hubben at koordinere

### Summary

Et Model Index er en metadatafil, der bruges af Hugging Face Hub til at beskrive og organisere modelkomponenter og konfigurationer.

## Key Concepts

- Metadata
- Pipeline-konfiguration
- Modelindlæsning
- Interoperabilitet

## Use Cases

- Definition af pipelines med flere komponenter
- Upload af komplekse modeller til Hubben
- Automatisk opdagelse og indlæsning af modeller
