## Definition

AutoML (Automated Machine Learning) strømliner udviklingen af ML-modeller ved at automatisere opgaver såsom datapræprocessing, feature-engineering, modelvalg og hyperparameterjustering. Det gør det muligt for ikke-eksperter.

### Summary

En metodologi, der automatiserer end-to-end-processen med at anvende maskinlæring på virkelige problemer og reducerer manuelt arbejde.

## Key Concepts

- Hyperparameterjustering
- Feature-engineering
- Modelvalg
- Demokratisering

## Use Cases

- Hurtig prototyping for forretningsanalytikere
- Optimering af produktionspipelines i stort omfang
- Sammenligning af flere algoritmer automatisk

## Code Example

```python
from auto_ml import Predictor
predictor = Predictor(type_of_estimator='classifier')
predictor.fit(dataframe, target='label')
```
