### Summary

Una API de Transferencia de Estado Representacional es un estilo arquitectónico para diseñar aplicaciones en red que se basa en métodos HTTP estándar.

## Key Concepts

- Ausencia de estado
- Métodos HTTP
- URI basados en recursos
- Serialización JSON

## Use Cases

- Comunicación entre microservicios
- Integración con el backend de aplicaciones móviles
- Acceso a datos de terceros
