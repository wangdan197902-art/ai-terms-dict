## Definition

H2O es una plataforma de memoria principal de código abierto muy utilizada para el aprendizaje automático distribuido y escalable, así como para el análisis predictivo. Desarrollada originalmente por dos estudiantes de doctorado de Harvard, proporciona un marco unificado que facilita la construcción, implementación y gestión de modelos de machine learning en entornos empresariales complejos.

### Summary

Una plataforma de aprendizaje automático de código abierto y distribuida que admite varios algoritmos para análisis predictivo y aprendizaje profundo.

## Key Concepts

- Computación en memoria
- AutoML
- Aprendizaje profundo
- Procesamiento distribuido

## Use Cases

- Construcción de modelos predictivos para la evaluación de riesgos financieros
- Ingeniería automatizada de características y selección de modelos
- Despliegue de modelos de aprendizaje profundo en grandes conjuntos de datos
