## Definition

Un conjunto de datos 'mantenido fuera' consiste en ejemplos excluidos intencionalmente de la fase de entrenamiento de un modelo de aprendizaje automático. Este subconjunto se utiliza para evaluar qué tan bien

### Summary

Muestras de datos reservadas del conjunto de entrenamiento para evaluar el rendimiento del modelo y prevenir el sobreajuste durante el desarrollo.

## Key Concepts

- Generalización
- Sobreajuste
- Conjunto de validación
- Evaluación no sesgada

## Use Cases

- Ajuste de hiperparámetros
- Comparación de diferentes arquitecturas de modelos
- Estimación final del rendimiento antes de la producción
