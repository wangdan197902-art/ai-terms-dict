## Definition

La clasificación binaria es un problema fundamental de aprendizaje automático donde la variable de salida es categórica con exactamente dos resultados posibles, como verdadero/falso o correo no deseado/no no deseado. Algoritmos como la regresión logística se utilizan comúnmente.

### Summary

Una tarea de aprendizaje supervisado cuyo objetivo es predecir una de dos clases posibles para cada instancia de entrada.

## Key Concepts

- Límite de Decisión
- Matriz de Confusión
- Regresión Logística
- Métricas de Precisión

## Use Cases

- Detección de Spam
- Diagnóstico Médico
- Detección de Fraude
