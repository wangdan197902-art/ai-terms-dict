## Definition

El conjunto de datos Altlex consiste en pares de oraciones que comparten el mismo significado subyacente pero utilizan vocabulario o estructuras sintácticas diferentes. Se utiliza principalmente para entrenar modelos de incrustación.

### Summary

Un conjunto de datos que contiene formas léxicas alternativas utilizadas para entrenar modelos en equivalencia semántica y detección de paráfrasis.

## Key Concepts

- Equivalencia semántica
- Detección de paráfrasis
- Variación léxica
- Similitud vectorial

## Use Cases

- Entrenamiento de motores de búsqueda semántica
- Mejora de sistemas de respuesta a preguntas
- Optimización de métricas de similitud textual
