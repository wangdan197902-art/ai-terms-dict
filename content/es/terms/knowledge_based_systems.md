## Definition

Los sistemas basados en el conocimiento (KBS) son una rama de la inteligencia artificial que incorpora conocimiento específico de un dominio en un sistema informático para realizar tareas que típicamente requieren experiencia humana. Estos sistemas capturan y almacenan el conocimiento experto de un campo particular, permitiendo que la máquina razon sobre ese conocimiento para tomar decisiones o proporcionar recomendaciones.

### Summary

Sistemas de IA que utilizan bases de conocimiento explícitas y motores de inferencia para resolver problemas complejos que requieren experiencia humana.

## Key Concepts

- Base de conocimiento
- Motor de inferencia
- Sistemas expertos
- Razonamiento basado en reglas

## Use Cases

- Apoyo al diagnóstico médico
- Solución de problemas industriales
- Evaluación de riesgos financieros
