## Definition

Un Feature Store actúa como puente entre los equipos de ingeniería de datos y los de aprendizaje automático, proporcionando una visión unificada de las características tanto para el entrenamiento por lotes como para la inferencia en tiempo real. Garantiza la consistencia mediante

### Summary

Un repositorio centralizado diseñado para gestionar, compartir y servir características de manera consistente en el entrenamiento y la inferencia del aprendizaje automático.

## Key Concepts

- Desajuste entrenamiento-servicio (Training-serving skew)
- Gestión centralizada
- Versionado de características
- Infraestructura MLOps

## Use Cases

- Plataformas de ML empresariales
- Sistemas de detección de fraude en tiempo real
- Equipos de desarrollo de modelos colaborativos
