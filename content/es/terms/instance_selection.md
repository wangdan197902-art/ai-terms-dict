## Definition

La selección de instancias tiene como objetivo mejorar la eficiencia computacional y el rendimiento del modelo eliminando puntos de datos redundantes o ruidosos. A diferencia de la selección de características, opera sobre las filas del conjunto de datos.

### Summary

Una técnica de preprocesamiento que reduce el tamaño de un conjunto de datos seleccionando un subconjunto de instancias representativas.

## Key Concepts

- Reducción de datos
- Eliminación de ruido
- Subconjunto representativo
- Eficiencia computacional

## Use Cases

- Preprocesamiento de conjuntos de datos a gran escala
- Aceleración de búsquedas de vecinos más cercanos
- Limpieza de conjuntos de datos desequilibrados
