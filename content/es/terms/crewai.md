## Definition

CrewAI proporciona un entorno estructurado para construir sistemas multiagente donde cada agente tiene un rol, objetivo y conjunto de herramientas específicos. Simplifica la creación de flujos de trabajo permitiendo a los desarrolladores...

### Summary

CrewAI es un marco de trabajo para orquestar agentes de IA autónomos que juegan roles para colaborar en tareas complejas.

## Key Concepts

- Sistemas Multiagente
- Agentes de Juego de Roles
- Delegación de Tareas
- Orquestación de Agentes

## Use Cases

- Flujos de trabajo de investigación automatizada
- Cadenas complejas de atención al cliente
- Pipelines de generación y revisión de código
