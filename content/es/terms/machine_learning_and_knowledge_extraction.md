## Definition

Este campo combina técnicas de aprendizaje automático con procesamiento del lenguaje natural y minería de datos para transformar datos en bruto en conocimiento accionable. Implica entrenar modelos para reconocer entidades, relaciones y conceptos clave, permitiendo la automatización de tareas complejas de comprensión de la información.

### Summary

El proceso de utilizar algoritmos de aprendizaje automático para identificar patrones automáticamente y derivar información estructurada a partir de grandes conjuntos de datos no estructurados.

## Key Concepts

- Reconocimiento de patrones
- Procesamiento del lenguaje natural
- Minería de datos
- Ingeniería de características

## Use Cases

- Extraer el sentimiento de los clientes a partir de publicaciones en redes sociales
- Identificar condiciones médicas a partir de notas clínicas
- Automatizar la clasificación de documentos en despachos de abogados
