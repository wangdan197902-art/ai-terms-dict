## Definition

La minimización de datos es un principio fundamental de privacidad que exige a las organizaciones limitar la recopilación de datos a lo que sea adecuado, relevante y necesario. En el ámbito de la IA, esto implica diseñar modelos que requieran menos información sensible para funcionar eficazmente.

### Summary

El principio de recopilar y procesar únicamente los datos personales estrictamente necesarios para un propósito específico.

## Key Concepts

- Limitación de Finalidad
- Privacidad desde el Diseño
- Reducción de Datos
- Cumplimiento Normativo

## Use Cases

- Diseño de conjuntos de datos anonimizados para investigación pública
- Implementación de aprendizaje federado para mantener los datos crudos locales
- Auditoría de sistemas de IA para detectar retención excesiva de datos
