## Definition

En el procesamiento del lenguaje natural, el contexto es crucial para resolver ambigüedades, como entender pronombres o modismos basándose en oraciones anteriores. Las arquitecturas modernas como los transformadores utilizan mecanismos de atención para capturar este contexto.

### Summary

El contexto se refiere a la información circundante o el entorno que ayuda a un modelo de IA a interpretar los datos de entrada con precisión y generar respuestas relevantes.

## Key Concepts

- Mecanismo de Atención
- Modelado de Secuencias
- Resolución de Ambigüedades
- Tamaño de Ventana

## Use Cases

- Chatbots conversacionales
- Resumen de documentos
- Asistentes de finalización de código
