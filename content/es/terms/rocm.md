## Definition

ROCm (Radeon Open Compute) es un conjunto de controladores y software desarrollado por AMD para habilitar la computación de alto rendimiento en GPUs AMD. Proporciona bibliotecas, compiladores y herramientas necesarias para desarrollar aplicaciones paralelas.

### Summary

La plataforma de software de código abierto de AMD para computación en GPU, que proporciona alternativas al ecosistema CUDA de NVIDIA.

## Key Concepts

- Computación en GPU
- Software de Código Abierto
- Procesamiento Paralelo
- Aceleración de Hardware

## Use Cases

- Entrenamiento de modelos de IA en GPUs AMD
- Simulaciones científicas
- Clústeres de computación de alto rendimiento
