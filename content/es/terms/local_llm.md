## Definition

Ejecutar un LLM Local implica desplegar modelos de peso abierto directamente en hardware de consumo, como PC, Mac o servidores locales. Este enfoque elimina la dependencia de proveedores de API de terceros, garantizando mayor privacidad.

### Summary

Un LLM Local se refiere a ejecutar modelos de lenguaje grandes en hardware personal en lugar de servicios en la nube, priorizando la privacidad de los datos y la accesibilidad sin conexión.

- Privacidad de datos

- Requisitos de hardware

## Use Cases

- Asistentes de toma de notas privadas
- Herramientas de investigación sin conexión
- Procesamiento seguro de datos empresariales
