## Definition

La adaptación de dominio aborda el desafío cuando los datos de entrenamiento y prueba provienen de distribuciones diferentes. Al alinear las representaciones de características entre un dominio fuente etiquetado y uno objetivo sin etiquetar o con pocos datos, se mejora la generalización.

### Summary

Un método de aprendizaje automático que mejora el rendimiento del modelo en un dominio objetivo aprovechando el conocimiento de un dominio fuente.

## Key Concepts

- Dominio Fuente
- Dominio Objetivo
- Desplazamiento de Distribución
- Alineación de Características

## Use Cases

- Traducción de imágenes médicas entre diferentes tipos de escáneres
- Adaptación del reconocimiento de voz a diferentes acentos
- Sistemas de recomendación multiplataforma
