## Definition

WebSocket es un protocolo de comunicaciones informáticas que permite una comunicación persistente y bidireccional entre un cliente y un servidor. A diferencia de HTTP, que requiere una nueva conexión para cada ciclo de solicitud-respuesta...

### Summary

Un protocolo que proporciona canales de comunicación dúplex completo sobre una única conexión TCP para el intercambio de datos en tiempo real.

## Key Concepts

- Dúplex completo
- Conexión persistente
- Baja latencia
- Protocolo TCP

## Use Cases

- Aplicaciones de chat en tiempo real
- Actualizaciones en vivo de resultados deportivos
- Juegos multijugador en línea
