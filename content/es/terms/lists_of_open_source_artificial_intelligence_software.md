## Definition

Estos se refieren a repositorios organizados, como temas de GitHub, listas 'Awesome' o wikias mantenidas por la comunidad, que agregan software de código abierto relacionado con la inteligencia artificial. Sirven como recursos esenciales para descubrir herramientas.

### Summary

Colecciones o directorios curados que catalogan herramientas, bibliotecas y marcos de trabajo de IA disponibles públicamente.

## Key Concepts

- Código abierto
- Curación de recursos
- Mantenimiento comunitario
- Descubrimiento de software

## Use Cases

- Descubrimiento de herramientas de investigación
- Selección de la pila tecnológica para startups
- Recopilación de recursos educativos
