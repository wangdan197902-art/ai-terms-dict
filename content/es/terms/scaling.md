## Definition

El escalado es la metodología activa de expansión de sistemas de IA añadiendo más capas, neuronas o ejemplos de entrenamiento. Incluye técnicas como el entrenamiento distribuido en múltiples GPUs para manejar el aumento

### Summary

El escalado es el proceso de ajustar el tamaño del modelo o el volumen de datos para mejorar las capacidades de aprendizaje y el rendimiento.

## Key Concepts

- Entrenamiento distribuido
- Complejidad del modelo
- Sobreajuste
- Aceleración por hardware

## Use Cases

- Aprendizaje profundo distribuido
- Aumento de la profundidad de redes neuronales
- Expansión de corpus de entrenamiento
