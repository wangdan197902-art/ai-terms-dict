## Definition

La Programación Inductiva, a menudo denominada Síntesis de Programas, implica crear código de software basándose en especificaciones proporcionadas como pares de entrada-salida en lugar de instrucciones explícitas. El sistema infiere la lógica subyacente necesaria para transformar las entradas en las salidas deseadas.

### Summary

Un campo de la síntesis de programas que genera automáticamente programas informáticos a partir de ejemplos de entrada-salida.

## Key Concepts

- Síntesis de Programas
- Ejemplos de Entrada-Salida
- Codificación Automatizada
- Inferencia Lógica

## Use Cases

- Herramientas de autocompletado de código
- Generación de scripts a partir de datos
- Ayudas educativas para programación
