## Definition

Lanzado en agosto de 2024, Llama 3.1 expande la familia Llama para incluir un masivo modelo de 405 mil millones de parámetros junto con variantes más pequeñas de 8B y 70B. Una característica destacada es la ventana de contexto extendida.

### Summary

Llama 3.1 es la actualización de agosto de 2024 de Meta que presenta modelos de 8B, 70B y 405B parámetros con una ventana de contexto ampliada de 128k y capacidades nativas de uso de herramientas.

## Key Concepts

- Ventana de contexto larga
- Llamada a funciones (Function calling)
- Uso de herramientas
- 405B parámetros

## Use Cases

- Análisis de documentos largos
- Agentes de integración de API
- Síntesis de código avanzada
