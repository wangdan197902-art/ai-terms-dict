## Definition

Este enfoque emplea técnicas de satisfacción de restricciones dentro de una base de conocimiento para garantizar que los productos ensamblados cumplan con todos los requisitos técnicos y del cliente. Previene combinaciones inválidas mediante reglas lógicas estrictas definidas en el sistema.

### Summary

Un proceso automatizado que utiliza bases de conocimiento específicas del dominio para generar configuraciones de productos válidas a partir de restricciones del usuario.

## Key Concepts

- Satisfacción de restricciones
- Modelado de productos
- Sistemas basados en reglas
- Razonamiento automatizado

## Use Cases

- Ensamblaje personalizado de hardware informático
- Selección de licencias y características de software
- Configuración de maquinaria industrial
