## Definition

MindsDB actúa como un puente entre las bases de datos relacionales tradicionales y los flujos de trabajo modernos de aprendizaje automático. Permite a los usuarios crear modelos predictivos utilizando consultas SQL estándar, eliminando la necesidad de mover datos fuera del entorno de la base de datos.

### Summary

MindsDB es una plataforma de código abierto que permite a los desarrolladores construir, entrenar e implementar modelos de aprendizaje automático directamente dentro de sus bases de datos existentes utilizando SQL.

## Key Concepts

- ML basado en SQL
- Integración de bases de datos
- AutoML
- Modelado predictivo

## Use Cases

- Detección de fraude en tiempo real en bases de datos
- Predicción de abandono de clientes
- Informes automatizados con pronósticos
