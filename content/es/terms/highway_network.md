## Definition

Las Redes de Autopista están diseñadas para abordar el problema del gradiente que desaparece en el aprendizaje profundo mediante la incorporación de puertas adaptativas que controlan el flujo de información. Similar a las celdas LSTM, estas puertas permiten que la señal pase directamente o sea transformada, facilitando el entrenamiento de redes extremadamente profundas sin degradación del rendimiento.

### Summary

Una arquitectura de red neuronal profunda que introduce mecanismos de puerta para facilitar el flujo de gradientes a través de redes muy profundas.

## Key Concepts

- Mecanismo de Puerta
- Gradiente que Desaparece
- Aprendizaje Profundo
- Flujo de Información

## Use Cases

- Redes neuronales profundas
- Reconocimiento de voz
- Visión por computadora
