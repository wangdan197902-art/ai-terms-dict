## Definition

AutoML (Aprendizaje Automático Automatizado) agiliza el desarrollo de modelos de ML automatizando tareas como la preprocesamiento de datos, la ingeniería de características, la selección de modelos y el ajuste de hiperparámetros. Permite

### Summary

Una metodología que automatiza el proceso integral de aplicación del aprendizaje automático a problemas del mundo real, reduciendo el esfuerzo manual.

## Key Concepts

- Ajuste de hiperparámetros
- Ingeniería de características
- Selección de modelos
- Democratización

## Use Cases

- Prototipado rápido para analistas de negocios
- Optimización de canales de producción a gran escala
- Comparación automática de múltiples algoritmos
