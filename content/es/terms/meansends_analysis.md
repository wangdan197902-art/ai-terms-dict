## Definition

El análisis medios-fines es una estrategia cognitiva utilizada en la inteligencia artificial y la psicología para resolver problemas complejos. Implica comparar el estado actual de un problema con el estado objetivo deseado, identificar las diferencias entre ellos y seleccionar operadores o acciones que reduzcan esas diferencias. Este proceso se repite recursivamente hasta que se alcanza el objetivo final.

### Summary

Un método heurístico de resolución de problemas que reduce la diferencia entre el estado actual y el estado objetivo identificando subobjetivos.

## Key Concepts

- Búsqueda en Espacio de Estados
- Descomposición de Subobjetivos
- Planificación Heurística
- Reducción de Diferencias

## Use Cases

- Demostración automática de teoremas
- Planificación de rutas en robótica
- Resolución de rompecabezas complejos
