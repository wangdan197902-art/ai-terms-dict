## Definition

El conjunto de datos WikiHow consta de aproximadamente 60.000 artículos de instrucciones recopilados del sitio web WikiHow. Es ampliamente utilizado en la investigación de procesamiento del lenguaje natural para tareas como el resumen

### Summary

Un conjunto de datos a gran escala compuesto por artículos de cómo hacer cosas de WikiHow, utilizado principalmente para tareas de resumen de texto y generación de instrucciones.

## Key Concepts

- Resumen de texto
- Texto procedimental
- Extracción de instrucciones

## Use Cases

- Entrenamiento de modelos de resumen abstractivo
- Generación de instrucciones paso a paso
- Comprensión de estructuras del lenguaje procedimental
