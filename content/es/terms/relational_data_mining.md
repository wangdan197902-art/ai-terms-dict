## Definition

La minería de datos relacionales se centra en extraer información útil de bases de datos donde los datos están organizados en múltiples tablas relacionadas, en lugar de una única tabla plana. Aprovecha las relaciones entre estas entidades para encontrar patrones complejos.

### Summary

El proceso de descubrir patrones y conocimiento a partir de datos que consisten en múltiples tablas o entidades interrelacionadas.

## Key Concepts

- Entidad-relación
- Aprendizaje multi-relacional
- Minería de grafos
- Integración de bases de datos

## Use Cases

- Análisis de redes sociales
- Detección de fraude en transacciones financieras
- Descubrimiento de fármacos y bioinformática
