Hugging Face Diffusers es un kit modular diseñado para simplificar el uso de modelos de difusión. Ofrece tuberías preentrenadas para tareas como la generación de texto a imagen, la inpintación de imágenes y la super-resol

### Summary

Una biblioteca dentro del ecosistema de Hugging Face que proporciona implementaciones de vanguardia de modelos de difusión para la generación de imágenes, audio y texto.

- Abstracción de Tubería (Pipeline)
- Modelos Preentrenados
- Proceso de Denoising (Eliminación de Ruido)
- Hub Comunitario
