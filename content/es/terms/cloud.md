## Definition

La computación en la nube proporciona infraestructura escalable para cargas de trabajo de IA, permitiendo a los desarrolladores acceder a potentes GPU y almacenamiento sin mantener centros de datos físicos. Soporta varios modelos de servicio como IaaS, PaaS y SaaS.

### Summary

La nube se refiere a servidores remotos alojados en Internet utilizados para almacenar, gestionar y procesar datos y modelos de IA en lugar de hardware local.

## Key Concepts

- Escalabilidad
- Infraestructura como Servicio
- Almacenamiento Remoto
- Elasticidad

## Use Cases

- Entrenamiento de grandes modelos de lenguaje
- Alojamiento de APIs de IA para aplicaciones web
- Almacenamiento de conjuntos de datos masivos para análisis
