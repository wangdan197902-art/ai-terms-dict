## Definition

Dentro del desarrollo de IA, un proceso denota el flujo de trabajo sistemático necesario para transformar datos sin procesar en modelos o información accionable. Esto incluye etapas como la ingestión de datos, el preprocesamiento, la extracción de características

### Summary

Una serie estructurada de acciones o pasos realizados para lograr un objetivo o resultado computacional específico.

## Key Concepts

- Pipeline (Tubería de datos)
- Workflow (Flujo de trabajo)
- Automatización
- Iteración

## Use Cases

- Pipeline MLOps de extremo a extremo
- Procedimientos de limpieza de datos
- Calendarios de reentrenamiento de modelos
