## Definition

Este paso crítico consiste en adjuntar metadatos significativos a los puntos de datos en bruto para que los algoritmos puedan aprender la relación entre la entrada y la salida. Por ejemplo, cajas delimitadoras alrededor de objetos en imágenes

### Summary

La anotación de datos es el proceso de etiquetar datos en bruto, como imágenes o texto, para hacerlos adecuados para el entrenamiento de aprendizaje automático supervisado.

## Use Cases

- Entrenamiento de modelos de detección de objetos
- Construcción de clasificadores de análisis de sentimientos
- Creación de conjuntos de datos para reconocimiento de voz
