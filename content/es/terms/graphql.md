## Definition

Desarrollado por Facebook, GraphQL proporciona una descripción completa y comprensible de los datos en tu API, otorgando a los clientes el poder de pedir exactamente lo que necesitan y nada más. Reemplaza múltiples puntos finales de API con uno solo.

### Summary

GraphQL es un lenguaje de consulta para APIs y un tiempo de ejecución para satisfacer esas consultas con datos existentes, permitiendo a los clientes solicitar exactamente lo que necesitan.

- Definición de Esquema
- Lenguaje de Consulta
- Único Punto de Entrada
- Seguridad de Tipos

## Use Cases

- Backends de aplicaciones web modernas
- Obtención de datos para aplicaciones móviles
- Orquestación de microservicios
