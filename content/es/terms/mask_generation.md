## Definition

La generación de máscaras implica producir máscaras espaciales o temporales que determinan qué elementos de un conjunto de datos son visibles o activos durante operaciones específicas. En visión por computadora, se utiliza para la segmentación de objetos

### Summary

El proceso de crear máscaras binarias o probabilísticas para ocultar o enfatizar selectivamente partes de los datos de entrada durante el procesamiento del modelo.

## Key Concepts

- Enmascaramiento binario
- Máscaras de atención
- Inpainting (Restauración de imágenes)
- Selección de características

## Use Cases

- Inpainting y restauración de imágenes
- Mecanismos de atención en Transformers
- Detección y segmentación de objetos
