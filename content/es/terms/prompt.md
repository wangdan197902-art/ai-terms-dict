## Definition

Un prompt sirve como la interfaz principal para interactuar con modelos de lenguaje grandes y otros sistemas de IA generativa. Define el contexto, el tono y las restricciones para la salida del modelo. Un prompt efectivo

### Summary

Un texto de entrada o instrucción proporcionado a un modelo de IA generativa para elicitar una respuesta o comportamiento específico.

## Key Concepts

- Ajuste por instrucciones
- Ventana de contexto
- Aprendizaje few-shot
- Tokenización

## Use Cases

- Generación de contenido de escritura creativa
- Completado de código y depuración
- Interacciones con chatbots de atención al cliente
