## Definition

Un linter es una utilidad que realiza un análisis estático en el código fuente sin ejecutarlo. Verifica errores de sintaxis, posibles errores, 'olores de código' (código de mala calidad) y desviaciones de las guías de estilo o mejores prácticas.

### Summary

Una herramienta de análisis estático de código utilizada para señalar errores de programación, errores, errores estilísticos y construcciones sospechosas.

## Key Concepts

- Análisis estático
- Calidad del código
- Aplicación de estilo
- Detección de errores

## Use Cases

- Revisión automática de código
- Ganchos previos al compromiso (pre-commit hooks)
- Validación en pipelines de CI/CD
