## Definition

### Summary

Un verificador de tipos es una herramienta de análisis estático que verifica si los tipos de variables y expresiones en el código cumplen con las reglas del sistema de tipos del lenguaje de programación.

## Key Concepts

- Análisis estático
- Seguridad de tipos
- Prevención de errores
- Calidad del código

## Use Cases

- Validación de formas de tensores en PyTorch
- Garantizar el cumplimiento de contratos de API
- Refactorización de grandes bases de código de IA
