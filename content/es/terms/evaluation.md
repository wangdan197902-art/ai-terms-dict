## Definition

La evaluación implica medir sistemáticamente qué tan bien realiza un modelo de IA tareas específicas utilizando métricas cuantitativas (por ejemplo, precisión, puntuación F1, BLEU) y evaluaciones cualitativas. Incluye la validación cruzada, las pruebas en conjuntos de datos de prueba independientes y la auditoría de sesgos para garantizar la equidad y la seguridad.

### Summary

La evaluación es el proceso de medir el rendimiento, la precisión y la robustez de un modelo de IA frente a métricas y conjuntos de datos predefinidos.

## Key Concepts

- Métricas
- Conjunto de Validación
- Generalización
- Establecimiento de Referencias

## Use Cases

- Comparar versiones de modelos durante el ajuste de hiperparámetros
- Auditar modelos en busca de equidad y sesgo
- Certificar sistemas de IA para el cumplimiento normativo
