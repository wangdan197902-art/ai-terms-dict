## Definition

En contextos de IA, 'fuente' denota típicamente la procedencia de conjuntos de datos de entrenamiento, bibliotecas de código abierto o pesos de modelos preentrenados. Rastrear las fuentes es crítico para la reproducibilidad, el cumplimiento de licencias y la auditoría de sesgos.

### Summary

Se refiere al origen de los datos, código o modelos utilizados en el desarrollo e implementación de IA.

## Key Concepts

- Procedencia de Datos
- Código Abierto
- Flujo de Entrada
- Reproducibilidad

## Use Cases

- Auditoría de datos de entrenamiento para detectar sesgos
- Gestión de versiones de modelos
- Citación de dependencias de código abierto
