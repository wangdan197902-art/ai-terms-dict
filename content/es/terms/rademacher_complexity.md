## Definition

La complejidad de Rademacher evalúa qué tan bien una clase de hipótesis puede correlacionarse con etiquetas aleatorias (ruido). Sirve como un proxy para la capacidad o flexibilidad del modelo. Una menor complejidad sugiere una mejor generalización.

### Summary

Una medida estadística utilizada en la teoría del aprendizaje para cuantificar la riqueza de una clase de funciones mediante su capacidad para ajustarse a ruido aleatorio.

## Key Concepts

- Límites de generalización
- Capacidad del modelo
- Sobreajuste
- Teoría del aprendizaje estadístico

## Use Cases

- Análisis teórico de las SVM
- Derivación de tasas de convergencia
- Comparación de clases de modelos
