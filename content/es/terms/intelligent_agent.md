## Definition

Un agente inteligente es un sistema capaz de percibir su entorno a través de sensores o entradas de datos, procesar esta información utilizando algoritmos de razonamiento y actuar sobre el entorno mediante actuadores

### Summary

Una entidad de software autónoma que percibe su entorno, razona sobre las acciones y ejecuta tareas para alcanzar objetivos específicos.

## Key Concepts

- Autonomía
- Bucle percepción-acción
- Orientación a objetivos
- Reactividad

## Use Cases

- Bots de trading autónomos que ejecutan estrategias de mercado
- Automatización robótica de procesos (RPA) que maneja flujos de trabajo multietapa
- Asistentes personales que gestionan tareas de calendario y correo electrónico
