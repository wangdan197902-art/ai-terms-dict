## Definition

Un agente de IA es una entidad de software que opera de forma autónoma dentro de un entorno definido para cumplir con objetivos predefinidos. Utiliza mecanismos de percepción para recopilar datos y procesa esta información para tomar decisiones y ejecutar acciones.

### Summary

Un sistema autónomo diseñado para percibir su entorno y realizar acciones para alcanzar objetivos específicos.

## Key Concepts

- autonomía
- bucle percepción-acción
- comportamiento orientado a objetivos
- interacción con el entorno

## Use Cases

- Bots de atención al cliente automatizados
- Navegación de vehículos autónomos
- Asistentes de generación de código
