## Definition

Un modelo impulsado por datos es un tipo de sistema de inteligencia artificial donde el comportamiento y las predicciones emergen de patrones identificados dentro de datos históricos, en lugar de estar definidos por reglas codificadas explícitamente o conocimientos previos teóricos. Estos modelos dependen de la cantidad y calidad de los datos disponibles para aprender.

### Summary

Un modelo computacional cuyos parámetros y estructura se aprenden directamente de datos empíricos en lugar de programación explícita.

## Key Concepts

- Aprendizaje Empírico
- Reconocimiento de Patrones
- Inferencia Estadística
- Modelos de Caja Negra

## Use Cases

- Predecir tendencias del mercado de valores utilizando datos históricos de precios
- Diagnosticar enfermedades a partir de escaneos de imágenes médicas
- Predecir patrones climáticos utilizando observaciones satelitales
