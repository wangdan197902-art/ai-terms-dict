## Definition

Desarrollado por Google, EfficientNet utiliza un método de escalado compuesto para equilibrar la profundidad de la red, el ancho y la resolución de las imágenes de entrada. Este enfoque permite al modelo alcanzar una precisión de última generación mientras

### Summary

EfficientNet es una familia de arquitecturas de redes neuronales convolucionales que escala uniformemente la profundidad, el ancho y la resolución para lograr mayor precisión con menos parámetros.

## Key Concepts

- Escalado compuesto
- Bloques MobileNetV2
- Atención SE
- Eficiencia de parámetros

## Use Cases

- Clasificación de imágenes en dispositivos móviles
- Detección de objetos en entornos con recursos limitados
- Extracción de características para conjuntos de datos visuales a gran escala
