## Definition

Una Interfaz de Línea de Comandos (CLI) permite a los usuarios controlar el software ingresando comandos textuales en lugar de utilizar elementos gráficos. En el desarrollo de IA, las CLI son esenciales para ejecutar scripts, gestionar modelos y automatizar tareas de entrenamiento.

### Summary

La Interfaz de Línea de Comandos es un método basado en texto para interactuar con programas de software escribiendo comandos en un terminal.

## Key Concepts

- Entrada de Texto
- Emulación de Terminal
- Scripting
- Automatización

## Use Cases

- Ejecución de scripts de inferencia de modelos
- Gestión de recursos de IA en la nube
- Automatización de pipelines de preprocesamiento de datos
