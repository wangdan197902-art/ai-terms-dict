## Definition

El aprendizaje activo reduce la cantidad de datos etiquetados necesarios permitiendo que el modelo elija las instancias más informativas para la etiquetatura humana. En lugar de recibir muestras aleatorias pasivamente, el algoritmo selecciona activamente qué datos necesita aprender.

### Summary

Una estrategia de aprendizaje automático donde el algoritmo consulta selectivamente a un usuario u oráculo para etiquetar nuevos puntos de datos, optimizando el proceso de entrenamiento.

## Key Concepts

- Estrategia de consulta
- Eficiencia de etiquetado
- Muestreo por incertidumbre
- Humano en el bucle

## Use Cases

- Anotación de imágenes médicas con radiólogos expertos
- Etiquetado de análisis de sentimiento para dialectos raros
- Selección de escenarios para conducción autónoma
