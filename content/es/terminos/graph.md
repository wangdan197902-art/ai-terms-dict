## Definition

Un grafo es una estructura de datos fundamental en IA compuesta por vértices (nodos) y aristas (enlaces) que denotan relaciones. Las Redes Neuronales de Grafos (GNN) aprovechan esta estructura para realizar aprendizaje en no-E

### Summary

Una estructura matemática compuesta por nodos conectados por aristas, utilizada para representar relaciones entre entidades.

## Key Concepts

- Nodos/Vértices
- Aristas
- Matriz de Adyacencia
- Topología

## Use Cases

- Grafos de Conocimiento
- Análisis de Redes Sociales
- Predicción de Propiedades Moleculares
