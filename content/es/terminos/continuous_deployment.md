## Definition

El despliegue continuo es una extensión de la entrega continua que automatiza todo el proceso de lanzamiento. Una vez que los cambios de código superan todas las barreras de calidad, incluidas las pruebas unitarias, de integración y de seguridad, se implementan automáticamente.

### Summary

Una práctica de ingeniería de software donde cada cambio de código que pasa las pruebas automatizadas se libera automáticamente en producción.

## Key Concepts

- Pruebas automatizadas
- Pipelines CI/CD
- Despliegue sin tiempo de inactividad
- Banderas de función (Feature Flags)

## Use Cases

- Plataformas de comercio algorítmico de alta frecuencia
- Aplicaciones web a gran escala
- Gestión de arquitecturas de microservicios
