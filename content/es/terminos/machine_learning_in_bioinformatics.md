## Definition

Este campo interdisciplinario utiliza el aprendizaje automático para procesar grandes volúmenes de datos biológicos, permitiendo a los investigadores predecir funciones génicas, clasificar enfermedades y comprender las interacciones moleculares, acelerando así el descubrimiento científico y médico.

### Summary

La aplicación de modelos computacionales para analizar datos biológicos, como secuencias genómicas y estructuras proteicas, con el fin de descubrir conocimientos biológicos.

## Key Concepts

- Análisis genómico
- Plegamiento de proteínas
- Modelado predictivo
- Datos de alto rendimiento

## Use Cases

- Predecir estructuras proteicas a partir de secuencias de aminoácidos
- Clasificar tipos de cáncer basándose en la expresión génica
- Identificar posibles dianas farmacológicas en bases de datos moleculares
