## Definition

En informática y aprendizaje profundo, un tensor es un objeto matemático que generaliza escalares, vectores y matrices a dimensiones superiores. Se caracteriza por su rango (número de dimensiones) y su forma.

### Summary

Una matriz multidimensional que sirve como estructura de datos fundamental para los marcos de aprendizaje profundo.

## Key Concepts

- Rango
- Forma
- Dimensionalidad
- Broadcasting (Difusión)

## Use Cases

- Procesamiento de imágenes (tensores 4D)
- Almacenamiento de pesos de redes neuronales
- Entrada de datos por lotes

## Code Example

```python
import torch
t = torch.tensor([[1, 2], [3, 4]])
```
