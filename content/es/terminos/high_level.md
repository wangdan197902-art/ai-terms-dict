## Definition

En IA, 'de alto nivel' denota abstracciones que simplifican procesos complejos. Los lenguajes de alto nivel (como Python) o las API permiten a los desarrolladores construir modelos sin gestionar la memoria

### Summary

Se refiere a representaciones abstractas o interfaces de programación que ocultan los detalles de implementación de bajo nivel al usuario.

## Key Concepts

- Abstracción
- API
- Ocultamiento de complejidad
- Productividad

## Use Cases

- Uso de API de alto nivel de PyTorch/TensorFlow
- Ingeniería de características en pipelines de ML
- Lenguajes específicos de dominio para IA
