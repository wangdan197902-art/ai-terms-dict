## Definition

La programación, también conocida como codificación, implica traducir la lógica y los requisitos humanos a un formato que las computadoras puedan ejecutar. Utiliza sintaxis y semántica específicas definidas por los lenguajes de programación.

### Summary

La programación es el proceso de escribir instrucciones para computadoras utilizando lenguajes de programación para crear software, aplicaciones o scripts.
