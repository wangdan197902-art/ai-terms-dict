## Definition

Un punto de referencia sirve como un punto de referencia estandarizado para comparar las capacidades de diferentes modelos o algoritmos de IA. Típicamente implica un conjunto de datos curado y métricas de evaluación específicas, como precisión o recall.

### Summary

Abreviatura de benchmark, un conjunto de pruebas estándar o métrica utilizada para evaluar el rendimiento de modelos de IA.

## Key Concepts

- Métricas de evaluación
- Pruebas estandarizadas
- Comparación de rendimiento
- Curación de conjuntos de datos

## Use Cases

- Benchmark GLUE para PLN
- ImageNet para visión por computadora
- Selección de modelos en producción
