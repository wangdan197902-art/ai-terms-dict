## Definition

Los aspectos estructurales definen cómo se organizan los datos o las capas de una red neuronal. En las redes neuronales gráficas, la estructura se refiere a las conexiones entre nodos; en el aprendizaje profundo, se refiere a la topología de las capas.

### Summary

Relativo a la organización subyacente, arquitectura o disposición de los componentes dentro de un sistema.

## Key Concepts

- Arquitectura
- Topología
- Organización

## Use Cases

- Redes Neuronales Gráficas (GNN)
- Búsqueda de Arquitectura Neuronal (NAS)
- Diseño de esquemas de datos
