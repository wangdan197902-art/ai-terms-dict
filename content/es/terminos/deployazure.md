## Definition

Desplegar en Azure implica utilizar herramientas nativas de la nube como Azure Machine Learning, Azure Kubernetes Service (AKS) o Azure Functions para servir modelos de ML a escala. Abarca la gestión de recursos de cómputo...

### Summary

El proceso de alojar y ejecutar modelos de aprendizaje automático en servicios de infraestructura en la nube de Microsoft Azure.

## Key Concepts

- Infraestructura en la Nube
- Servicio de Modelos (Model Serving)
- Escalabilidad
- Pipelines CI/CD

## Use Cases

- Puntos finales de API en tiempo real para inferencia
- Puntuación por lotes de grandes conjuntos de datos
- Gestión de modelos de grado empresarial
