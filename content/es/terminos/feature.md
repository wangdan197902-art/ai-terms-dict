## Definition

En el aprendizaje automático, una característica es un atributo o variable distinta que describe una instancia dentro de un conjunto de datos. Las características pueden ser numéricas, categóricas o textuales, y sirven como la entrada fundamental para los modelos.

### Summary

Una propiedad o característica medible individual de un fenómeno observado, que sirve como datos de entrada para los modelos de aprendizaje automático.

## Key Concepts

- Variable de entrada
- Atributo
- Columna del conjunto de datos
- Entrada ML

## Use Cases

- Entradas para modelado predictivo
- Análisis y visualización de datos
- Procesos de selección de características
