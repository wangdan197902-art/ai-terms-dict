## Definition

Server-Sent Events (SSE) maken eenrichtingscommunicatie mogelijk van de server naar de client, waarbij de server gegevens continu kan streamen zonder dat de client herhaaldelijk moet pollen. Het maakt gebruik van gewone HTTP, wat het eenvoudig te implementeren maakt.

### Summary

Een standaardprotocol waarmee webservers realtime-updates naar clients kunnen pushen via één enkele HTTP-verbinding.

## Key Concepts

- Gebeurtenisstroom
- Eenrichtingscommunicatie
- Automatische herbinding
- Op HTTP gebaseerd

## Use Cases

- Live aandelenkoersen
- Realtime-meldingen
- Updates voor voortgangsregistratie
