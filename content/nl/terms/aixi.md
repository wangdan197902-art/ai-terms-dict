## Definition

AIXI is een theoretisch kader dat door Marcus Hutter is voorgesteld en een geïdealiseerde intelligente agent definieert. Het combineert Solomonoff-inductie voor het voorspellen van de omgeving met versterkingsleren (reinforcement learning) voor het maken van beslissingen die de verwachte beloning maximaliseren.

### Summary

AIXI is een wiskundige theorie over algemene kunstmatige intelligentie die een optimale agent modelleert die met zijn omgeving interageert.

## Key Concepts

- Solomonoff-inductie
- Versterkingsleren
- Kolmogorov-complexiteit
- Optimale agent

## Use Cases

- Theoretisch onderzoek naar AGI
- Benchmarken van RL-algoritmen
- Begrip van de grenzen van intelligentie
