## Definition

In kunstmatige intelligentie is een vector een fundamentele gegevensstructuur die wordt gebruikt om informatie numeriek weer te geven. Het bestaat uit een geordende lijst van getallen (elementen) die kenmerken van een entiteit mappen naar

### Summary

Een wiskundig object dat een hoeveelheid vertegenwoordigt met zowel grootte als richting, vaak gebruikt in AI om datapunten in een hoge-dimensionale ruimte te coderen.

## Key Concepts

- Dimensionaliteit
- Embedding
- Grootte
- Richting

## Use Cases

- Semantisch zoeken in databases
- Embeddings voor natuurlijke taalverwerking
- Trekkings van kenmerken bij beeldherkenning

## Code Example

```python
import numpy as np
vec = np.array([1.0, 2.0, 3.0])
print(vec.shape)
```
