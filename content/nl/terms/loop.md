## Definition

Een fundamentele structuur voor besturingsstromen in de informatica en AI-ontwikkeling. Een lus stelt algoritmen in staat door datasets te itereren, herhaalde berekeningen uit te voeren of trainingsperioden (epochs) uit te voeren. Veelvoorkomende typen zijn for-lussen en while-lussen.

### Summary

Een programmeerconstructie die een blok code meerdere keren herhaalt tot aan een bepaalde voorwaarde is voldaan.

## Key Concepts

- Iteratie
- Besturingsstroom
- Epochs
- Batchverwerking

## Use Cases

- Het trainen van neurale netwerken over meerdere epochs
- Het doorlopen van steekproeven in een dataset
- Stap-voor-stap uitvoering in reinforcement learning

## Code Example

```python
for epoch in range(10):
    for batch in dataloader:
        train_step(batch)
```
