## Definition

In de statistische leertheorie vertegenwoordigt een leerbaar functieklassen de hypothese-ruimte die beschikbaar is voor een algoritme. Het definieert het bereik van patronen of afbeeldingen die het model potentieel kan vastleggen.

### Summary

Een leerbaar functieklassen is een verzameling wiskundige functies, gedefinieerd door een specifieke modelarchitectuur en parameterruimte, die een leeralgoritme kan optimaliseren.

- Hypothese-ruimte
- Modelcapaciteit
- VC-dimensie

## Use Cases

- Selecteren van geschikte modelarchitecturen
- Theoretische analyse van generalisatiegrenzen
- Begrijpen van modelbeperkingen
