## Definition

In de computerwetenschap en deep learning is een tensor een wiskundig object dat scalairs, vectoren en matrices generaliseert naar hogere dimensies. Het wordt gekenmerkt door zijn rang (aantal dimensies) en vorm.

### Summary

Een multidimensionaal array dat dient als de fundamentele gegevensstructuur voor deep learning-frameworks.

## Key Concepts

- Rang
- Vorm
- Dimensionaliteit
- Broadcasting

## Use Cases

- Beeldverwerking (4D-tensors)
- Opslag van neurale netwerkgewichten
- Gebatchte gegevensinvoer

## Code Example

```python
import torch
t = torch.tensor([[1, 2], [3, 4]])
```
