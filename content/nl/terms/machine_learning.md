## Definition

Machine Learning (ML) stelt computers in patronen te leren uit historische gegevens en beslissingen of voorspellingen te doen op nieuwe, onbekende gegevens. Het omvat verschillende technieken, waaronder supervised learning, unsupervised learning en reinforcement learning.

### Summary

Een deelgebied van AI gericht op het bouwen van systemen die uit gegevens leren om de prestaties te verbeteren zonder expliciete programmering.

## Key Concepts

- Supervised Learning
- Unsupervised Learning
- Modeltraining
- Voorspelling

## Use Cases

- Fraudedetectie
- Aanbevelingssystemen
- Spraakherkenning
