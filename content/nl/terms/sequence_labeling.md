## Definition

Bij sequencielabeling wordt voor elk token in een gegeven invoersequentie, zoals woorden in een zin of tekens in een string, een categorisch label voorspeld. Veelvoorkomende toepassingen zijn onder andere Named Entity Recognition (herkenning van eigennamen) en Part-of-Speech tagging (woordsoortbepaling).

### Summary

Een taak voor natuurlijke taalverwerking waarbij aan elk element in een invoerreeks een label wordt toegewezen.

## Key Concepts

- Tokenclassificatie
- Contextuele afhankelijkheid
- Bidirectionele codering
- CRF-lagen (Conditional Random Fields)

## Use Cases

- Named Entity Recognition
- Part-of-Speech Tagging
- Syntactische chunking
