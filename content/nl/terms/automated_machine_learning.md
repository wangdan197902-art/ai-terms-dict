## Definition

AutoML (Automated Machine Learning) stroomlijnt de ontwikkeling van ML-modellen door taken zoals gegevensvoorverwerking, feature engineering (kenmerkengineering), modelselectie en hyperparameter-tuning te automatiseren. Het maakt het mogelijk (voor bredere groepen ontwikkelaars).

### Summary

Een methodologie die het end-to-end proces van het toepassen van machine learning op echte problemen automatiseert, waardoor handmatig werk wordt verminderd.

## Key Concepts

- Hyperparameter-tuning
- Feature engineering
- Modelselectie
- Democratisering

## Use Cases

- Snelle prototyping voor bedrijfsanalisten
- Optimaliseren van productiepipelines op grote schaal
- Automatisch vergelijken van meerdere algoritmen

## Code Example

```python
from auto_ml import Predictor
predictor = Predictor(type_of_estimator='classifier')
predictor.fit(dataframe, target='label')
```
