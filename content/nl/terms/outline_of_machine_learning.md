## Definition

Deze term beschrijft de structurele indeling van machine learning in supervised, unsupervised, semi-supervised en reinforcement learning. Het omvat kernfamilieën van algoritmen zoals lineaire regressie...

### Summary

Een uitgebreide classificatie van machine learning-paradigma's, algoritmen en evaluatiemetrieken.

## Key Concepts

- Supervised Learning (Toezichtend leren)
- Unsupervised Learning (Onbeheerd leren)
- Reinforcement Learning (Versterkend leren)
- Model Evaluation (Modelbeoordeling)

## Use Cases

- Gidsen voor algoritmeselectie
- Planning van data science-workflows
- Creëren van academische taxonomieën
