## Definition

Beschrijvingslogica's (DL) zijn beslisbare fragmenten van de eerste-orde logica die de theoretische basis vormen voor ontologieën, met name de Web Ontology Language (OWL). Ze maken een precieze definitie en redenering over concepten mogelijk.

### Summary

Een familie van formele talen voor kennisrepresentatie, gebruikt om de conceptuele structuur van domeinen weer te geven en hierover te redeneren.

## Key Concepts

- Ontologie
- Redeneren
- Subsumptie
- Concepthiërarchie

## Use Cases

- Bouwen van semantische webontologieën
- Medische kennisgrafieken
- Geautomatiseerde classificatiesystemen
