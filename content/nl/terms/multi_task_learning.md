### Summary

Multitask learning is een machine learning-paradigma waarbij een model gelijktijdig wordt getraind op meerdere gerelateerde taken om de generalisatie te verbeteren.

## Key Concepts

- Gedeelde representatie
- Inductieve bias
- Taakinterferentie
- Generalisatie

## Use Cases

- Natuurlijke taalverwerking (bijv. NER en POS-tagging)
- Computervisie (bijv. detectie en segmentatie)
- Spraakherkenning en -synthese
