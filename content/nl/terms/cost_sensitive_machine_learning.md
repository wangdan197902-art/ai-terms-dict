## Definition

Kostengevoelig machine leren breidt traditioneel supervised learning uit door verschillende straffen toe te kennen aan verschillende soorten fouten. In echte scenario's hebben vals-positieven en vals-negatieven vaak verschillende economische gevolgen.

### Summary

Een machine learning-paradigma dat kosten voor verkeerde classificaties in het trainingsproces integreert om te optimaliseren op economische impact in plaats van alleen nauwkeurigheid.

## Key Concepts

- Aanpassing van verliesfunctie
- Klassenongelijkheid
- Kosten voor verkeerde classificatie
- Optimalisatiedoelstelling

## Use Cases

- Fraudedetectie in de bankwereld
- Medisch ziektescreening
- Spamfiltering met hoge kosten voor vals-positieven
