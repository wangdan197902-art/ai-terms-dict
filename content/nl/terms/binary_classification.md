## Definition

Binair classificeren is een fundamenteel machine learning-probleem waarbij de uitvoervariabele categorisch is met precies twee mogelijke uitkomsten, zoals waar/onwaar of spam/niet spam. Algoritmen zoals logistische regressie worden hier vaak voor gebruikt.

### Summary

Een taak binnen supervised learning waarbij het doel is om voor elk invoervoorwerp één van twee mogelijke klassen te voorspellen.

## Key Concepts

- Beslisgrens
- Confusiematrix
- Logistische regressie
- Accuratesse-metrieken

## Use Cases

- Spamdetectie
- Medische diagnose
- Fraudedetectie
