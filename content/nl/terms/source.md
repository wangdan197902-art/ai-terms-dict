## Definition

In AI-contexten duidt 'bron' meestal op de herkomst van trainingsdatasets, open-source bibliotheken of voorgecompileerde modelgewichten. Het volgen van bronnen is cruciaal voor reproduceerbaarheid, licentienaleving...

### Summary

Verwijst naar de oorsprong van data, code of modellen die worden gebruikt bij de ontwikkeling en implementatie van AI.

## Key Concepts

- Gegevensherkomst
- Open Source
- Invoerstream
- Reproduceerbaarheid

## Use Cases

- Controleren van trainingsdata op bias
- Beheer van modelversies
- Noemen van open-source afhankelijkheden
