## Definition

In de context van AI en data science verwijst numeriek naar gegevenstypen of methoden die kwantitatieve waarden omvatten, zoals gehele getallen, drijvende-kommagetallen en decimalen. In tegenstelling tot categorische of tekstuele data.

### Summary

Betrekking hebbend op, betrokken bij of gebruikmakend van getallen of wiskundige berekeningen in plaats van symbolische logica of tekst.

## Key Concepts

- Drijvende-kommagetallen
- Gehele getallen
- Gegevenstypen
- Kwantitatieve data

## Use Cases

- Statistische modellering
- Financiële prognoses
- Verwerking van sensordata

## Code Example

```python
import numpy as np
arr = np.array([1.5, 2.7, 3.9])
mean_val = np.mean(arr)
print(mean_val)
```
