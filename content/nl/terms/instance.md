## Definition

In machine learning verwijst een instantie naar één specifiek voorbeeld uit de dataset. Het bestaat uit een reeks invoerkenmerken (attributen) en eventueel een doellabel. Instanties zijn de fundamentele eenheid.

### Summary

Een enkel datapunt of observatie dat wordt gebruikt in machine learning-taken, meestal voorgesteld als een vector van kenmerken.

## Key Concepts

- Datapunt
- Kenmerkvector
- Trainingsgegevens
- Observatie

## Use Cases

- Supervised learning-training (Toezichtleren)
- Opsplitsen van datasets
- Voorbeelden bij anomaliedetectie
