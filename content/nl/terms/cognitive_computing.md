## Definition

Cognitieve computing is een tak van kunstmatige intelligentie die streeft naar natuurlijke interactie met mensen terwijl tegelijkertijd menselijke hersenprocessen worden gesimuleerd. Deze systemen maken gebruik van machine learning, deep learning en andere technieken.

### Summary

Cognitieve computing verwijst naar systemen die menselijke denkprocessen simuleren in een gecomputeriseerd model, waarbij AI wordt gebruikt om functies van de menselijke hersenen na te bootsen.

## Key Concepts

- Machine Learning
- Natural Language Processing
- Mens-machine-interactie
- Omgaan met ambiguïteit

## Use Cases

- Ondersteuning bij medische diagnose
- Klantenservice-chatbots
- Analyse van financiële risico's
