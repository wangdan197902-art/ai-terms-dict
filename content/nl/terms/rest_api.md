## Definition

REST-API's stellen clients en servers in staat om te communiceren door gebruik te maken van staatloze operaties via HTTP-protocollen zoals GET, POST, PUT en DELETE. Ze structureren bronnen als URI's en gebruiken standaardformaten.

### Summary

Een Representational State Transfer-API is een architecturale stijl voor het ontwerpen van netwerktoepassingen die vertrouwt op standaard HTTP-methoden.

## Key Concepts

- Staatloosheid
- HTTP-methoden
- Op bronnen gebaseerde URI's
- JSON-serialisatie

## Use Cases

- Communicatie tussen microservices
- Backend-integratie voor mobiele apps
- Toegang tot gegevens van derden
