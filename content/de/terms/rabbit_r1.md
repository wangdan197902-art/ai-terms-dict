## Definition

Das Rabbit r1 ist ein dediziertes Hardwaregerät, das von Rabbit Inc. eingeführt wurde und auf seinem proprietären Large Action Model (LAM) basiert. Im Gegensatz zu allgemeinen Smartphones konzentriert es sich darauf, bestimmte digitale Aufgaben auszuführen.

### Summary

Ein handgehaltenes, KI-gestütztes Gerät, das das Large Action Model (LAM) nutzt, um Aufgaben autonom auszuführen.

## Key Concepts

- Large Action Model
- KI-Hardware
- Intentionserkennung
- Aufgabenautomatisierung

## Use Cases

- Autonome Buchungs- und Reservierungsdienste
- Aufgabenausführung über Apps hinweg
- Sprachgesteuerte digitale Assistenz
