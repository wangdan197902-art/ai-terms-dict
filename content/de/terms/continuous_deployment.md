## Definition

Continuous Deployment ist eine Erweiterung von Continuous Delivery, die den gesamten Release-Prozess automatisiert. Sobald Code-Änderungen alle Qualitätskontrollen bestehen, einschließlich Unit-Tests, Integrationstests und Sicherheitsprüfungen, werden sie automatisch live geschaltet.

### Summary

Eine Praxis der Softwareentwicklung, bei der jeder Code-Change, der automatisierte Tests besteht, automatisch in die Produktionsumgebung bereitgestellt wird.

## Key Concepts

- Automatisierte Tests
- CI/CD-Pipelines
- Ausfallfreie Bereitstellung (Zero-downtime Deployment)
- Feature Flags

## Use Cases

- Plattformen für Hochfrequenzhandel
- Webanwendungen im großen Maßstab
- Verwaltung von Microservices-Architekturen
