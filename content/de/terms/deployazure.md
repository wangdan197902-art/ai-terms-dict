## Definition

Die Bereitstellung auf Azure umfasst die Nutzung cloudnativer Tools wie Azure Machine Learning, Azure Kubernetes Service (AKS) oder Azure Functions, um ML-Modelle skalierbar bereitzustellen. Dazu gehört auch das Management von Rechenressourcen und die Überwachung der Modelle.

### Summary

Der Prozess des Hostings und Ausführens von Machine-Learning-Modellen auf der Cloud-Infrastruktur von Microsoft Azure.

## Key Concepts

- Cloud-Infrastruktur
- Modellbereitstellung (Model Serving)
- Skalierbarkeit
- CI/CD-Pipelines

## Use Cases

- Echtzeit-API-Endpunkte für Inferenzen
- Batch-Bewertung großer Datensätze
- Unternehmensklassiges Modellmanagement
