## Definition

Das Lernen von Aktionsmodellen beinhaltet, dass ein Agent eine interne Darstellung davon aufbaut, wie seine Aktionen die Umgebung von einem Zustand in einen anderen überführen. Im Gegensatz zur passiven Beobachtung nutzt diese Methode die Interaktion des Agenten mit der Umgebung, um ein Modell der Umgebungsregeln zu erstellen.

### Summary

Eine Technik des verstärkenden Lernens, bei der ein Agent die Dynamik seiner Umgebung durch Beobachtung der Auswirkungen seiner eigenen Aktionen lernt.

## Key Concepts

- Zustandsübergänge
- Umgebungsdynamik
- Modellbasiertes RL
- Prädiktives Modellieren

## Use Cases

- Roboternavigation in unbekanntem Gelände
- Spiel-KI, die Physik-Engines lernt
- Steuerungssysteme der industriellen Automatisierung
