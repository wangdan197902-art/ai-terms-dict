## Definition

In der künstlichen Intelligenz ist ein Vektor eine grundlegende Datenstruktur, die zur numerischen Darstellung von Informationen dient. Er besteht aus einer geordneten Liste von Zahlen (Elementen), die Merkmale eines Objekts abbilden.

### Summary

Ein mathematisches Objekt, das eine Größe mit sowohl Betrag als auch Richtung darstellt und häufig in der KI verwendet wird, um Datenpunkte im hochdimensionalen Raum zu kodieren.

## Key Concepts

- Dimensionalität
- Einbettung
- Betrag
- Richtung

## Use Cases

- Semantische Suche in Datenbanken
- Einbettungen im Natural Language Processing
- Merkmalsextraktion bei der Bilderkennung

## Code Example

```python
import numpy as np
vec = np.array([1.0, 2.0, 3.0])
print(vec.shape)
```
