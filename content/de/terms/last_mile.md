## Definition

Das „Last-Mile“-Problem bezieht sich auf die Herausforderungen bei der Bereitstellung von Modellen in der Produktion, einschließlich der Integration in bestehende Infrastrukturen, der Sicherstellung einer Latenzarmen Inferenz und der Handhabung von Edge Cases (Sonderfällen).

### Summary

Die letzte Phase der Bereitstellung von KI-Lösungen von der Entwicklungsumgebung hin zu Endanwendern in realen Betriebsumgebungen.

## Key Concepts

- Modellbereitstellung
- MLOps
- Produktionsreife
- Integration

## Use Cases

- Bereitstellung von Empfehlungsmaschinen
- Integration von Betrugserkennungs-APIs
- Implementierung von Edge-AI-Geräten
