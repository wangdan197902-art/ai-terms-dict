## Definition

Feingranulare Analyse beinhaltet das Identifizieren und Kategorisieren von Objekten oder Konzepten auf Subklassenebene anstatt nur auf der Hauptklassebene. Zum Beispiel die Unterscheidung zwischen spezifischen Hunderassen oder verschiedenen Arten von Vögeln innerhalb derselben Gattung.

### Summary

Beschreibt Analyse- oder Klassifizierungsaufgaben, die das Unterscheiden zwischen subtilen Unterschieden innerhalb einer breiten Kategorie erfordern.

## Key Concepts

- Subklassifikation
- Intraklassenvarianz
- Detaillierte Merkmale
- Hierarchische Labels

## Use Cases

- Artenidentifikation in der Ökologie
- Erkennung von Produktvarianten im E-Commerce
- Mikroanalyse von Gesichtsausdrücken
