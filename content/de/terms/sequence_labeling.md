## Definition

Bei der Sequenzbeschriftung wird für jedes Token in einer gegebenen Eingabesequenz, wie etwa Wörter in einem Satz oder Zeichen in einem String, ein kategorisches Label vorhergesagt. Häufige Anwendungen sind die Named Entity Recognition (NER), die Parts-of-Speech-Tags (POS) und das syntaktische Chunking.

### Summary

Eine Aufgabe im Natural Language Processing (NLP), bei der jedem Element einer Eingabesequenz ein Label zugewiesen wird.
