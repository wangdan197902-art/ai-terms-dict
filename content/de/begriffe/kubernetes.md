## Definition

Kubernetes (oft abgekürzt als K8s) ist ein Container-Orchestrierungssystem, das ursprünglich von Google entwickelt wurde. Es automatisiert die Bereitstellung, Skalierung und den Betrieb von Anwendungcontainern über Cluster hinweg...

### Summary

Eine Open-Source-Plattform zur Automatisierung des Bereitstellens, Skalierens und Verwaltens von containerisierten Anwendungen.

## Key Concepts

- Container-Orchestrierung
- Pods
- Cluster
- Mikrodienste

## Use Cases

- Bereitstellung großer Webdienste
- Verwaltung von Mikrodienst-Architekturen
- Automatisches Skalieren von KI-Workloads
