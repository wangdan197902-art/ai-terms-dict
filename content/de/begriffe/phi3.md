## Definition

Phi-3 ist eine Reihe kleiner Sprachmodelle (SLMs), die von Microsoft veröffentlicht wurden. Sie sind darauf ausgelegt, eine hohe Leistung zu erzielen, die mit größeren Modellen vergleichbar ist, jedoch deutlich weniger Rechenressourcen erfordert.

### Summary

Eine Familie von Open-Weight-Modellen für große Sprachmodelle, entwickelt von Microsoft Research.

## Key Concepts

- Kleine Sprachmodelle
- Microsoft Research
- Edge AI
- Training mit synthetischen Daten

## Use Cases

- Ausführung von LLMs auf Mobilgeräten oder Laptops
- Kosteneffiziente Inferenz für Unternehmensanwendungen
- Echtzeitverarbeitung mit geringen Latenzanforderungen
