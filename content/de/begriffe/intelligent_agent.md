## Definition

Ein intelligenter Agent ist ein System, das in der Lage ist, seine Umgebung durch Sensoren oder Dateneingaben wahrzunehmen, diese Informationen mithilfe von Algorithmen zur logischen Schlussfolgerung zu verarbeiten und durch Aktuatoren auf die Umgebung einzuwirken

### Summary

Eine autonome Softwareeinheit, die ihre Umgebung wahrnimmt, über Aktionen nachdenkt und Aufgaben zur Erreichung spezifischer Ziele ausführt.

## Key Concepts

- Autonomie
- Perception-Action Loop
- Goal Orientation
- Reactivity

## Use Cases

- Autonome Trading-Bots, die Marktstrategien ausführen
- Robotic Process Automation, die mehrstufige Workflows abwickelt
- Persönliche Assistenten, die Kalender- und E-Mail-Aufgaben verwalten
