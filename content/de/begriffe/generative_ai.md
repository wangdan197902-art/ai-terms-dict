### Summary

Generative KI ist eine Form der künstlichen Intelligenz, die neue Inhalte wie Text, Bilder, Audio und Code basierend auf Mustern erstellen kann, die aus Trainingsdaten gelernt wurden.

## Key Concepts

- Inhalts-Synthese
- Musternlernen
- LLMs (Große Sprachmodelle)
- Diffusionsmodelle

## Use Cases

- Textgenerierung
- Bilderschaffung
- Code-Vervollständigung
