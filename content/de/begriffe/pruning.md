## Definition

Pruning beinhaltet das Identifizieren und Eliminieren von Neuronen, Verbindungen oder Filtern in einem neuronalen Netzwerk, die nur minimal zur Genauigkeit der Ausgabe beitragen. Durch das Entfernen dieser redundanten Elemente wird das Modell kompakter und effizienter.

### Summary

Eine Technik zur Modellkompression, die redundante oder weniger bedeutende Parameter entfernt, um die Größe zu reduzieren und die Inferenzgeschwindigkeit zu verbessern.

## Key Concepts

- Modellkompression
- Entfernung von Redundanzen
- Beschleunigung der Inferenz
- Sparsity (Spärlichkeit)

## Use Cases

- Bereitstellung von KI auf Mobilgeräten
- Optimierung für Edge Computing
- Reduzierung der Cloud-Inferenzkosten
