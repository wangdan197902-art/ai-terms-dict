## Definition

Server-Sent Events (SSE) ermöglichen eine einseitige Kommunikation vom Server zum Client, bei der der Daten kontinuierlich streamen kann, ohne dass der Client wiederholt Polling betreiben muss. Es nutzt einfaches HTTP, was es einfach zu implementieren macht.

### Summary

Ein Standardprotokoll, das Webservern ermöglicht, Echtzeit-Updates über eine einzelne HTTP-Verbindung an Clients zu senden.

## Key Concepts

- Event-Stream
- Einwegkommunikation
- Automatische Wiederherstellung der Verbindung
- HTTP-basiert

## Use Cases

- Live-Börsenkurse
- Echtzeit-Benachrichtigungen
- Fortschrittsupdates
