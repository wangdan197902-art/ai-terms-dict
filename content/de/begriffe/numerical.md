## Definition

Im Kontext von KI und Data Science bezieht sich numerisch auf Datentypen oder Methoden, die quantitative Werte wie Ganzzahlen, Gleitkommazahlen und Dezimalzahlen umfassen. Im Gegensatz zu kategorialen oder textuellen Daten sind numerische Daten messbar.

### Summary

Bezieht sich auf Zahlen oder mathematische Berechnungen im Gegensatz zu symbolischer Logik oder Text.

## Key Concepts

- Gleitkommazahlen
- Ganzzahlen
- Datentypen
- Quantitative Daten

## Use Cases

- Statistische Modellierung
- Finanzprognosen
- Verarbeitung von Sensordaten

## Code Example

```python
import numpy as np
arr = np.array([1.5, 2.7, 3.9])
mean_val = np.mean(arr)
print(mean_val)
```
