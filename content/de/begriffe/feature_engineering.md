## Definition

Feature Engineering ist die Kunst, Domänenexpertise zu nutzen, um Rohdaten in Features zu transformieren, die die zugrunde liegenden Muster für Machine-Learning-Algorithmen besser repräsentieren. Dieser Prozess umfasst die Erstellung neuer Variablen aus bestehenden Daten.

### Summary

Die Praxis, Domänenwissen zu nutzen, um neue Features zu erstellen oder bestehende zu modifizieren, um die Leistung von Machine-Learning-Modellen zu verbessern.

## Key Concepts

- Domänenwissen
- Datentransformation
- Modellleistung
- Variablenerstellung

## Use Cases

- Verbesserung der Genauigkeit von Regressionsmodellen
- Verstärkung von Klassifizierungsgrenzen
- Optimierung der Zeitreihenvorhersage

## Code Example

```python
df['new_feature'] = df['feature_a'] * df['feature_b']
```
