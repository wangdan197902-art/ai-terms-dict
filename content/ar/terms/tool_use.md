## Definition

يتيح استخدام الأدوات لنماذج اللغة التفاعل مع بيئات البرمجيات الخارجية من خلال استدعاء دوال محددة مسبقاً، مثل الآلات الحاسبة، ومحركات البحث، أو استعلامات قواعد البيانات. يوسع هذا النهج القدرات

### Summary

نموذج حيث تختار وكلاء الذكاء الاصطناعي وتنفذ دوال خارجية أو واجهات برمجة تطبيقات لأداء مهام محددة تتجاوز قدراتهم الأصلية.

## Key Concepts

- استدعاء الدوال
- هندسة الوكلاء
- تكامل واجهة برمجة التطبيقات
- المعرفة الخارجية

## Use Cases

- روبوتات دعم العملاء الآلية
- مساعدي تحليل البيانات
- أتمتة المنزل الذكي
