## Definition

يتضمن النشر على منصة Azure استخدام أدوات أصلية للسحابة مثل Azure Machine Learning و Azure Kubernetes Service (AKS) أو Azure Functions لتقديم نماذج تعلم الآلة على نطاق واسع. ويشمل ذلك إدارة موارد الحوسبة...

### Summary

عملية استضافة وتشغيل نماذج تعلم الآلة على خدمات البنية التحتية للسحابة الإلكترونية من مايكروسوفت Azure.

## Key Concepts

- البنية التحتية للسحابة
- تقديم النموذج
- القابلية للتوسع
- خطوط أنابيب التكامل والنشر المستمر (CI/CD)

## Use Cases

- نقاط نهاية واجهة برمجة التطبيقات (API) في الوقت الفعلي للاستنتاج
- التقييم الدفعي لمجموعات البيانات الكبيرة
- إدارة النماذج بمعايير المؤسسات
