## Definition

في سياقات الذكاء الاصطناعي وعلوم الحاسوب، يشير مصطلح 'معطى' إلى الحالة الأولية أو مجموعة البيانات أو المعاملات المقدمة للنموذج أو الدالة قبل بدء الحساب. وهو يحدد شروط الحدود للعملية.

### Summary

مصطلح يشير إلى البيانات الموجودة مسبقاً أو القيود أو السياق المقدم كمدخل لعملية خوارزمية.

## Key Concepts

- بيانات الإدخال
- قيود
- سياق
- تهيئة

## Use Cases

- إعداد التعلم بعدد قليل من الأمثلة
- مشاكل إشباع القيود
- حسابات الاحتمالات الشرطية
