## Definition

تتيح أحداث موجهة من الخادم (SSE) اتجاهاً واحداً للاتصال من الخادم إلى العميل، حيث يمكن للخادم بث البيانات باستمرار دون الحاجة من العميل لإجراء استعلامات متكررة. تعتمد على بروتوكول HTTP العادي، مما يجعلها بسيطة وفعالة.

### Summary

بروتوكول قياسي يسمح للخوادم الويب بدفع تحديثات فورية إلى العملاء عبر اتصال HTTP واحد.

## Key Concepts

- تدفق الأحداث
- اتصال أحادي الاتجاه
- إعادة الاتصال التلقائي
- قائم على HTTP

## Use Cases

- تغذية أسعار الأسهم المباشرة
- الإشعارات الفورية
- تحديثات تتبع التقدم
