## Definition

يتضمن هذا الممارسة ربط نماذج الذكاء الاصطناعي، مثل نماذج اللغات الكبيرة، ببيئات تطوير البرمجيات مثل VS Code أو IntelliJ. يتيح ذلك ميزات مثل إكمال الكود الذكي، والتنقيح الآلي، وتحليل الأخطاء.

### Summary

يشير تكامل بيئة التطوير المتكاملة إلى دمج أدوات مدعومة بالذكاء الاصطناعي مباشرة داخل بيئات التطوير لمساعدة المبرمجين في مهام البرمجة.

## Key Concepts

- الاقتراح التلقائي
- إكمال الكود
- نموذج لغة كبير
- تجربة المطور

## Use Cases

- اقتراحات الكود الذكية
- إعادة الهيكلة الآلية
- تنفيذ الاستعلامات بلغة طبيعية
