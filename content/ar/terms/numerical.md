## Definition

في سياق الذكاء الاصطناعي وعلوم البيانات، يشير المصطلح العددي إلى أنواع البيانات أو الطرق التي تتضمن قيمًا كمية، مثل الأعداد الصحيحة والكسور العشرية. وعلى عكس البيانات الفئوية أو النصية، فإن البيانات العددية قابلة للقياس الرياضي المباشر.

### Summary

يتعلق بالأرقام أو يتضمنها أو يستخدمها، بدلاً من المنطق الرمزي أو النص.

## Key Concepts

- الكسور العشرية (Floats)
- الأعداد الصحيحة (Integers)
- أنواع البيانات (Data Types)
- البيانات الكمية (Quantitative Data)

## Use Cases

- النمذجة الإحصائية
- التنبؤ المالي
- معالجة بيانات المستشعرات

## Code Example

```python
import numpy as np
arr = np.array([1.5, 2.7, 3.9])
mean_val = np.mean(arr)
print(mean_val)
```
