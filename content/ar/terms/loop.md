## Definition

هي بنية أساسية لتدفق التحكم في علوم الحاسوب وتطوير الذكاء الاصطناعي، حيث تسمح الحلقات للخوارزميات بالتكرار عبر مجموعات البيانات، أو إجراء حسابات متكررة، أو تشغيل فترات تدريبية. الأنواع الشائعة تشمل

### Summary

بنية برمجية تكرر كتلة من التعليمات البرمجية عدة مرات حتى يتحقق شرط معين.

## Key Concepts

- التكرار
- تدفق التحكم
- الفترات التدريبية (Epochs)
- المعالجة الدفعية

## Use Cases

- تدريب الشبكات العصبية عبر فترات متعددة
- التكرار عبر عينات مجموعة البيانات
- التنفيذ خطوة بخطوة في التعلم التعزيزي

## Code Example

```python
for epoch in range(10):
    for batch in dataloader:
        train_step(batch)
```
