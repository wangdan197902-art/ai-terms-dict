## Definition

في الذكاء الاصطناعي، المتجه هو بنية بيانات أساسية تُستخدم لتمثيل المعلومات رقمياً. يتكون من قائمة مرتبة من الأرقام (العناصر) التي تربط ميزات الكيان بـ

### Summary

كائن رياضي يمثل كمية لها كل من المقدار والاتجاه، وغالباً ما يُستخدم في الذكاء الاصطناعي لترميز نقاط البيانات في فضاء عالي الأبعاد.

## Key Concepts

- البعدية
- التضمين
- المقدار
- الاتجاه

## Use Cases

- البحث الدلالي في قواعد البيانات
- تضمينات معالجة اللغات الطبيعية
- استخراج ميزات التعرف على الصور

## Code Example

```python
import numpy as np
vec = np.array([1.0, 2.0, 3.0])
print(vec.shape)
```
