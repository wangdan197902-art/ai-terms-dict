## Definition

يعمل الاختبار المرجعي كنقطة مرجعية موحدة لمقارنة قدرات نماذج أو خوارزميات الذكاء الاصطناعي المختلفة. يتضمن عادةً مجموعة بيانات مُنتقاة بعناية ومقاييس تقييم محددة مثل الدقة أو السرعة.

### Summary

اختصار لمصطلح Benchmark، وهو مجموعة اختبار قياسية أو مقياس يُستخدم لتقييم أداء نماذج الذكاء الاصطناعي.

## Key Concepts

- مقاييس التقييم
- الاختبار الموحد
- مقارنة الأداء
- انتقاء مجموعات البيانات

## Use Cases

- اختبار GLUE لمعالجة اللغات الطبيعية
- مجموعة بيانات ImageNet للرؤية الحاسوبية
- اختيار النماذج للإنتاج
