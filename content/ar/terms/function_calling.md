## Definition

يُمكّن استدعاء الدوال نماذج اللغات الكبيرة من التفاعل مع الأدوات الخارجية وواجهات برمجة التطبيقات (APIs) من خلال توليد مخرجات هيكلية، مثل كائنات JSON، تحدد الدالة التي يجب تنفيذها والحجج المطلوبة.

### Summary

آلية تتيح لنماذج اللغات الكبيرة (LLMs) إخراج بيانات هيكلية تُشغّل دوال برمجية محددة.

## Key Concepts

- المخرجات الهيكلية
- تكامل واجهة برمجة التطبيقات
- استخدام الأدوات
- مخطط JSON

## Use Cases

- حجز الرحلات الجوية عبر واجهات برمجة تطبيقات السفر
- إجراء حسابات رياضية معقدة
- الاستعلام عن سجلات قاعدة البيانات
