## Definition

يسرع تعلم الآلة الآلي (AutoML) تطوير نماذج تعلم الآلة من خلال أتمتة مهام مثل معالجة البيانات المسبقة، وهندسة الميزات، واختيار النموذج، وضبط المعاملات الفائقة. مما يمكّن غير المتخصصين من استخدام التقنيات المتقدمة.

### Summary

منهجية تؤتمت العملية الكاملة لتطبيق تعلم الآلة على المشكلات الواقعية، مما يقلل من الجهد اليدوي.

## Key Concepts

- ضبط المعاملات الفائقة
- هندسة الميزات
- اختيار النموذج
- إتاحة الوصول (Democratization)

## Use Cases

- النمذجة السريعة لمحللي الأعمال
- تحسين خطوط أنابيب الإنتاج واسعة النطاق
- مقارنة خوارزميات متعددة تلقائياً

## Code Example

```python
from auto_ml import Predictor
predictor = Predictor(type_of_estimator='classifier')
predictor.fit(dataframe, target='label')
```
