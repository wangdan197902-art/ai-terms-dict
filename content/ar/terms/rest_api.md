## Definition

تتيح واجهات برمجة تطبيقات REST التواصل بين العملاء والخوادم من خلال استخدام عمليات عديمة الحالة عبر بروتوكولات HTTP مثل GET وPOST وPUT وDELETE. وهي تنظم الموارد كعناوين URL وتستخدم تنسيقات قياسية.

### Summary

واجهة برمجة تطبيقات نقل الحالة التمثيلية هي نمط معماري لتصميم التطبيقات الشبكية التي تعتمد على طرق HTTP القياسية.

## Key Concepts

- عدم وجود حالة (Statelessness)
- طرق HTTP
- عناوين URL القائمة على الموارد
- تسلسل JSON

## Use Cases

- التواصل بين الخدمات المصغرة
- تكامل الخلفية لتطبيقات الجوال
- الوصول إلى بيانات الأطراف الثالثة
