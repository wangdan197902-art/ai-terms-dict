## Definition

في سياق الذكاء الاصطناعي، يشير التحليل إلى الفحص المنهجي للبيانات أو توقعات النموذج أو سلوكيات النظام لفهم الأنماط الكامنة، أو تشخيص المشكلات، أو استخلاص رؤى قابلة للتنفيذ.

### Summary

عملية فحص البيانات أو مخرجات النموذج لاستخلاص رؤى وأنماط ذات معنى.

## Key Concepts

- استخلاص الرؤى
- قابلية تفسير النموذج
- فحص البيانات
- التقييم التشخيصي

## Use Cases

- تصحيح أخطاء النماذج
- الذكاء التجاري (BI)
- الذكاء الاصطناعي القابل للتفسير (XAI)
