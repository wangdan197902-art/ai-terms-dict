## Definition

Code Search Net هي مجموعة بيانات شاملة تم إنشاؤها لتطوير أبحاث استرجاع الكود. تحتوي على ملايين الأزواج من الأوصاف باللغة الطبيعية ومقتطفات الكود المقابلة لها بلغة Java. تُستخدم هذه البيانات لتقييم دقة الأنظمة في فهم واسترجاع الكود.

### Summary

مجموعة بيانات معيارية مصممة لتقييم أنظمة بحث الكود من خلال اقتران استعلامات اللغة الطبيعية بمقتطفات كود ذات صلة.

## Key Concepts

- استرجاع الكود
- البحث الدلالي
- كود Java
- معالجة اللغة الطبيعية

## Use Cases

- بناء إضافات لبيئات التطوير المتكاملة (IDE)
- أبحاث فهم الكود
- التوثيق الآلي للكود
