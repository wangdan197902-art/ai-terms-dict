## Definition

هندسة الميزات هي فن الاستفادة من الخبرة في المجال لتحويل البيانات الخام إلى ميزات تمثل الأنماط الأساسية بشكل أفضل لخوارزميات التعلم الآلي. تشمل هذه العملية إنشاء متغيرات جديدة أو تعديل الموجودة لتحسين دقة النموذج.

### Summary

ممارسة استخدام معرفة المجال لإنشاء ميزات جديدة أو تعديل الميزات الموجودة لتعزيز أداء نماذج التعلم الآلي.

## Key Concepts

- معرفة المجال
- تحويل البيانات
- أداء النموذج
- إنشاء المتغيرات

## Use Cases

- تحسين دقة نماذج الانحدار
- تعزيز حدود التصنيف
- تحسين التنبؤ بالسلاسل الزمنية

## Code Example

```python
df['new_feature'] = df['feature_a'] * df['feature_b']
```
