## Definition

صُممت النماذج متعددة اللغات للتعامل مع مدخلات لغوية متنوعة دون الحاجة إلى نماذج منفصلة لكل لغة. وعادةً ما تستخدم هذه الأنظمة تضمينات مشتركة أو محاذاة عبر اللغات

### Summary

يشير مصطلح متعدد اللغات في الذكاء الاصطناعي إلى النماذج القادرة على معالجة أو فهم أو توليد محتوى بلغات طبيعية متعددة.

## Key Concepts

- النقل عبر اللغات
- المفردات المشتركة
- تحديد اللغة
- الترجمة بدون أمثلة سابقة (Zero-Shot)

## Use Cases

- خدمات الترجمة الآلية
- روبوتات الدردشة لخدمة العملاء العالمية
- محركات البحث عبر اللغات
