## Definition

Virtauspohjaiset generatiiviset mallit rakentavat monimutkaisia todennäköisyysjakaumia soveltamalla sarjaa kääntyviä ja derivoituvia muunnoksia yksinkertaiseen perusjakaumaan, kuten Gaussiin. Koska muunnokset ovat kääntyviä, tarkka todennäköisyys voidaan laskea.

### Summary

Generatiivisten mallien luokka, joka käyttää kääntyviä muunnoksia yksinkertaisten jakaumien muuntamiseksi monimutkaisiksi datanjakaumiksi.

## Key Concepts

- Kääntyvät muunnokset
- Tarkka todennäköisyysfunktio
- Normalisoivat virtaukset
- Muuttujanvaihto

## Use Cases

- Kuvagenerointi
- Tiheyden estimointi
- Anomalioiden havaitseminen
