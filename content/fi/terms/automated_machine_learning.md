## Definition

AutoML (Automated Machine Learning) tehostaa ML-mallien kehittämistä automatisoimalla tehtäviä kuten datan esikäsittelyä, piirteiden insinööritaidot, mallin valinnan ja hyperparametrien säädön. Se mahdollistaa...

### Summary

Menetelmä, joka automatisoi koneoppimisen soveltamisen kokonaisvaltaisesti todellisiin ongelmiin vähentäen manuaalista työtä.

## Key Concepts

- Hyperparametrien säätö
- Piirteiden insinööritaidot
- Mallin valinta
- Demokratisointi

## Use Cases

- Nopeat prototyypit liiketoiminnan analyytikoille
- Suurten tuotantoputkien optimointi
- Useiden algoritmien automaattinen vertailu

## Code Example

```python
from auto_ml import Predictor
predictor = Predictor(type_of_estimator='classifier')
predictor.fit(dataframe, target='label')
```
