## Definition

Prompt-insinööritaidot sisältävät tiettyjen syötteiden, joita kutsutaan prompteiksi, muotoilua generatiivisten tekoälymallien saamiseksi tuottamaan tarkkoja, relevantteja ja korkealaatuisia vastauksia. Se vaatii ymmärrystä siitä, miten mallit tulkitsevat ohjeistusta ja kontekstia.

### Summary

Käytäntö, jossa suunnitellaan ja optimoidaan syötekstiä ohjaamaan suurta kielimallia haluttuihin tuloksiin.

## Key Concepts

- Kontekstuaalinen kehys
- Vähäisen esimerkin oppiminen
- Ohjetunointi
- Token-optimaatio

## Use Cases

- Automaattiset asiakaspalvelun chatbotit
- Koodin luomisen avustajat
- Luovan kirjoittamisen apuvälineet

## Code Example

```python
prompt = "Translate the following English text to French: 'Hello world'"
response = llm.generate(prompt)
```
