## Definition

Object Detection (kohteiden tunnistus) laajentaa kuvien luokittelua siten, että se ei vain määritä, mitkä kohteet läsnä ovat, vaan myös missä ne sijaitsevat. Se tuottaa havaittuihin kohteisiin liittyvät rajauskoordinaatit sekä niiden luokittelutunnisteet.

### Summary

Tietokonenäkötekniikka, joka tunnistaa ja paikantaa kohteita kuvissa tai videovirroissa käyttämällä rajauslaatikoita (bounding boxes).

## Key Concepts

- Rajauslaatikot
- Luokkatunnisteet
- YOLO (You Only Look Once)
- Faster R-CNN

## Use Cases

- Itseajavien ajoneuvojen navigointi
- Kaupan varastonhallinta
- Turvakameravalvonta
