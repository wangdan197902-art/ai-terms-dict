## Definition

Ominaisuusinsinööritaidot ovat taitoa hyödyntää alaosaamista raakadatan muuntamiseksi ominaisuuksiksi, jotka edustavat paremmin taustalla olevia kuvioita koneoppimisalgoritmeille. Tämä prosessi sisältää uusien muuttujien luomista.

### Summary

Toimintatapa, jossa hyödynnetään alaosaamista uusien ominaisuuksien luomiseen tai olemassa olevien muokkaamiseen koneoppimismallien suorituskyvyn parantamiseksi.

## Key Concepts

- Alaosaaminen
- Datan muunnos
- Mallin suorituskyky
- Muuttujan luominen

## Use Cases

- Regressiomallin tarkkuuden parantaminen
- Luokittelurajojen parantaminen
- Aikasarjan ennusteen optimointi

```python
df['new_feature'] = df['feature_a'] * df['feature_b']
```
