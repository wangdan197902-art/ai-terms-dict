## Definition

REST API:t mahdollistavat viestinnän asiakkaiden ja palvelimien välillä hyödyntämällä tilattomia operaatioita HTTP-protokollan kautta, kuten GET, POST, PUT ja DELETE. Ne rakenteistavat resurssit URI-muodossa ja käyttävät standardoituja muotoja datan vaihtoon.

### Summary

Representational State Transfer -rajaupinta (REST API) on arkkitehtuurinen tyyli verkkosovellusten suunnitteluun, joka perustuu vakiintuneisiin HTTP-menetelmiin.

## Key Concepts

- Tilattomuus
- HTTP-menetelmät
- Resurssipohjaiset URI:t
- JSON-sarjastus

## Use Cases

- Mikropalveluiden välinen viestintä
- Mobiilisovellusten taustajärjestelmäintegraatio
- Kolmansien osapuolten tietojen käyttö
