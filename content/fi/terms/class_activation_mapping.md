## Definition

CAM luo syötekuvien päälle asetettavia lämpökarttoja, jotka näyttävät, mitkä pikselit vaikuttivat eniten mallin päätökseen tietyn luokan osalta. Se toimii soveltamalla globaalia keskiarvopoolingausta lopullisiin konvoluutio-

### Summary

Luokkien aktivaatiokartoitus (CAM) on visualisointitekniikka, joka korostaa syötekuvassa niitä alueita, jotka ovat eniten vastuussa tietystä ennustetusta luokasta.

## Key Concepts

- Lämpökarttavizualisointi
- Tulkittavuus
- Ominaisuuden merkitys
- Globaali keskiarvopoolingaustus

## Use Cases

- Lääketieteellisen kuvadiagnoosin varmistus
- Autonomisen objektintunnistuksen vianetsintä
- Selitettävän tekoälyn raportointi
