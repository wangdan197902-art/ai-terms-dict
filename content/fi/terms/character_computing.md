## Definition

Tämä käsite keskittyy tekstin manipulointiin, jossa laskennan perusyksikkö on yksi merkki. Sitä käytetään usein tehtävissä, jotka vaativat hienojakoista tekstanalyysiä, kuten oikeinkirjoituksen tarkistuksessa.

### Summary

Merkkikohtainen laskenta tarkoittaa tekstin käsittelyä, generointia tai analysointia yksittäisten merkkien tasolla sanan tai lauseen sijaan.

## Key Concepts

- Jäsennys (Tokenization)
- Tekstigenerointi
- Hienojakoinen analyysi
- Alasanan yksiköt

## Use Cases

- Käsinkirjoituksen tunnistus (OCR)
- Vähävaraiskielisten mallinnus
- Salakirjoituksen analyysi
