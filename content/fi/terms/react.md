## Definition

ReAct-kehyksen avulla kielimallit (LLM) tuottavat sekä päättelyjälkiä että tehtäväkohtaisia toimintoja vuorottelevassa järjestyksessä. Simuloimalla ihmismäisiä ajatusprosesseja mallit pystyvät vuorovaikutukseen ulkoisten ympäristöjen kanssa.

### Summary

ReAct on kehotteiden (prompting) paradigma, joka yhdistää päättelyn ja toiminnan parantaakseen suurten kielimallien suorituskykyä monimutkaisissa tehtävissä.

## Key Concepts

- Vuorotteleva päättely
- Toimintojen generointi
- Observaatioiden integrointi
- Ajatusketju (Chain-of-Thought)

## Use Cases

- Monimutkaiset kysymys-vastaus-järjestelmät
- Itseohjautuvat agentit
- Totuudenmukaisuuden tarkistusprosessit
