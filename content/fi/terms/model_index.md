## Definition

Indeksitiedosto, nimeltään tyypillisesti 'model_index.json', sisältää rakenteellista tietoa mallin arkkitehtuurista, mukaan lukien putkityyppi, alimallit ja konfiguraatiopolut. Se mahdollistaa Hubin yhdistämään monimutkaisia malleja oikein latausprosessin aikana.

### Summary

Model Index on metatietotiedosto, jota Hugging Face Hub käyttää kuvaamaan ja järjestämään mallikomponentteja ja konfiguraatioita.

## Key Concepts

- Metatiedot
- Putkikonfiguraatio
- Mallin lataus
- Yhteentoimivuus

## Use Cases

- Monikomponenttisten putkien määrittely
- Monimutkaisten mallien lataaminen Hubiin
- Automaattinen mallien etsintä ja lataus
