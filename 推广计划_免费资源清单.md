## 🎯 推广路线图（按优先级）

### 🔥 P0 立即执行（0~24 小时内）

#### 1. 修复 GSC 验证 meta + 部署 Ahrefs 验证（代码侧）
**动作**：在 `layouts/partials/seo.html` 添加两个 meta 标签
```html
<!-- GSC 新域名验证（terms.ai-term-hub.com） -->
<meta name="google-site-verification" content="Po21SS8ccODMgK6x0_LBsBX0t5qz7bdqRzASwE-ngyo" />
<!-- Ahrefs Webmaster Tools 验证 -->
<meta name="ahrefs-site-verification" content="ad7fa89d439de59a24f2c0539f078231d409b57876cd4088b2e41eafe03df321" />
```
**验证**：部署后访问 `view-source:https://terms.ai-term-hub.com/en/` 应能看到这两个 meta。

#### 2. 用户手动：在 Google Search Console 提交新域名
**步骤**：
1. 访问 https://search.google.com/search-console
2. 添加资源 → 网址前缀 → 输入 `https://terms.ai-term-hub.com`
3. 选择"HTML 标签"验证方式 → 等待自动验证（meta 已部署）
4. 验证成功后 → 左侧菜单"站点地图" → 提交 `https://terms.ai-term-hub.com/sitemap.xml`
5. 左侧菜单"网址检查" → 输入 `https://terms.ai-term-hub.com/en/` → 点击"请求编入索引"

**预期效果**：24~72 小时内 Google 开始抓取，1~2 周内出现首批收录。

#### 3. 用户手动：注册 Bing Webmaster Tools
**步骤**：
1. 访问 https://www.bing.com/webmasters/
2. 用 Microsoft 账号登录（或用 Google 账号 OAuth）
3. 添加站点 → 输入 `https://terms.ai-term-hub.com/`
4. 验证方式选"HTML meta tag" → Bing 会给一个 content 值（形如 `12345ABCDEF...`）
5. **把 content 值告知我**，我添加到 seo.html
6. 部署后在 Bing 后台点击"Verify"
7. 提交 sitemap：`https://terms.ai-term-hub.com/sitemap.xml`

**Bing 优势**：Bing Webmaster Tools 提供 **免费 IndexNow API**，可主动推送 URL，加速收录。

#### 4. 用户手动：注册百度搜索资源平台
**步骤**：
1. 访问 https://ziyuan.baidu.com/
2. 用百度账号登录
3. 用户中心 → 站点管理 → 添加网站 → `https://terms.ai-term-hub.com`
4. 验证方式选"HTML 标签" → 百度会给一个 content 值（形如 `codeva-xxxxxxx`）
5. **把 content 值告知我**，我添加到 seo.html
6. 部署后在百度后台点击"完成验证"
7. 提交 sitemap：链接提交 → 自动提交 → sitemap → `https://terms.ai-term-hub.com/sitemap.xml`

**百度特殊说明**：
- 百度对境外服务器抓取频率较低，可能需 1~4 周才收录
- 百度优先收录中文页面（`/zh/`），建议把中文版作为重点提交
- 百度支持"主动推送 API"，每天可推送 1000 条 URL，加速收录

#### 5. 用户手动：注册 Yandex Webmaster
**步骤**：
1. 访问 https://webmaster.yandex.com/
2. 用 Yandex 账号登录（可先注册 yandex.com 邮箱）
3. 添加站点 → `https://terms.ai-term-hub.com/`
4. 验证方式选"HTML meta tag" → Yandex 给一个 content 值
5. **把 content 值告知我**，我添加到 seo.html
6. 部署后验证 → 提交 sitemap

**Yandex 价值**：俄罗斯/东欧市场流量，对俄语（`/ru/`）页面效果好。
